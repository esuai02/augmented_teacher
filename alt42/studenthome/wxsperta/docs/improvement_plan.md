## 📋 상세 실행 계획

### 🔤 1단계: 용어 순화 (즉시 적용 가능)

#### 페이지 타이틀 변경
```
현재: "21개 AI 에이전트 프로젝트 시스템"
변경: "나의 성장 정원 🌱"
부제: "스스로 발견하고, 가꾸고, 성장하는 공간"
```

#### 카테고리 이름 변경
| 현재 | 변경 | 의미 |
|------|------|------|
| 미래설계 에이전트 (1-4) | 🔮 꿈 심기 | 미래를 상상하고 씨앗을 심는 곳 |
| 실행 에이전트 (5-11) | ⚡ 오늘 가꾸기 | 매일의 행동으로 새싹을 키우는 곳 |
| 브랜딩 에이전트 (12-14) | 🌸 나 피우기 | 나만의 색깔로 꽃을 피우는 곳 |
| 지식관리 에이전트 (15-21) | 🌳 지혜 숲 | 배움을 나무처럼 키우는 곳 |

#### 버튼 텍스트 변경
```
현재: "프로젝트 보기"
변경: "함께 해볼까요?" 또는 "시작하기 🌱"
```

#### 네비게이션 변경
```
현재: "← AI 대화 매트릭스로 돌아가기"
변경: "← 내 정원으로 돌아가기 🏡"
```

```
- 호버 시 싹트는 애니메이션
- 완료/진행중 상태 표시 (싹 → 꽃)
```

### 🚀 3단계: 온보딩 & 사용 흐름 개선

#### 첫 방문자 환영
```html
<div class="welcome-overlay">
  <h2>안녕하세요! 🌱</h2>
  <p>여기는 나만의 성장 정원이에요.</p>
  
  <div class="mood-check">
    <p>오늘 기분이 어때요?</p>
    <button>😊 좋아요!</button>
    <button>😐 그냥요</button>
    <button>😔 힘들어요</button>
  </div>
  
  <p class="hint">기분에 맞는 활동을 추천해드릴게요</p>
</div>
```

#### 추천 시작점
```
기분 좋을 때 → "새로운 꿈을 그려볼까요?" (미래 정원)
평범할 때 → "오늘 작은 것 하나 해볼까요?" (오늘 가꾸기)
힘들 때 → "잠깐 쉬면서 정리해볼까요?" (지혜 숲)
```

#### 진행 상태 표시
```
🌱 씨앗 → 시작하지 않음
🌿 새싹 → 시작함
🌳 나무 → 진행 중
🌸 꽃 → 완료
🍎 열매 → 나눔/공유
```

### 🔗 4단계: 관계 시각화

#### 도구 간 연결 표시
```
시간 수정체(미래 상상)
       ↓
타임라인 합성기(계획 세우기)
       ↓
일일 사령부(오늘 할 일)
       ↓
실행 파이프라인(실행하기)
       ↓
성장 엘리베이터(성장 확인)
```

#### 나의 여정 타임라인
```
[3일 전] 🌱 "미래의 나에게 편지" 시작
[2일 전] 🌿 첫 번째 편지 작성 완료
[어제]   📝 답장 상상해보기
[오늘]   💭 추천: "꿈 지도 그리기"
```
