# 🌌 마이 궤도 (My Orbit) - 컨셉 기획서

> **작성일**: 2025-12-06

---

## 1. 왜 "마이 궤도"인가?

### 1.1 한국 학생들의 정서 분석

| 현실 | 갈증 | 우리가 줄 것 |
|------|------|-------------|
| 입시 부담, 경쟁 | "내 속도로 가고 싶어" | 남과 비교 없는 **나만의 궤도** |
| 정해진 길만 강요 | "내 길을 찾고 싶어" | 탐험과 발견의 **우주 여행** |
| 관리당하는 느낌 | "응원받고 싶어" | 함께하는 **우주 동료들** |
| 미래 불안 | "내가 뭘 좋아하는지 모르겠어" | 나를 알아가는 **항해 일지** |

### 1.2 "마이 궤도"의 의미

```
🌍 지구 = 지금의 나
🚀 로켓 = 오늘의 행동
🪐 행성들 = 나의 목표와 꿈
⭐ 별들 = 내가 모은 지식과 경험
🌌 우주 = 무한한 가능성

"남들과 똑같은 길이 아니라, 
 나만의 궤도를 돌며 성장한다"
```

## 2. 세계관 설정

### 2.1 스토리

> 너는 막 **[너의 이름] 행성**을 발견한 우주 탐험가야.  
> 이 행성은 아직 미지의 땅이 많아.  
> 
> 탐험하면서 자원을 모으고, 기지를 세우고,  
> 언젠가는 다른 행성까지 닿을 수 있는  
> 너만의 우주를 만들어가는 거야.
> 
> 걱정 마. 혼자가 아니야.  
> **관제탑**과 **AI 크루**가 언제나 함께해.

### 2.2 주요 공간

| 장소 | 설명 | 현재 명칭 매핑 |
|------|------|---------------|
| 🏠 **나의 행성** | 홈 화면. 오늘의 상태와 미션 | 메인 대시보드 |
| 🗺️ **항해 지도** | 목표와 계획을 시각화 | 미래설계 카테고리 |
| 🚀 **미션 센터** | 오늘 할 일, 실행 | 실행 카테고리 |
| 📡 **관제탑** | AI와 대화, 조언 받기 | AI 대화 |
| 🌟 **자원 창고** | 모은 지식과 자료 | 지식관리 카테고리 |
| 🎨 **나의 깃발** | 개인 브랜딩, 표현 | 브랜딩 카테고리 |
