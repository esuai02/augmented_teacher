# AI 에이전트 프로젝트 시스템

21개의 AI 에이전트를 통한 통합 성장 플랫폼

## 개요

이 프로젝트는 개인의 성장과 발전을 위한 21개의 AI 에이전트로 구성된 통합 시스템입니다. 각 에이전트는 특정 영역에서 사용자의 성장을 도우며, 서로 유기적으로 연결되어 시너지 효과를 창출합니다.

## 시스템 구조

### 1. 미래설계 에이전트 (1-4)
- **시간 수정체**: 미래 자아 시각화
- **타임라인 합성기**: 계획 현실화
- **성장 엘리베이터**: 성장 분석
- **성과지표 엔진**: 목표 수치화

### 2. 실행 에이전트 (5-11)
- **동기 엔진**: 동기 부여
- **SWOT 분석기**: 전략 분석
- **일일 사령부**: 일정 관리
- **내면 브랜딩**: 정체성 구축
- **수직 탐사기**: 본질 탐구
- **자원 정원사**: 자료 정리
- **실행 파이프라인**: 작업 자동화

### 3. 브랜딩 에이전트 (12-14)
- **외부 브랜딩**: 개인 브랜딩
- **성장 트리거**: 도전 과제
- **경쟁 생존 전략가**: 경쟁 전략

### 4. 지식관리 에이전트 (15-21)
- **시간수정체 CEO**: AI CEO
- **AI 정원사**: 지식 가꾸기
- **신경망 설계사**: 학습 설계
- **정보 허브**: 정보 허브
- **지식 연결망**: 지식 연결
- **지식 수정체**: 지식 결정화
- **유연한 백본**: 자동화 시스템

## 프로젝트 구조

각 AI 에이전트는 3개의 대표 프로젝트를 가지며, 각 프로젝트는 3개의 하위 프로젝트로 구성됩니다:

```
ai_agents/
├── index.php                    # 메인 대시보드
├── styles/                      # CSS 스타일
├── scripts/                     # JavaScript 파일
├── api/                         # API 엔드포인트
├── data/                        # 사용자 데이터 저장
├── future_design/               # 미래설계 에이전트 (1-4)
├── execution/                   # 실행 에이전트 (5-11)
├── branding/                    # 브랜딩 에이전트 (12-14)
└── knowledge_management/        # 지식관리 에이전트 (15-21)
```

## 설치 방법

1. 프로젝트를 웹 서버 디렉토리에 복사합니다.
2. 데이터베이스를 생성하고 `database_schema.sql`을 실행합니다.
3. 웹 브라우저에서 `index.php`에 접속합니다.

## 주요 기능

- **프로젝트 진행도 추적**: 각 프로젝트와 하위 프로젝트의 완료 상태를 추적
- **에이전트 간 연계**: 관련된 에이전트들 간의 시너지 효과 시각화
- **개인화된 도구**: 각 에이전트별 특화된 실행 도구 제공
- **진행 상황 저장**: 로컬 스토리지 및 데이터베이스에 진행 상황 자동 저장

## API 사용법

### 진행 상황 조회
```
GET /api/progress_api.php
GET /api/progress_api.php?card_id=01_time_capsule
```

