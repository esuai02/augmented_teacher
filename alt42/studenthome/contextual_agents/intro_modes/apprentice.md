# 7) 🔍 인지적 도제형 – World Expert Intelligence 설계

## W: 세계관 (Worldview) 정렬

* 핵심 신념: **“지식은 주입이 아니라 ‘사고 방법’을 전수하는 것이다.”**
* 목표: 교사의 사고 과정을 시연하고, 점진적으로 지원을 줄이며, 학생이 독립적으로 사고·문제 해결이 가능하도록 훈련.
* 교사 모토(원문 연결): *말하는 수학, 설명하는 수학 중심 · 과정 중심 평가와 다양한 풀이 허용.*
* 전략적 관점:

  * **사고의 메커니즘**을 학습자 뇌 속에 복제.
  * 교사는 코치+모델, 학생은 견습생.
* 보조 세계관 연결:

  * 📚 체계적 진도형: 개념·맥락 기반 안정화
  * 🎯 개인맞춤형: 사고 패턴 취약 영역 맞춤 지원
  * ✏️ 성과 집중형: 시험 대비 시 사고 기반 고난도 문제 대응
  * ⚡ 목표달성형: 사고 훈련을 짧은 미션으로
  * 🧠 사고력 중심형: 메타인지 강화
  * 🚀 자율학습형: 독립 설계형 사고 훈련

---

## X: 문맥지능 (Context)

* 필수 컨텍스트 수집:

  * 학생 풀이 과정의 구체성·논리성, 다양한 풀이 시도 빈도, 질문·설명 능력, 지원 수준(의존도), 전이 성과.
* 스위칭 트리거:

  * 풀이 과정 설명 불능 → 🔍 집중
  * 전이 실패 반복 → 🧠 병행
  * 시험 D-14 → ✏️ 결합
  * 계획 품질·실행률 저하 → 📚·🎯 지원
  * 동기 부족 → ⚡ 미션형 사고 블록 삽입

---

## S: 구조지능 (Structure)

* 표준 변수:

  * `모델링빈도`, `코칭세션수`, `스캐폴딩단계`, `전이성공률`, `다양한풀이비율`.
* KPI:

  * 전이 성공률 ≥ 80%
  * 다양한 풀이 비율 ≥ 60%
  * 풀이 과정 발화 점수 ≥ 4/5
  * 스캐폴딩 단계 최종 0단계(완전 독립)
* 데이터 모델 예시:

```json
{
  "mode":"cognitive",
  "modelingFrequency":2,
  "coachingSessions":3,
  "scaffoldingLevel":1,
  "transferSuccessRate":0.82,
  "multipleSolutionsRate":0.65
}
```

---

## P: 절차지능 (Procedure)

* 5단계 도제 절차:

  1. **모델링(Modeling)**: 교사가 문제 풀이 사고 과정을 시연
  2. **코칭(Coaching)**: 학생이 풀이하며 이유를 발화, 교사가 피드백
  3. **스캐폴딩(Scaffolding)**: 점진적으로 지원·힌트 감소
  4. **명료화(Articulation)**: 학생이 전략·개념을 언어로 정리
  5. **탐색(Exploration)**: 열린 문제·다양한 풀이 탐색
* 주간 루프:

  * 월·수: 모델링·코칭
  * 금: 스캐폴딩·명료화
  * 주말: 탐색 과제

## E: 실행지능 (Execution)

* 교사 체크리스트:

  * [ ] 주간 모델링·코칭 횟수 기록
  * [ ] 스캐폴딩 레벨 조정
  * [ ] 전이·다양한 풀이 비율 모니터링
  * [ ] 학생 발화·설명 품질 피드백
* 학생 루틴(🔍 학생모드 부연):

  1. 모델링: 교사 사고 과정 관찰
  2. 코칭: 풀이 이유 발화
  3. 스캐폴딩: 점진적 지원 제거
  4. 명료화: 전략·개념 되돌아보기
  5. 탐색: 다양한 풀이 시도
* 자동화 매핑:

  * **mathking**: 풀이 과정 시뮬레이션·다양한 풀이 비교
  * **management**: 사고 과정 기록·패턴 분석

```

규칙: ① 모델링 ② 코칭 ③ 스캐폴딩 ④ 명료화 ⑤ 탐색

```


```
입력: 문제·개념

```

## R: 성찰지능 (Reflection)

* 주간 리플렉션 질문:

  1. 이번 주 모델링에서 배운 핵심 사고 패턴
  2. 스스로 찾은 대체 풀이 1개
  3. 전이 성공/실패 사례 1개씩
  4. 다음 주 스캐폴딩 목표
* 개선 규칙:

  * 전이 실패율 > 20% → 🧠 병행
  * 다양한 풀이 비율 < 50% → 탐색 블록 강화
  * 발화 품질 저조 → 코칭 세션 증량
