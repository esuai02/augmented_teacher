# 4) ⚡ 목표 달성형 – World Expert Intelligence 설계

## W: 세계관 (Worldview) 정렬

* 핵심 신념: **“작은 성취의 연쇄가 큰 도약을 만든다.”**
* 목표: 단기 성취 → 즉시 피드백 → 도파민 동기 부스트 → 다음 미션으로 연결.
* 교사 모토(원문 연결): *맞춤형 포모도로 설정 및 자기 조절력 강화 · 명확한 결과 피드백 제공.*
* 전략적 관점:

  * **짧은 주기·명확한 보상** 구조로 루틴 없는 학생도 빠르게 몰입.
  * 실행률 유지가 최우선, 내용 강도는 점진적으로 상승.
* 보조 세계관 연결:

  * 📚 체계적 진도형: 장기 커리큘럼과 연동
  * 🎯 개인맞춤형: 미션 설계시 약점 보정 요소 반영
  * ✏️ 성과 집중형: 시험 대비 기간에 고강도 미션 패키지
  * 🧠 사고력 중심형: 사고 전환·창의형 미션 삽입
  * 🚀 자율학습형: 상위권 자율 미션 셋 구성
  * 🔍 인지적 도제형: 과정 설명·모델링 미션 포함

---

## X: 문맥지능 (Context)

* 필수 컨텍스트 수집:

  * 현재 집중시간, 동기 수준, 최근 달성률, 선호 학습 스타일, 과목별 급한 과제·목표.
* 스위칭 트리거:

  * 실행율 < 80% → ⚡ 우선 적용
  * 시험 D-14 → ✏️와 결합, 고빈출 미션 패키지
  * 기초 결손·약점 지속 → 🎯와 병행
  * 사고력·풀이과정 개선 필요 → 🔍/🧠 결합
  * 자율성 높은 학생 → 🚀 자율 미션으로 전환

---

## S: 구조지능 (Structure)

* 표준 변수:

  * `missionsPerDay`, `포모도로비율`, `미션성공률`, `스트릭일수`, `보상규칙`.
* KPI:

  * 미션 성공률 ≥ 85%
  * 스트릭 유지 ≥ 7일
  * 주간 학습시간 +20% 상승
  * 보상 목표 달성률 ≥ 90%
* 데이터 모델 예시:

```json
{
  "mode":"mission",
  "missionsPerDay":5,
  "focusBreak":"10:5",
  "streakDays":7,
  "reward":"주간 성취 보상",
  "successRate":0.85
}
```

---

## P: 절차지능 (Procedure)

* 1일 루틴:

  1. **미션 세팅(아침)**: 하루 5개, 전날 실패시 7개
  2. **실행 블록**: 포모도로 10분 집중/5분 휴식
  3. **즉시 피드백**: 성공/실패 체크, 다음날 반영
  4. **보상 부여(주간)**: 80% 이상 달성 시 보상 지급
* 1주 루프:

  * 월\~토: 미션 실행·기록
  * 일요일: 주간 성취 집계·보상·다음 주 난이도 조정

---

## E: 실행지능 (Execution)

* 교사 체크리스트:

  * [ ] 미션 난이도·수량 조정
  * [ ] 학생별 스트릭·성공률 점검
  * [ ] 피드백·보상 적용 여부 확인
  * [ ] 병행모드 전환 판단
* 학생 루틴(⚡ 학생모드 부연):

  1. 하루 5개 미션, 실패 시 7개
  2. 스트릭 최소 7일 유지
  3. 포모도로 10:5
  4. 달성률 80% 미만 시 난이도 재조정
  5. 주간 보상 시스템 유지
* 자동화 매핑:

  * **mathking**: 미션 단위 과제 제공 + 피드백 자동 누적
  * **management**: 미션 완료율 체크, 짧은 주기 성취 기록

## 실행용 에이전트 프롬프트 세트

1. **MissionGenerator**

```
역할: ⚡ 목표 달성형 미션 설계자
입력: 학생 약점·목표·집중시간·성향
규칙: ① 하루 5개 미션(실패시 7개) ② 포모도로 10:5 적용 ③ 난이도 점진 상승
출력: 일간 미션표(과제·소요시간·난이도·피드백슬롯)
```

2. **StreakMonitor**

```
입력: 스트릭일수, 성공률
트리거: 스트릭 중단/성공률 <80%
처방: 난이도 조정 + 🎯 맞춤 보강 삽입
```

3. **RewardAgent**

```
입력: 주간 성취율
처방: 80% 이상 → 보상 지급 / 미만 → 피드백·보정미션 제공
```

## R: 성찰지능 (Reflection)

* 주간 리플렉션 질문:

  1. 이번 주 가장 재미있었던 미션 1개
  2. 실패율이 높았던 미션과 이유
  3. 보상 효과 체감 정도(0\~5점)
  4. 다음 주 추가/삭제할 미션 유형
* 개선 규칙:

  * 실패율 > 30% → 난이도 하향 조정
  * 성공률 100% 2주 지속 → 난이도 상향
  * 보상 효과 저조 → 보상 방식을 변경

---

## T: 트래픽 지능 (Traffic & Transfer)

* 정보흐름:

  * 미션 완료 데이터 → 🎯/📚/✏️로 전송
  * 병행 모드 진입 시 즉시 통합
* J-커브:

  * 스트릭 유지 기간에 난이도·미션량을 살짝 올려 곡선 폭발
* 단절지점:

  * 2일 이상 미션 미완료 → 긴급 개입 세션

## A: 추상화 지능 (Aftermath)

* 월간: 미션 성공률·스트릭 패턴 분석 → 개인 동기유발 요소 DB 업데이트
* 축적 자산: 과제 난이도 스케일, 보상-성취 상관 데이터, 성향별 최적 미션 목록
