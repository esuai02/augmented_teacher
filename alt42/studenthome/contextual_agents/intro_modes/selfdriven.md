# 6) 🚀 자율 학습형 – World Expert Intelligence 설계

## W: 세계관 (Worldview) 정렬

* 핵심 신념: **“학습의 주인은 학생 자신이다.”**
* 목표: 커리큘럼 수립부터 오답 관리까지 학생이 전 과정을 설계·주도.
* 교사 모토(원문 연결): *선택의 장단점 성찰 지원 · 분기 단위 딜레마 요소 최적화.*
* 전략적 관점:

  * 교사는 가이드·피드백·리스크 관리자 역할.
  * 자율성을 주되 **성과·품질 검증** 체계를 병행.
* 보조 세계관 연결:

  * 📚 체계적 진도형: 자율 계획에 구조적 기반 제공
  * 🎯 개인맞춤형: 자율 계획에 약점·성향 반영
  * ✏️ 성과 집중형: 시험 기간 전환
  * ⚡ 목표달성형: 루틴 붕괴 시 긴급 동기 부스트
  * 🧠 사고력 중심형: 사고·전략 설계형 프로젝트 병행
  * 🔍 인지적 도제형: 사고 시연·스캐폴딩으로 자율 학습 품질 향상

---

## X: 문맥지능 (Context)

* 필수 컨텍스트 수집:

  * 학생 자율 실행율, 계획 품질(구체성·현실성), 피드백 반영률, 성취 데이터, 자기평가·타인피드백 기록.
* 스위칭 트리거:

  * 실행율 < 70% → ⚡ 병행
  * 계획 품질 저조(모호·비현실) → 📚·🎯 개입
  * 시험 D-30 → ✏️ 전환
  * 전략 필요·창의적 탐구 욕구 ↑ → 🧠/🔍 병행

---

## S: 구조지능 (Structure)

* 표준 변수:

  * `자율비중(%)`, `계획품질점수`, `실행율`, `피드백반영률`, `성과KPI`.
* KPI:

  * 자율 계획 실행율 ≥ 80%
  * 계획 품질점수 ≥ 4/5
  * 피드백 반영률 ≥ 90%
  * 목표 성취율 ≥ 85%
* 데이터 모델 예시:

```json
{
  "mode":"selfled",
  "selfLedShare":0.7,
  "planQuality":4.2,
  "executionRate":0.82,
  "feedbackApplyRate":0.93,
  "targetAchievement":0.87
}
```

---

## P: 절차지능 (Procedure)

* 1분기 루프:

  1. **계획 수립**: 학생이 주간·월간 계획 작성
  2. **검수·조율**: 교사가 목표·구조·리스크 검토
  3. **실행·기록**: 학습·오답·메타인지 기록 유지
  4. **피드백·수정**: 월 2회 피드백 세션
  5. **성과 분석**: 분기 종료 후 회고·다음 분기 설계
* 병행 규칙:

  * 계획 실패율↑ → 📚·🎯 모드 개입
  * 동기 저하 → ⚡ 삽입
  * 고난도·창의 프로젝트 → 🧠/🔍 결합

---

## E: 실행지능 (Execution)

* 교사 체크리스트:

  * [ ] 주간 계획 수립 여부 확인
  * [ ] 계획 품질 채점
  * [ ] 실행·기록·오답 관리 점검
  * [ ] 피드백 적용 결과 확인
* 학생 루틴(🚀 학생모드 부연):

  1. 주간 학습 계획 직접 수립·공유
  2. 실패 계획 원인 분석 후 수정
  3. 자기 주도 학습시간 ≥ 70%
  4. 월 2회 멘토/동료 피드백
  5. 분기별 포트폴리오 제작
* 자동화 매핑:

  * **mathking**: 플랜 템플릿 제공 + 커스텀 루트 설계
  * **management**: 목표 vs 실행 비교, 피드백 순환 설계

## 실행용 에이전트 프롬프트 세트

1. **SelfLedPlanner**

```
역할: 🚀 자율 학습형 계획 코치
입력: 목표·범위·자율비중·성향·과거 성취데이터
규칙: ① 주간 계획 초안 학생 작성 ② 교사 검수·조율 ③ 분기별 회고·재설계
출력: 주간 계획표·위험요소·개선안
```

2. **PlanQualityRater**

```
입력: 계획 세부항목
출력: 품질점수(1~5), 부족항목, 개선 가이드
```

3. **FeedbackLoopAgent**

```
입력: 실행데이터·피드백기록
출력: 반영률·효과분석·다음 계획 수정안
```

## R: 성찰지능 (Reflection)

* 월 2회 피드백 세션 질문:

  1. 계획 대비 실행율 %와 이유
  2. 가장 성과가 좋았던 활동
  3. 실패 원인 1\~2개
  4. 다음 달 개선 전략
* 개선 규칙:

  * 실행율 < 80% → ⚡ 병행
  * 피드백 반영률 < 90% → 📚·🎯 재설계
  * 계획 품질 저조 → 계획 템플릿 보강

## T: 트래픽 지능 (Traffic & Transfer)

* 정보흐름:

  * 자율 계획·성과 → 📚·🎯·✏️·🧠로 환류
  * 분기별 자산 → 전 학습모드 공유
* J-커브:

  * 자율성·성과 동반 상승 시 난이도·범위 확대
* 단절지점:

  * 계획 누락·실행 기록 부재 → 긴급 개입 세션
