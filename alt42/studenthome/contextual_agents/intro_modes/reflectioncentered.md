# 5) 🧠 사고력 중심형 – World Expert Intelligence 설계

## W: 세계관 (Worldview) 정렬

* 핵심 신념: **“정답보다 중요한 건 ‘어떻게 거기까지 갔는가’다.”**
* 목표: 학생이 스스로 질문·응답하며 사고과정을 명료화하고 문제 해결 전략을 자기화.
* 교사 모토(원문 연결): *시험 및 주간 목표 중심 주제 가이드 · 시험 대비 모드 스위칭 구간 설정*
* 전략적 관점:

  * \*\*메타인지(자기 이해)\*\*와 **전이 가능성**을 끌어올리는 수업 구조.
  * 단기 점수보다 장기적 사고력·문제해결력 향상이 KPI.
* 보조 세계관 연결:

  * 📚 체계적 진도형: 개념 구조·맥락 기반 강화
  * 🎯 개인맞춤형: 사고 과정 취약 영역에 맞춤 설계
  * ✏️ 성과 집중형: 시험 전 고난도 문제 대비
  * ⚡ 목표달성형: 사고 훈련을 짧은 미션으로 쪼개 실행
  * 🚀 자율학습형: 사고 탐구 프로젝트 설계
  * 🔍 인지적 도제형: 과정 시연·스캐폴딩 병행

---

## X: 문맥지능 (Context)

* 필수 컨텍스트 수집:

  * 최근 풀이 패턴(속도·정확도), 풀이 과정 설명 능력, 전이 성공/실패 사례, 사고전략 레퍼토리, 자기성찰 기록(학습일지).
* 스위칭 트리거:

  * 전이 실패 2회 이상 → 🧠 모드 집중
  * 고난도 문제 해설 이해 불능 → 🔍 도제형 병행
  * 시험 D-14 → ✏️ 모드 일부 결합
  * 동기 저하·루틴 붕괴 → ⚡ 단기 사고 미션 삽입
  * 자율 탐구 욕구 상승 → 🚀 프로젝트 전환

---

## S: 구조지능 (Structure)

* 표준 변수:

  * `전이성공률`, `풀이과정설명점수`, `메타인지체크리스트점수`, `성찰일지작성률`, `사고전략목록`.
* KPI:

  * 전이 성공률 ≥ 80%
  * 풀이 과정 설명 점수 ≥ 4/5
  * 성찰일지 작성률 ≥ 90%
  * 전략 레퍼토리 확장 건수 ≥ 월 3개
* 데이터 모델 예시:

```json
{
  "mode":"reflection",
  "transferSuccessRate":0.8,
  "processExplanationScore":4.2,
  "metaChecklistScore":85,
  "reflectionLogRate":0.9,
  "strategiesLearned":["거꾸로 추론","조건 분해"]
}
```

---

## P: 절차지능 (Procedure)

* 1회 세션 루틴:

  1. **질문 유도**: 오늘 주제와 관련된 개방형 질문 1\~2개
  2. **풀이 과정 발화**: 학생이 문제를 풀며 생각 과정을 설명
  3. **전략 명료화**: 교사가 핵심 전략·개념을 라벨링
  4. **전이 실험**: 비슷하지만 다른 유형 문제 적용
  5. **성찰 기록**: 오늘 전략·전이 결과·개선점 기록
* 주간 루프:

  * 월\~목: 사고 훈련 세션
  * 금: 메타인지 체크리스트 작성
  * 주말: 전략 포트폴리오 업데이트

---

## E: 실행지능 (Execution)

* 교사 체크리스트:

  * [ ] 전이 실패·성공 패턴 기록
  * [ ] 풀이 과정 설명 훈련 주 2회 이상
  * [ ] 전략 라벨링·정리 피드백 제공
  * [ ] 성찰일지 점검·피드백
* 학생 루틴(🧠 학생모드 부연):

  1. 매일 10분 학습 일지 작성
  2. 주간 메타인지 체크리스트 작성
  3. 실행하지 않은 계획은 ‘실패 기록’
  4. 월 1회 학습 전략 재검토
  5. 생각-행동 갭 측정·개선
* 자동화 매핑:

  * **mathking**: 학습일지·자기피드백 작성 기능
  * **management**: 일지 작성 여부 체크, 키워드 분석

## 실행용 에이전트 프롬프트 세트

1. **ThoughtTrainer**

```
역할: 🧠 사고력 중심형 훈련 코치
입력: 주제, 최근 오답, 전이 실패 유형
규칙: ① 개방형 질문 ② 풀이 과정 발화 ③ 전략 라벨링 ④ 전이 실험 ⑤ 성찰 기록
출력: 세션 계획·질문 리스트·전이 문제·전략 메모
```

2. **MetaChecklistAgent**

```
입력: 주간 학습 내용·성취율
출력: 메타인지 체크리스트 채점·개선 권고
```

3. **StrategyPortfolioBuilder**

```
입력: 주간 전략·전이 결과
출력: 전략 포트폴리오 업데이트, 누락 전략 보완
```

## R: 성찰지능 (Reflection)

* 주간 리플렉션 질문:

  1. 이번 주 가장 잘 작동한 사고 전략 1개
  2. 실패한 전이 사례와 원인
  3. 전략 포트폴리오에 추가할 내용
  4. 다음 주 적용할 새로운 전략
* 개선 규칙:

  * 전이 실패율 > 20% → 🔍 도제형 집중 세션
  * 전략 확장 없음 2주 지속 → 난이도·문제 범위 조정
  * 성찰일지 누락 2회 이상 → ⚡ 미션형 작성 과제

---

## T: 트래픽 지능 (Traffic & Transfer)

* 정보흐름:

  * 전략·전이 데이터 → 📚/🎯/✏️ 모드로 환류
  * 시험 대비 시 고난도 문제 전략 전송
* J-커브:

  * 전략 적용 성공률이 오를 때 난이도·유형 확장
* 단절지점:

  * 전략 설명 거부·성찰일지 미작성 → 즉시 개입

---

## A: 추상화 지능 (Aftermath)

* 분기별: 전략 라이브러리 업데이트, ‘사고전환 성공 사례집’ 제작
* 축적 자산: 전략별 전이 성공률 DB, 사고 패턴 분석 리포트

## 상황별 블렌딩 레시피

* **전이 실패 빈발** → 🧠 + 🔍
* **시험 대비** → 🧠 + ✏️ (고난도 대비)
* **기초 보정 필요** → 🧠 + 📚/🎯
* **루틴 붕괴** → 🧠 + ⚡ (짧은 사고 미션)
* **자율 상위권** → 🧠 + 🚀 (사고 탐구 프로젝트)
