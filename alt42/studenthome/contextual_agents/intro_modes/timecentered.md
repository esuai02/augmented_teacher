# 8) 🕒 시간 피드백 중심형 – World Expert Intelligence 설계

## W: 세계관 (Worldview) 정렬

* 핵심 신념: **“시간은 학습의 생명선이자 성과의 가속기다.”**
* 목표: 학습 과정의 **시간 사용 효율·밀도·반복 주기**를 최적화하여, 뇌의 골든 타임을 활용한 장기기억 전환율 극대화.
* 철학:

  * 시간은 단순한 흐름이 아니라 **작업기억→장기기억 전환의 촉매제**.
  * 밀도 높은 상호작용 + 적정 간격 반복 = 학습 곡선 J-커브.
* 보조 세계관 연결:

  * 📚 체계적 진도형: 시간 기반 커리큘럼 관리
  * ✏️ 성과 집중형: 시험 D-데이 시간 배분·속도 훈련
  * 🎯 개인맞춤형: 개인별 최적 집중·반복 주기 설계
  * ⚡ 목표달성형: 단기 시간미션 부여
  * 🧠 사고력 중심형: 깊이 사고 시간 블록 설계
  * 🚀 자율학습형: 자기주도 시간 설계권 부여
  * 🔍 인지적 도제형: 모델링·코칭에 시간밀도 적용

## X: 문맥지능 (Context)

* 필수 컨텍스트 수집:

  * 과목별 집중 지속 시간, 학습 속도(문항/분), 반복 주기, 시간당 정답률, 시간 분배 패턴, 학습 중 피로도 곡선.
* 스위칭 트리거:

  * 집중 지속 시간 급감 → ⚡ 병행
  * 시험 임박 → ✏️ 속도·배분 집중
  * 복습 전이 실패율↑ → 반복 간격 재설계
  * 자율 계획 미흡 → 📚·🎯 지원

## S: 구조지능 (Structure)

* 표준 변수:

  * `집중블록길이`, `휴식비율`, `시간당문항수`, `반복간격`, `시간밀도지수`.
* KPI:

  * 집중블록 완수율 ≥ 90%
  * 시간밀도지수 ≥ 0.8
  * 반복 간격 준수율 ≥ 85%
  * 시간당 정답률 ≥ 목표치(과목별 설정)
* 데이터 모델 예시:

## P: 절차지능 (Procedure)

* 1일 루틴:

  1. **집중 블록 설계**: 과목·난이도별 시간 설정
  2. **시간 추적**: 실시간 타이머·집중도 체크
  3. **반복 간격 복습**: 1-3-7-14일 간격 복습 큐 자동 생성
  4. **세션 피드백**: 블록별 소요시간·효율 기록
* 1주 루프:

  * 시간 로그 분석 → 밀도·간격 조정
  * 주간 평균 집중 시간·효율 보고
  * 병행 모드(📚·✏️·🎯 등)와 시간 블록 재배치
