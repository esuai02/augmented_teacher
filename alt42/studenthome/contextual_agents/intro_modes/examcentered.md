# 2) ✏️ 성과 집중형 – World Expert Intelligence 설계

## W: 세계관 (Worldview) 정렬

* 핵심 신념: **“시험은 프로젝트, 점수는 결과물.”**
* 목표: 제한된 시간 내 **최대 점수 효율**을 뽑아내는 전술 운영.
* 교사 모토(원문 연결): *시험 전 1주일 동안 모든 과정을 정렬하고 실전 테스트 레벨업 + 보충학습 구조 운영.*
* 전략적 관점:

  * 단기 폭발력 → 장기 커리큘럼보다 **D-데이 기준 역산 플랜**이 우선.
  * **실전 감각 + 점수 회수율**이 KPI.
* 보조 세계관 연결:

  * 📚 체계적 진도형: 시험 이후 커리 리셋·기본기 보강
  * 🎯 개인맞춤형: 취약 파트 급속 보정
  * ⚡ 목표달성형: 집중력 회복·스트릭 관리
  * 🧠 사고력 중심형: 고난도·창의형 대비
  * 🔍 인지적 도제형: 사고과정 정교화(고득점 안정성)
  * 🚀 자율학습형: 상위권이 스스로 모의전 실행·조율

---

## X: 문맥지능 (Context)

* 필수 컨텍스트 수집:

  * 시험일, 출제 범위·비중, 기출 분석(단원별 빈도·오답률), 현재 점수대, 과목별 점수목표, 시간 배분 데이터.
* 스위칭 트리거:

  * 시험 D-30: 📚→✏️ 전환
  * 모의고사/단원테스트 점수 상승폭 정체: 🎯 맞춤형 보강
  * 고난도 문제 연속 실패: 🧠/🔍 사고력·도제 삽입
  * 동기저하·체력저하: ⚡ 미션형 스트릭 가동

---

## S: 구조지능 (Structure)

* 표준 변수:

  * `Dday`, `목표점수`, `점수차이`, `출제빈도맵`, `약점리스트`, `회독횟수(목표: 3)`, `백지복습빈도(일일)`, `실전모의(주2)`.
* KPI:

  * 기출/예상문제 정답률 ≥ 90%
  * 약점영역 점수 상승 ≥ 20점
  * 백지복습 완성률 ≥ 95%
  * 실전모의 시험 평균점수 ≥ 목표점수-5점
* 데이터 모델 예시:

```json
{
  "mode":"exam",
  "Dday":30,
  "targetScore":95,
  "currentScore":82,
  "gap":13,
  "weakness":["벡터", "확률"],
  "finalReviewList":40,
  "mockTestsPerWeek":2
}
```

---

## P: 절차지능 (Procedure)

* **D-30\~D-15**:

  * 기출·유형별 문제집 1회독 완성
  * 취약 단원 백지복습 + 즉시 보충
* **D-14\~D-7**:

  * 실전모의 2회/주, 시간 제한 적용
  * 오답 즉시 분석(48시간 내 완료)
  * 고빈출 유형 집중 스팟트레이닝
* **D-6\~D-2**:

  * 파이널 리뷰(40문항), 감각유지 퀵세션
  * 컨디션 루틴(수면·식사·운동) 적용
* **D-1\~시험일**:

  * 복습은 오직 ‘확실한’ 범위, 새로운 내용 금지
  * 시험 당일 리허설·심리 안정화

## E: 실행지능 (Execution)

* 교사 체크리스트:

  * [ ] 시험 일정·범위 확정
  * [ ] 기출빈도·약점리스트 완성
  * [ ] 회독·모의고사 진행률 점검
  * [ ] 백지복습 누락 여부 확인
  * [ ] 컨디션 관리 루틴 적용
* 학생 루틴(✏️ 학생모드 부연):

  1. D-30부터 시작하는 철저한 시험 대비
  2. 매일 백지 복습
  3. 기출문제 3회독
  4. 컨디션 루틴 확립
  5. 시험 후 오답 분석 48시간 내 완료
* 자동화 매핑:

  * **mathking**: 단원별 출제 빈도 분석, Final 리뷰 구성
  * **management**: 시험 3\~4주 전 계획 리마인드, 예상문제 정확도 추적

## 실행용 에이전트 프롬프트 세트

1. **ExamPlanner**

```
역할: ✏️ 성과 집중형 시험 대비 설계자
입력: 시험일, 출제범위, 기출분석, 현재점수, 목표점수
규칙: ① D-30 역산 플랜 ② 기출 3회독·백지복습 매일 ③ 실전모의 주 2회
출력: D-데이별 학습표(일자/단원/유형/회독/모의), KPI, 보완안
```

2. **WeaknessBooster**

```
입력: 약점리스트, 오답률
처방: 🎯맞춤형 단원보강 + ⚡미션(10문항 클리어·포모도로 25:5)
```

3. **FinalReviewBuilder**

```
입력: 기출빈도, 예상문제리스트
출력: 파이널 리뷰(40문항)·풀이시간·난이도 조정안
```

4. **ConditionManager**

```
입력: 학생 수면/식사/운동/심리 데이터
출력: D-7~시험일까지 컨디션 관리 루틴
```

## R: 성찰지능 (Reflection)

* 시험 후 48h 내 리플렉션 5문:

  1. 예상과 실제 점수 차이, 원인 1\~2개
  2. 가장 많이 맞힌 유형·틀린 유형 비교
  3. 시험 중 시간 배분 오류 여부
  4. 백지복습 효과 체감(0\~5점)
  5. 다음 시험 대비 개선 1\~2개
* 개선 규칙:

  * 반복 오답 유형 → 📚 체계적 진도형 리셋
  * 시간 부족 문제 → 모의실전 속도 훈련 강화
  * 집중력 하락 → ⚡ 목표달성형 블록 삽입

---

## T: 트래픽 지능 (Traffic & Transfer)

* 정보흐름:

  * **D-30**: 계획·기출·약점 → 강도 높이기
  * **D-14**: 모의고사 → 오답→보완
  * **D-7**: 파이널리뷰·컨디션 루틴
* J-커브 준비:

  * D-7\~D-1 집중폭발(모의+리뷰) → 점수 상승 곡선 극대화
* 단절지점:

  * 회독 지연, 모의고사 누락, 백지복습 미수행 → 경고·개입

---

## A: 추상화 지능 (Aftermath)

* 시험 후:

  * 점수데이터·오답패턴·시간배분 로그를 📚 체계적 진도형과 🎯 맞춤형 모드로 전달
  * 파이널 리뷰·고빈출 매트릭스 갱신 → 다음 시험 대비 자산화

---

## 상황별 블렌딩 레시피

* **기초 약점 심각** → ✏️ + 🎯 + 📚 병행
* **고난도 대비** → ✏️ + 🧠/🔍
* **동기저하** → ✏️ + ⚡ (단기 미션·포모도로)
* **상위권 자율** → ✏️ + 🚀 (모의 계획권 일부 위임)

## 변형·최적화 파라미터

* 회독 횟수: 3(기본) / 상위권 2, 하위권 4
* 모의고사 주기: 주 2회(기본) / 고득점 유지 1회
* 파이널 리뷰 문항 수: 40(기본) / 범위·시간에 맞춰 ±10
* 포모도로: 25:5(기본) / 집중력 약한 학생 10:5
