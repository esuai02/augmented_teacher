# 3) 🎯 개인 맞춤형 – World Expert Intelligence 설계

## W: 세계관 (Worldview) 정렬

* 핵심 신념: **“모든 학생은 서로 다른 지도를 들고 있다.”**
* 목표: 개별 학습 수준·패턴·취약점에 맞춰 **NEXT STEP**을 정밀 설계.
* 교사 모토(원문 연결): *학생 고유 UX 축적 · 시험 대비 로드맵 모순점 제거*
* 전략적 관점:

  * 획일적 진도보다 **개별 최적화 루트** 우선.
  * 목표는 점수, 그러나 수단은 개성화.
* 보조 세계관 연결:

  * 📚 체계적 진도형: 기초·커리큘럼 안정화
  * ✏️ 성과 집중형: 시험 임박 시 점수 극대화
  * ⚡ 목표달성형: 짧은 주기 동기 유지
  * 🧠 사고력 중심형: 이해 기반 결손 보완
  * 🚀 자율학습형: 자기주도형 상위권 확장
  * 🔍 인지적 도제형: 사고 패턴 교정·심화

---

## X: 문맥지능 (Context)

* 필수 컨텍스트 수집:

  * 기초 진단 점수, 과목별 세부 단원 숙련도, 오답 유형, 학습 성향(시각·청각·실습형), 집중 시간대, 학습 히스토리.
* 스위칭 트리거:

  * 진도 격차 > 15% → 📚 병행
  * 시험 D-30 → ✏️ 전환
  * 집중력 불안정·실행율 저하 → ⚡ 삽입
  * 사고 전환 필요 → 🧠/🔍 투입
  * 자율성↑ → 🚀 확대

---

## S: 구조지능 (Structure)

* 표준 변수:

  * `기초점수`, `약점리스트`, `학습성향`, `목표달성일`, `개별로드맵`, `개입주기`, `성취로그`.
* KPI:

  * 개별 목표 달성률 ≥ 85%
  * 약점영역 보정 후 향상률 ≥ 20%
  * 학습 지속율 ≥ 90%
  * 학습성향 맞춤 활동 비중 ≥ 70%
* 데이터 모델 예시:

```json
{
  "mode":"custom",
  "baselineScore":65,
  "weakness":["확률","수열"],
  "learningStyle":"visual",
  "targetDate":"2025-12-15",
  "personalRoadmap":[{"week":1,"focus":"확률 기초","hours":8}],
  "checkInCycle":"weekly"
}
```

---

## P: 절차지능 (Procedure)

* 1단계: **진단(주0)** → 2단계: **개별 로드맵 생성** → 3단계: **실행·모니터링** → 4단계: **보정·최적화**
* 수업 루틴:

  1. 진단평가(지필·구술·AI 자동분석)
  2. 약점별 맞춤 자료/문제 배치
  3. 주간 피드백 → 로드맵 미세조정
  4. 월간 성취로그·UX 데이터 축적
* 병행 규칙:

  * 시험 D-30 진입 시 ✏️ 병행
  * 기초 과락 영역 발생 시 📚 강제 주입
  * 동기저하 시 ⚡로 단기 회복

---

## E: 실행지능 (Execution)

* 교사 체크리스트:

  * [ ] 진단 데이터 최신화
  * [ ] 약점별 맞춤 콘텐츠 제공
  * [ ] 주간 성취·이탈률 점검
  * [ ] 병행 모드 전환 여부 판단
* 학생 루틴(🎯 학생모드 부연):

  1. 진단 결과를 그대로 수용
  2. 하루 2h 이상 기초 개념 반복
  3. “무지 노트” 작성
  4. 주 3회 이상 AI 튜터 세션
  5. 작은 성취 기록 유지
* 자동화 매핑:

  * **mathking**: 진단평가 → 맞춤 콘텐츠 자동 제공
  * **management**: 학습 이탈 경보, 히스토리 기반 개입 시점 자동화

## R: 성찰지능 (Reflection)

* 주간 리플렉션 질문:

  1. 가장 개선된 영역 1개
  2. 여전히 어려운 영역 1개와 원인
  3. 성향별 활동 중 효과 최고/최저
  4. 병행모드 스위칭 필요성 여부
  5. 다음 주 개선점 1\~2개
* 개선 규칙:

  * 개선 정체 2주 지속 → 다른 모드 블렌딩
  * 성향별 활동 효과 저조 → 활동 유형 변경
  * 약점 반복 → 📚 or 🔍 강화

---

## T: 트래픽 지능 (Traffic & Transfer)

* 정보흐름:

  * 진단→로드맵→실행→피드백→재설계
  * 시험 D-30 데이터 → ✏️ 모드 전송
* 단절지점:

  * 무지 노트 누락, 주간 피드백 미참여, 이탈률 >15% → 경고·보정
* J-커브:

  * 개인 성취곡선 분석 후 **성취보상 이벤트**로 곡선 급상승 유도

## A: 추상화 지능 (Aftermath)

* 월간: 개인 UX 로그 → 📚·🧠 모드로 환류
* 축적 자산: 성향별 학습패턴 데이터베이스, 약점 보정 템플릿, 성취그래프
