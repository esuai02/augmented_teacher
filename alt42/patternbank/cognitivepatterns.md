# 수학 인지관성 도감 - 60개 학습 패턴 상세 문서

## 개요
이 문서는 ShiningStars 시스템에서 사용되는 60개의 수학 학습 인지관성 패턴을 체계적으로 정리한 것입니다. 각 패턴은 학습 과정에서 발생하는 인지적 장애물을 식별하고, 구체적인 해결 방안을 제시합니다.

## 카테고리별 분류

### 1. 인지 과부하 (Cognitive Overload) - 10개 패턴
### 2. 자신감 왜곡 (Confidence Distortion) - 10개 패턴  
### 3. 실수 패턴 (Mistake Patterns) - 7개 패턴
### 4. 접근 전략 오류 (Approach Errors) - 10개 패턴
### 5. 학습 습관 (Study Habits) - 10개 패턴
### 6. 시간/압박 관리 (Time Pressure) - 6개 패턴
### 7. 검증/확인 부재 (Verification Absence) - 4개 패턴
### 8. 기타 장애 (Other Obstacles) - 3개 패턴

## 상세 패턴 설명

### 카테고리 1: 인지 과부하 (Cognitive Overload) 🧠

#### 패턴 1: 아이디어 해방 자동발화형
- **설명**: 번쩍이는 아이디어가 떠오르면 검증 없이 바로 써 내려가 결국 오답을 양산하는 패턴
- **우선순위**: 높음
- **해결 방법**: 
  - 아이디어가 떠오르면 5초 멈춤
  - 아이디어를 한 줄로 요약 후, '약점 가설' 1개를 곧바로 적기
  - 문제 지문을 다시 읽고, 가설과 비교
- **확인 방법**: 5초 멈춤→가설 쓰기 루틴을 세 번 성공했는지 확인. 요약이 적절했는지 짧게 피드백
- **오디오 스크립트**: "번쩍이는 아이디어가 떠오르면 즉시 적지 말고 5초간 멈춰서 생각해보세요. 핵심 논거를 한 줄로 정리하고, 혹시 틀릴 수 있는 부분은 없는지 오류 가설을 하나 세워보세요."

#### 패턴 6: 작업기억 ⅔ 할당형
- **설명**: 다음 일정·잡생각이 머릿속을 스치며 2/3만 집중하는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - 떠오른 일정은 포스트잇에 적고 덮어두기
  - 25분 집중 / 5분 휴식 Pomodoro 타이머 사용
  - 휴식 때만 메모 확인·업데이트
- **확인 방법**: 25분 집중 세션 3번 돌렸는지, 중간에 잡생각 메모를 몇 번 했는지 확인

#### 패턴 23: 과다 정보 섭취형
- **설명**: 한 문제를 풀며 해설·영상·블로그 등 여러 자료를 동시에 열어 인지 부하가 폭발하는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 문제당 참고자료 최대 2개 원칙
  - 추가 자료가 필요하면 기존 2개 중 1개를 닫고 새로 연다
  - 학습 끝나면 참고자료 목록을 요약 5줄로 정리

#### 패턴 42: 노이즈 필터 실패형
- **설명**: 지문 속 중요치 않은 숫자·문장이 작업기억을 점유해 핵심 정보를 덮어버리는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 문제를 처음 읽을 때 밑줄(핵심) / 연필 흐림선(노이즈) 2단계 표시
  - 풀이 중 노이즈 부분은 괄호로 접어두기
  - 최종 검산 시 노이즈가 풀이에 영향을 줬는지 체크

#### 패턴 56: 전략 중복 추적 피로형
- **설명**: 동시에 3가지 이상 풀이를 전개하다 작업기억이 분산-탈진하는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 동시에 2개 풀이만 허용, 3번째 아이디어는 대기 메모 칸에 보류
  - 두 풀이 중 하나가 막히면 대기 칸에서 1개만 꺼내 진행
  - 세션 종료 후 사용 안 한 아이디어를 '보류 로그'로 분류‧검토

### 카테고리 2: 자신감 왜곡 (Confidence Distortion) 😰

#### 패턴 2: 3초 패배 예감형
- **설명**: '못 풀 것 같다'는 느낌이 3초 만에 뇌를 잠그고, 관련 개념 연결이 끊어지는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - '포기 신호'를 감지하면 3분 타이머를 켜고 문제 해석을 처음부터 다시 적기
  - 막힌 부분을 눈으로 3분간 응시하며 조건·단어를 재색인
- **오디오 스크립트**: "포기하고 싶은 마음이 들면 바로 타이머를 3분으로 설정하세요. 그리고 문제를 다시 읽으면서 다른 관점에서 해석해보세요."

#### 패턴 3: 과신-시야 협착형
- **설명**: 과한 자신감으로 숫자·기호의 미세한 차이를 인식하지 못하는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 풀이 착수 전 심호흡 10회
  - 비슷한 기호·수치를 색펜으로 구분 표시
  - 계산 단계마다 '작은 차이 체크' 칸에 ✔︎

#### 패턴 5: 모순 확신-답불가형
- **설명**: '틀린 곳이 없다'는 집착으로 시야가 좁아져 교정을 못 하는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 답이 안 나올 때 '간단 실수 90%' 문장을 써서 관점을 전환
  - 풀이를 거꾸로 읽으며 '사소한 실수 찾기' 게임화
  - 한 번은 다른 색 펜으로 다시 써보기

#### 패턴 29: 자기긍정 과열형
- **설명**: '이건 내가 잘하던 유형'이라는 자기암시로 검산·근거 검토를 생략하는 패턴
- **우선순위**: 낮음
- **해결 방법**:
  - '익숙유형' 생각이 들면 문제 번호 옆에 검산 플래그★ 표시
  - 풀이 후 ★이 있는 문제는 역대입·조건 체크 2단계 검산 필수
  - 주간 회고에서 ★ 문제의 실제 정답률을 통계로 기록

### 카테고리 3: 실수 패턴 (Mistake Patterns) ❌

#### 패턴 4: 무의식 연쇄 실수형
- **설명**: 손이 먼저 움직여 사소한 계산 실수가 꼬리를 무는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - 숫자 한 줄 쓸 때마다 펜을 내려놓고 1초 휴식
  - 매일 풀이 후 '실수 장면' 1개 기록
  - 다음 날 첫 학습 전에 그 기록을 재확인

#### 패턴 28: 공간-시각 혼선형
- **설명**: 도형·그래프·좌표를 머릿속에 잘못 배치해 관계를 뒤집어 버리는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 문제를 읽자마자 A6 메모지에 빠른 스케치(축·꼭짓점·변수 기입)
  - 변수나 길이 변화가 생길 때마다 스케치를 즉시 업데이트
  - 풀이 완료 후 스케치 ↔ 최종 답을 색펜 화살표로 연결

#### 패턴 38: 단위 무시형
- **설명**: 길이·각도·π 변환 등 단위 체크를 생략해 결과가 엇갈리는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - 단위 변환이 필요할 때마다 둥근 박스로 원·목표 단위 표시
  - 변환 후 박스 옆에 '변환 OK' 스탬프(✔︎) 찍기
  - 답안 작성 직전 모든 박스를 훑어 미검증 박스=0 확인

#### 패턴 39: 시각화 회피형
- **설명**: 그래프·도형 그리기를 귀찮아해 공간적 관계를 착시·오독하는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 도형·그래프 문제는 A6 메모지에 60초 제한 스케치를 필수
  - 변수 값이 변할 때마다 색펜으로 동적 업데이트
  - 풀이 후 스케치와 알지브라식 답을 화살표 연결해 일치 여부 확인

#### 패턴 48: 관성적 읽기 스킵형
- **설명**: 익숙해 보이는 문제라 생각해 지문의 끝을 읽지 않고 풀이를 시작하는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 문장 끝마다 '／' 표시해 끝까지 시각적으로 확인
  - 표시 후 마지막 문장을 큰 소리로 1번 읽고 착수
  - 풀이 중 조건 충돌이 생기면 스킵 여부를 체크표로 기록

#### 패턴 50: 단계 통합 과속형
- **설명**: 두세 단계를 한 줄로 압축해 적으면서 오류 추적이 불가능해지는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 2단계 이상은 반드시 화살표 대신 연속 번호(①②)로 구분
  - 통합 줄 작성 후 각 번호 옆에 중간 결과를 따로 산출
  - 검산 시 중간 결과와 최종 결과 간 일관성을 확인

#### 패턴 53: 계산 체계 혼합형
- **설명**: 분수↔소수, 라디안↔도 등 단위를 혼용하다 값이 뒤섞이는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 변환이 일어날 때마다 변환표를 노트 옆에 작성
  - 최종 계산 단계에서 '최종 단위 일관?' 체크박스를 ✔︎
  - 혼합 오류가 나오면 변환표를 색펜으로 강조 재정리

### 카테고리 4: 접근 전략 오류 (Approach Errors) 🎯

#### 패턴 7: 반(半)포기 창의 탐색형
- **설명**: '어차피 틀릴 것'이라며 낮은 확률의 창의 풀이만 헤매는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 정석 접근 A안을 먼저 10분 시도
  - 실패 시 A안 문제점 1줄 정리
  - B안 스케치 → B안도 막히면 과감히 답안·해설 구조 분석

#### 패턴 10: 불확실 강행형
- **설명**: 근거 부족인데도 '일단 적용'해서 오류가 연쇄되는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 근거 약하면 노란 포스트잇에 '확신 ★☆☆' 등급 표시
  - 별 1‧2개인 줄은 풀이 끝에 재검산 표시(✔︎)
  - 검산 단계에서 ★ 1‧2 지점 우선 점검

#### 패턴 13: 징검다리 난도적형
- **설명**: 청킹 없이 산발적으로 추론해 전역 구조를 놓치는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 문제를 3~4개 '청크'로 나누고 각 단계에 번호(①②③…) 붙이기
  - 단계 끝마다 '다음 단계 조건'을 한 줄 메모
  - 최종 답 후 번호 순서를 거꾸로 점검(③→②→①)

#### 패턴 16: 확률적 답안 던지기형
- **설명**: 근거가 부족한데도 '일단 찍어보자' 식으로 답을 기입해 오류가 연쇄되는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 근거가 약할 때는 노란 포스트잇에 '확신 ★☆☆' 등급 표시
  - ★ 1·2개가 붙은 줄은 풀이 끝에 재검산 필수
  - 최종 제출 전, ★ 표시가 있는 줄만 모아서 1분 스피드 셀프 퀴즈

#### 패턴 24: 이론-연산 전도형
- **설명**: 개념 증명·이론에 깊게 몰입하다가 정작 필수 계산을 뒤로 밀어 실수를 유발하는 패턴
- **우선순위**: 낮음
- **해결 방법**:
  - 증명 줄이 10줄을 넘기면 바로 계산 단계 체크 박스 작성
  - 증명 ↔ 계산을 N:1 교차(10줄마다 계산 1번) 구조로 강제
  - 최종 답 후 증명·계산 단계를 색깔 다른 하이라이터로 구분 표시

#### 패턴 32: 역추적 단절형
- **설명**: 답을 먼저 보고 거꾸로 이유를 찾다 논리 사다리가 중간에서 끊기는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 답 확인 전 역방향 체크리스트를 빈칸으로 작성
  - 체크리스트를 채우며 필요 근거를 파란색, 이미 있는 근거를 검정으로 표시
  - 빈칸이 남으면 앞단계를 정·역방향 교차 검토

#### 패턴 33: 사다리 건너뛰기형
- **설명**: 중간 논증을 생략하고 결론으로 직행, 근거 빈칸을 스스로 인식하지 못하는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - 논증 단계에 번호와 화살표를 모두 명시
  - 결론에 도달하면 ①부터 화살표를 역방향으로 따라가며 근거 문장 점검
  - 빠진 단계가 있으면 빨간펜으로 'Missing Step!' 태그

#### 패턴 37: 개념-문제 불일치 간과형
- **설명**: 문제에서 요구하는 개념과 다른 영역 해법을 고집해 방향이 어긋나는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 문제 읽자마자 상단에 '필수 개념' 1줄 제목 작성
  - 풀이 중 개념이 바뀌면 제목 옆에 🚨표시 후 이유 메모
  - 최종 답 후 제목과 실제 사용 개념이 일치? 불일치? 이중 체크

### 카테고리 5: 학습 습관 (Study Habits) 📚

#### 패턴 8: 해설지-혼합 착각형
- **설명**: 내 생각과 해설 내용을 섞어 쓰다 근거가 뒤섞이는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 내 풀이=파란색, 해설=빨간색 두 색깔 분리 기록
  - 해설을 읽을 때 '왜 다른가?' 차이 2개 메모
  - 하루 뒤, 파란·빨간 노트를 다시 읽어 통합 정리

#### 패턴 9: 연습 회피 관성형
- **설명**: '이해했어' 착각으로 반복 연습을 건너뛰고 넘어가는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - 새 개념 배우면 즉시 난이도 Low·Mid·High 1문제씩 풀기
  - Low / Mid 틀리면 해당 개념 '불완전'로 표시 후 재학습
  - 주간 체크리스트: 개념당 최소 3회 재방문

#### 패턴 14: 무의식 재현 루프형
- **설명**: 예전에 성공했던 공식을 맹목적으로 재사용하며 문제 특성을 무시하는 패턴
- **우선순위**: 낮음
- **해결 방법**:
  - 공식 사용할 때 '조건 동일?' 체크박스를 옆에 그리기
  - 조건이 다르면 즉시 다른 방법 후보를 메모
  - 학습 후 '조건 불일치 발견 목록'을 주간 로그에 기록

#### 패턴 19: 과거 방식 고착형
- **설명**: 새로운 유형도 과거에 익숙했던 공식·방법만 고집하는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 문제를 읽고 30초 간 '이 유형을 처음 본다면?' 스스로 질문
  - 떠오른 대안 풀이를 메모 2줄로 적어보기
  - 실제 풀이 후 기존 공식 vs 대안 풀이의 장·단점 비교 작성

#### 패턴 25: 단일 예시 착시형
- **설명**: 특정 예제에서 성공한 방식을 새 문제에 그대로 적용해 예외 상황을 놓치는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 새 문제 시작 시 '예시와 다른 점 3개'를 빠르게 메모
  - 풀이 중 3개의 차이가 모두 반영됐는지 중간·최종에 체크
  - 주간 회고 때 '예시 착시 → 교정 성공 사례'를 포트폴리오에 기록

#### 패턴 35: 공식 암기 과신형
- **설명**: 문제 특성과 무관하게 외운 공식만 기계적으로 대입, 오적용 위험이 큰 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 공식을 적을 때 오른쪽에 '출처·조건'을 1줄 주석
  - 공식 사용 전 조건 매칭 질문 3개 답 체크
  - 매주 사용한 공식·조건 목록을 통계로 정리

#### 패턴 41: 지식-실행 단절형
- **설명**: 개념은 이해했지만 문제 적용 단계에서 멈칫해 '알아도 못 푸는' 상황이 반복되는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - 새 개념 학습 직후 예제 1문제를 즉시 해결(3분 제한)
  - 예제가 막히면 '개념 → 절차 → 예시' 흐름을 음성으로 20초 복창
  - 복창 후 다시 풀어 보고 성공 여부를 O/X로 기록

#### 패턴 57: 목표-행동 단절형
- **설명**: '개념 학습'이 '풀이 수집'으로 변질돼 원래 목표를 잊는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - 학습 시작 전 '오늘 목표 1문장'을 화면 상단에 고정
  - 30분마다 목표 문장을 소리 내어 읽고 현재 행동과 매칭 여부 체크
  - 세션 끝에 목표 달성도를 0~100%로 자평·기록

### 카테고리 6: 시간/압박 관리 (Time Pressure) ⏰

#### 패턴 11: 속도 압박 억제형
- **설명**: 시험 시간이 눈에 들어올 때마다 '빨리 해야 한다'는 압박이 새 아이디어와 기억을 눌러 버리는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - 시작과 동시에 손목시계·휴대폰 시계 뒤집기
  - 조용 타이머를 15분 간격으로 설정(삐 소리 X, 진동 O)
  - 타이머 울릴 때마다 현재 문제를 1문장으로 요약 후 진행 여부 판단

#### 패턴 12: 시험 트라우마 악수형
- **설명**: 과거에 시험을 망친 기억이 문제 순서·전략에 투영돼 '악수'를 두는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - 시작 2분 내에 '가장 쉬운 2문제'를 골라 먼저 해결
  - 성공감이 생기면 그다음 문제를 난도별 라벨링(L·M·H) 후 착수
  - 45분 세션 후 성공 → 어려움 순서를 다시 리뷰

#### 패턴 26: 시간 왜곡 긴장형
- **설명**: 제한 시간을 실제보다 덜/더 급하게 느껴 불필요한 조급함·지연을 만드는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 세션 60분을 45분 타이머 + 15분 여유로 나누기
  - 45분 타이머 종료 시 현재 진행도를 %로 적기
  - 남은 15분은 검산·보완 전용 영역으로만 사용

#### 패턴 46: 전환 비용 과소평가형
- **설명**: 여러 문제·풀이법을 빈번히 바꾸며 작업기억을 재로딩, 집중 에너지를 낭비하는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 문제 전환 전 현재 풀이를 2줄 요약해 노트 여백에 작성
  - 새 문제로 넘어갈 때 요약 옆에 타임스탬프 기록
  - 하루 학습 끝에 전환 횟수와 소요 시간을 막대그래프로 시각화

#### 패턴 59: 다중 문제 스위칭 과부하형
- **설명**: 시험 직전에 여러 문제를 빠르게 훑다 인지 세트업이 실패하는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - 시험 전날 최대 3세트(L·M·H 각 1세트)만 선정
  - 각 세트 완료 후 5분 정리 노트로 핵심만 요약
  - 요약 노트를 아침 리콜(5분)로 다시 읽고 시험장 입장

### 카테고리 7: 검증/확인 부재 (Verification Absence) ✔️

#### 패턴 15: 조건 회피-추론 생략형
- **설명**: 복잡한 조건을 '시야 밖'으로 밀어두고 직감만으로 추론을 강행하는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - 문제의 각 조건 옆에 ✔︎를 표시하고 한글로 5-7단어 요약 작성
  - 풀이 중 조건을 사용할 때마다 ✔︎ 색깔을 검정 → 초록으로 변경
  - 남은 검정 ✔︎가 있으면 풀이 완료 전 반드시 조건 재적용

#### 패턴 20: 불완전 개념 종결형
- **설명**: 정의·조건을 끝까지 읽지 않고 '충분해'라고 판단해 풀이를 서둘러 종결하는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - 문제에 나온 용어·명제는 노트 하단에 정의 원문을 그대로 필사
  - 풀이 중 해당 정의를 적용할 때 밑줄 + 옆에 페이지 참조 표시
  - 풀이 후 '정의 적용 위치'를 하이라이트 색으로 모두 확인

#### 패턴 31: 개념-용어 혼동형
- **설명**: 정의·기호를 모호하게 기억해 비슷한 단어와 혼동, 조건 매칭에 실패하는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 개념 등장 시 색상 코드 지정: 정의(파란), 정리(초록), 예외(보라)
  - 유사 용어는 노트 오른쪽에 '헷갈림 리스트'로 별도 기록
  - 학습 종료 전 헷갈림 리스트를 퀴즈 카드로 3분 복습

#### 패턴 34: 조건 재정렬 미흡형
- **설명**: 복합 조건의 순서를 무시해 필수·보조 정보를 혼선시키는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 모든 조건 앞에 순번 스티커(①②③) 부착 후 순서 고정
  - 풀이 중 해당 조건을 사용하면 순번 옆에 체크✔︎
  - 체크되지 않은 조건이 남으면 순서를 재검토해 적용 위치 보완

#### 패턴 36: 근사치 타협형
- **설명**: '대략 맞겠지' 하고 근사 계산으로 풀이를 종료, 오차 검증을 생략하는 패턴
- **우선순위**: 낮음
- **해결 방법**:
  - 근사값을 쓸 때마다 옆에 '±오차 범위'를 바로 기입
  - 최종 답 전 오차 ≤ 목표 허용치? 체크박스에 ✔︎
  - 오차 초과 시 정확 계산 또는 더 정밀한 근사법 재적용

#### 패턴 47: 반례 무시형
- **설명**: 풀이가 순조로우면 '예외 없겠지'라며 반례 검증을 생략하는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - 풀이 과정 중 '반례 가능성 칸'을 만들고 최소 1개 쓰기
  - 최종 답 전 반례를 실제로 수치·그림으로 확인
  - 반례가 존재하면 풀이를 분기해 조건 보강 또는 전략 수정

#### 패턴 49: 조건 재해석 과잉형
- **설명**: 애매한 문구를 자의적으로 해석해 핵심 의미를 빗나가는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 애매 문구는 즉시 질문 카드 작성 → 교사·AI 튜터에게 전송
  - 답변을 받을 때까지 임시 해석에 '?' 마크 붙여 진행
  - 확정 해석 후 '?' 마크 부분을 빨간펜 정정

#### 패턴 51: 중간점검 생략형
- **설명**: 풀이가 절반쯤 진행됐을 때 검산 없이 끝까지 돌진, 오류를 초기에 놓치는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - 문제 착수와 동시에 자동 알람을 풀이 예상시간의 50% 지점에 설정
  - 알람이 울리면 즉시 진행 중인 식에 역대입 검증 수행
  - 검산 결과를 O／Δ／X 기호로 표시 후 계속 진행

#### 패턴 52: 검산 회피형
- **설명**: 시간 아까워 검산을 건너뛰어 정답률이 흔들리는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - 최종 답 기입 직후 검산 메뉴 3개(역대입·단위·추가 조건) 중 1개를 무조건 실행
  - 검산 완료 시 문제 번호 옆에 ✔︎ 스탬프 찍기
  - 주간 회고 때 검산 스탬프 개수와 실제 정답률을 분석 그래프로 비교

#### 패턴 54: 음운 혼동형
- **설명**: 'sine'↔'sign', 'root'↔'route' 등 비슷한 발음을 착각해 기호·용어를 바꾸는 패턴
- **우선순위**: 낮음
- **해결 방법**:
  - 유사 음 용어를 색깔로 구분
  - 필기 시 발음을 속삭이며 기호를 다시 한번 확인
  - 학습 후 유사 음 용어 목록을 퀴즈 카드로 2분 복습

#### 패턴 55: 참조 프레임 불일치형
- **설명**: 좌표 원점·축 방향 전환을 놓쳐 그래프·변수를 잘못 배치하는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 좌표 변환이 나오면 작은 스케치로 새 원점·축을 즉시 표시
  - 변수·길이를 옮길 때마다 스케치 상에 마커 펜으로 업데이트
  - 풀이 완료 후 스케치와 대수식 관계를 검산 화살표로 연결

---

### 카테고리 8: 기타 장애 (Other Obstacles) 🔧

#### 패턴 17: 방심 단기 기억 증발형
- **설명**: 잠깐 산만해지면서 방금 세운 관계식이나 조건을 잊어버리는 패턴
- **우선순위**: 낮음
- **해결 방법**:
  - 새 식·조건을 세울 때마다 왼쪽 여백에 번호 목록으로 기록
  - 산만함을 느끼면 즉시 목록을 큰 소리로 1줄 복창
  - 풀이 종료 후 목록과 실제 풀이를 체크‧매칭

#### 패턴 18: 도구 의존 과적형
- **설명**: CAS·계산기에 과도하게 의존해 개념 이해·추론 회로가 비활성화되는 패턴
- **우선순위**: 낮음
- **해결 방법**:
  - CAS 입력 전에 예상 결과 범위·부호·대략 값을 손으로 스케치
  - 계산 결과가 나오면 예상 vs 결과를 3초 비교해 차이를 표시
  - 차이가 크면 계산 단계나 모델링 방식을 수작업으로 한 번 더 검산

#### 패턴 21: 피로-오답 포용형
- **설명**: 체력이 떨어질수록 오류 감지력이 급감해 '이 정도면 됐겠지' 하고 넘어가는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 30분 집중 + 2분 눈·목 스트레칭 루틴(타이머 필수)
  - 피로 신호 느끼면 물 3모금 + 10초 눈감기
  - 세션 마지막 5분은 반드시 검산 전용으로 예약

#### 패턴 22: 감정 전염 스트레스형
- **설명**: 옆 친구·교사 표정 / 소음에 불안이 증폭돼 작업기억 용량이 급락하는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 불안을 느끼면 즉시 4-7-8 호흡법(4초 들숨-7초 정지-8초 날숨) 1회
  - 집중 음악(화이트노이즈·Lo-fi) 1곡 반복 설정
  - 방해 요소가 지속되면 A6 메모지에 감정 상태 한 단어 적고 덮기

#### 패턴 27: 보상 심리 도박형
- **설명**: 앞선 실수를 만회하려는 집착으로 복잡한 해법을 억지로 채택하는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - '분노 수정' 감정을 느끼면 2분 워킹 브레이크(자리서 20걸음 왕복)
  - 돌아와서 현재 문제 난이도를 L·M·H 중 다시 판단
  - 고난도('H')로 변질되면, 바로 새로운 문제로 전환 후 나중에 재도전

#### 패턴 30: 메타인지 고갈형
- **설명**: 문제 진행 중 '내가 뭘 모르는지' 평가 기능이 고갈돼 학습이 무의식적 반복으로 변하는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 20분마다 알람 → '내가 모르는 부분 1문장 메모' 루틴
  - 메모한 문장을 과녁표 (🎯) 표시 목록에 모으기
  - 세션 종료 후 과녁표 항목을 자료 탐색·질문 리스트로 전환

#### 패턴 40: 메모 불능 기억 과신형
- **설명**: '머릿속에 다 있어'라며 메모 없이 진행, 항목 순서가 뒤섞이는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 조건·중간값을 '미니 메모칩'(포스트잇)으로 즉시 기록
  - 풀이 단계 전환 때마다 메모칩을 눈으로 터치 확인
  - 풀이 후 메모칩을 순서대로 재정렬하며 논리 흐름 검산

#### 패턴 43: 인터럽트 리셋 불능형
- **설명**: 알림·대화 등 외부 방해 후 이전 맥락을 복구하지 못해 흐름이 끊기는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 방해를 받기 전 단계를 한 줄로 요약해 상단 포스트잇에 써둔다
  - 방해가 끝나면 포스트잇을 소리 내어 읽고 동일 단계에서 재시작
  - 포스트잇을 떼어 노트 하단에 붙이며 'Context Restored' 체크

#### 패턴 44: 감정 보상 과다형
- **설명**: 작은 성공에 과도한 도파민 보상이 발생해 주의력이 이완되고 다음 단계가 느슨해지는 패턴
- **우선순위**: 낮음
- **해결 방법**:
  - 성공 시 10초 셀프 칭찬(속삭이기) 후 바로 타이머 재가동
  - 다음 단계 착수 전 '다음 할 일 5단어' 메모
  - 학습 끝에 총 셀프 칭찬 시간을 분 단위로 기록(1분 이내 목표)

#### 패턴 45: 휴식 부족 저하형
- **설명**: 장시간 집중 후 인지 피로가 누적돼 오류 검출률이 급락하는 패턴
- **우선순위**: 높음
- **해결 방법**:
  - 90분 세션 → 15분 휴식 'Pomodoro Plus' 스케줄 설정
  - 휴식 시간엔 스트레칭 + 물 1컵 + 창밖 2분 바라보기 수행
  - 휴식 후 첫 문제를 검산 문제로 선택해 집중도 회복 확인

#### 패턴 58: 피드백 과민형
- **설명**: 작은 지적에도 불안이 급등해 작업기억 용량이 급락하는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 부정적 피드백을 받으면 30초 눈 감고 복식호흡
  - 노트에 '교정 = 성장' 문장을 써서 시야에 두기
  - 피드백을 '사실' '해석' '다음 행동' 3열 표로 분리 기록

#### 패턴 60: 자기평가 누적 오류형
- **설명**: 진행 중 정확도 추정이 계속 어긋나 자기효능감이 왜곡되는 패턴
- **우선순위**: 중간
- **해결 방법**:
  - 각 문제 해결 후 난이도·정확도 5점 척도 자체 채점
  - 세션 끝에 실제 채점 결과와 산포도 그래프로 비교
  - 편향(과·과소 평가)을 발견하면 다음 세션 보정 목표 설정
