# Shining Stars - AI 교육 에이전트 시스템 아키텍처

## 시스템 개요
수학 학습에 대한 부정적 인식을 제거하고 뇌가소성을 활용하여 학생들의 잠재력을 최대화하는 AI 기반 교육 시스템

### 핵심 목표
1. **심리적 변화**: 수학에 대한 무의식적 부정 인식 제거
2. **도파민 시스템 활성화**: 
   - Tonic 도파민: 기본적인 동기부여 수준 향상
   - Phasic 도파민: 상황별 성취감과 보상 제공
3. **뇌가소성 촉진**: 지속적인 성찰과 피드백을 통한 신경 연결 강화
4. **개인화된 학습**: AI 에이전트의 맞춤형 가이드

## 아키텍처 구성

### 1. 프론트엔드 레이어
- **인터페이스**: 게임화된 별자리 맵 UI
- **성찰 도구**: 감정과 생각을 표현하는 입력 시스템
- **피드백 시각화**: AI 응답의 시각적 표현

### 2. AI 에이전트 레이어
- **대화 엔진**: GPT-4 기반 자연어 처리
- **심리 분석기**: 학생의 감정 상태 파악
- **맞춤형 응답기**: 개인화된 피드백 생성
- **진행 추적기**: 학습 여정 모니터링

### 3. 데이터 레이어
- **학생 프로필**: 개인별 학습 특성 저장
- **성찰 기록**: 시간별 응답 데이터
- **진행 상태**: 여정 맵 진행도
- **AI 피드백**: 생성된 응답 히스토리

### 4. 보안 및 인증
- Moodle LMS 통합 인증
- 역할 기반 접근 제어 (학생/교사/관리자)

## 주요 기능

### 학생 경험
1. **몰입형 UI**: 우주 테마의 여정 맵
2. **자유로운 표현**: 제약 없는 성찰 입력
3. **즉각적 피드백**: AI의 공감적 응답
4. **진행 시각화**: 성장 과정 확인

### AI 에이전트 특징
1. **공감적 대화**: 학생의 감정 이해
2. **격려와 동기부여**: 긍정적 강화
3. **통찰력 제공**: 메타인지 촉진
4. **개인화**: 학생별 맞춤 접근

### 교사 도구
1. **진행 모니터링**: 학급 전체 현황
2. **인사이트 대시보드**: AI 분석 결과
3. **개입 알림**: 특별 관심 필요 학생

## 기술 스택
- **프론트엔드**: HTML5, CSS3, JavaScript (ES6+)
- **백엔드**: PHP 7.4+
- **AI**: OpenAI GPT-4 API
- **데이터베이스**: MySQL/MariaDB
- **인증**: Moodle LMS Integration
- **캐싱**: Redis (선택사항)
