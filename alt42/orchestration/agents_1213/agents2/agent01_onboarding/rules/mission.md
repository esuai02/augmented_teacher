# Agent 01 - Onboarding Mission (수학학원 시스템 특화)

## 목적
- 신규/대상 학생의 기본 프로필과 학습 이력 로딩 시 교사의 노하우를 반영하여 초기 맥락을 정확히 수집한다.
- **수학학원 시스템에 특화된 정보 수집**: 수학 학습 스타일, 학원 정보, 수학 성적, 교재 정보 등
- 담임 선생님에게 수집된 내용을 자동으로 전달한다.
- 등록전 학생의 경우 추가정보 제공 등을 통하여 등록을 유도한다.

## 핵심 점검 항목

### 기본 프로필
- 프로필: 이름, 이메일, 최근 접속, MBTI/학습성향
- 학원에 기대하는 내용과 향후 변화를 원하는 욕구 요소

### 수학학원 시스템 특화 항목
- **수학 학습 스타일**: 계산형, 개념형, 응용형 분류
- **학원 정보**: 학원명, 학원 등급(반), 학원 수업 일정
- **수학 성적/수준**: 최근 시험 점수, 등수, 취약 단원
- **학습 교재**: 학교 교과서, 학원 교재(쎈, 개념원리, RPM 등), 문제집
- **단원별 마스터링**: 완료 단원, 진행 중 단원, 미완료 단원 (단원 연계성 고려)

## 데이터 소스
- Moodle 사용자(`mdl_user`) + 학생 프로필 확장 테이블
- 최근 활동 로그(표준 로그)/완료 현황
- Exam System 데이터
- Learning Type 데이터
- **수학학원 시스템 데이터**: 학원 정보, 수학 학습 스타일, 단원별 마스터링

## 표현 정책
- 학생에게는 긍정 프레이밍
- 교사에게는 구체적 지표와 개선 포인트
