## 에이전트 개요

**목적**: 다른 에이전트들의 분석 결과를 종합하여 개입 타겟팅

**주요 기능**:
- 여러 에이전트 데이터 종합 분석
- 개입 필요성 판단
- 타겟팅 우선순위 결정

**특징**: 직접적인 DB 테이블 없이 다른 에이전트 데이터를 종합 사용
