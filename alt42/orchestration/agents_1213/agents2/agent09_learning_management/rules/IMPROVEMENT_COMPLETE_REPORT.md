## 🎯 핵심 개선 사항

### 1. 데이터 부재 시나리오 대응

#### 학생 설문 자동 생성 (S0_R5)
- **상황**: DB 데이터 밀도 50% 미만
- **대응**: 학습 습관, 어려움, 목표/동기, 만족도 설문 자동 생성
- **효과**: 데이터 부족 시 보완 가능

#### 선생님 체크리스트 수집 (S0_R6)
- **상황**: 선생님이 직접 관찰한 내용 입력
- **대응**: 학생 상태, 학습 태도, 어려움/강점 체크리스트 수집
- **효과**: 정성적 데이터 수집 가능

#### 텍스트 입력 분석 (S0_R7)
- **상황**: 자유 형식 텍스트 입력
- **대응**: 자연어 처리 (키워드 추출, 감정 분석, 주제 분석)
- **효과**: 자유 형식 피드백 수집 가능

#### 다중 소스 통합 (S0_R8)
- **상황**: DB + 설문 + 체크리스트 + 텍스트 입력
- **대응**: 소스별 신뢰도 가중치 기반 통합 분석
- **효과**: 분석 정확도 향상

- 위험도 등급: 낮음/주의/경고/긴급

#### 조기 경고 시스템 (S1_R2)

- 단계별 맞춤형 개입 전략

- 쉬운 목표부터 시작
