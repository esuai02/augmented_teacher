#### ⚠️ 부족한 점
- **수학 단원별 난이도 DB 부재**:

- **학생 수준(L)의 정량적 측정 부재**:
  - 선생님은 "이 학생은 중상위권, 함수는 약하고 도형은 강해"를 수치화
  - 현재 룰에는 L∈[0,1] 언급만 있고, 실제 측정/업데이트 로직 없음
- **교재별 특성 DB 부재**:
  - 선생님은 "쎈은 유형 많고, 개념원리는 설명 좋고" 같은 교재 특성 알고 있음

## 💡 현직 선생님의 핵심 역량 vs 현재 룰

### 현직 선생님이 보유한 사전 지식 (경험 DB)

1. **단원별 난이도 지도**
   ```
   중1: 정수와 유리수(쉬움) → 일차방정식(보통) → 함수(어려움)
   중2: 일차함수(보통) → 연립방정식(보통) → 도형(어려움)
   중3: 이차함수(어려움) → 이차방정식(보통) → 통계(쉬움)
   ```

2. **단원별 평균 학습 시간**
   ```
   함수 단원: 개념 4시간 + 유형 6시간 + 심화 4시간 = 총 14시간
   통계 단원: 개념 2시간 + 유형 3시간 + 심화 2시간 = 총 7시간
   ```

3. **출제 비중 경험**
   ```
   중간고사: 함수 30%, 도형 25%, 통계 15%, 기타 30%
   기말고사: 전체 범위, 함수/도형 비중 높음
   ```

4. **학생 수준별 학습 방법**
   ```
   하위권: 교과서 중심, 반복 연습, 개념 정리 강조
   중위권: 문제집 중심, 유형 연습, 오답 정리
   상위권: 심화 문제집, 변형 문제, 창의 문제
   ```

**현재 룰은 이러한 사전 지식 DB가 전혀 없음** ⚠️
