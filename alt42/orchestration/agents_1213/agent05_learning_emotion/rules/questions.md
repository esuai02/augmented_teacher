💬 포괄형 질문 1

“이 학생의 학습활동 전반에서 감정의 흐름은 어떤 패턴으로 움직이고 있을까?”
→ 개념 이해부터 문제풀이, 복습, 귀가검사까지의 감정 곡선을 분석해
집중-피로-회복-몰입의 순환 패턴이 안정적인지 진단하고,
학생이 몰입에 진입하거나 빠져나오는 ‘감정 전환 구간’을 시각적으로 도출.
이를 기반으로 시그너처루틴 레벨 감정패턴을 모델링하고,
현재 감정 루틴의 안정도 점수를 계산해 다른 에이전트에 전달.

💬 포괄형 질문 2

“이 학생의 감정 페르소나를 기반으로, 어떤 방식의 피드백과 상호작용이 학습 효율을 극대화할까?”
→ 학생의 감정형 페르소나(예: 불안-보완형, 도전-몰입형, 회피-저항형)를 식별한 뒤
TTS 톤, 인터페이스 타이밍, 과제 순서, 휴식 삽입 시점 등
감정 반응에 맞는 피드백 구조를 최적화함.
특히, 감정형 페르소나별 피드백 수용률·회복속도 데이터를 기반으로
선생님/AI의 개입 방식(논리형·격려형·리듬형)을 맞춤 제안.

💬 포괄형 질문 3

“이 학생이 장기적으로 학습 감정의 자기조절 능력을 키우기 위해 지금부터 어떤 루틴이 필요할까?”
→ 감정 회복속도, 피로 누적 패턴, 메타인지적 감정 인식도를 종합 분석해
정서적 회복 루틴과 자기통제 루틴(예: 감정 명시 → 행동 전환 → 성취 인식)을 설계.
이를 통해 학습감정의 자가 피드백 시스템을 형성하고,
장기적으로 감정의 파동이 학습 효율에 미치는 영향을 예측·조정하도록
“감정 기반 자기조절 성장 로드맵”을 생성.

# 🧠 학습감정 분석 에이전트 (Agent 05)

## 1️⃣ 이 에이전트를 필요로 하는 상황

| 구분                   | 호출 시점                     | 감정분석 목적           | 연계되는 다른 에이전트   |
| -------------------- | ------------------------- | ----------------- | -------------- |
| ① 개념이해 직후            | 개념은 배웠는데 집중 흐름이 일정하지 않을 때 | 이해 수준과 감정 몰입도 파악  | 개념노트, 목표·계획 분석 |
| ② 유형학습 중 난이도 편차 발생 시 | 학생이 갑자기 흥미를 잃거나 과몰입 상태일 때 | 동기 수준·좌절감 식별      | 유형학습, 침착도 분석   |
| ③ 문제풀이 도중 멈춤 시간이 길 때 | 불안·혼란·확신부족 원인 식별          | 문제노트, 질의응답        |                |

| ⑤ 질의응답 직전·직후         | 질문 망설임/자신감·정서유형 탐색        | 질의응답, 개념노트        |                |
| ⑥ 복습 루틴 진입 전         | 복습에 대한 부담감·저항감 감정 탐지      | 복습루틴, 휴식루틴        |                |
| ⑦ 포모도르 종료 시점         | 몰입도-피로도의 균형 확인            | 포모도르, 휴식루틴        |                |
| ⑧ 귀가검사 직전            | 하루 학습의 정서 마무리 상태 파악       | 귀가검사, 피로감 분석      |                |

| ⑩ 장기간 학습패턴 변화 감지 시   | 정서기복, 루틴 붕괴 조기 경보         | 휴식루틴, 침착도, 학습관리   |                |

## 2️⃣ 상황별 맞춤형 복합질문 세트

(→ 각 질문은 감정탐색형 + 메타인지 유도형으로 구성)

---

### 🟢 ① 개념이해 후

**상황:** 학생이 개념공부를 끝냈지만 표정이 약간 공허하거나, 스스로 정리가 안 됨.
**복합질문 예시:**

> “지금 이 개념을 정리했을 때, 머릿속이 어떤 느낌이야?
> 완전히 ‘정리됐다’ 쪽이야, 아니면 ‘조금 헷갈리는데 괜찮겠지’ 하는 느낌이야?”
> → (이 질문으로 이해정도 + 자신감지수 + 피로감 3요소를 동시에 식별함)

---

### 🟡 ② 유형학습 중 집중 흐름 변화

**상황:** 집중도가 들쭉날쭉하거나, 같은 유형을 지루해하는 모습.
**복합질문 예시:**

> “지금 문제들이 조금 지겨워졌어? 아니면 익숙해서 빨라졌어?
> 이럴 땐 ‘지루한 몰입’이랑 ‘루틴 몰입’이 헷갈릴 때가 많거든.”
> → (흥미도·도전감·루틴 감정 구분)

---

### 🔵 ③ 문제풀이 중 멈춤 상태

**상황:** 풀이 도중 멈추거나 눈동자가 고정된 채 움직이지 않음.
**복합질문 예시:**

> “지금 멈춘 이유가 뭐 같아?
> (1) 막혀서 답답, (2) 실수할까봐 불안, (3) 아직 생각 정리가 안 돼서?”
> → (불안형/집중형/혼란형 감정 구분 후 문제노트에 전달)

---

### 🟣 ④ 오답노트 작성 전후

**상황:** 틀린 문제를 마주할 때 표정이 무표정하거나 피로한 기색.
**복합질문 예시:**

> “이 문제 다시 보는 게 지금 좀 부담돼 보여.
> ‘왜 틀렸는지’ 알고 싶은 쪽이야, ‘다시 하기 싫은 쪽’이야?”
> → (자기성찰형 vs 회피형 식별 / 감정저항도 산출)

---

### 🟠 ⑤ 질의응답 시점

**상황:** 학생이 질문을 망설이거나 눈치를 보는 경우.
**복합질문 예시:**

> “지금 물어보려던 거, 머릿속에선 이미 질문이 만들어졌는데
> ‘지금 물어봐도 될까?’ 하는 생각이 먼저 들었지?”
> → (불안형/자신형 판단 + 질문시점 데이터 전송)

---

### 🟤 ⑥ 복습 루틴 진입 전

**상황:** 복습 시작 버튼을 눌러야 하는데 주저하는 경우.
**복합질문 예시:**

> “지금 복습 들어가기 전, 느낌이 어때?
> ‘이거만 하면 완벽해질 것 같아’ 쪽이야, 아니면 ‘또 해야 돼?’ 쪽이야?”
> → (동기 지속도와 감정저항도 계산)

---

### 🔵 ⑦ 포모도르 종료 직후

**상황:** 세션 종료 시 표정·시선·손의 움직임이 느려짐.
**복합질문 예시:**

> “이번 세션 끝났을 때, 머릿속은 맑아졌어, 아니면 좀 퍼석한 느낌이야?
> 이 피로감이 ‘성취감 섞인 피로’인지 그냥 ‘지침’인지 구분해보자.”
> → (몰입형 vs 소진형 감정 분리 / 회복루틴 전달)

---

### ⚫ ⑧ 귀가검사 전

**상황:** 하루를 마무리하며 집중도는 괜찮지만 감정이 무겁게 느껴질 때.
**복합질문 예시:**

> “오늘 하루 공부를 딱 떠올리면, 어떤 한 장면이 제일 강하게 남아?
> 그게 ‘잘됐다’의 순간이야, ‘힘들었다’의 순간이야?”
> → (하루 감정 클로징 데이터 / 성장지향 vs 소진형 판정)

---

### 🔴 ⑨ 시험 대비 2~3주 전

**상황:** 긴장감과 불안이 증가하는 시기.
**복합질문 예시:**

> “요즘 공부할 때, ‘시간이 모자랄 것 같아’ 하는 생각이 자주 들어?
> 아니면 ‘이제 감이 좀 잡힌다’는 생각이 많아?”
> → (불안-준비도 매핑 / 침착도 및 계획조정 에이전트로 전달)

---

### ⚪ ⑩ 루틴 붕괴 감지 시

**상황:** 일정 기간 동안 학습 리듬이 흐트러짐.
**복합질문 예시:**

> “요즘 공부할 때, 예전처럼 ‘몰입되는 순간’이 잘 안 오는 느낌이야?
> 아니면 ‘의욕은 있는데 집중이 안 된다’ 쪽이야?”
> → (정서적 관성/인지적 피로/동기 저하 유형 식별 후 재가동 시나리오 생성)

---

## 💡 정리 요약

* 이 에이전트는 **‘학습 중 감정의 미세한 파동’을 탐지해주는 센서**임.
* 복합질문은 **상황+감정+메타인지** 3요소로 설계되어야 함.
* 모든 질문의 응답은 **페르소나 DB 갱신 → 감정코드 생성 → 연결 에이전트로 전달**
  (예: 불안형 감정코드 F2 → 침착도 분석에 전송 → 휴식루틴 자동 제안)
