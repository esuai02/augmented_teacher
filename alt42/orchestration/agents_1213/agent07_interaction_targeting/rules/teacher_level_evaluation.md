### 2️⃣ 수학 학습 단계별 타겟팅 (3/20점, F)

#### ❌ 현저히 부족

**현직 선생님의 학습 단계별 타겟팅 전략:**

1. **개념 학습 단계**
   - 타겟팅: 집중도 높은 시간대 + 쉬운 문제로 시작
   - 상호작용 모드: 도제학습/탐구학습
   - 현재: 전혀 없음 (-5점)

2. **유형 연습 단계**
   - 타겟팅: 목표 설정 후 + 성취감 강조
   - 상호작용 모드: 단기미션/자기주도
   - 현재: 전혀 없음 (-5점)

3. **심화 문제 단계**
   - 타겟팅: 에너지 충분할 때 + 실수 용인 분위기
   - 상호작용 모드: 자기성찰/맞춤학습
   - 현재: 전혀 없음 (-5점)

4. **기출 문제 단계**
   - 타겟팅: 시험 D-day 고려 + 시간 관리 강조
   - 상호작용 모드: 시험대비/시간성찰
   - 현재: 전혀 없음 (-5점)

**데이터 가정**:
- 현재 학습 단계 감지 (Agent 04의 활동 유형 정보 활용)
- 단계별 학생 반응 데이터 (어떤 단계에서 어떤 상호작용이 효과적이었는지)
