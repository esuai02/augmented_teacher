## 🎯 영역별 상세 평가

### 1️⃣ 침착도 측정 및 해석 (15/20점, B)

#### ✅ 강점
- **기본 침착도 측정 방식**: 정답 선택 후 실제 정답 비율 기반 측정
  - 현직 선생님도 "확신 없이 찍은 건 아닌지" 확인하는 방식과 유사
- **침착도 수준별 해석 기준**: 95+, 90~94, 85~89, 80~84, 75~79, <75 구분
  - 선생님의 일반적 판단 기준과 유사
- **기준선 대비 추이 분석**: +5/-5 기준으로 고효율/피로 구분
  - 선생님도 평소 대비 컨디션 비교 사용

#### ⚠️ 부족한 점

1. **수학 문제 유형별 침착도 패턴 DB 부재**
   - 선생님: "계산 문제는 침착도 높고, 증명 문제는 낮아"
   - 현재: 문제 유형 구분 없이 통합 측정
   - 필요: 계산형/개념형/응용형/증명형별 기준선 설정

2. **단원별 침착도 특성 미반영**
   - 선생님: "함수 단원은 침착도 낮아지기 쉬워, 통계는 비교적 안정적"
   - 현재: 단원별 특성 고려 없음
   - 필요: 단원별 침착도 기준선 DB

3. **학생 수준별 침착도 기준 부재**
   - 선생님: "상위권은 90이 정상, 하위권은 80이면 양호해"
   - 현재: 절대 기준만 사용
   - 필요: 학생 수준별 상대적 기준 설정

**점수: 15/20점** (75%)

   - 현재: 모든 문제 유형에 동일 기준 적용
   - 필요: 유형별 풀이 속도/정답률 기준 상이하게 설정

   - 필요: 개념학습/유형연습/심화단계별 기준 차별화

   - 선생님: "쎈은 문제 많아서 침착도 떨어지기 쉬워, 개념원리는 설명 중심이라 안정적"

### 3️⃣ 오류 원인 정밀 분석 (11/15점, B-)

#### ✅ 강점
- **연속 오답 패턴 분석**: 불안정한 판단 패턴 감지 (R1)
- **확인 단계 생략 분석**: 검증 과정 누락 파악 (R1a)
- **주저 패턴 분석**: 확신 부족형 문제 감지 (R3)
- **화이트보드 로그 분석**: 사고 생략 구간 파악 (R7)

#### ⚠️ 부족한 점

1. **수학 오류 유형별 침착도 패턴 미분류**
   - 선생님: "계산 실수는 침착도 낮을 때 자주 나, 개념 오류는 침착도 높아도 발생해"
   - 현재: 오류 원인 분류만 있고 침착도와의 연관성 분석 부재
   - 필요: 계산실수/개념오류/이해부족/착각별 침착도 패턴 분석

2. **문제 난이도와 침착도 저하 연관성 미반영**
   - 선생님: "기본 문제에서 침착도 떨어지면 컨디션 문제, 심화 문제에서 떨어지면 정상이야"
   - 현재: 문제 난이도 정보 활용 없음
   - 필요: 난이도별 침착도 기준선 차별화

3. **오답 원인 선택 항목의 수학 특화 부족**
   - 현재: 시간압박, 근거부족, 자신감과잉, 착각, 피로 (R7b)
   - 선생님은 추가로 고려: 계산 실수, 개념 혼동, 조건 놓침, 수식 오류
   - 필요: 수학 특화 오류 원인 항목 추가

**점수: 11/15점** (73%)

## 💡 현직 선생님의 핵심 역량 vs 현재 룰

### 현직 선생님이 보유한 사전 지식 (경험 DB)
