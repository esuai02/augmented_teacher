# 교재별 활동 완료 전략 룰

## 목적
수학 교재별 특성(쎈 A/B/C, 개념원리 등)을 고려하여 활동 완료 전략을 수립하고 시간을 산정합니다.

## 데이터 소스
- `textbook_strategies.yaml`: 교재별 단계별 완료 전략, 시간 산정, 정답률 목표
- 학생 설문: 사용 중인 교재 정보
- 선생님 체크리스트: 교재별 단계별 완료 순서

## 교재별 완료 전략

### 쎈 (Senn)
**특징**: 문제집, A → B → C 단계별 진행

#### A단계 (기본형)
- **목표 정답률**: 80%
- **문제당 평균 시간**: 3분
- **완료 전략**: A단계 완료 후 B단계 진행
- **우선순위**: 1 (최우선)

#### B단계 (유형)
- **목표 정답률**: 70%
- **문제당 평균 시간**: 4.5분
- **완료 전략**: A단계 완료 후 B단계 진행
- **우선순위**: 2

#### C단계 (심화)
- **목표 정답률**: 60%
- **문제당 평균 시간**: 6분
- **완료 전략**: B단계 완료 후 C단계 진행
- **우선순위**: 3

**완료 규칙**: A → B → C 순서 필수

### 개념원리 (Concept Principle)
**특징**: 개념서, 개념 이해 중심

#### 개념 단계
- **페이지당 평균 시간**: 20분
- **완료 전략**: 페이지 순서대로 진행
- **우선순위**: 1

**완료 규칙**: 개념 완료 후 문제집(쎈 등) 진행 권장

### 라이트쎈 (Light Senn)
**특징**: 문제집, 기본형 → 유형 진행

#### 기본형
- **목표 정답률**: 75%
- **문제당 평균 시간**: 2.5분
- **완료 전략**: 기본형 완료 후 유형 진행
- **우선순위**: 1

#### 유형
- **목표 정답률**: 65%
- **문제당 평균 시간**: 4분
- **완료 전략**: 기본형 완료 후 유형 진행
- **우선순위**: 2

**완료 규칙**: 기본형 → 유형 순서

## 시간 산정 알고리즘

### 문제집 교재 (쎈, 라이트쎈 등)
```
예상 소요 시간 = 문제 수 × 문제당 평균 시간
```

**예시**:
- 쎈 A단계 20문제: 20 × 3분 = 60분
- 쎈 B단계 15문제: 15 × 4.5분 = 67.5분

### 개념서 교재 (개념원리 등)
```
예상 소요 시간 = 페이지 수 × 페이지당 평균 시간
```

**예시**:
- 개념원리 10페이지: 10 × 20분 = 200분 (약 3.3시간)

---

## 시간 제약별 조정 전략

### 시간 충분 (가용 시간 ≥ 예상 시간의 1.2배)
- **전략**: 모든 단계 완료 (A → B → C)
- **예시**: 쎈 A, B, C 모두 완료

### 시간 적당 (가용 시간 = 예상 시간의 0.8~1.2배)
- **전략**: A, B단계 완료, C단계 선택적
- **예시**: 쎈 A, B 완료, C는 시간 여유 시만

### 시간 부족 (가용 시간 < 예상 시간의 0.8배)
- **전략**: A단계만 완료, B, C단계 연기
- **예시**: 쎈 A만 완료, B, C는 다음 시간 창으로

## 활동 조정 규칙

### Rule 1: 단계별 순서 준수
- **조건**: 문제집 교재 사용 시
- **규칙**: 낮은 단계(A/기본형) 완료 후 높은 단계(B/C/유형) 진행
- **예외**: 없음 (순서 필수)

### Rule 2: 개념서 우선 전략
- **조건**: 개념서 + 문제집 조합 사용 시
- **규칙**: 개념서 완료 후 문제집 진행
- **예시**: 개념원리 완료 후 쎈 진행

### Rule 3: 정답률 목표 기반 조정
- **조건**: 목표 정답률 미달 시
- **규칙**: 해당 단계 반복 학습 또는 이전 단계 복습
- **예시**: 쎈 A단계 정답률 70% (목표 80% 미달) → A단계 재학습

### Rule 4: 시간 부족 시 단계 축소
- **조건**: 가용 시간이 모든 단계 완료 시간보다 부족할 때
- **규칙**: 낮은 단계 우선 완료, 높은 단계 연기
- **예시**: 쎈 A만 완료, B, C 연기

## 활동 조정 프로세스

### Step 1: 교재 정보 확인
1. 각 잔여 활동의 교재 정보 확인
2. 교재별 단계 정보 확인
3. `textbook_strategies.yaml`에서 교재별 전략 조회

### Step 2: 시간 산정
1. 문제 수 또는 페이지 수 확인
2. 교재별 문제당/페이지당 평균 시간 확인
3. 예상 소요 시간 계산

### Step 3: 완료 순서 확인
1. 교재별 완료 규칙 확인
2. 선행 단계 완료 여부 확인
3. 완료 가능한 단계 판정

### Step 4: 시간 제약 분석
1. 가용 시간 확인
2. 예상 소요 시간과 비교
3. 시간 제약 정도 판정

### Step 5: 활동 계획 수립
1. 완료 가능한 단계 선별
2. 시간 제약별 조정 전략 적용
3. 최종 활동 계획 생성

## 코칭 템플릿

### 쎈 교재 사용 시
- "쎈은 A단계(기본형)를 먼저 완료한 후 B단계(유형)로 진행합시다."
- "시간이 부족하니 A단계만 완료하고, B, C단계는 내일로 연기합시다."
- "A단계 정답률이 80% 목표에 미달하니 A단계를 다시 한 번 복습합시다."

### 개념원리 + 쎈 조합 시
- "개념원리를 먼저 완료하여 개념을 이해한 후 쎈으로 문제 풀이를 진행합시다."
- "개념 이해가 완료되었으니 이제 쎈 A단계 문제를 풀어봅시다."

### 시간 충분 시
- "시간이 충분하니 쎈 A, B, C단계를 모두 완료합시다."
- "모든 단계를 완료하면 실력이 크게 향상됩니다."

### 시간 부족 시
- "시간이 부족하니 쎈 A단계만 완료하고, B, C단계는 다음 시간 창에서 진행합시다."
- "A단계를 확실히 완료하는 것이 B, C단계로 넘어가는 것보다 중요합니다."

## 예시 시나리오

### 시나리오 1: 쎈 A단계 완료, B단계 진행
**상황**: 
- 가용 시간: 1.5시간
- 쎈 A단계: 완료됨
- 쎈 B단계: 15문제 남음

**시간 산정**:
- B단계 예상 시간: 15 × 4.5분 = 67.5분 (약 1.1시간)

**활동 계획**:
- B단계 완료 가능 (1.5시간 > 1.1시간)
- 목표 정답률 70% 달성 목표

**코칭**: "A단계를 완료하셨으니 이제 B단계(유형) 문제를 풀어봅시다. 목표 정답률은 70%입니다."

### 시나리오 2: 시간 부족 - A단계만 완료
**상황**:
- 가용 시간: 1시간
- 쎈 A단계: 20문제 남음
- 쎈 B단계: 15문제 남음

**시간 산정**:
- A단계 예상 시간: 20 × 3분 = 60분 (1시간)
- B단계 예상 시간: 15 × 4.5분 = 67.5분 (약 1.1시간)
- 총 예상 시간: 약 2.1시간

**활동 계획**:
- 시간 부족 (1시간 < 2.1시간)
- A단계만 완료, B단계 연기

**코칭**: "시간이 부족하니 A단계만 완료하고, B단계는 내일로 연기합시다. A단계를 확실히 완료하는 것이 중요합니다."

### 시나리오 3: 개념원리 + 쎈 조합
**상황**:
- 가용 시간: 2시간
- 개념원리: 5페이지 남음
- 쎈 A단계: 20문제 남음

**시간 산정**:
- 개념원리 예상 시간: 5 × 20분 = 100분 (약 1.7시간)
- 쎈 A단계 예상 시간: 20 × 3분 = 60분 (1시간)
- 총 예상 시간: 약 2.7시간

**활동 계획**:
- 시간 부족 (2시간 < 2.7시간)
- 개념원리 우선 완료 (개념 이해가 선행되어야 함)
- 쎈 A단계는 남은 시간에 진행 또는 연기

**코칭**: "개념원리를 먼저 완료하여 개념을 이해한 후 쎈으로 문제 풀이를 진행합시다. 개념 이해가 문제 풀이의 기초가 됩니다."
