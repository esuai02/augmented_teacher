   - 필요: 수학 특화 지표
   - **데이터 가정**: 학생설문/선생님 체크리스트 입력

     - "최근 학습 단계: 개념학습 / 유형연습 / 심화 / 기출"

2. **학생 수준(L)의 정량적 측정 부재** (-2점)

   - **데이터 가정**: Agent 01 온보딩 데이터 연계

3. **학습 감정의 수학 특화 분석 부족** (-2점)

   - 필요: 수학 학습 관련 감정 구분

### 2️⃣ 수학 학습 단계별 세계관 매핑 (5/20점, D)

#### ❌ 현저히 부족

**현재 상태**: 9가지 세계관(커리큘럼, 맞춤학습, 시험대비 등)만 존재, 수학 학습 단계와 매핑 없음

**필요한 매핑**:

1. **개념 학습 단계** (-5점)
   - **권장 세계관**: 탐구학습 모드 (호기심 기반 개념 탐색)
   - **대안 세계관**: 도제학습 모드 (선생님-제자 관계로 개념 전달)
   - **데이터 가정**: 선생님 체크리스트 입력
     - "개념 학습 단계: 탐구학습 모드 권장"
     - "개념 학습 시 학생이 질문을 많이 할 때: 도제학습 모드"
   - **룰 필요**: 학습 단계 감지 → 세계관 자동 매핑

2. **유형 연습 단계** (-5점)
   - **권장 세계관**: 단기미션 모드 (구체적 목표, 즉각적 성취)
   - **대안 세계관**: 자기주도 모드 (자율적 연습, 반복 선택)
   - **데이터 가정**: 선생님 텍스트 입력
     - "유형 연습: 단기미션 모드 (오늘 3개 유형 완료 목표)"
     - "유형 연습 시 동기 저하: 자기주도 모드로 전환"
   - **룰 필요**: 유형 연습 감지 → 단기미션/자기주도 매핑

3. **심화 문제 단계** (-3점)
   - **권장 세계관**: 자기성찰 모드 (실수→성장 서사)
   - **대안 세계관**: 맞춤학습 모드 (개인 수준 맞춤)
   - **데이터 가정**: 선생님 체크리스트
     - "심화 문제: 자기성찰 모드 (틀려도 성장)"
     - "심화 문제 풀이 중 피로: 맞춤학습 모드로 난이도 조정"

4. **기출 문제 단계** (-2점)
   - **권장 세계관**: 시험대비 모드 (D-day 기반 긴장 완화)
   - **대안 세계관**: 시간성찰 모드 (시간 관리 강조)
   - **데이터 가정**: 선생님 텍스트 입력
     - "기출 문제: 시험대비 모드 (D-14일 이내)"
     - "기출 문제 풀이 시간 초과: 시간성찰 모드"

**점수: 5/20점** (25%)

### 3️⃣ 학원 수업 맥락 고려 (0/15점, F)

#### ❌ 완전 부재

**현직 선생님의 실제 준비**:
- 학원 수업 전: 예습 가이드, 예상 난이도 안내
- 학원 수업 후: 복습 상호작용, 학원 수업 내용 통합

**필요한 기능**:

1. **학원 수업 전 상호작용 준비** (-8점)
   - **필요 데이터**: 학원 수업 일정, 예상 학습 단원
   - **데이터 가정**: 학생설문 입력
     - "학원 수업 전 상호작용이 필요하신가요? (예/아니오)"
     - "학원 수업 전 어떤 도움이 필요하신가요?"
       - A) 예습 가이드
       - B) 개념 리마인드
       - C) 문제 선행 학습
   - **세계관 매핑**: 
     - 예습 가이드 → 맞춤학습 모드
     - 개념 리마인드 → 도제학습 모드
     - 문제 선행 학습 → 단기미션 모드

2. **학원 수업 후 상호작용 준비** (-7점)
   - **필요 데이터**: 학원 수업 내용, 학생 이해도
   - **데이터 가정**: 선생님 체크리스트 입력
     - "학원 수업 후 상호작용 준비 체크리스트"
     - "학원 수업 내용: __단원"
     - "학생 이해도: 완전 이해 / 부분 이해 / 이해 부족"
   - **세계관 매핑**:
     - 완전 이해 → 자기주도 모드 (자율 복습)
     - 부분 이해 → 도제학습 모드 (추가 설명)
     - 이해 부족 → 자기성찰 모드 (재학습)

**점수: 0/15점** (0%)

### 4️⃣ 문제 풀이 중 실시간 준비 (2/15점, F)

**현재**: 사후 상호작용만 (문제 풀이 완료 후 피드백)

**필요한 기능**:

1. **막힘 감지 시 즉시 상호작용 준비** (-8점)

   - **데이터 가정**: 선생님 체크리스트 입력

     - "막힘 감지 시 상호작용 모드: 도제학습 / 단기미션"
   - **세계관 매핑**:
     - 개념 문제 막힘 → 도제학습 모드 (힌트 제공)
     - 계산 문제 막힘 → 단기미션 모드 (단계별 안내)
   - **룰 필요**: 실시간 문제 풀이 상태 모니터링 → 막힘 감지 → 세계관 선택

2. **계산 실수 패턴 감지** (-3점)
   - **필요 데이터**: 계산 실수 빈도, 실수 유형
   - **데이터 가정**: 선생님 텍스트 입력

     - "계산 실수 유형: 부호 실수 / 계산 과정 오류 / 답안 옮기기 실수"
   - **세계관 매핑**: 자기성찰 모드 (실수→개선 서사)

3. **문제 풀이 진도 기반 준비** (-2점)
   - **필요 데이터**: 문제 풀이 속도, 진도 예측
   - **데이터 가정**: 자동 수집 (문제 풀이 로그)
   - **세계관 매핑**: 시간성찰 모드 (시간 관리 안내)

**점수: 2/15점** (13%)
