# Agent 03 - Goals Analysis Actions

## 수행 가능한 액션 목록

### 목표 분석
- `analyze_goal_quality`: 목표 품질 분석
- `calculate_completion_rate`: 달성률 계산
- `check_category_balance`: 카테고리 균형 확인

### 목표 조정
- `breakdown_long_term_goals`: 장기 목표 세분화
- `suggest_realistic_timeline`: 현실적 기간 제안
- `rebalance_categories`: 카테고리 균형 조정

### 제안
- `recommend_goal_structure`: 목표 구조 추천
- `suggest_intermediate_milestones`: 중간 마일스톤 제안
