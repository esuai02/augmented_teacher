# Agent 12 - Rest Routine Actions

## 수행 가능한 액션 목록

### 분석
- `analyze_rest_patterns`: 휴식 패턴 분석
- `identify_rest_type`: 휴식 유형 식별
- `assess_rest_quality`: 휴식 품질 평가

### 최적화
- `optimize_rest_routine`: 휴식 루틴 최적화
- `suggest_rest_timing`: 휴식 타이밍 제안
- `create_custom_rest_rule`: 맞춤형 휴식 규칙 생성
