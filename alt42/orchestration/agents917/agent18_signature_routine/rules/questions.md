💬 포괄형 질문 1

“이 학생의 몰입 패턴과 감정 리듬을 종합해서, 지금 시점에서 가장 자연스럽게 몰입이 일어나는 조건을 찾아줘.”
→ 온보딩 데이터와 최신 선호동 정보를 결합하여, 학생이 몰입에 진입하는 **심리 트리거(감정·환경·행동요소)**를 분석하고, 시작 루틴–집중 루틴–회복 루틴 구조로 정리된 시그너처 루틴 초안을 제시

💬 포괄형 질문 2

“이 학생의 시그너처 루틴을 안정적으로 유지·강화하기 위해 어떤 감정적 강화 요소와 상호작용 방식을 적용해야 할까?”
→ 학습 중 감정곡선, 몰입 지속시간, 강화 피드백 반응 패턴을 기반으로 **루틴 유지 요인(강화 자극·시간대·감정톤)**을 탐색하고, 루틴이 무너질 때 빠르게 회복할 수 있는 보상 루프 설계 및 피드백 전략을 도출

💬 포괄형 질문 3

“이 학생이 장기적으로 몰입을 지속하며 성장하기 위해, 시그너처 루틴을 어떤 방향으로 진화시켜야 할까?”
→ 루틴 반복 안정성, 감정 진폭, 인지 부하 패턴을 종합 분석하여 성취형→탐구형→의미형 루틴으로 이어지는 루틴 진화 경로를 제시하고, 각 단계에서 필요한 정서·인지적 보정 포인트를 함께 제공

## 🧭 1. 이 에이전트를 필요로 하는 상황

| 구분                                  | 상황 설명                          | 시그너처 루틴이 필요한 이유                      |
| ----------------------------------- | ------------------------------ | ------------------------------------ |
| **① 초기 온보딩 후 첫 몰입 설계 단계**           | 학생이 아직 자기 스타일을 모를 때            | 초반 몰입 성공 경험을 설계해야 ‘자기효능감 루프’ 형성 가능   |
| **② 학습 루틴이 흔들릴 때**                  | 집중이 잘 안 되고 진도 대비 체감 성취감이 낮을 때  | 루틴의 감정 트리거(흥미·자기확인·도전 등)를 다시 재조립해야 함 |
| **③ 포모도로나 일일 계획이 반복되지만 성장체감이 없을 때** | 학습의 흐름은 있으나 감정적 피드백이 결여된 상태    | 단순 반복이 아닌 ‘의미가 있는 반복’으로 전환 필요        |
| **④ 상호작용 콘텐츠의 몰입률이 떨어질 때**          | 학생이 TTS, 퀴즈, 필기 활동에서 반응이 미미할 때 | 기존 루틴과 콘텐츠의 감정/인지 리듬 불일치             |
| **⑤ 시험 직전 루틴이 무너질 때**               | 불안감 증가, 비효율적 공부 패턴             | 안정감 중심 루틴(회복형 루틴) 재탐색 필요             |
| **⑥ 장기 학습에서 슬럼프 발생 시**              | 성취감 둔화, 흥미 상실                  | 루틴 내 ‘보상 회로’의 감정-행동 매칭을 다시 맞춰야 함     |
| **⑦ 새로운 과목이나 단원 진입 시**              | 익숙한 루틴이 낯선 콘텐츠에 안 맞을 때         | 이전 루틴의 구조를 변형해 재적용 필요                |

## 🎯 2. 각 상황별 맞춤형 복합질문 예시 (에이전트에게 요청할 형태)

이 질문들은 **에이전트 18이 Drilling을 통해 ‘심리 요인 → 루틴 패턴 → 콘텐츠 설계 가이드라인’**을 자동으로 도출하도록 하는 트리거 역할을 해.

| 상황                        | 에이전트에게 던질 복합질문                                                                                               |
| ------------------------- | ------------------------------------------------------------------------------------------------------------ |
| **① 초기 온보딩 후 첫 몰입 설계 단계** | “이 학생의 온보딩 데이터 중 감정적 반응과 몰입 유지시간 패턴을 결합해서, 처음 학습 몰입을 유도할 수 있는 심리 트리거를 찾아줘. 이때 루틴의 시작 조건과 강화 조건을 함께 제안해줘.”    |
| **② 학습 루틴이 흔들릴 때**        | “최근 학습 로그에서 몰입이 깨지는 시점(행동·시간·감정)을 분석해서, 이전 루틴 중 유지해야 할 핵심 요소와 교체해야 할 요소를 구분해줘. 그리고 새로 적응할 수 있는 전환 루틴을 제시해줘.” |
| **③ 성장 체감이 없을 때**         | “학생의 최근 활동 중 ‘시간 대비 만족도’가 낮은 패턴을 찾아, 정서적 보상 회로를 복원할 수 있는 루틴을 설계해줘. 감정적 강화 포인트를 3단계로 제시해.”                    |
| **④ 콘텐츠 몰입률 저하 시**        | “콘텐츠별 몰입 로그와 선호동 데이터를 분석해서, 학생이 반응하는 감각요소(청각, 시각, 상호작용 빈도)를 정렬해줘. 그걸 기반으로 시그너처 루틴에 맞는 상호작용 요소 조합을 만들어.”      |
| **⑤ 시험 직전 루틴 붕괴 시**       | “최근 학습기록에서 불안정 신호(지연, 무입력, 재시도 등)를 찾아서, 안정감 중심의 ‘회복 루틴’을 제안해줘. 루틴의 리듬(시작–집중–이완) 구조를 포함해줘.”                   |
| **⑥ 장기 슬럼프 시**            | “지난 한 달간 학습 감정 데이터를 기준으로, 동기 저하의 주요 요인을 감정→행동→결과 흐름으로 시각화해줘. 그리고 이 학생의 내적 동기유형에 맞는 시그너처 루틴 복원안을 제시해.”        |
| **⑦ 새로운 단원/과목 진입 시**      | “이전 루틴에서 유지해야 할 기본 리듬과 새 단원에 맞게 조정할 부분을 비교해줘. 감정적 전이(익숙함→낯섦)를 완화할 전이 루틴을 구성해.”                               |
