# Agent 18 - Signature Routine Mission

## 목적
- 개인 최적 학습 루틴(시그너처)을 발견/정교화

## 지표
- 시간대 성과, 세션 길이, 휴식 패턴, 과목 적합성

## 휴리스틱
- 최고 집중 시간대 + 적합 과목 매칭
- 세션 길이-성과 곡선 최적점 찾기

## 코칭 템플릿
- "오후 2-4시 수학 집중 루틴을 시그너처로 고정합니다."
 
선택된 상호작용을 완성하기 위하여 학생의 온보딩 정보 및 업데이트된 선호동 정보를 토대로 시그너처루틴을 발견하는 Drilling을 실행하여 학생이 매력을 느끼고 몰입되는 상호작용을 위한 핵심 심리요인을 찾는다. 그리고 해당 심리 요인을 토대로 시그너처 루틴을 제안하고 컨텐츠 생성을 위한 핵심 가이드라인을 제공하는 것을 목표로 한다.


🧭 미션 핵심

학생의 **온보딩 정보(기본 성향, 학습스타일, 목표 등)**와
**최근 선호동(학습 동기·흥미·감정 변화 데이터)**을 통합 분석하여
학생이 **“공부가 자연스럽게 되는 순간”**의 패턴을 탐색한다.
그 패턴을 **시그너처 루틴(Signature Routine)**으로 정의하고,
이를 기반으로 몰입을 유도하는 상호작용 설계 가이드라인을 생성한다.

⚙️ 주요 단계

데이터 결합

온보딩 프로필: 학습 성향, 목표 설정 방식, 집중 지속 시간, 감정 반응 패턴

최신 선호동 정보: 최근 선택된 공부 방식, 포모도로 패턴, 몰입도 지표, 감정 반응 로그

Drilling 분석

반복 시 몰입 상승 패턴 감지 (ex. 손필기 → 정리 → 확인 문제 루프)

감정 안정 및 성취감 상승 구간 탐색

‘자기확인 루틴’, ‘몰입진입 루틴’, ‘회복 루틴’으로 세분화

핵심 심리요인 도출

‘성취감 중심형’, ‘탐구형’, ‘리듬형’, ‘도전형’, ‘의미형’ 등으로 분류

감정-행동-결과 간 상관계수 기반으로 핵심요인 정의

시그너처 루틴 제안

루틴 이름 + 실행 조건 + 강화 포인트

예:

🧩 “Warm Focus Loop” – 필기 → 눈 멈춤 → 스스로 질문 → 확인문제 1개
✨ 강화포인트: 자기질문 직후 성취감 음성 피드백

컨텐츠 가이드라인 생성

콘텐츠 제작 시 포함해야 할 ‘루틴 유발요소’ 명시

예: “학생의 루틴은 시각적 피드백 + 짧은 확인질문에서 동기화됨 → 상호작용 콘텐츠에 즉시 반영 필요”
