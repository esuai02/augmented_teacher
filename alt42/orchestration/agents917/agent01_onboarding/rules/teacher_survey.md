# 선생님 설문 조사 - 의견이 필요한 부분

## 개요
이 문서는 `agent01_onboarding_rules.yaml`의 규칙들이 실제 현장 선생님의 처리 방식을 얼마나 잘 반영하고 있는지 확인하기 위해 선생님의 의견이 필요한 부분들을 정리한 것입니다.

생성일: 2025-01-21
목표: rules.yaml을 현장 선생님 처리방식 99% 구현

---

## 1. 규칙별 검증 필요 사항

### 1.1 수학 수준 기반 루틴 추천 (R1, R1b, R1c)

**현재 규칙**:
- R1: 수학이 어려운 학생 → 개념 중심 루틴
- R1b: 상위권 학생 → 심화 루틴
- R1c: 중위권 학생 → 균형잡힌 루틴

**선생님께 질문**:
1. 수학 수준 분류가 "수학이 어려워요", "중위권", "상위권", "상위권 이상" 4단계로 충분한가요?
2. 각 수준별로 실제로 어떤 루틴을 추천하시나요?
3. 수학이 어려운 학생에게 개념 중심 루틴 외에 추가로 고려하시는 요소가 있나요?
4. 상위권 학생에게 심화 루틴을 추천할 때 주의사항이 있나요?

---

### 1.2 수학 자신감 기반 정서 강화 (R2, R2b)

**현재 규칙**:
- R2: 자신감 5점 이하 → 정서 강화 루틴
- R2b: 자신감 8점 이상 → 도전 과제 제안

**선생님께 질문**:
1. 자신감 점수(1-10점) 기준이 적절한가요? (현재 5점 이하, 8점 이상)
2. 자신감이 낮은 학생에게 어떤 방식으로 정서 강화를 하시나요?
3. 자신감이 높은 학생에게 도전 과제를 줄 때 주의사항이 있나요?
4. 자신감과 수학 수준이 불일치하는 경우(예: 수준은 높은데 자신감 낮음) 어떻게 대응하시나요?

---

### 1.3 시험 대비 스타일 기반 루틴 (R3, R3b, R3c, R3d)

**현재 규칙**:
- R3: 벼락치기 → 단기 집중 루틴
- R3c: 꾸준한 준비 → 장기 지속 루틴
- R3d: 전략적 집중 → 효율적 집중 루틴

**선생님께 질문**:
1. 시험 대비 스타일 분류가 적절한가요? (벼락치기, 꾸준한 준비, 전략적 집중)
2. 벼락치기 성향 학생에게 단기 집중 루틴 외에 추가 조언이 있나요?
3. 꾸준한 준비 스타일 학생에게 장기 루틴을 제안할 때 구체적으로 어떤 내용을 포함하나요?
4. 전략적 집중 스타일 학생의 "전략"을 어떻게 파악하시나요?

---

### 1.4 부모 개입 스타일 대응 (R4, R4a, R4b, R11a, R11b, R11c)

**현재 규칙**:
- R4: 적극 개입 → 부모 피드백 채널 동기화
- R4b: 부분 지원 → 선택적 피드백 채널
- R11c: 자율 존중 + 상위권 → 최소 개입 모드

**선생님께 질문**:
1. 부모 개입 스타일 분류가 적절한가요? (적극 개입, 부분 지원, 자율 존중)
2. 적극 개입 부모님과 소통할 때 어떤 방식이 효과적인가요?
3. 부분 지원 부모님에게는 어떤 시점에 피드백을 드리시나요?
4. 자율 존중 부모님의 경우, 언제 개입하시나요?
5. 부모 스타일과 학생 성향이 맞지 않을 때 어떻게 조율하시나요?

---

### 1.5 학습 시간 기반 경고 및 추천 (R5, R5a, R5b)

**현재 규칙**:
- R5: 주간 10시간 미만 → 학습시간 부족 경고
- R5b: 주간 15시간 이상 → 학습 효율성 향상 루틴

**선생님께 질문**:
1. 주간 학습 시간 기준(10시간, 15시간)이 적절한가요?
2. 학습 시간이 부족한 학생에게 어떤 방식으로 시간을 확보하도록 유도하시나요?
3. 학습 시간은 충분하지만 효율이 낮은 학생을 어떻게 판단하시나요?
4. 학습 시간 외에 고려해야 할 요소가 있나요? (집중도, 학습 환경 등)

---

### 1.6 진도 비교 및 분석 (R10a, R10b, R10c)

**현재 규칙**:
- R10a: 개념 진도와 심화 진도 비교 분석
- R10b: 혼란 개념 예측 (중위권 이하)
- R10c: 진도 격차 경고 (개념 중등 vs 심화 미적분)

**선생님께 질문**:
1. 개념 진도와 심화 진도가 어느 정도 차이까지 허용 가능한가요?
2. 진도 격차가 클 때 어떤 방식으로 보완하시나요?
3. 현재 진도에서 혼란을 겪을 개념을 예측할 때 어떤 기준을 사용하시나요?
4. 진도 불균형을 어떻게 판단하시나요? (예: 개념 중등2 vs 심화 미적분1)

---

### 1.7 개념 강의 vs 문제풀이 우선순위 (R9b, R9c, R9d)

**현재 규칙**:
- R9c: 수학 어려움 + 자신감 낮음 → 개념 강의 우선
- R9d: 상위권 + 자신감 높음 → 문제풀이 우선

**선생님께 질문**:
1. 개념 강의와 문제풀이의 우선순위를 결정할 때 어떤 기준을 사용하시나요?
2. 수학이 어려운 학생에게도 문제풀이를 먼저 시작하는 경우가 있나요?
3. 상위권 학생에게도 개념 강의가 필요한 경우가 있나요?
4. 개념과 문제풀이의 비율을 어떻게 조절하시나요?

---

### 1.8 장기 목표 기반 설계 (R12a, R12b, R12c)

**현재 규칙**:
- R12a: 장기 목표와 현재 성향 연결 분석
- R12b: 경시대회 준비 현실성 평가
- R12c: 목표 달성 필수 요건 분석

**선생님께 질문**:
1. 학생의 장기 목표를 파악할 때 어떤 질문을 하시나요?
2. 경시대회 준비 목표 학생의 현실성을 평가할 때 어떤 요소를 보시나요?
3. 목표와 현재 상태의 격차가 클 때 어떻게 접근하시나요?
4. 목표를 달성하기 위한 단계별 계획을 어떻게 수립하시나요?

---

### 1.9 복합 상황 대응 (R14a, R14b)

**현재 규칙**:
- R14a: 수학 어려움 + 낮은 자신감 → 종합 지원 루틴
- R14b: 상위권 + 시간 부족 → 효율 학습 루틴

**선생님께 질문**:
1. 여러 요인이 복합적으로 작용할 때 우선순위를 어떻게 정하시나요?
2. 수학이 어렵고 자신감이 낮은 학생에게 가장 먼저 해결해야 할 것은 무엇인가요?
3. 잠재력은 있지만 시간이 부족한 학생에게 어떻게 접근하시나요?
4. 복합 상황에서 선생님이 가장 중요하게 생각하시는 요소는 무엇인가요?

---

## 2. 규칙 우선순위 검증

**현재 우선순위 범위**: 75-96

**선생님께 질문**:
1. 규칙 우선순위가 실제 판단 순서와 일치하나요?
2. 가장 우선적으로 고려해야 할 요소는 무엇인가요?
3. 상황에 따라 우선순위가 달라지는 경우가 있나요?

---

## 3. 누락된 규칙 확인

**선생님께 질문**:
1. 현재 규칙에 반영되지 않은 중요한 판단 기준이 있나요?
2. 특정 상황에서만 발생하는 예외 케이스가 있나요?
3. 학생 유형별로 추가로 고려해야 할 요소가 있나요?

---

## 4. 규칙 신뢰도(Confidence) 검증

**현재 신뢰도 범위**: 0.75-0.94

**선생님께 질문**:
1. 각 규칙의 신뢰도가 적절한가요?
2. 어떤 규칙이 더 확실하고, 어떤 규칙이 예외가 많은가요?
3. 신뢰도가 낮은 규칙(0.75-0.85)에 대해 추가 조건이 필요한가요?

---

## 5. 액션(Action) 구체화 필요 사항

**현재 액션 예시**:
- "개념 이해 중심 학습 + 짧은 주기 피드백 루프"
- "정서 강화 루틴"
- "단기 집중 루틴"

**선생님께 질문**:
1. 각 액션이 구체적으로 어떤 내용을 포함해야 하나요?
2. "짧은 주기 피드백"이 구체적으로 몇 일 주기인가요?
3. "정서 강화 루틴"의 구체적인 구성 요소는 무엇인가요?
4. 루틴을 제안할 때 선생님이 학생에게 전달하는 메시지 톤은 어떤가요?

---

## 6. 예외 상황 처리

**선생님께 질문**:
1. 규칙이 적용되지 않는 예외 상황이 있나요?
2. 학생이 여러 조건을 동시에 만족할 때 어떻게 처리하시나요?
3. 규칙 간 충돌이 발생할 때 우선순위를 어떻게 정하시나요?

---

## 7. 데이터 수집 방법 개선

**선생님께 질문**:
1. 현재 수집하는 데이터로 충분한가요?
2. 추가로 수집하면 좋을 데이터가 있나요?
3. 데이터 수집 시점(온보딩 시, 수업 중, 정기적으로)에 대한 의견이 있나요?

---

## 8. 피드백 루프

**선생님께 질문**:
1. 규칙 적용 결과에 대한 피드백을 어떻게 수집하면 좋을까요?
2. 규칙이 잘못 적용된 경우를 어떻게 감지하고 수정할 수 있을까요?
3. 선생님의 판단과 AI 규칙의 차이를 어떻게 학습시킬 수 있을까요?
