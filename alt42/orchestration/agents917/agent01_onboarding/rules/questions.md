💬 포괄형 질문 1

“이 학생의 현재 수학 학습 맥락을 종합해서, 첫 수업에서 무엇을 어떻게 시작해야 할지 알려줘.”
→ 온보딩 정보, 개념/심화 진도, 학습 스타일, 시험 대비 성향, 자신감 수준 등을 반영해서 수업 도입 루틴, 설명 전략, 자료 유형 등을 포함한 총체적 개입 전략 도출

💬 포괄형 질문 2

“이 학생의 성향과 목표를 기반으로, 커리큘럼과 루틴을 어떤 방향으로 최적화해야 할까?”
→ 목표(단기/중기/장기), 학습 성향, 스트레스/자신감, 부모 개입도 등을 고려하여 진도별 우선순위, 학습 흐름, 문제 유형 비중 조절 등 커스터마이징 설계 도출

💬 포괄형 질문 3

“이 학생이 중장기적으로 성장하기 위해 지금부터 어떤 부분을 특히 신경 써야 할까?”
→ 경시 준비, 진학 목표, 수학 자존감 성장, 피로 누적 방지, 루틴 유지 여부 등을 포괄적으로 분석하여 조기 리스크 예측 및 트래킹 우선요소 추천

## 🎯 2. 상황별 맞춤형 복합질문 예시

### 🟦 S0. 수학 특화 정보 수집 단계 (신규 추가)

* "이 학생의 수학 학습 스타일을 파악해주세요. (계산형/개념형/응용형)"
* "다니고 있는 수학학원 정보를 수집해주세요. (학원명, 등급, 수업 일정)"
* "최근 수학 시험 성적과 취약한 단원을 알려주세요."
* "사용 중인 수학 교재를 알려주세요. (학교 교과서, 학원 교재: 쎈/개념원리 등)"
* "지금까지 배운 수학 단원 중 완료한 단원, 진행 중인 단원을 알려주세요."

### 🟦 S1. 신규 학생 등록 직후

* "이 학생은 어떤 수학 진도에 있고 어떤 학습 스타일을 가지고 있는지 한 번에 요약해줄 수 있나요? (수학 학습 스타일 포함)"
* "초기 수업을 시작할 때 어떤 내용을 중심으로 준비하면 좋을까요? (학원 진도와 학교 진도 연계 고려)"

### 🟦 S2. 수업 전 학습 설계 필요

* "시험 스타일, 수학 자신감, 부모 개입 정도를 종합해서 어떤 루틴이 적절할까요? (학원 수업 일정 반영)"
* "이 학생에게는 개념 강의 위주로 갈지, 문제풀이를 먼저 해야 할지 판단해주세요. (수학 학습 스타일 반영)"

### 🟦 S3. 개념/심화 진도 판단

* "개념 진도와 심화 진도를 비교해서 현재 난이도 밸런스는 적절한가요? (단원 선후관계 고려)"
* "지금 진도에서 자주 혼란을 겪는 개념은 무엇일 가능성이 높나요? (단원 난이도 DB 활용)"

### 🟦 S4. 학부모 상담 정보 추출

* "이 학생의 학습 성향과 부모님의 스타일을 바탕으로 상담 전략을 알려주세요. (학원 정보 포함)"
* "학부모님과 주기적인 피드백 루틴이 필요한 학생인가요? (학원 수업 이해도 고려)"

### 🟦 S5. 장기 목표 기반 설계

* "이 학생의 장기 목표와 현재 성향을 연결해서 신경 써야 할 요소가 있다면 알려주세요. (학원 진도와 목표 연계)"
* "경시대회 준비가 목표라고 했는데, 지금 상태로 보면 어떤 접근이 현실적일까요? (학원 수업과 병행 전략)"

## 🔍 3. Rules.yaml에 없지만 다른 값들과 연계해서 사용되는 질문들

### 📌 학원 정보 연계 질문 (academy_grade, academy_schedule)

**연계 필드**: `academy_name`, `academy_experience`, `math_learning_style`, `concept_progress`, `advanced_progress`

* "이 학생이 다니는 학원의 등급(반)과 수업 일정을 고려해서, 학원 수업과 개인 학습을 어떻게 연계하면 좋을까요? (학원 진도와 개념/심화 진도 비교)"
* "학원 등급과 수업 일정을 바탕으로, 이 학생의 수학 학습 스타일(계산형/개념형/응용형)에 맞는 보완 학습 계획을 설계해주세요."
* "학원 수업 일정과 현재 개념/심화 진도를 비교해서, 학원 수업 전후로 어떤 내용을 보완하면 좋을까요?"

### 📌 진도 레벨 연계 질문 (advanced_level, concept_level, course_level, grade_detail)

**연계 필드**: `concept_progress`, `advanced_progress`, `math_level`, `math_unit_mastery`, `student_grade`

* "이 학생의 개념 레벨과 심화 레벨, 그리고 현재 과정 레벨을 종합해서, 적절한 난이도 밸런스를 제안해주세요. (단원 마스터리 수준 반영)"
* "학년 상세 정보와 개념/심화 레벨을 비교해서, 현재 진도가 적절한지 판단해주세요. (수학 수준과 단원 마스터리 연계)"
* "과정 레벨과 학년 상세 정보를 바탕으로, 개념 진도와 심화 진도의 적절한 비율을 제안해주세요."

### 📌 학습 감정/상태 연계 질문 (boredom, calmness, pace_anxiety, satisfaction)

**연계 필드**: `stress_level`, `math_confidence`, `study_style`, `exam_style`, `math_learning_style`

* "이 학생의 지루함 수준과 평온함 수준을 파악해서, 학습 동기 유지 전략을 제안해주세요. (수학 자신감과 학습 스타일 연계)"
* "학습 속도 불안감과 만족도를 종합해서, 적절한 학습 페이스를 제안해주세요. (스트레스 수준과 시험 대비 스타일 고려)"
* "지루함, 평온함, 속도 불안감, 만족도를 종합해서, 이 학생의 학습 감정 상태를 진단하고 개선 방안을 제시해주세요. (수학 자신감과 부모 개입 스타일 반영)"

### 📌 학습 방법 연계 질문 (pomodoro, learning_notes, feedback_preference)

**연계 필드**: `study_hours_per_week`, `weekly_hours`, `study_style`, `exam_style`, `parent_style`

* "이 학생의 뽀모도로 기법 사용 여부와 학습 메모 작성 습관을 고려해서, 효과적인 학습 루틴을 제안해주세요. (주당 학습 시간과 학습 스타일 연계)"
* "학습 메모 작성 습관과 피드백 선호도를 종합해서, 이 학생에게 적합한 피드백 방식을 제안해주세요. (학습 스타일과 부모 개입 스타일 고려)"
* "뽀모도로 기법, 학습 메모, 피드백 선호도를 바탕으로, 이 학생의 자기주도 학습 능력을 평가하고 개선 방안을 제시해주세요. (시험 대비 스타일과 주당 학습 시간 반영)" 

### 📌 인지/집중력/협업 역량 연계 질문 (cognitive_total, concentration_score, collaboration_score, completion_time)

**연계 필드**: `math_learning_style`, `study_style`, `math_confidence`, `stress_level`, `exam_style`, `weekly_hours`

* "이 학생의 인지 능력 총점과 집중력 점수를 종합해서, 학습 세션의 최적 시간과 휴식 간격을 제안해주세요. (스트레스 수준과 주당 학습 시간 고려)"
* "인지 능력과 학습 완료 시간 패턴을 분석해서, 이 학생에게 적합한 문제 난이도 조절 전략을 제안해주세요. (수학 학습 스타일과 수학 자신감 연계)"
* "협업 점수와 학습 스타일을 종합해서, 이 학생이 그룹 학습과 개인 학습 중 어느 것이 더 효과적인지 판단해주세요. (시험 대비 스타일 반영)"
* "집중력 점수와 완료 시간 데이터를 바탕으로, 이 학생의 학습 피로도를 예측하고 예방 전략을 제시해주세요. (스트레스 수준과 수학 자신감 고려)"
* "인지 능력 총점, 집중력, 협업 점수를 종합 분석하여, 이 학생의 학습 역량 프로필을 생성하고 강점/약점 기반 맞춤 학습 전략을 제안해주세요."
