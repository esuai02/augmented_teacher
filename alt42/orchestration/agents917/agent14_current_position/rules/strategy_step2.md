# Agent 14 - Current Position Strategy Step 2
# 학원 맥락 통합 및 학생 수준별 차별화 전략

## 목적
현직 수학선생님이 학생의 현재 위치를 평가할 때 고려하는 학원 맥락 및 학생 수준별 차별화 요소를 반영한 분석 전략

---

## 1. 학원 맥락 통합 전략

### 1.1 학원 수업 이해도와 학습 진행 연계 분석

#### 1.1.1 데이터 수집
- **학생 설문**: "오늘 학원 수업을 이해하셨나요? (A) 완전히 이해 (B) 대부분 이해 (C) 반만 이해 (D) 거의 이해 못함"

#### 1.1.2 분석 지표
- **이해도 등급**: 완전히 이해(4점), 대부분 이해(3점), 반만 이해(2점), 거의 이해 못함(1점)
- **이해도-지연 상관계수**: 학원 수업 이해도와 학습 지연 상관 분석
- **이해도-만족도 상관계수**: 학원 수업 이해도와 학습 만족도 상관 분석

#### 1.1.3 인사이트 생성
- **이해 못함 → 지연 패턴**: 학원 수업을 이해 못하면 집에서 학습 지연
- **이해 못함 → 불만족 패턴**: 학원 수업을 이해 못하면 학습 만족도 하락

#### 1.1.4 대응 전략
- **이해 못함 감지**: 예습/복습 강화, 선행 학습 필요
- **이해도 하락**: 학원 수업 전 예습, 학원 수업 후 복습 강화
- **Agent 09 전달**: 학원 수업 이해도 기반 목표 조정 제안

---

### 1.2 학원 과제 부담과 학습 진행 연계 분석

#### 1.2.1 데이터 수집
- **학생 설문**: "오늘 학원 과제 양이 적절하신가요? (A) 적절함 (B) 조금 많음 (C) 너무 많음"

#### 1.2.2 분석 지표
- **과제 부담 등급**: 적절함(1점), 조금 많음(2점), 너무 많음(3점)
- **과제 부담-지연 상관계수**: 학원 과제 양과 학습 지연 상관 분석
- **과제 부담-만족도 상관계수**: 학원 과제 양과 학습 만족도 상관 분석

#### 1.2.3 인사이트 생성
- **과제 과다 → 지연 패턴**: 학원 과제가 많으면 학교 학습 지연
- **과제 과다 → 불만족 패턴**: 학원 과제가 많으면 학습 만족도 하락

#### 1.2.4 대응 전략
- **과제 과다 감지**: 과제 우선순위 조정, 학교 학습과 병행 전략
- **과제 부담 완화**: 필수 과제 우선, 선택 과제 후순위
- **Agent 09 전달**: 학원 과제 부담 기반 계획 조정 제안

---

### 1.3 학원 진도와 학교 진도 비교 분석

#### 1.3.1 데이터 수집
- **학생 설문**: "학원 진도와 학교 진도가 얼마나 차이나나요?"
- **선생님 텍스트 입력**: "학원 진도: 함수 단원 60%, 학교 진도: 함수 단원 40%"

#### 1.3.2 분석 지표
- **진도 차이**: 학원 진도 - 학교 진도 (양수: 학원이 앞섬, 음수: 학교가 앞섬)
- **진도 정렬도**: 학원 진도와 학교 진도의 일치 정도
- **현재 위치 파악**: 학원 진도와 학교 진도 비교하여 현재 학습 위치 파악

#### 1.3.3 인사이트 생성
- **학원 진도 앞섬**: 학교 진도 따라잡기 필요, 학교 수업 이해도 향상 필요
- **학교 진도 앞섬**: 학원 진도 따라잡기 필요, 학원 수업 이해도 향상 필요
- **진도 불일치**: 연계 학습 전략 수립 필요

#### 1.3.4 대응 전략
- **진도 불일치 감지**: 학원-학교 진도 연계 학습 전략 수립
- **진도 정렬**: 학원 진도와 학교 진도 일치화 전략
- **Agent 09 전달**: 학원-학교 진도 연계 기반 목표 조정 제안

## 2. 학생 수준별 차별화 전략

### 2.1 수학 성적 수준별 분석

#### 2.1.1 데이터 수집
- **학생 설문**: "현재 수학 성적 수준은? (A) 하위권 (B) 중위권 (C) 상위권"

#### 2.1.2 분석 지표
- **수준별 완료율 기대치**: 하위권(50%), 중위권(65%), 상위권(80%)
- **수준별 지연 허용 범위**: 하위권(±60분), 중위권(±40분), 상위권(±20분)
- **수준별 진행 패턴**: 하위권(지연 빈번), 중위권(적절), 상위권(원활)

#### 2.1.3 인사이트 생성
- **하위권 특성**: 지연 빈번, 완료율 낮음 → 쉬운 문제부터, 기초 개념 강화
- **중위권 특성**: 적절한 진행, 완료율 중간 → 유형별 집중 연습
- **상위권 특성**: 원활 진행, 완료율 높음 → 심화 학습, 추가 도전 과제

#### 2.1.4 대응 전략
- **하위권**: 쉬운 문제부터, 기초 개념 강화, 완료율 기대치 조정
- **중위권**: 유형별 집중 연습, 적절한 난이도 유지
- **상위권**: 심화 학습, 추가 도전 과제, 완료율 기대치 상향

---

### 2.2 수학 학습 스타일별 분석

#### 2.2.1 데이터 수집
- **학생 설문**: "수학 문제를 풀 때 어떤 방식이 편하신가요? (A) 계산을 정확하게 하는 것이 중요해요 (B) 개념을 이해하고 연결하는 것이 중요해요 (C) 다양한 문제 유형을 풀어보는 것이 중요해요"

#### 2.2.2 분석 지표
- **학습 스타일 분류**: 계산형, 개념형, 응용형
- **스타일별 진행 패턴**: 계산형(반복 연습 원활), 개념형(개념 학습 원활), 응용형(다양한 유형 원활)
- **스타일별 지연 패턴**: 계산형(개념 학습 지연), 개념형(반복 연습 지연), 응용형(단조로운 문제 지연)

#### 2.2.3 인사이트 생성
- **계산형 학생**: 반복 연습에서 원활, 개념 학습에서 지연 → 개념 정리 강화
- **개념형 학생**: 개념 학습에서 원활, 반복 연습에서 지연 → 반복 연습 강화
- **응용형 학생**: 다양한 유형에서 원활, 단조로운 문제에서 지연 → 다양한 유형 제공

#### 2.2.4 대응 전략
- **계산형**: 반복 연습 강화, 개념 정리 보완
- **개념형**: 개념 정리 강화, 반복 연습 보완
- **응용형**: 다양한 유형 제공, 단조로운 문제 보완

---

## 3. 종합 분석 및 리포트 생성

### 3.1 통합 분석 지표
- **학원 맥락 통합 지표**: 학원 수업 이해도, 학원 과제 부담, 학원-학교 진도 비교
- **학생 수준별 지표**: 수학 성적 수준, 수학 학습 스타일
- **맥락-수준 교차 분석**: 학원 맥락과 학생 수준의 교차 분석

### 3.2 종합 인사이트 생성
- **학원 맥락 기반 인사이트**: 학원 수업 이해도/과제 부담/진도가 학습 진행에 미치는 영향 분석
- **학생 수준 기반 인사이트**: 학생 수준별 맞춤 진행 패턴 분석
- **맥락-수준 교차 인사이트**: 학원 맥락과 학생 수준의 교차 영향 분석

### 3.3 종합 대응 전략 수립
- **학원 맥락 기반 전략**: 학원 수업 이해도 향상, 학원 과제 부담 완화, 학원-학교 진도 연계
- **학생 수준 기반 전략**: 수준별 맞춤 진행 전략, 수준별 기대치 조정
- **맥락-수준 교차 전략**: 학원 맥락과 학생 수준을 종합한 맞춤 전략
