# Agent 01 - Onboarding Actions

## 수행 가능한 액션 목록

### 초기화 액션
- `initialize_support_mode`: 지원 모드 초기화
- `recommend_path`: 학습 경로 추천
- `display_message`: 메시지 표시

### 분석 액션
- `generate_description`: 설명 생성 (exam_style_summary, parent_support_summary, study_hours_summary 등)
- `analyze`: 분석 수행 (exam_gap_areas 등)

### 추천 액션
- `recommend`: 루틴/전략 추천 (error_log_routine 등)
- `boost_mode`: 정서 강화 모드 활성화
- `strategy_type`: 전략 유형 설정

### 통신 액션
- `communication_mode`: 통신 모드 설정 (부모 피드백 채널 동기화 등)
- `alert`: 경고 메시지 표시

### 기타
- `suggest_challenge`: 도전 과제 제안
- `monitor_interval_minutes`: 모니터링 간격 설정
