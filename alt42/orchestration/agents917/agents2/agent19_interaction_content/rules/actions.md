# Agent 19 - Interaction Content Actions

## 수행 가능한 액션 목록

### 생성
- `generate_interaction_content`: 상호작용 컨텐츠 생성
- `create_advanced_problem_set`: 심화 문제 세트 생성
- `create_visualization_card`: 시각화 카드 생성

### 최적화
- `align_with_goals`: 목표 정렬
- `adjust_difficulty`: 난이도 조정
- `optimize_feedback_timing`: 피드백 타이밍 최적화
