# Agent 08 Calmness 지식파일

## 목적
- 침착도(집중/정서 안정) 지표로 학습 수행 적합도를 판단하고 개입을 제안

## 해석 기준 (교사용)
- 95+: 매우 침착, 고난도/심화 추천
- 90~94: 침착, 표준 학습 진행 권장
- 85~89: 안정, 가벼운 복습 → 점진적 난이도 상승
- 80~84: 보통, 쉬운 과제로 워밍업 후 본학습
- 75~79: 낮음, 3~5분 휴식/호흡 후 재시작
- <75: 긴급 복구, 휴식·수분섭취·스트레칭 권장

## 패턴 휴리스틱
- 기준선 대비 +5 이상: 고효율 상태 → 심화 컨텐츠 배치
- 기준선 대비 -5 이하: 피로 누적 → 휴식 루틴 삽입

## 코칭 템플릿
- "지금 효율이 높아요. 어려웠던 개념을 공략해봅시다."
- "짧은 휴식이 필요해 보여요. 호흡 3분 후 다시 시작해요."
