💬 포괄형 질문 1

“이 학생의 현재 수학 학습 진행 상태를 정밀하게 진단해줘.”
→ 수학일기 기록, 진도율, 지연 패턴, 감정곡선, 포모도르 리듬 등을 종합 분석해
지금 ‘이 학생이 어디까지 와 있고, 어느 지점에서 속도나 이해가 끊겼는지’를
현직 수학 선생님 수준으로 설명해줘.
(출력: 진행 요약, 지연 원인, 학습 리듬 해석, 감정 기반 위험 신호)

💬 포괄형 질문 2

“이 학생의 학습 단계별 강약점을 구조적으로 정리해줘.”
→ 단원별/난이도별/단계별(개념–유형–심화–기출) 데이터를 토대로
각 단계에서의 이해도·진도율·소요시간·오류유형을 분석해
‘어디서 병목이 발생하는지’, ‘어떤 구간이 원활한지’를 시각적으로 구분해줘.
(출력: 단계별 점수, 단원별 병목지도, 난이도별 패턴 요약)

💬 포괄형 질문 3

“지금 이 학생의 학습 위치를 기준으로, 다음 수업과 개입 방향을 어떻게 설계해야 할까?”
→ 진행률, 감정 곡선, 리듬 점수, 위험도 지수를 종합하여
현재 시점에서 ‘심화로 갈지, 복습으로 돌아갈지, 휴식 리듬을 조정할지’를
교사 수준으로 의사결정 로직을 제시해줘.
(출력: 권장 루틴 변화안, 단기 개입 전략, 연계 에이전트 전달 데이터 요약)

# 🧭 Agent 14 – 현재 위치 평가 활성 조건 및 복합질문

---

## 🩵 1. 학습이 계획 대비 지연 또는 불균형일 때

### 📍 상황

* 포모도르 진행률이 예상보다 느림
* 특정 단원이나 활동에서 시간 편차 발생
* 오늘 목표 달성률이 70% 미만

### 💬 에이전트 질문

> “현재 진행 궤적에서 예상 시간 대비 얼마나 지연되고 있나요?
> 지연이 발생한 구간은 어떤 활동 유형(개념 공부/문제풀이/정리)인가요?
> 최근 3개 포모도르 기준 집중 밀도와 만족도는 어떻게 변화했나요?
> 이 패턴이 지속될 경우 향후 1시간 내 이탈 위험도는 몇 등급으로 예측되나요?”

→ **출력:**
지연 분석 리포트 + 리듬 점수 + 예측 위험도 + 보정 권고

---

## 💛 2. 학습 활동이 멈춘 것 같이 보일 때

### 📍 상황

* 최근 30분 이상 입력 없음
* 마지막 tend 후 후속 활동 없음
* Eye Tracker나 필기 데이터 공백 발생

### 💬 에이전트 질문

> “현재 마지막 활동이 종료된 이후 몇 분간 무활동이 지속되었나요?
> 무활동 전후의 감정 상태 변화는 어땠나요?
> 동일한 패턴이 이번 주 몇 회 반복되었나요?
> 이 정체 구간이 습관화 패턴인지 일시적 피로인지 분류해줄 수 있나요?”

→ **출력:**
무활동 리포트 (지속시간, 정서 상관도, 패턴 유형) → Agent 13, 12 전송

---

## 🧡 3. 학생의 감정 곡선이 불안정하거나 부정적 피드백이 늘 때

### 📍 상황

* ‘불만족’ 응답 비율 상승
* 집중도 하락 또는 잦은 루틴 이탈
* 학습일기 피드백 내용에서 부정 키워드 증가

### 💬 에이전트 질문

> “최근 5개 포모도르 의 정서 곡선을 그려주세요.
> 감정 하락 구간은 어떤 단원이나 문제 유형과 겹치나요?
> 불만족 비율이 지연 구간과 얼마나 상관되나요?
> 정서 하락의 주요 원인을 인지적 요소(난이도, 시간압박)와 정서적 요소(피로, 좌절감)로 구분해 제시해주세요.”

→ **출력:**
정서 곡선 그래프 + 원인 분류 + 위험 요인 라벨링 → Agent 13 및 Rest Routine 공유

---

## 💚 4. 학습 계획 재조정이 필요할 때

### 📍 상황

* 계획 수립은 되어 있지만 실제 흐름이 다르게 진행
* 교사 또는 시스템이 “오늘 계획을 현실적으로 다시 맞춰야 함”을 감지

### 💬 에이전트 질문

> “현재 완료된 포모도르 대비 잔여 시간으로 달성 가능한 목표 범위는 어디까지인가요?
> 어떤 블록을 축소하거나 연기하면 지연 률을 20% 이내로 줄일 수 있나요?
> 감정 곡선과 진행 패턴을 기준으로 ‘지금 즉시 유지해야 할 핵심 활동’은 무엇인가요?”

→ **출력:**
조정 가능 목표 리스트 + 축소/연기 제안 + 핵심 유지 추천 → Agent 09 전송

---

## 💙 5. 일일 결산 또는 보고 시점

### 📍 상황

* 학습 세션 종료 또는 귀가 검사 직전
* 교사 또는 Agent 09가 일일 요약을 요청

### 💬 에이전트 질문

> “오늘 전체 학습의 진행곡선을 5분 단위로 요약해주세요.
> 시간 대비 완료율, 감정 변화, 집중 밀도의 상관 분석을 제공해주세요.
> 하루 종료 시점의 리듬 점수와 위험 등급 요약본을 보내주세요.”

→ **출력:**
일일 통합 리포트 (진행 그래프 + 정서 트렌드 + 위험도 요약) → Agent 09 및 교사 대시보드 전송

---

## 🩷 6. 장기 패턴 예측 또는 주간 리포트 생성 전

### 📍 상황

* Agent 09 또는 13이 “최근 패턴 기반 예측 모델 업데이트”를 요청
* 주간 학습 리듬 또는 이탈 예방 시뮬레이션 필요

### 💬 에이전트 질문

> “최근 7일간의 현재 위치 분석 데이터를 통합해 패턴 벡터를 생성해주세요.
> 지연도, 감정곡선, 리듬점수의 시간 상관계수를 계산해주세요.
> 다음 3일간 예상되는 이탈 위험도 추세를 예측해주세요.”

→ **출력:**
주간 트렌드 모델 + 예측 위험 지수 + 권장 루틴 변경안 → Agent 13, 09 전송
