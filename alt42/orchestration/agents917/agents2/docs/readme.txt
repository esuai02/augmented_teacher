| 폴더                 | 역할                                   |
| ------------------ | ------------------------------------ |
| **ontology/**      | 각 에이전트의 온톨로지 데이터 저장소 (개념/관계/규칙/SWRL) |
| **llm/**           | 해당 에이전트의 프롬프트, 예시, 응답 캐시 관리          |
| **knowledge/**     | 학습 관련 지식, 패턴, FAQ 등                  |
| **common/**        | 공용 실행 로직 (브릿지, 룰엔진, 상태관리)            |
| **ontology_core/** | 모든 에이전트가 공유하는 교육 도메인 온톨로지            |
| **llm_core/**      | 모든 에이전트가 공유하는 프롬프트 템플릿과 파서           |
| **config/**        | 시스템 및 에이전트 전역 설정                     |
| **scripts/**       | 온톨로지 검증/백업/LLM 프롬프트 자동 생성 스크립트       |
