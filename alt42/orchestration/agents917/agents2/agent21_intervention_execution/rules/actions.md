# Agent 21 - Intervention Execution Actions

## 수행 가능한 액션 목록

### 실행
- `execute_intervention`: 개입 실행
- `send_message`: 메시지 전송
- `deliver_resources`: 리소스 전달

### 기록
- `record_execution_time`: 실행 시각 기록
- `record_target_response`: 대상 반응 기록
- `track_performance_metrics`: 성과 지표 추적

### 후속
- `schedule_followup`: 후속 조치 일정 조정
- `evaluate_effectiveness`: 효과 평가
