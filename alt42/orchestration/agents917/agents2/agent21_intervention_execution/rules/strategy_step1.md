# Agent 21 - 수학 학습 단계별 개입 실행 전략 (Step 1)

## 개요

현직 수학선생님의 실제 개입 실행 방식을 분석하여 수학 학습 단계별로 차별화된 개입 실행 전략을 정의합니다.

**데이터 소스**: 선생님 설문, 체크리스트, 텍스트 입력을 통한 현직 선생님 경험 데이터

---

## 1. 개념 학습 단계 개입 실행 전략

### 현직 선생님의 실행 패턴

**선생님 텍스트 입력 기반 데이터:**
> "개념 학습 단계: 이해 확인 질문 먼저, 이해 안 되면 개념 재설명, 이해되면 예제 풀이 안내"

### 실행 순서

1. **이해 확인 질문 (Understanding Check Question)**
   - 실행 시점: 개념 설명 후 즉시
   - 실행 방식: 간단한 확인 질문 ("이 개념 이해되셨나요?")
   - 대기 시간: 5분
   - 목적: 학생의 이해도 즉시 파악

2. **개념 재설명 (Concept Review)**
   - 실행 조건: 이해 확인 질문에 "이해 안 됨" 응답
   - 실행 방식: 다른 방식으로 개념 재설명 (시각화, 예시 등)
   - 목적: 이해도 향상

3. **예제 풀이 안내 (Example Guidance)**
   - 실행 조건: 이해 확인 질문에 "이해됨" 응답
   - 실행 방식: 쉬운 예제부터 풀이 안내
   - 목적: 이해도 검증 및 적용

### 개입 실행 룰 매핑

- `MATH_STAGE_01_concept_learning_execution` (priority: 95)
- 실행 순서: 이해 확인 → 대기 → 조건부 실행 (재설명 또는 예제 안내)

---

## 2. 유형 연습 단계 개입 실행 전략

### 현직 선생님의 실행 패턴

**선생님 텍스트 입력 기반 데이터:**
> "유형 연습 단계: 막히면 힌트 제공, 재시도 권장, 틀리면 오류 원인 분석"

### 막힘 시간별 개입 전략

1. **3분 막힘 → 힌트 제공**
   - 실행 방식: 간단한 힌트 제공
   - 목적: 풀이 방향 제시

2. **5분 막힘 → 오류 원인 분석**
   - 실행 방식: 현재 풀이 과정 분석 및 오류 지점 지적
   - 목적: 오류 인식 및 수정

3. **틀린 후 → 유사 문제 제공**
   - 실행 조건: 힌트 제공 후에도 틀림
   - 실행 방식: 비슷하지만 쉬운 유사 문제 제공
   - 목적: 이해도 강화

### 개입 실행 룰 매핑

- `MATH_STAGE_02_type_practice_execution` (priority: 94)
- 실행 순서: 막힘 시간 감지 → 단계별 개입 실행

---

## 3. 심화 학습 단계 개입 실행 전략

### 현직 선생님의 실행 패턴

**선생님 텍스트 입력 기반 데이터:**
> "심화 학습 단계: 도전 의욕 확인, 단계별 접근법 제시, 포기 직전 격려"

### 실행 순서

1. **도전 의욕 확인 (Challenge Motivation Check)**
   - 실행 시점: 심화 문제 시작 전
   - 실행 방식: "이 문제 도전해볼까요?" 같은 의욕 확인
   - 목적: 동기 부여

2. **단계별 접근법 제시 (Stepwise Approach)**
   - 실행 조건: 막힘 발생 시
   - 실행 방식: 문제를 작은 단계로 나누어 접근법 제시
   - 목적: 복잡한 문제 단순화

3. **포기 직전 격려 (Encouragement Before Giveup)**
   - 실행 조건: 포기 의도 감지
   - 실행 방식: "조금만 더 해보면 풀 수 있어요" 같은 격려
   - 목적: 지속성 유지

4. **부분 성공 강조 (Partial Success Highlight)**
   - 실행 조건: 완전 풀이는 실패했으나 부분적 진전 있음
   - 실행 방식: "이 부분은 정확하게 하셨어요" 같은 부분 성공 강조
   - 목적: 자신감 유지

### 개입 실행 룰 매핑

- `MATH_STAGE_03_advanced_learning_execution` (priority: 93)
- 실행 순서: 의욕 확인 → 막힘 시 접근법 → 포기 직전 격려 → 부분 성공 강조

## 4. 기출 문제 풀이 단계 개입 실행 전략

### 현직 선생님의 실행 패턴

**선생님 텍스트 입력 기반 데이터:**
> "기출 문제 풀이 단계: 시간 관리 안내, 실전 전략 제시, 오답 분석 후 재풀이"

### 실행 순서

1. **시간 관리 안내 (Time Management Guidance)**
   - 실행 시점: 기출 문제 풀이 시작 전
   - 실행 방식: "이 문제는 5분 안에 풀어야 해요" 같은 시간 안내
   - 목적: 실전 시간 감각 향상

2. **실전 전략 제시 (Exam Strategy Provision)**
   - 실행 시점: 문제 풀이 중
   - 실행 방식: "시험에서는 이렇게 접근하세요" 같은 전략 제시
   - 목적: 실전 대비

3. **오답 분석 (Wrong Answer Analysis)**
   - 실행 조건: 틀린 문제 발생 시
   - 실행 방식: 오류 원인 분석 및 설명
   - 목적: 오류 인식 및 수정

4. **재풀이 권장 (Retry Recommendation)**
   - 실행 조건: 오답 분석 후
   - 실행 방식: "다시 풀어보세요" 같은 재풀이 권장
   - 목적: 이해도 강화

### 개입 실행 룰 매핑

- `MATH_STAGE_04_exam_practice_execution` (priority: 92)
- 실행 순서: 시간 관리 → 실전 전략 → 오답 분석 → 재풀이

- "각 학습 단계에서 어떤 순서로 개입하시나요?"
- "개념 학습 단계에서 가장 효과적인 개입 방식은 무엇인가요?"
- "유형 연습 단계에서 막힘 시 어떻게 대응하시나요?"

### 선생님 체크리스트
- [ ] 개념 학습: 이해 확인 질문 먼저

- [ ] 심화 학습: 도전 의욕 확인 후 단계별 접근법
- [ ] 기출 풀이: 시간 관리 안내 후 실전 전략

- "심화 학습 단계: 도전 의욕 확인, 단계별 접근법 제시, 포기 직전 격려"
- "기출 문제 풀이 단계: 시간 관리 안내, 실전 전략 제시, 오답 분석 후 재풀이"

## 효과성 평가 기준

### 단계별 효과성 지표

1. **개념 학습 단계**
   - 이해 확인 질문 응답률
   - 개념 재설명 후 이해도 향상률
   - 예제 풀이 성공률

2. **유형 연습 단계**
   - 힌트 제공 후 재시도 성공률
   - 오류 분석 후 유사 문제 성공률
   - 막힘 시간 단축률

3. **심화 학습 단계**
   - 도전 의욕 확인 후 지속률
   - 단계별 접근법 적용 성공률
   - 포기 직전 격려 후 완료률

4. **기출 문제 풀이 단계**
   - 시간 관리 준수률
   - 실전 전략 적용률
   - 재풀이 성공률
