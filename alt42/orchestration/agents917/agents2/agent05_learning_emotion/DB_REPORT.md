## 데이터베이스 구조

### 참조 테이블

Agent 05는 다른 에이전트의 데이터를 종합하여 사용하거나, 별도의 감정 관련 테이블을 사용할 수 있습니다.

**가능한 데이터 소스**:
- `mdl_abessi_today`: 오늘 목표/검사 데이터 (감정 점수 포함 가능)
- `mdl_alt42_student_profiles`: 학생 프로필 (감정 상태 포함 가능)
- 기타 감정 추적 테이블 (구현 상태 확인 필요)
