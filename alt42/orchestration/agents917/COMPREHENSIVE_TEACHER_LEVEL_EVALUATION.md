### 선생님 체크리스트 항목

#### 단원 연계성
1. "학년별 단원 선후관계를 정의해주세요"
   - 예: 중1 정수→방정식→함수
   - 예: 중2 일차함수→연립방정식

#### 단원별 난이도
2. "단원별 난이도와 평균 학습 시간을 입력해주세요"
   - 중1 함수 단원: 난이도 8/10, 평균 학습 시간 6시간
   - 중2 통계 단원: 난이도 4/10, 평균 학습 시간 3시간

#### 출제 빈도
3. "학교/선생님별 출제 패턴을 입력해주세요"
   - 함수 단원: 항상 3문제 출제
   - 기본형 60%, 유형 30%, 심화 10% 비율

#### 학생 수준별 전략
4. "학생 수준별 학습 전략을 입력해주세요"
   - 하위권: 개념 중심, 목표 정답률 60%
   - 상위권: 심화 중심, 목표 정답률 90%

### 선생님 텍스트 입력 항목

#### 학원 정보
1. "학원 진도와 학교 진도를 비교해주세요"
   - 예: 학원 진도: 함수 단원, 학교 진도: 방정식 단원 → 선행 학습 중

#### 문제 재정의
2. "학생의 문제를 수학 학습 관점에서 재정의해주세요"
   - 예: 표면적 문제: 수학 점수 하락
   - 근본 원인: 함수 개념 이해 부족 → 함수 단원 재학습 필요

#### 피드백 템플릿
3. "수학 단원별, 오류 유형별 피드백 템플릿을 입력해주세요"
   - 함수 단원 오류: 함수식 이해 부족 → 개념 재학습 필요
   - 계산 실수: 반복 연습 필요, 풀이 과정 확인 강화
