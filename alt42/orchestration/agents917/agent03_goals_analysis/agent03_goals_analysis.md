# Agent 03 Goals Analysis 지식파일

## 목적
- 목표의 질과 진행률을 분석해 달성 전략을 최적화

## 핵심 지표
- 달성률, 진행상태, 평균 목표기간, 카테고리 균형, 목표 품질 점수

## 해석/휴리스틱
- 달성률 < 60%: 목표 세분화, 기간 현실화
- 장기 목표 다수: 단계 목표로 분해, 중간 성취 도출
- 한 카테고리 치우침: 균형 보정(학업/기술/성장)

## 코칭 템플릿
- "목표를 2~3단계로 쪼개어 작은 성공을 쌓아봅시다."
- "카테고리 균형을 위해 이번 주엔 보완 영역에 20% 시간을 배정해요."
