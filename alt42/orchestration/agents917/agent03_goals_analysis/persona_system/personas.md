# Agent 03 - Goals Analysis Personas

**Version:** 1.0
**Purpose:** 목표 분석 에이전트의 학생 페르소나 정의
**Reference:** rules.yaml, contextlist.md, Agent03PersonaEngine.php

---

## 개요

Agent 03 Goals Analysis는 학생의 목표 설정, 진행, 달성 과정을 분석하고 최적의 전략을 제안합니다. 이 문서는 목표 관련 상황에서 학생들이 보이는 다양한 페르소나를 정의합니다.

### 페르소나 분류 체계

| 상황 코드 | 상황 설명 | 페르소나 수 |
|-----------|-----------|-------------|
| G0 | 목표 설정 단계 | 6개 |
| G1 | 목표 진행 중 | 5개 |
| G2 | 목표 정체/위기 | 4개 |
| G3 | 목표 재설정 | 2개 |
| CRISIS | 위기 상황 | 2개 |

---

## G0: 목표 설정 단계 페르소나

새로운 목표를 설정하는 단계에서 나타나는 학생 유형

### G0_P1: 야심찬 과목표 설정자 (Ambitious Overreacher)

**특성:**
- 매우 높은 목표를 다수 설정
- "전교 1등", "100점", "완벽하게" 등의 표현 사용
- 현실적 제약 고려 부족
- 높은 초기 열정

**발화 패턴:**
```
"이번에는 진짜 전교 1등 할 거예요"
"모든 과목 100점 목표예요"
"완벽하게 다 해내고 싶어요"
"목표는 10개 정도 세웠어요"
```

**권장 개입:**
- 톤: Gentle
- 개입 패턴: GapAnalysis
- 전략: 중간 마일스톤 제안, 우선순위 설정 유도

**위험 요소:**
- 초기 실패로 인한 의욕 상실
- 번아웃 가능성
- 자기효능감 저하

---

### G0_P2: 모호한 목표 설정자 (Vague Dreamer)

**특성:**
- 추상적이고 불명확한 목표
- 측정 가능한 기준 부재
- 기한 설정 미흡
- 구체적 실행 계획 부재

**발화 패턴:**
```
"그냥 잘하고 싶어요"
"어떻게든 나아지게요"
"뭔가 달라지면 좋겠어요"
"좀 더 열심히 할게요"
```

**권장 개입:**
- 톤: Professional
- 개입 패턴: GoalSetting
- 전략: SMART 기준 안내, 구체화 질문

**위험 요소:**
- 진행 상황 측정 불가
- 달성 여부 판단 어려움
- 동기 유지 어려움

---

### G0_P3: 외부 동기 기반 목표 설정자 (External-Driven)

**특성:**
- 부모, 선생님, 친구 등 타인의 기대에 의한 목표
- 개인적 의미 연결 부족
- 소극적 참여 태도
- "해야 해서" 식 표현

**발화 패턴:**
```
"부모님이 하래서요"
"선생님이 정해주셨어요"
"다들 이렇게 하니까요"
"꼭 해야 하는 거래요"
```

**권장 개입:**
- 톤: Collaborative
- 개입 패턴: GoalSetting
- 전략: 내재적 동기 탐색, 개인 가치 연결

**위험 요소:**
- 외부 압력 제거 시 포기
- 진정한 성취감 부재
- 장기적 동기 유지 어려움

---

### G0_P4: 현실적 계획자 (Realistic Planner)

**특성:**
- 구체적이고 측정 가능한 목표
- 기한과 제약 조건 인식
- 단계별 계획 수립
- 자기 능력에 대한 현실적 평가

**발화 패턴:**
```
"한 달 안에 80점 이상 목표예요"
"시간이 부족하니까 중요한 것부터요"
"매일 30분씩 하려고요"
"지금 실력으로 여기까지는 가능할 것 같아요"
```

**권장 개입:**
- 톤: Professional
- 개입 패턴: PlanDesign
- 전략: 기존 계획 정교화, 효율성 향상

**강점:**
- 높은 달성 확률
- 자기 관리 능력
- 적응적 조정 가능

---

### G0_P5: 두려움 기반 목표 회피자 (Fear-Based Avoider)

**특성:**
- 실패에 대한 강한 두려움
- 최소한의 안전한 목표만 설정
- 도전 회피 경향
- 낮은 자기효능감

**발화 패턴:**
```
"못할 것 같아요"
"실패하면 어떡해요"
"자신 없어요"
"그냥 작은 거만 할래요"
```

**권장 개입:**
- 톤: Encouraging
- 개입 패턴: EmotionalSupport
- 전략: 자신감 구축, 작은 성공 경험 제공

**위험 요소:**
- 성장 기회 제한
- 낮은 자존감 유지
- 학습된 무기력

---

### G0_P6: 전략적 최적화 추구자 (Strategic Optimizer)

**특성:**
- 데이터와 분석 중시
- 효율성과 최적화 추구
- 메트릭스와 지표에 관심
- 합리적 의사결정

**발화 패턴:**
```
"가장 효율적인 방법이 뭐예요?"
"데이터로 분석해주세요"
"우선순위를 정해주세요"
"시간 대비 효과가 어떨까요?"
```

**권장 개입:**
- 톤: Professional
- 개입 패턴: PlanDesign
- 전략: 분석적 프레임워크 제공, 데이터 기반 전략

**강점:**
- 논리적 접근
- 자원 최적화 능력
- 객관적 평가 가능

---

## G1: 목표 진행 중 페르소나

목표를 향해 진행 중인 상황에서 나타나는 학생 유형

### G1_P1: 꾸준한 달성자 (Consistent Achiever)

**특성:**
- 높은 목표 달성률 (70% 이상)
- 일관된 진행 패턴
- 자기 규율 우수
- 긍정적 피드백 루프

**발화 패턴:**
```
"이번 목표도 달성했어요!"
"매일 하니까 습관이 됐어요"
"다음 목표도 도전해볼까요?"
"더 어려운 것도 해보고 싶어요"
```

**권장 개입:**
- 톤: Professional
- 개입 패턴: SkillBuilding
- 전략: 스트레치 목표 제안, 성장 기회 확대

**활용 포인트:**
- 롤모델로 활용 가능
- 고급 전략 적용 가능
- 자기주도 학습 촉진

---

### G1_P2: 어려움 속 인내자 (Struggling Perseverer)

**특성:**
- 낮은 달성률에도 불구하고 노력 지속
- 높은 노력 수준
- 좌절에도 포기하지 않음
- "열심히 하는데..." 표현

**발화 패턴:**
```
"열심히 하는데 잘 안 돼요"
"포기하고 싶지 않아요"
"뭐가 문제인지 모르겠어요"
"계속 해볼게요"
```

**권장 개입:**
- 톤: Warm
- 개입 패턴: EmotionalSupport + GapAnalysis
- 전략: 노력 인정, 장애물 분석, 방법 개선

**주의점:**
- 노력을 당연시하지 않기
- 구체적 개선점 제시
- 정서적 지지 우선

---

### G1_P3: 일관성 없는 시작자 (Inconsistent Starter)

**특성:**
- 빈번한 시작과 중단 반복
- 낮은 일관성 점수
- 많은 포기된 목표
- "또 다시..." 패턴

**발화 패턴:**
```
"또 못 했어요"
"다시 시작할게요"
"이번엔 정말 할 거예요"
"까먹었어요"
```

**권장 개입:**
- 톤: Gentle
- 개입 패턴: BehaviorModification
- 전략: 목표 단순화, 습관 시스템 구축

**핵심 전략:**
- 최소 실행 단위 설정
- 트리거-행동-보상 연결
- 환경 설계 지원

---

### G1_P4: 과부하 멀티태스커 (Overloaded Multitasker)

**특성:**
- 8개 이상의 동시 진행 목표
- 분산된 집중력
- 압도감과 스트레스
- "너무 많다" 표현

**발화 패턴:**
```
"다 하려니까 벅차요"
"시간이 부족해요"
"뭐부터 해야 할지 모르겠어요"
"너무 많은 것 같아요"
```

**권장 개입:**
- 톤: Calm
- 개입 패턴: GapAnalysis
- 전략: 우선순위 설정, 목표 축소 제안

**긴급 조치:**
- 즉각적인 목표 수 점검
- 상위 3개 우선순위 선정
- 나머지 목표 보류 권고

---

### G1_P5: 완료 직전 목표 보유자 (Near Completion)

**특성:**
- 80% 이상 진행된 목표 2개 이상
- 마지막 단계에서 정체
- 완료 직전 긴장감

**발화 패턴:**
```
"거의 다 했어요"
"마지막이 어려워요"
"끝내고 싶어요"
"조금만 더 하면 돼요"
```

**권장 개입:**
- 톤: Encouraging
- 개입 패턴: SkillBuilding
- 전략: 완료 축하 준비, 다음 목표 논의

---

## G2: 목표 정체/위기 상황 페르소나

목표 진행이 멈추거나 위기에 처한 상황

### G2_P1: 정체 좌절형 (Plateau Frustrated)

**특성:**
- 3주 이상 진행 정체
- 좌절감과 답답함
- "왜 안 되지" 자문
- 방법 변경 필요 인식

**발화 패턴:**
```
"왜 안 늘어요?"
"제자리예요"
"답답해요"
"뭘 해도 안 돼요"
```

**권장 개입:**
- 톤: Warm
- 개입 패턴: GapAnalysis
- 전략: 정체 원인 분석, 전략 전환 제안

---

### G2_P2: 포기 직전형 (Giving Up) - 최우선

**특성:**
- 명시적 포기 의사 표현
- 극도로 낮은 동기
- 무기력 징후
- 즉각적 개입 필요

**발화 패턴:**
```
"포기할래요"
"더 이상 못하겠어요"
"소용없어요"
"그만둘래요"
```

**권장 개입:**
- 톤: Warm
- 페이스: Very Slow
- 개입 패턴: EmotionalSupport
- 전략: 압력 제거, 작은 성공에 집중

**긴급 프로토콜:**
1. 목표 논의 일시 중단
2. 정서 상태 우선 확인
3. 무조건적 지지 표현
4. 아주 작은 목표로 재시작

---

### G2_P3: 비교 고통형 (Comparison Distressed)

**특성:**
- 타인과의 비교로 인한 고통
- 자존감 저하
- "나만 못해" 인식
- 사회적 비교 집착

**발화 패턴:**
```
"친구는 벌써 했는데..."
"다른 애들은 다 하는데"
"나만 뒤처져요"
"왜 나만 못하죠?"
```

**권장 개입:**
- 톤: Warm
- 개입 패턴: EmotionalSupport
- 전략: 개인 성장 관점 전환, 비교 중단 유도

---

### G2_P4: 외부 장애 직면형 (External Blocker)

**특성:**
- 시간, 환경 등 외부 제약
- 의지와 무관한 장애
- 현실적 조정 필요

**발화 패턴:**
```
"시간이 없어서요"
"시험 기간이라"
"학교 일정 때문에"
"상황이 안 돼요"
```

**권장 개입:**
- 톤: Collaborative
- 개입 패턴: PlanDesign
- 전략: 현실적 일정 조정, 미시 기회 활용

---

## G3: 목표 재설정 상황 페르소나

### G3_P1: 성찰적 조정자 (Reflective Adjuster)

**특성:**
- 자발적 목표 조정 의사
- 현실적 자기 평가
- 성숙한 태도

**발화 패턴:**
```
"다시 생각해보니..."
"조정해야 할 것 같아요"
"너무 무리였어요"
"현실적으로 다시 정할게요"
```

**권장 개입:**
- 톤: Professional
- 개입 패턴: GoalSetting
- 전략: SMART 기준 재적용, 조정 지원

---

### G3_P2: 방어적 합리화형 (Defensive Rationalizer)

**특성:**
- 실패 합리화 경향
- 원래 목표 축소 해석
- 방어 기제 작동

**발화 패턴:**
```
"원래 그렇게 중요한 건 아니었어요"
"그냥 해본 거였어요"
"별로 신경 안 썼어요"
```

**권장 개입:**
- 톤: Gentle
- 개입 패턴: EmotionalSupport
- 전략: 노력 인정 우선, 부드러운 현실 점검

---

## CRISIS: 위기 상황 페르소나 (최우선)

### CRISIS_P1: 목표 불안 위기형 (Goal Anxiety Crisis)

**특성:**
- 목표로 인한 극심한 불안
- 수면 장애, 스트레스 증상
- 즉각적 정서 안정 필요

**발화 패턴:**
```
"목표 생각하면 불안해요"
"잠이 안 와요"
"스트레스 받아요"
"압박감이 너무 커요"
```

**긴급 개입:**
- 톤: Warm
- 페이스: Very Slow
- 개입 패턴: SafetyNet
- 즉각 조치: 목표 논의 중단, 정서 안정 우선

---

### CRISIS_P2: 번아웃 징후형 (Burnout Signs)

**특성:**
- 신체적, 정서적 탈진
- 전반적 의욕 상실
- "다 그만두고 싶다" 표현
- 장기적 회복 필요

**발화 패턴:**
```
"다 지쳤어요"
"아무것도 하기 싫어요"
"그냥 다 포기하고 싶어요"
"의욕이 없어요"
```

**긴급 개입:**
- 톤: Warm
- 페이스: Very Slow
- 개입 패턴: CrisisIntervention
- 즉각 조치: 모든 목표 축소/보류, 휴식 권고, 전문 상담 연계 검토

---

## 페르소나 전환 트리거

| 전환 유형 | 감지 신호 | 조치 |
|-----------|-----------|------|
| 동기 저하 | 응답 열정 감소, 참여도 하락 | 격려 모드 전환 |
| 성공 모멘텀 | 연속 달성, 자신감 상승 | 스트레치 목표 제안 |
| 번아웃 경고 | 갑작스런 과부하, 탈진 언급 | 위기 모드 전환 |
| 불안 상승 | 짧은 응답, 회피 언급 | 정서 지원 강화 |
