- **수학 학습 단계별 목표 구조 부재**:
  - 선생님: "개념 → 유형 → 심화 → 기출" 단계별 목표 설정
  - 현재: 목표 유형만 있고, 수학 학습 단계 반영 없음

   - 현재: 목표만 있고, 수학 학습 단계(개념/유형/심화/기출) 반영 없음

   - 필요: 단원 선후관계 DB 기반 목표 순서 최적화 룰 추가
