 
💬 Agent 03 – 포괄형 질문 세트
💬 포괄형 질문 1

“이 학생의 목표와 계획이 실제 학습 리듬 속에서 얼마나 유기적으로 작동하고 있나요?”
→ 분기·주간·오늘 목표 간 연결성, 계획의 현실성, 일정 변동 대응력, 회복탄력성 지표를 함께 분석해
실제 루틴이 궤도 안에서 작동 중인지 진단하고, 필요 시 구조적 재설계를 제안.
(핵심 포인트: 목표-계획 연계율 / 계획 품질 / 회복 루프 / 실행 지속성)

💬 포괄형 질문 2

“현재 학생의 학습 궤도가 장기 커리큘럼과 목표 궤적 위에 안정적으로 놓여 있나요?”
→ 분기목표와 장기 커리큘럼의 정합성, 시험일정 반영률, 진학 목표와의 궤도 일치 정도를 통합 분석해
“이탈·지연·과속” 여부를 판단하고, 커리큘럼·루틴·시간배분을 어떻게 조정해야 궤도 재진입이 가능한지 제시.
(핵심 포인트: 커리큘럼 정합성 / 분기목표 적합도 / 진학 방향 연계 / 궤도 안정성)

💬 포괄형 질문 3

“이 학생의 목표 설계 구조는 변동 상황에서도 회복 가능하고, 선생님의 노하우가 반영된 형태인가요?”
→ 목표 설계의 단계성·탄력성, 일정 변화 대응 전략, 감정 기반 회복 패턴, 교사 피드백 반영률을 분석해
목표 시스템이 ‘무너지지 않고 다시 일어서는 구조’로 되어 있는지를 평가하고,
교사 노하우와 연계된 회복 전략 자동생성 로직을 제안.
(핵심 포인트: 회복탄력성 / 정서적 복귀속도 / 교사 노하우 반영률 / 자기조정능력)

## 🧭 1. 이 에이전트를 필요로 하는 상황 (Trigger Conditions)

| 상황 코드                    | 상황 설명                                             | 주요 신호                         |
| ------------------------ | ------------------------------------------------- | ----------------------------- |
| **S1. 목표–계획 불일치 감지**     | 분기목표와 주간/오늘 목표 간의 연결이 약하거나, 계획이 실현되지 않는 패턴이 반복될 때 | 주간 목표 달성률 < 70%, 일정 변동률 > 30% |
| **S2. 시간 부족 딜레마 발생**     | 학습 일정이 압축되어 목표가 현실적으로 불가능해질 때                     | “시간이 모자라요”, “숙제가 밀렸어요” 등의 로그  |
| **S3. 회복탄력성 저하 감지**      | 변수(시험, 일정 변경, 컨디션 저하 등) 발생 시 복귀가 느릴 때             | 일정 복귀 T₍avg₎ > 3일             |
| **S4. 목표 과부하 또는 미루기 증가** | 목표량이 많거나 실행 지연이 누적될 때                             | 미루기 로그 3회 이상 / 포모도르 종료율 ↓     |
| **S5. 루틴 저항도 상승**        | 귀가검사 / 루틴 데이터에서 부정 감정 ↑ 또는 참여도 ↓                  | 감정 점수 < 5 / 10                |
| **S6. 커리큘럼-목표 불일치 감지**   | 장기 커리큘럼과 분기 목표의 흐름이 어긋날 때                         | 커리큘럼 진도 편차 > 20%              |
| **S7. 교사 피드백 불일치**       | 교사가 판단한 ‘핵심목표’와 시스템이 인식한 목표 방향이 다를 때              | 교사 노트 vs AI 추천 불일치 > 2회       |
| **S8. 시험 직전 목표 재정렬 필요**  | 시험 8주 전 모드로 진입할 때                                 | 시험일정 데이터 인식 Trigger           |

## 🧩 2. 각 상황별 맞춤형 복합질문

### 🟦 S1 : 목표–계획 불일치 감지

> 🔍 “왜 계획대로 흐르지 않았는가?”를 다층적으로 진단

1. 분기 목표와 주간 목표가 어디서 엇나갔는가요?
2. 주간 계획을 세울 때 시험 일정이나 컨디션 같은 변수를 충분히 반영했나요?
3. 이번 주 실행 데이터를 기준으로 보면 어느 단계에서 흐름이 끊겼나요?
4. 목표를 수정해야 할까요, 아니면 계획 방식을 바꿔야 할까요?

---

### 🟨 S2 : 시간 부족 딜레마 발생

> 🕒 “시간 한계 속 유연한 회복력 있는 대안”을 유도

1. 이번 주 가장 시간을 압박한 요인은 무엇인가요? (학교, 피로, 계획 오류 중 선택)
2. 우선순위를 조정한다면 어떤 활동을 줄이거나 연기할 수 있을까요?
3. 회복탄력 전략으로 짧은 세션이나 보충시간을 활용할 수 있을까요?
4. 일정 압박이 학습 동력으로 작용하고 있나요, 아니면 부담으로 작용하나요?

---

### 🟧 S3 : 회복탄력성 저하 감지

> 🌱 “충격 후 복귀 속도와 심리적 회복력” 분석

1. 일정이 흐트러졌을 때 어떤 감정이 먼저 느껴졌나요? (좌절, 무관심, 의욕 중?)
2. 복귀 과정에서 도움을 받은 부분이 있었나요? (교사 조언, 루틴, 포모도르)
3. 평소 여유 시간을 어디에 배치해 두었나요? 그게 회복에 도움이 됐나요?
4. 다시 리듬을 만들기 위해 이번 주 어떤 작은 단계부터 시작할 수 있을까요?

---

### 🟥 S4 : 목표 과부하 또는 미루기 증가

> ⚖️ “목표 양 vs 실행력 의 균형” 점검

1. 목표 개수가 너무 많다고 느껴지나요, 아니면 집중이 잘 안 되나요?
2. 미루게 된 이유가 시간 부족인가요, 동기 저하인가요?
3. 현재 목표 중 가장 핵심 2개만 골라본다면 무엇인가요?
4. 주간 계획에서 어디까지 ‘선택적 생략’을 허용해볼 수 있을까요?

---

### 🟩 S5 : 루틴 저항도 상승

> 💬 “감정 신호와 행동 저항” 탐지

1. 요즘 루틴을 시작할 때 가장 귀찮게 느껴지는 부분은 어디인가요?
2. 수업준비 또는 포모도르 초기 5분에 집중을 깨는 요소가 있나요?
3. 학습 리듬을 다시 올리기 위해 변화를 주고 싶은 부분이 있나요?
4. 감정 패턴상 “힘든 날” 후 회복 속도는 어떤가요?

---

### 🟪 S6 : 커리큘럼–목표 불일치 감지

> 🧩 “장기 커리큘럼 과 단기 목표 의 정합성” 점검

1. 이번 분기 목표가 장기 커리큘럼과 어디서 엇갈리고 있을까요?
2. 최근 시험 결과가 커리큘럼 조정에 영향을 줬나요?
3. 진학 목표에 비해 현재 커리큘럼의 속도나 깊이는 적절한가요?
4. 경쟁 커리큘럼 데이터와 비교했을 때 보완해야 할 영역은 어디인가요?

---

### 🟫 S7 : 교사 피드백 불일치

> 🤝 “AI 분석 vs 교사 직관 의 차이 조정”

1. 선생님이 보기에 이 학생의 핵심 목표는 무엇인가요?
2. AI 가 추천한 목표와 차이가 나는 이유는 무엇이라고 생각하나요?
3. 교사의 판단 근거 (정서, 집중력, 지속성 등)를 모델에 어떻게 반영할까요?

---

### ⚫ S8 : 시험 직전 재정렬

> 🎯 “시험 궤도 진입을 위한 목표–시간 재배치”

1. 시험 8주 전 현재 목표 중 가장 효율이 낮은 부분은 무엇인가요?
2. 시험 기간 동안 ‘핵심 개념 – 유형 – 기출’ 루프가 완성되어 있나요?
3. 일정 압박을 줄이기 위해 지금 선제적으로 조정할 부분은 있나요?
4. 분기목표와 시험목표 간의 시간축 불일치는 없나요?
