# Agent 10 Concept Notes 지식파일

## 목적
- 개념노트(화이트보드 필기) 데이터를 기반으로 개념 이해도와 학습 흐름을 해석

## 핵심 지표 해석
- 총 필기량(nstroke): 활동 강도/몰입도 지표
- 마지막 필기시점(tlaststroke): 세션 종료 근접도/최근성
- 생성시각(timecreated): 기록 시작 타임라인
- usedtime : 노트에 사용된 총 시간
- 개념 제목(contentstitle): 단원/주제 맥락

## 해석 휴리스틱
- 필기량 상위 항목: 집중 또는 난해 구간(컨텍스트로 분별)
- 마지막 필기 최근 + 필기량 하위: 가벼운 탐색 가능성 → 보강 권장
- 오래된 항목 + 필기량 중간: 회상 복습 후보

## 코칭 템플릿
- "필기가 많았던 개념을 요약 카드로 정리해봅시다."
- "최근 필기가 적었던 개념을 10분 복습으로 보완해요."
