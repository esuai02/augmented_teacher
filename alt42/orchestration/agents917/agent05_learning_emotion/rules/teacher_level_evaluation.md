- **감정 원인 분석 깊이 부족**:

### 2️⃣ 수학 학습 특화 감정 (12/25점, D+)

#### ✅ 부분적 반영
- **문제풀이 멈춤 감지 (R3)**: 문제풀이 중 멈춤 시간 감지
- **유형학습 흥미 손실 (R2)**: 유형학습 중 흥미도 감지

#### ❌ 현저히 부족한 점

**수학 특화 감정 패턴 누락:**

1. **문제 유형별 감정 패턴 미반영**
   - 선생님: "함수 문제에서 좌절하는 학생 vs 도형 문제에서 자신감을 보이는 학생"
   - 현재: 문제 유형과 감정 패턴 연결 없음
   - 설문 필요: "어떤 유형의 문제를 풀 때 가장 자주 막히나요?"
     - (A) 계산 문제
     - (B) 개념 이해 문제
     - (C) 응용 문제
     - (D) 증명 문제

2. **계산 실수 vs 개념 오류 감정 차이 미반영**
   - 선생님: "계산 실수로 틀렸을 때는 아쉬움, 개념 오류로 틀렸을 때는 좌절감"
   - 현재: 오답노트 감정만 있고 오류 유형별 감정 구분 없음
   - 설문 필요: "문제를 틀렸을 때, 계산 실수였을 때와 개념을 몰랐을 때 감정이 다르나요?"

3. **난이도별 감정 반응 패턴 부재**
   - 선생님: "기본 문제: 자신감, 유형 문제: 도전감, 심화 문제: 불안 또는 흥미"
   - 현재: 난이도와 감정 패턴 연결 없음
   - 설문 필요: "어려운 문제를 마주했을 때 어떤 감정이 드나요?"
     - (A) 도전하고 싶어요
     - (B) 포기하고 싶어요
     - (C) 조금씩 풀어보고 싶어요
     - (D) 먼저 쉬운 문제부터 하고 싶어요

4. **풀이 과정별 감정 변화 추적 부재**
   - 선생님: "문제 읽기 → 풀이 시작 → 중간 계산 → 답 확인" 각 단계별 감정 변화 관찰
   - 현재: 활동 완료 후 감정만 탐지, 과정 중 감정 변화 추적 없음
   - 설문 필요: "문제를 풀 때, 어느 단계에서 가장 스트레스를 받나요?"
     - (A) 문제를 읽을 때
     - (B) 풀이 방법을 생각할 때
     - (C) 계산하는 과정
     - (D) 답을 확인할 때

5. **시간 압박 감정 미반영**
   - 선생님: "시험 시간이 부족할 때의 불안과 문제가 어려워서의 불안은 다르다"
   - 현재: 시간 관련 감정 요소 부재
   - 설문 필요: "시간이 부족할 때 어떤 감정이 드나요?"
     - (A) 긴장감이 높아져요
     - (B) 집중력이 올라가요
     - (C) 당황해서 실수를 해요
     - (D) 포기하고 싶어요
