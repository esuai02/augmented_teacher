# Agent 22 - Module Improvement Actions

## 수행 가능한 액션 목록

### 수집
- `collect_improvement_ideas`: 개선 아이디어 수집
- `identify_issues`: 이슈 식별
- `assess_impact_effort`: 영향도-노력도 평가

### 우선순위화
- `prioritize_improvements`: 개선안 우선순위화
- `create_impact_effort_matrix`: 영향도-노력도 매트릭스 생성
- `select_top_improvements`: 상위 개선안 선택
