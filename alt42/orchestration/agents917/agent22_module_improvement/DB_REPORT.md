## 에이전트 개요

**목적**: 에이전트 모듈 개선을 위한 로그 분석 및 룰 파일 분석

**주요 기능**:
- 에이전트 실행 로그 분석
- 룰 파일 분석
- 모듈 개선 제안

**특징**: 직접적인 DB 테이블 없이 로그 기반 분석

---

## 데이터베이스 구조

### 참조 데이터 소스

Agent 22는 다음 데이터 소스를 사용합니다:

1. **에이전트 실행 로그**: 시스템 로그 파일
2. **룰 파일**: 각 에이전트의 rules.yaml 파일
3. **사용자 테이블**: `mdl_user` (학생 기본 정보)

---

### 1. 사용자 테이블: `mdl_user`

**목적**: 학생 기본 정보

**주요 필드**:
- `id`: 사용자 ID
- `firstname`: 이름
- `lastname`: 성
