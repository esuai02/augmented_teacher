💬 학습관리 분석 에이전트용 포괄형 질문 3가지
💬 포괄형 질문 1

“이 학생의 학습관리 데이터 전체를 종합했을 때, 지금 가장 취약하거나 불안정한 부분은 어디이며 어떤 원인 패턴이 작동하고 있는지 분석해줘.”
→ 데이터 생성 빈도, 루틴 유지율, 포모도르 누락률, 감정기복, 시스템 신뢰도 등을 함께 고려하여
‘학습 관리력 저하의 근본 원인’을 구조적으로 진단하고 등급화된 리스크 요약 리포트 도출

💬 포괄형 질문 2

“이 학생의 학습관리 데이터를 기반으로, 어떤 피드백 루프를 우선적으로 재설계해야 지속적 성장을 만들 수 있을까?”
→ 데이터 불균형(개념/문제/복습 간), 루틴 붕괴 주기, 자동화 저항 패턴 등을 분석하여
우선순위 높은 피드백 영역(계획-실행-검증 중 어디인지)과 그에 맞는 행동전략/알림 방식 제시

💬 포괄형 질문 3

“이 학생이 학습관리 안정성을 높이기 위해 지금부터 어떤 루틴적 습관과 시스템 연동 방식을 강화해야 할까?”
→ 최근 4주간의 데이터 일관성, 루틴 유지 성공률, 피드백 반영률, 시스템 의존도, 회복속도 등을 종합하여
‘관리 안정성 향상 시나리오(습관화 루틴 + 자동화 개입 포인트)’를 단계별로 제안
---

### **1️⃣ 이 에이전트를 필요로 하는 상황**

| 구분                  | 상황 설명                            | 주요 징후                          |
| ------------------- | -------------------------------- | ------------------------------ |
| **① 활동데이터 희박형**     | 최근 활동결과, 포모도르, 질문기록 등이 거의 없는 상태  | 학습 로그 단절, 학습 의욕 저하, 심리적 거리감 발생 |
| **② 데이터는 많은데 불균형형** | 활동은 많으나 일부 데이터(예: 복습·정리)가 지속 누락됨 | 학습루틴의 편향, 자기점검 부재              |
| **③ 패턴불안정형**        | 어떤 주는 몰입도가 높고, 어떤 주는 거의 없음       | 감정기복, 외부요인 영향                  |
| **④ 자동화 저항형**       | 시스템이 제시한 루틴이나 피드백을 무시·미이행        | 학습시스템 신뢰도 낮음, 자기조절력 약화         |
| **⑤ 퇴원·중단위험형**      | 장기 결석, 로그 공백, 포모도르 미작성 누적        | 학습관리 의존도 약화, 이탈 가능성 높음         |
| **⑥ 고성취 지속형(긍정사례)** | 데이터가 꾸준하고 학습밀도 높은 학생             | 최적 루틴 분석을 통한 모범 패턴 도출 필요       |

### **2️⃣ 각 상황별 맞춤형 복합질문 예시 (에이전트에게 요청할 질문)**

| 상황              | 에이전트에 던질 복합질문 예시                                                                              |
| --------------- | --------------------------------------------------------------------------------------------- |
| **① 활동데이터 희박형** | “최근 2주간 학습데이터 밀도가 30% 이하로 떨어진 학생의 주요 누락 항목과 그 원인(시간관리, 심리요인, 환경요인)을 추정하고, 개선 피드백 시나리오를 생성해줘.” |
| **② 데이터 불균형형**  | “개념노트와 문제노트 생성 비율이 비정상적으로 차이 나는 학생의 학습루틴 불균형 원인을 분석하고, ‘균형 회복’ 중심의 피드백을 설계해줘.”                |
| **③ 패턴불안정형**    | “최근 4주간 포모도르 지속시간과 활동결과 밀도의 변동폭이 큰 학생들의 공통 패턴을 찾아내고, 감정리듬과 학습리듬을 동기화시키는 개입전략을 제시해줘.”          |
| **④ 자동화 저항형**   | “시스템 피드백 실행률이 50% 이하인 학생의 피드백 저항 이유를 유형화하고, ‘내적 동기 재활성화’ 중심의 맞춤 피드백 문구 세트를 만들어줘.”             |
| **⑤ 퇴원·중단위험형**  | “최근 1개월간 로그 공백·포모도르 미작성·귀가검사 누락이 동시에 나타난 학생의 심리의존도 등급을 재평가하고, ‘재진입 유도’ 메시지 시나리오를 설계해줘.”       |
| **⑥ 고성취 지속형**   | “지속적으로 높은 데이터 일관성을 보이는 학생들의 공통 루틴 패턴을 도출하고, 이를 ‘학습관리 우수사례 피드백’으로 다른 학생에게 전파할 수 있게 구성해줘.”      |
