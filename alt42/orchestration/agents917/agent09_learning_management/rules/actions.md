# Agent 09 - Learning Management Actions

## 수행 가능한 액션 목록

### 종합 분석
- `analyze_learning_patterns`: 학습 패턴 분석
- `assess_attendance_pattern`: 출결 패턴 평가
- `evaluate_goal_achievement`: 목표 달성 평가

### 루틴 설계
- `redesign_routine`: 루틴 재설계
- `adjust_time_slots`: 시간대 조정
- `optimize_pomodoro`: 포모도로 최적화

### 보완 전략
- `create_review_routine`: 복습 루틴 생성
- `suggest_time_allocation`: 시간 배분 제안
- `recommend_weakness_focus`: 약점 집중 권장
