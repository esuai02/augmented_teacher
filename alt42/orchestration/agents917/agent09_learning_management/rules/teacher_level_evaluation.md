## 🎯 영역별 상세 평가

### 1️⃣ 데이터 수집 및 분석 (12/20점, C)

#### ✅ 강점
- **5가지 핵심 지표 수집**: 출결, 목표, 포모도로, 오답노트, 시험 패턴
  - 현직 선생님도 주요 관찰 지표로 사용하는 항목들 포함
- **기본 통계 산출**: 달성률, 완성률, 평균값 등 기본 수치 제공
- **시간대별 패턴 분석**: 요일별, 시간대별 패턴 식별

#### ⚠️ 부족한 점

1. **데이터 깊이 분석 부재**
   - 선생님: "출석률 80%인데 월요일만 빠지면 자주 결석하는 패턴이야"
   - 현재: 단순 출석률만 표시, 패턴 심층 분석 없음
   - **필요**: 연속 결석 패턴, 요일별 일관성, 계절/시기별 변화 추이

2. **데이터 간 상관관계 분석 부재**
   - 선생님: "출석률 떨어지면 포모도로도 같이 안 돼. 심리적 의존도 낮아진 거야"
   - 현재: 각 지표 독립 분석, 상관관계 없음
   - **필요**: 출결↔포모도로, 목표↔시험점수, 오답노트↔학습시간 등 상관 분석

3. **정성적 데이터 수집 부재**
   - 선생님: "학생이 요즘 집중이 안 돼. 시험 전 불안해하는 것 같아"
   - 현재: 정량적 데이터만 수집
   - **필요**: 학생 설문, 선생님 관찰 체크리스트, 텍스트 입력 (에이전트 요청 시)

4. **데이터 신뢰도 평가 부재**
   - 선생님: "데이터는 많은데 실제 학습 효과와 연결 안 돼. 가짜 노력이야"
   - 현재: 데이터만 수집, 신뢰도 검증 없음
   - **필요**: 데이터 밀도, 일관성, 효과성 검증 로직

**점수: 12/20점** (60%)

### 2️⃣ 학습 패턴 진단 (8/20점, D+)

#### ✅ 부분적 반영
- **기본 휴리스틱**: mission.md에 4가지 규칙 언급
  - 출결 저하 + 포모도로 불완성 → 루틴 재설계
  - 목표 달성률 낮음 + 기간 과다 → 세분화
  - 오답 개념오해 ↑ → 개념 재학습
  - 테스트 점수 저조 → 복습 계획

#### ❌ 현저히 부족한 점

1. **패턴 유형 분류 체계 부재**
   - 선생님: "이 학생은 활동데이터 희박형, 저 학생은 데이터 불균형형"
   - 현재: questions.md에 6가지 유형 언급만 있고 실제 룰 없음
   - **필요**: 각 유형별 진단 룰, 자동 분류 로직

2. **위험도 등급 체계 부재**
   - 선생님: "위험도 1단계: 주의, 2단계: 경고, 3단계: 긴급 개입 필요"
   - 현재: risk_level 'low'만 표시, 등급 체계 없음
   - **필요**: 다단계 위험도 산정, 단계별 대응 전략

3. **패턴 변화 추적 부재**
   - 선생님: "최근 2주간 패턴이 급격히 변했어. 외부 요인 있나봐"
   - 현재: 현재 상태만 분석, 변화 추이 없음
   - **필요**: 시계열 분석, 변화율 계산, 이상 징후 탐지

4. **원인 분석 로직 부재**
   - 선생님: "출석률 떨어진 이유: 시간관리 문제? 동기 저하? 환경 변화?"
   - 현재: 증상만 파악, 원인 분석 없음
   - **필요**: 다차원 원인 분석, 근본 원인 도출, 우선순위 평가

**점수: 8/20점** (40%)

## 🔧 개선 우선순위 제안
