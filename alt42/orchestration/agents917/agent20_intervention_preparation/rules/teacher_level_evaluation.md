## 🎯 영역별 상세 평가

### 1️⃣ 학생 상태 진단 및 분석 (12/20점, C)

#### ✅ 강점
- **기본 준비 상태 확인 (R1)**: 체크리스트, 리소스, 메시지 준비 상태 확인
- **학생 활동 패턴 분석 (R3c)**: 활동 시간대 분석을 통한 타이밍 추천
- **긴급도 기반 판단 (R3d)**: 긴급 상황 감지 및 즉시 개입 추천

#### ❌ 현저히 부족한 점

**수학 학습 특화 진단 부재:**

1. **학습 정지 원인 분석 미흡**
   - 현직 선생님: "계산 실수인지, 개념 이해 부족인지, 문제 해석 오류인지 구분 필요"
   - 현재: 데이터 트리거만 확인 (침착도 하락, 필기 지연 등), 원인 분석 없음
   - 필요: 수학 학습 맥락에서 정지 원인 세분화 룰 추가

2. **오개념 여부 파악 부재**
   - 현직 선생님: "이 학생은 함수와 방정식을 혼동하는 것 같아" → 오개념 교정 자료 준비
   - 현재: 개입 목적에 오개념 교정 언급 없음
   - 필요: 오개념 진단 및 교정 개입 룰 추가

3. **학습 수준별 상태 진단 부족**
   - 현직 선생님: "하위권 학생은 개념 재학습, 중위권은 유형 연습, 상위권은 심화 문제"
   - 현재: 학생 수준(L) 기반 차별화 전략 없음
   - 필요: 수준별 개입 전략 룰 추가

4. **단원별 취약점 분석 부재**
   - 현직 선생님: "함수 단원에서 막히는구나, 도형은 잘 하는데" → 단원별 맞춤 개입
   - 현재: 단원별 취약점 정보 활용 룰 없음
   - 필요: 단원별 취약점 기반 개입 자료 추천 룰 추가

**점수: 12/20점** (60%)
