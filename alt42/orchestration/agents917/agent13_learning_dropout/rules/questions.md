Agent 13 – Learning Dropout Mission은 “학습 흐름이 끊기기 전·중·후의 모든 국면에서 조기 탐지와 회복을 설계하는 총체적 접근”이야.
그래서 이 에이전트용 포괄형 질문은 아래처럼 설계할 수 있어 👇

💬 포괄형 질문 1

“이 학생의 학습 몰입 흐름이 어느 시점에서 끊기고 있는지, 그 원인을 어떻게 조기 탐지·예방해야 할까?”
→ 최근 포모도르 패턴, 감정 변화율, 필기량 추세, 목표 입력 지연, 루틴 붕괴 시점 등을 통합 분석하여
이탈 전조(지연, 피로, 권태, 산만 등)를 예측하고 즉각적 개입 타이밍·메시지·행동 루틴을 제안.

💬 포괄형 질문 2

“이 학생의 집중 회복력을 높이기 위해 어떤 루틴과 환경 변수를 조정해야 할까?”
→ 복귀 성공률, 이탈 후 복귀 지연시간, 감정 회복속도, 피로도 누적지수, 루틴 유지 안정도를 종합 분석하여
‘복귀 루프(이탈→개입→복귀→유지)’ 성공 확률을 높이는 맞춤형 루틴 보정 전략(시간·강도·보상형)을 설계.

💬 포괄형 질문 3

“이 학생이 장기적으로 이탈이 줄고, 자기조절 루틴이 자리잡도록 하기 위해 지금 어떤 피드백 체계를 구축해야 할까?”
→ 주간 이탈 트렌드, 감정·휴식 데이터, 교사 개입 효과, 피드백 내성도, 복귀 루프 완결률 등을 기반으로
단기 개입(즉시 리포커스)과 중장기 개입(자기조절 루틴·정서 회복 루틴)의 병행 설계를 제안.

## 🧩 1️⃣ 이 에이전트를 필요로 하는 주요 상황

| 상황 코드   | 상황 설명                           | 발생 시점   | 관련 데이터 소스                   |
| ------- | ------------------------------- | ------- | --------------------------- |
| **S1**  | 학생이 로그인했지만 5분 이상 아무 활동 없음       | 세션 초반   | timecreated, tlaststroke    |
| **S2**  | 포모도로 중간에 갑자기 학습 중단(탭 이동·무입력 지속) | 학습 중    | npomodoro, ninactive        |
| **S3**  | 집중 루틴이 유지되지 않아 매일 같은 시점에 중단 반복  | 반복 패턴   | nlazy, pmresult             |
| **S4**  | 개념노트나 문제풀이노트에서 필기량 급감           | 중간세션    | nstroke, usedtime           |
| **S5**  | 학습감정 에이전트에서 ‘권태/불안’ 감정 감지       | 감정 연동   | emotion_state, decay_factor |
| **S6**  | 목표 입력을 미루거나 생략함                 | 세션 시작   | goal_input_delay            |
| **S7**  | 하루 2회 이상 High 등급 발생             | 지속 위험   | risk_grade_history          |
| **S8**  | 수업 종료 후 복귀가 느림(재접속까지 3시간↑)      | 세션 종료 후 | treentry                    |
| **S9**  | 주간 패턴에서 특정 요일 집중력 급락            | 주간 분석   | ninactive_trend             |
| **S10** | 학습 전환 실패 (개념→문제, 문제→오답 복귀 실패)   | 루틴 전환 시 | transition_fail_flag        |

## 🧠 2️⃣ 각 상황별 맞춤형 복합질문 예시

| 상황                    | 선생님/시스템이 Agent 13에 던질 복합질문                                                                            |
| --------------------- | ----------------------------------------------------------------------------------------------------- |
| **S1: 로그인 후 무활동**     | “학생이 로그인 후 5분째 입력이 없는데, 최근 루틴 데이터와 감정지수를 기반으로 이탈 위험을 평가하고, 바로 실행 가능한 리포커스 메시지를 추천해줘.”                 |
| **S2: 포모도로 중 중단**     | “현재 포모도로 도중 입력이 끊겼어. 이전 포모도로 패턴(npomodoro, ninactive, emotion_delta)을 종합해 중단 원인을 추정하고 재진입 메시지를 생성해줘.” |
| **S3: 반복 시점 중단**      | “학생이 매일 같은 시점(예: 25분 이후)에 멈춰. 3일간의 루틴 데이터로 이탈 트리거를 식별하고, 루틴 보정 제안을 줘.”                                |
| **S4: 필기 급감**         | “이번 세션에서 필기량이 급격히 줄었어. 이전 개념노트·문제노트 패턴과 비교해서 몰입도 저하 원인을 진단하고, 쉬운 승리 과제 하나 추천해줘.”                      |
| **S5: 감정 불안/권태 감지**   | “학습감정 에이전트에서 불안/권태가 감지됐어. 이탈 위험도와 연계해서 감정 안정용 인터벤션 메시지와 리포커스 루틴(5분 활동)을 제시해줘.”                        |
| **S6: 목표 입력 지연**      | “오늘 로그인 후 목표 입력이 누락됐어. 이전 목표-루틴 연계율을 참고해 시작 장벽의 원인(피로/의욕저하/불확실성)을 구분하고, 맞춤 시작 유도 문장을 만들어줘.”           |
| **S7: High 등급 2회 이상** | “최근 2일 연속 High 위험이야. 감정·루틴·휴식 데이터를 결합해서 보호자 알림 메시지 초안과 함께 장기 보정 전략을 제시해줘.”                            |
| **S8: 복귀 지연**         | “이탈 후 복귀까지 시간이 오래 걸렸어. treentry와 ninactive 히스토리를 기준으로 자기조절 루틴 수준을 평가하고, 다음 세션의 ‘복귀 강화’ 시나리오를 추천해줘.”   |
| **S9: 요일별 집중력 급락**    | “이번 주 목요일마다 ninactive가 급증하고 npomodoro가 급감해. 학습 감정 및 일정 변수를 분석해 그날만의 보정 전략을 제시해줘.”                     |
| **S10: 학습 전환 실패**     | “문제풀이 후 오답노트로 전환하지 못했어. 전환 실패율, emotion_delta, 루틴 데이터를 기반으로 전환 저항 원인을 구분하고 개입 메시지를 설계해줘.”             |
