## 🔧 개선 우선순위 제안

### 🔴 긴급 (현직 수준 도달 필수)

1. **수학 단원별 이탈 패턴 분석 룰 추가**
   - 단원 정보 수집 (학생설문/체크리스트)
   - 단원별 이탈 빈도 집계
   - 어려운 단원에서 이탈 증가 패턴 분석
   - 예: `S1_R1`: 단원별 이탈 패턴 분석
   - 예상 점수 향상: +5점

2. **문제 난이도와 이탈 연계 분석 룰 추가**
   - 문제 난이도 수집 (학생설문/체크리스트)
   - 난이도별 이탈 빈도 분석
   - 심화 문제에서 이탈 시 기본형 전환 제안
   - 예: `S2_R1`: 난이도별 이탈 분석 및 전환 제안
   - 예상 점수 향상: +4점

3. **학원 수업 이해도와 이탈 연계 룰 추가**
   - 학원 수업 이해도 수집 (학생설문)
   - 이해도 낮을 때 이탈 증가 패턴 분석
   - 이해 못한 내용 복습 우선 제안
   - 예: `S3_R1`: 학원 수업 이해도 기반 이탈 분석
   - 예상 점수 향상: +6점

### 🟡 중요 (수준 향상 필요)

4. **학습 단계별 이탈 패턴 분석 룰 추가**
   - 학습 단계(개념/유형/심화/기출) 수집
