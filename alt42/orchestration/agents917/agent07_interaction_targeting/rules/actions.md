# Agent 07 - Interaction Targeting Actions

## 수행 가능한 액션 목록

### 타겟 선정
- `select_interaction_target`: 개입 타겟 선정
- `determine_urgency`: 긴급도 판단
- `assess_readiness`: 준비도 평가

### 활동 배치
- `schedule_by_time`: 시간대별 활동 배치
- `match_difficulty_to_state`: 상태별 난이도 매칭
- `recommend_tools`: 도구 추천

### 최적화
- `optimize_interaction_timing`: 개입 타이밍 최적화
- `balance_workload`: 작업량 균형 조정
