# 학원 과제와 학교 활동 통합 조정 룰

## 목적
학원 과제와 학교 활동을 통합적으로 고려하여 우선순위를 설정하고 시간을 배분하여 효율적인 활동 조정을 수행합니다.

---

## 데이터 소스
- `academy_school_integration.yaml`: 학원/학교 활동 우선순위 및 시간 배분 전략
- 학생 설문: 학원 과제 제출 일정, 우선순위 선호도
- 선생님 체크리스트: 학원/학교 활동 시간 배분 비율

---

## 활동 유형 구분

### 학원 과제 (academy_homework)
- **기본 우선순위**: 0.8 (높음)
- **일반적 제출 일정**: 다음 날
- **난이도 배수**: 1.2 (일반적으로 학교보다 어려움)
- **시간 비중**: 60%

### 학교 숙제 (school_homework)
- **기본 우선순위**: 0.6 (중간)
- **일반적 제출 일정**: 이번 주
- **난이도 배수**: 1.0 (기준)
- **시간 비중**: 40%

---

## 우선순위 계산 알고리즘

### 통합 우선순위 점수 계산

```
통합 우선순위 점수 = (
  제출 일정 점수 × 0.4 +      # 제출 일정 40%
  활동 유형 점수 × 0.3 +      # 학원/학교 구분 30%
  난이도 점수 × 0.2 +         # 난이도 20%
  시험 관련도 × 0.1           # 시험 관련도 10%
)
```

### 제출 일정 점수
- 당일 제출: +0.3
- 다음 날 제출: +0.2
- 이번 주 제출: +0.1
- 유연함: +0.0

### 활동 유형 점수
- 학원 과제: +0.2
- 학교 숙제: +0.1

### 난이도 점수
- 높음: +0.1
- 중간: +0.05
- 낮음: +0.0

### 시험 관련도 점수
- 높음: +0.1
- 중간: +0.05
- 낮음: +0.0

## 시간 배분 전략

### 전략 1: 균형 배분 (Balanced)
- **조건**: 시간 충분할 때
- **비율**: 학원 60%, 학교 40%
- **적용**: 일반적인 상황

### 전략 2: 학원 과제 우선 (Academy Priority)
- **조건**: 학원 과제 제출 일정 임박 (다음 날)
- **비율**: 학원 70%, 학교 30%
- **적용**: 학원 과제가 급할 때

### 전략 3: 학교 숙제 우선 (School Priority)
- **조건**: 학교 시험 임박 또는 학원 과제 없음
- **비율**: 학원 40%, 학교 60%
- **적용**: 학교 시험이 가까울 때

## 활동 조정 규칙

### Rule 1: 제출 일정 기반 우선순위
```
if 학원_제출일 == "다음날" and 학교_제출일 == "이번주":
    우선순위 = 학원_과제
elif 학교_제출일 == "다음날" and 학원_제출일 == "이번주":
    우선순위 = 학교_숙제
else:
    우선순위 = 기본_우선순위_사용
```

### Rule 2: 난이도 기반 시간 조정
```
학원_예상시간 = 기본_시간 × 학원_난이도_배수(1.2)
학교_예상시간 = 기본_시간 × 학교_난이도_배수(1.0)
```

### Rule 3: 시간 부족 시 조정
```
if 가용시간 < 필요시간:
    if 학원_제출일 == "다음날":
        학교_숙제_축소 또는 비급한_학교_활동_연기
    else:
        최저_우선순위_활동_축소
```

## 활동 조정 프로세스

### Step 1: 활동 유형 구분
1. 각 잔여 활동의 유형 확인 (학원 과제 / 학교 숙제)
2. 제출 일정 확인
3. 난이도 평가

### Step 2: 우선순위 계산
1. 제출 일정 점수 계산
2. 활동 유형 점수 계산
3. 난이도 점수 계산
4. 시험 관련도 점수 계산
5. 통합 우선순위 점수 계산

### Step 3: 시간 배분 전략 선택
1. 제출 일정 분석
2. 적절한 시간 배분 전략 선택
3. 학원/학교 활동 시간 배분

### Step 4: 활동 계획 수립
1. 우선순위 상위 활동부터 시간 배분
2. 시간 초과 시 하위 우선순위 활동 조정
3. 최종 활동 계획 생성

## 예시 시나리오

### 시나리오 1: 학원 과제 제출 임박
**상황**: 
- 가용 시간: 2시간
- 학원 과제: 함수 문제 10개 (제출일: 내일, 난이도: 높음)
- 학교 숙제: 통계 문제 5개 (제출일: 이번 주, 난이도: 중간)

**우선순위 계산**:
- 학원 과제: 제출일(다음날, +0.2) × 0.4 + 활동유형(학원, +0.2) × 0.3 + 난이도(높음, +0.1) × 0.2 = 0.16
- 학교 숙제: 제출일(이번주, +0.1) × 0.4 + 활동유형(학교, +0.1) × 0.3 + 난이도(중간, +0.05) × 0.2 = 0.08

**시간 배분**: 학원 우선 전략 (학원 70%, 학교 30%)
- 학원 과제: 2시간 × 0.7 = 1.4시간
- 학교 숙제: 2시간 × 0.3 = 0.6시간

**최종 계획**: 학원 과제 우선 완료, 학교 숙제는 남은 시간에 진행

### 시나리오 2: 학교 시험 임박
**상황**:
- 가용 시간: 3시간
- 학원 과제: 도형 문제 8개 (제출일: 이번 주, 난이도: 중간)
- 학교 숙제: 함수 문제 12개 (제출일: 이번 주, 난이도: 높음, 시험 관련도: 높음)

**우선순위 계산**:
- 학원 과제: 제출일(이번주, +0.1) × 0.4 + 활동유형(학원, +0.2) × 0.3 + 난이도(중간, +0.05) × 0.2 = 0.12
- 학교 숙제: 제출일(이번주, +0.1) × 0.4 + 활동유형(학교, +0.1) × 0.3 + 난이도(높음, +0.1) × 0.2 + 시험관련도(높음, +0.1) × 0.1 = 0.11

**시간 배분**: 학교 우선 전략 (학원 40%, 학교 60%)
- 학원 과제: 3시간 × 0.4 = 1.2시간
- 학교 숙제: 3시간 × 0.6 = 1.8시간

**최종 계획**: 학교 숙제 우선 완료, 학원 과제는 남은 시간에 진행
