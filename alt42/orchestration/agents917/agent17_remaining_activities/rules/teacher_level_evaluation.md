## 🎯 영역별 상세 평가

### 1️⃣ 활동 조정 전략 다양성 (18/20점, A-)

#### ✅ 강점
- **5가지 조정 전략 구비**: 질문하기, 도제학습 전환, 활동축소, 하이튜터링, 징검다리 활동 설정
  - 현직 선생님도 상황별로 다양한 전략을 사용하는 것을 잘 반영
- **전략 선택 의사결정 트리**: 진행 상태, 학생 특성, 시간 제약 기반 선택 로직 우수
- **복합 전략 지원**: 여러 전략을 조합하여 사용 가능

#### ⚠️ 부족한 점
- **수학 단원별 전략 차별화 부재**: 
  - 선생님: "함수 단원은 개념이 중요해서 질문하기, 통계는 문제 풀이 중심으로 활동축소"
  - 현재: 일반적 전략만 있고 단원별 특화 전략 없음

**점수: 18/20점** (90%)

---

### 2️⃣ 병목 식별 및 분할 (13/15점, B+)

#### ✅ 강점
- **병목 유형 구분**: 시간 병목, 난이도 병목, 의존성 병목, 정서적 병목 잘 구분
- **병목 점수 계산식**: 다차원적 병목 평가 우수
- **분할 전략**: 시간/난이도/의존성 기반 분할 전략 구비

#### ⚠️ 부족한 점
- **수학 단원별 병목 패턴 DB 부재**:
  - 선생님: "함수 단원은 개념 이해 병목이 많고, 도형은 증명 병목이 많아"
  - 현재: 일반적 병목만 식별, 단원별 특화 병목 패턴 미반영

**점수: 13/15점** (87%)

### 3️⃣ 수학 단원별 우선순위 설정 (3/15점, F)

#### ❌ 현저히 부족

**현재 상태:**
- 일반적 우선순위 계산식만 존재
- `활동 우선순위 점수 = (목표 기여도 × 0.4) + (취약도 × 0.3) + (시험 중요도 × 0.2) + (완료 용이성 × 0.1)`
- 단원별 난이도, 출제 비중 기반 우선순위 설정 부재

**현직 선생님의 실제 판단:**

1. **단원별 출제 비중 기반 우선순위**
   - 선생님: "함수 단원은 시험에서 30% 출제되니까 우선순위 높아야 해"
   - 현재: `시험 중요도`만 있고, 단원별 출제 비중 DB 없음

2. **단원별 난이도 기반 우선순위**
   - 선생님: "어려운 단원(함수)은 시간 여유 있을 때, 쉬운 단원(통계)은 시간 부족할 때"
   - 현재: 단원별 난이도 정보 없음

3. **단원별 학습 시간 기반 우선순위**
   - 선생님: "함수 단원은 평균 6시간, 통계는 3시간. 시간 부족하면 통계부터"
   - 현재: 단원별 평균 학습 시간 DB 없음

**필요한 데이터 (학생설문/선생님 체크리스트/텍스트 입력):**

```
선생님 텍스트 입력 예시:
- "함수 단원: 출제 비중 30%, 난이도 8/10, 평균 학습 시간 6시간 → 우선순위 높음"
- "통계 단원: 출제 비중 15%, 난이도 4/10, 평균 학습 시간 3시간 → 우선순위 낮음"
- "도형 단원: 출제 비중 25%, 난이도 7/10, 평균 학습 시간 5시간 → 우선순위 중간"

학생 설문 예시:
- "어떤 단원이 시험에서 많이 나오나요?"
- "어떤 단원이 가장 어렵나요?"
```

**점수: 3/15점** (20%)

### 4️⃣ 학원 과제와 학교 활동 통합 (5/15점, D)

#### ❌ 현저히 부족

**현재 상태:**
- 학교 활동만 고려
- 학원 과제 우선순위 반영 로직 부재

**현직 선생님의 실제 판단:**

1. **학원 과제 우선순위 반영**
   - 선생님: "학원 과제는 보통 내일 제출이니까 학교 숙제보다 우선순위 높아"
   - 현재: 학원 과제 구분 없음

2. **시간 배분 조정**
   - 선생님: "학원 과제 2시간, 학교 숙제 1시간 남았으면 학원부터 해야 해"
   - 현재: 활동 유형별 시간 배분만 있고, 학원/학교 구분 없음

3. **과제 난이도 통합 고려**
   - 선생님: "학원 문제가 더 어려우면 더 시간 필요하고, 학교는 기본형이면 빠르게"
   - 현재: 활동 난이도만 있고, 학원/학교별 난이도 차이 고려 없음

**필요한 데이터 (학생설문/선생님 체크리스트/텍스트 입력):**

```
학생 설문:
- "학원 과제와 학교 활동 중 어떤 것을 우선순위로 하시나요?"
- "학원 과제 제출 일정은 어떻게 되나요?"

선생님 체크리스트:
- [ ] 학원 과제 우선순위 반영 (예: 학원 과제 > 학교 숙제)
- [ ] 학원 과제 제출 일정 고려
- [ ] 학원/학교 활동 시간 배분 비율 설정

선생님 텍스트 입력:
- "학원 과제는 보통 내일 제출이므로 학교 숙제보다 우선순위 높음"
- "학원 과제와 학교 활동 시간 배분: 60% 학원, 40% 학교"
```

**점수: 5/15점** (33%)

### 5️⃣ 수학 교재별 활동 완료 전략 (7/15점, D+)

#### ⚠️ 부족한 점

**현재 상태:**
- 시간 기반 활동 완료 전략만 존재
- 교재별 특성 반영 부재

**현직 선생님의 실제 판단:**

1. **교재별 단계별 완료 전략**
   - 선생님: "쎈은 A단계 완료 후 B단계, 개념원리는 페이지 순서대로"
   - 현재: 교재별 단계 구분 없음

2. **교재별 문제 수 및 시간 산정**
   - 선생님: "쎈 A단계는 문제당 3분, B단계는 4.5분, C단계는 6분"
   - 현재: 평균 시간만 있고, 교재별/단계별 시간 차이 없음

3. **교재별 정답률 목표**
   - 선생님: "쎈 A단계는 80% 목표, B단계는 70%, C단계는 60%"
   - 현재: 정답률 목표 설정 없음

**필요한 데이터 (학생설문/선생님 체크리스트/텍스트 입력):**

```
선생님 체크리스트:
- [ ] 쎈: A단계 완료 후 B단계 진행
- [ ] 개념원리: 개념 완료 후 쎈 진행
- [ ] 라이트쎈: 기본형 완료 후 유형 진행

선생님 텍스트 입력:
- "쎈 A단계: 문제당 평균 3분, 목표 정답률 80%"
- "쎈 B단계: 문제당 평균 4.5분, 목표 정답률 70%"
- "쎈 C단계: 문제당 평균 6분, 목표 정답률 60%"
- "개념원리: 페이지당 평균 20분, 개념 이해 중심"
```

**점수: 7/15점** (47%)

## 🔧 개선 우선순위 제안

1. **수학 단원별 활동 우선순위 설정 룰 추가** (-12점 → +12점)
   - 단원별 난이도, 출제 비중, 평균 학습 시간 DB 구축
   - 우선순위 계산식에 단원 정보 반영
   - 설문/체크리스트: 선생님 텍스트 입력
   - 예상 점수 향상: 3 → 15점

2. **학원 과제와 학교 활동 통합 조정 로직 추가** (-10점 → +10점)
   - 학원 과제 구분 및 우선순위 설정
   - 학원/학교 활동 시간 배분 조정
   - 설문 항목: "학원 과제와 학교 활동을 어떻게 우선순위화하시나요?"
   - 예상 점수 향상: 5 → 15점

3. **단원 연계성 기반 활동 순서 조정 룰 추가** (-6점 → +6점)
   - 단원 선후관계 기반 순서 조정
   - 연계 단원 동시 학습 전략
   - 설문/체크리스트: 선생님 텍스트 입력
   - 예상 점수 향상: 2 → 10점

### 🟡 중요 (수준 향상 필요)

4. **수학 교재별 활동 완료 전략 룰 추가** (-8점 → +8점)
   - 교재별(쎈 A/B/C 등) 단계별 완료 전략
   - 교재별 문제 수 및 시간 산정
   - 설문/체크리스트: 선생님 체크리스트
