## 🎯 주요 개선 사항

### 1. 수학 단원별 활동 우선순위 설정 (+12점)

#### 개선 내용
- **신규 파일**: `math_unit_priority.yaml`, `unit_priority_rules.md`
- 단원별 난이도, 출제 비중, 학습 시간 데이터베이스 구축
- 시간 제약별 우선순위 조정 알고리즘 추가
- 시험 일정 임박 시 출제 비중 높은 단원 우선 배정

#### 주요 기능
- **우선순위 계산식**: 출제 비중(40%) + 난이도 역수(20%) + 완료 용이성(20%) + 병목 위험도 역수(20%)
- **시간 제약별 조정**: 시간 충분 시 어려운 단원 우선, 시간 부족 시 쉬운 단원 우선
- **활동 조정 프로세스**: 단원 정보 확인 → 시간 제약 분석 → 우선순위 조정 → 활동 계획 수립

#### 데이터 수집 방법
- **학생 설문**: "어떤 단원이 시험에서 많이 나오나요?", "어떤 단원이 가장 어렵나요?"
- **선생님 체크리스트**: 단원별 출제 비중, 난이도, 평균 학습 시간
- **선생님 텍스트 입력**: "함수 단원: 출제 비중 30%, 난이도 8/10, 평균 학습 시간 6시간 → 우선순위 높음"

- 제출 일정 기반 시간 배분 전략 추가
- 난이도 기반 시간 조정 로직 추가

#### 주요 기능
- **활동 유형 구분**: 학원 과제(우선순위 0.8), 학교 숙제(우선순위 0.6)
- **우선순위 계산식**: 제출 일정(40%) + 활동 유형(30%) + 난이도(20%) + 시험 관련도(10%)
- **시간 배분 전략**: 균형 배분(60:40), 학원 우선(70:30), 학교 우선(40:60)

#### 데이터 수집 방법
- **학생 설문**: "학원 과제와 학교 활동 중 어떤 것을 우선순위로 하시나요?", "학원 과제 제출 일정은?"
- **선생님 체크리스트**: 학원 과제 우선순위 반영, 시간 배분 비율 설정
- **선생님 텍스트 입력**: "학원 과제는 보통 내일 제출이므로 학교 숙제보다 우선순위 높음"

### 3. 수학 교재별 활동 완료 전략 (+8점)

#### 개선 내용
- **신규 파일**: `textbook_strategies.yaml`, `textbook_completion_strategy.md`
- 교재별(쎈 A/B/C, 개념원리 등) 단계별 완료 전략 구축
- 교재별 시간 산정 알고리즘 추가
- 시간 제약별 단계 축소 전략 추가

#### 주요 기능
- **쎈 교재**: A단계(기본형, 80% 목표) → B단계(유형, 70% 목표) → C단계(심화, 60% 목표)
- **개념원리**: 개념 완료 후 문제집(쎈 등) 진행 권장
- **시간 산정**: 문제집은 문제당 평균 시간, 개념서는 페이지당 평균 시간

#### 데이터 수집 방법
- **학생 설문**: "사용 중인 수학 교재는 무엇인가요?", "교재의 어느 단계까지 완료했나요?"
- **선생님 체크리스트**: "쎈: A단계 완료 후 B단계 진행", "개념원리: 개념 완료 후 쎈 진행"
- **선생님 텍스트 입력**: "쎈 A단계: 문제당 평균 3분, 목표 정답률 80%"
