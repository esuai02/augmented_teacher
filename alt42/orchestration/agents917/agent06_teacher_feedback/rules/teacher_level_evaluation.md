## 🎯 영역별 상세 평가

### 1️⃣ 수학 교과 특화 피드백 (13/25점, D+)

#### ✅ 강점
- **기본 피드백 구조**: 칭찬/개선 구분, 정서 톤 설정 등 기본 구조는 완비
- **시험 대비 피드백**: 시험 전략, 금지 패턴 등 시험 맥락 피드백 존재

#### ❌ 현저히 부족한 점

**수학 특화 요소 누락:**

1. **수학 오류 유형별 피드백 템플릿 부재** (-6점)
   - 현직 선생님: "계산 실수는 반복 연습 강조, 개념 오류는 개념 재학습 필요"
   - 현재: R3e에서 오류 패턴 감지만 있고, 수학 특화 피드백 템플릿 없음
   - **필요 데이터**: 선생님 체크리스트 입력
     - "계산 실수: 풀이 과정 확인, 반복 연습 권장"
     - "개념 오류: 개념 설명 재제시, 예제 단계별 설명"
     - "적용 오류: 유형별 문제 다각도 제시"

2. **단원별 피드백 특성 미반영** (-4점)
   - 현직 선생님: "함수 단원은 개념 이해가 중요, 도형은 시각화 강조"
   - 현재: 단원별 차별화 없음
   - **필요 데이터**: 선생님 텍스트 입력
     - "함수 단원: 함수식 의미, 그래프 이해 강조"
     - "도형 단원: 시각적 이해, 도형 그리기 연습 강조"
     - "통계 단원: 데이터 해석, 표/그래프 읽기 강조"

3. **문제 유형별 피드백 부재** (-2점)
   - 현직 선생님: "기본형은 정확성, 유형은 패턴 인식, 심화는 사고력"
   - 현재: 문제 유형 구분 없음
   - **필요 데이터**: 선생님 체크리스트
     - 기본형: 정확한 계산, 단계별 풀이 확인
     - 유형: 유형별 패턴 인식, 풀이 전략 강화
     - 심화: 사고 과정, 다각도 접근법

**점수: 13/25점** (52%)

### 2️⃣ 학생 관찰 및 진단 능력 (16/20점, B)

#### ✅ 강점
- **학생 변화 감지 (S1)**: 집중도, 감정, 태도, 피로, 동기, 환경 등 포괄적 감지
- **학습 부진 패턴 분석 (S4)**: 성적 저하, 질문 감소, 자신감 저하 등 종합 분석
- **맥락 해석 요청**: 교사에게 맥락 해석을 요청하는 구조 우수

#### ⚠️ 부족한 점

1. **수학 학습 특화 관찰 부재** (-2점)
   - 현직 선생님: "문제 풀이 속도, 풀이 과정, 오답 패턴 등을 종합 관찰"
   - 현재: 일반적 학습 태도만 관찰, 수학 문제 풀이 특화 관찰 없음
   - **필요 데이터**: 학생 설문 + 선생님 체크리스트
     - "문제를 풀 때 어떤 단계에서 막히나요?" (이해 → 풀이 → 계산 → 검증)
     - "오답을 확인할 때 주로 어떤 부분이 틀렸나요?" (개념/계산/적용)

2. **단원별 취약점 정밀 진단 부족** (-2점)
   - 현직 선생님: "함수에서 함수식 이해는 되는데 그래프 해석이 약함"
   - 현재: 단원별 취약점만 파악, 세부 역량 구분 없음
   - **필요 데이터**: 선생님 텍스트 입력
     - "함수 단원: 함수식 이해(O), 그래프 해석(X), 응용 문제(X)"

**점수: 16/20점** (80%)

### 3️⃣ 맞춤형 피드백 생성 (14/20점, C+)

#### ✅ 강점
- **피드백 타겟 구분**: 학부모/학생별 피드백 구분 (R3a, R3c)
- **정서 톤 선택**: 격려/경고/균형/도전 선택 가능 (R3b)
- **칭찬 포인트 추출**: 강점 식별 및 강화 (R3d)

#### ⚠️ 부족한 점

1. **학생 수준별 피드백 차별화 부족** (-4점)
   - 현직 선생님: 
     - 하위권: "작은 성공 강조, 기본 개념 칭찬, 단계별 안내"
     - 중위권: "유형별 성취 강조, 오답 정리 가이드, 목표 제시"
     - 상위권: "도전 문제 제시, 사고 과정 칭찬, 창의적 접근 권장"
   - 현재: 수준별 차별화 없음
   - **필요 데이터**: 선생님 체크리스트 입력
     - "하위권 피드백 스타일: 격려 중심, 구체적 성취 강조"
     - "중위권 피드백 스타일: 균형, 목표 제시"
     - "상위권 피드백 스타일: 도전, 심화 제시"

2. **학습 단계별 피드백 부재** (-2점)
   - 현직 선생님: "개념 학습 단계 vs 유형 연습 단계 vs 심화 단계별로 피드백 다름"
   - 현재: 학습 단계 구분 없음
   - **필요 데이터**: 선생님 텍스트 입력
     - "개념 학습: 이해 확인, 예제 안내"
     - "유형 연습: 패턴 인식, 오답 분석"
     - "심화: 사고 과정, 창의적 접근"

**점수: 14/20점** (70%)

   - 문제 유형별 피드백 (기본형, 유형, 심화)
   - **데이터 수집**: 선생님 체크리스트 + 텍스트 입력
   - **예상 점수 향상**: +8점

   - 학교-학원 피드백 통합 로직
   - 다중 선생님 피드백 충돌 해결
   - **데이터 수집**: 선생님 텍스트 입력
   - **예상 점수 향상**: +6점

3. **학생 수준별 피드백 차별화 룰 추가**
   - 하위권/중위권/상위권별 피드백 전략
   - 수준별 칭찬 포인트 차별화
   - 수준별 개선 안내 차별화
   - **데이터 수집**: 선생님 체크리스트
   - **예상 점수 향상**: +4점

### 🟡 중요 (수준 향상 필요)

   - **데이터 수집**: 선생님 텍스트 입력
   - **예상 점수 향상**: +2점

## 💡 현직 선생님의 핵심 역량 vs 현재 룰

### 현직 선생님이 보유한 사전 지식 (경험 DB)

1. **수학 오류 유형별 대응 전략**
   ```
   계산 실수: 풀이 과정 확인, 반복 연습, 계산 연습 강화
   개념 오류: 개념 재설명, 예제 단계별 설명, 개념 연결
   적용 오류: 유형별 문제 다각도 제시, 패턴 인식 강화
   ```

2. **단원별 피드백 특성**
   ```
   함수: 함수식 의미, 그래프 이해, 함수 관계성 강조
   도형: 시각적 이해, 도형 그리기, 공간 인식 강조
   통계: 데이터 해석, 표/그래프 읽기, 통계적 사고 강조
   ```

3. **학생 수준별 피드백 전략**
