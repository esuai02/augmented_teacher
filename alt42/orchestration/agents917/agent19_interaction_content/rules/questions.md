💬 포괄형 질문 1

“현재 감지된 학습 상황(S1~S7)을 종합해, 어떤 상호작용 유형과 템플릿으로 대응하는 게 최적일까?”
→ 학습 이탈, 지연, 오답 반복, 침착도 저하 등 다중 신호를 분석해
가장 적합한 상호작용 유형(텍스트 전달 / 루틴 개선 / 비선형 등)을 알고리즘으로 자동 선택하고,
이에 맞는 템플릿과 룰 링크 매핑까지 포함한 상호작용 유형 선택 전략 도출.

💬 포괄형 질문 2

“선택된 상호작용 유형을 기반으로, 개인 특성(MBTI·학습 스타일·정서 상태)에 맞게 템플릿을 어떻게 패키징해야 할까?”
→ MBTI, 집중 시간대, 학습 스타일, 정서 로그를 활용해
기존 템플릿 라이브러리 재사용 가능성 평가 → 맞춤형 UI / 톤 / 링크 구성 →
HTML·CSS·JS 코드 수준의 패키징 전략 제시.

💬 포괄형 질문 3

“생성된 상호작용 컨텐츠가 실제로 학습 행동 변화에 효과적인지 어떻게 검증하고 추적해야 할까?”
→ 상호작용 결과 트래킹 지표(참여율, 클릭률, 재진입 성공률) 기반으로
효과성 분석 → 템플릿 효율 평가 → 룰 보정 및 다음 개입에 반영하는
피드백 루프 설계 제안.

## 💬 2. 상황별 Agent 19 맞춤형 복합질문 세트

각 상황은 Agent 19에게 단순히 “무엇을 만들어줘”가 아니라,
“**이 상황에서 어떤 유형의 상호작용이 가장 효과적이며, 어떤 템플릿으로 구현할까?**”라는 복합 질의로 접근해야 해.

---

### 🧩 S1. 학습 이탈 조짐 감지 후 재진입 유도

> **복합질문 예시**

```
Agent 19, 학생의 최근 이탈 징후가 감지됐어. 
이전 30분 동안 학습 집중도와 입력 이벤트가 급감했고, 감정 상태는 '권태'로 기록돼.  
지금 상태를 기반으로 “재진입을 유도할 상호작용 컨텐츠”를 제안하고,
적합한 상호작용 유형과 템플릿을 선택해줘.
룰 파일에서 연결 가능한 쉬운 승리 구간 링크도 포함시켜줘.
```

* 예상 상호작용 유형: **상호작용 컨텐츠 or 활동반려**
* 템플릿: “미니 재진입 챌린지” / “가벼운 루틴 재시작 인터페이스”

---

### 🧩 S2. 현재 위치 지연 감지

> **복합질문 예시**

```
Agent 19, 오늘의 목표 대비 진행률이 65%로 낮고,
학생은 현재 ‘지연 구간’에 있어. 
부담을 완화하면서 동기 회복을 유도할 상호작용 유형을 자동 선택해줘.  
이 상황에 맞는 타임쉬프팅 템플릿과 대체 학습 링크를 함께 구성해.
```

* 예상 상호작용 유형: **타임쉬프팅 or 루틴 개선**
* 템플릿: “페이스 조정 + 긍정 메시지” UI

---

### 🧩 S3. 휴식 루틴 이상 탐지

> **복합질문 예시**

```
Agent 19, 학생의 휴식 패턴이 비정상적으로 짧거나 누락되고 있어.
최근 피로 지표가 누적 상승 중이야.  
이 데이터를 기반으로 ‘휴식 리셋 상호작용’을 설계해줘.  
휴식 루틴 개선 템플릿을 추천하고, 적절한 시간 조정(타임쉬프팅)을 포함해.
```

* 예상 상호작용 유형: **루틴 개선 + 타임쉬프팅 복합**
* 템플릿: “집중 리듬 리셋 인터페이스”

---

### 🧩 S4. 오답 패턴 반복

> **복합질문 예시**

```
Agent 19, 학생이 동일 유형의 오답을 3회 이상 반복했어.  
Agent 11의 분석에 따르면, 개념 이해 단계에서 필기 체류시간이 짧아.  
이 원인을 개선할 ‘학습 루틴 개선형 상호작용’을 설계해줘.  
룰 파일에서 관련 개념보강 링크를 포함시켜.
```

* 예상 상호작용 유형: **루틴 개선**
* 템플릿: “오답 루프 탈출 가이드형 상호작용”

---

### 🧩 S5. 정서적 침착도 저하

> **복합질문 예시**

```
Agent 19, 학생의 침착도 점수가 지난주 대비 25% 하락했어.
문제 풀이 중 선택 오류가 빈번하고 감정 로그에 ‘조급함’이 기록됐어.  
이런 경우 감정 안정형 상호작용 컨텐츠를 만들어줘.  
MBTI가 INFP형인 학생에게 적합한 어조와 UI로 구성해.
```

* 예상 상호작용 유형: **비선형 상호작용 알고리즘**
* 템플릿: “감정-인지 리밸런싱 대화형 컨텐츠”

---

### 🧩 S6. 목표 대비 활동 불균형

> **복합질문 예시**

```
Agent 19, 학생의 목표 대비 활동 분포를 보니 
‘개념 공부’에 과도하게 집중하고 ‘문제풀이’는 미흡해.  
학습 지점을 변경하거나 새로운 활동을 제안하는 상호작용을 생성해줘.  
룰 파일에서 대체 활동 링크를 불러와 포함시켜.
```

* 예상 상호작용 유형: **활동반려 또는 학습지점 변경**
* 템플릿: “활동 균형 재배치 제안형 UI”

---

### 🧩 S7. 시그너처 루틴 형성 시점

> **복합질문 예시**

```
Agent 19, 학생이 몰입 루틴(시그너처 루틴)을 형성했어.
이 루틴을 강화하고 지속하도록 돕는 상호작용 컨텐츠를 설계해줘.  
루틴 강화형 멀티턴 상호작용 템플릿을 사용하고,  
룰 파일에서 보상형 콘텐츠 링크를 연결해.
```

* 예상 상호작용 유형: **멀티턴 상호작용**
* 템플릿: “루틴 강화 챕터형 상호작용”
