# Agent 21 - 학원 수업 전후 및 실시간 개입 실행 전략 (Step 2)

## 개요

현직 수학선생님의 학원 수업 전후 개입 타이밍과 실시간 문제 풀이 개입 전략을 정의합니다.

**데이터 소스**: 선생님 설문, 체크리스트, 텍스트 입력을 통한 현직 선생님 경험 데이터

---

## 1. 학원 수업 전후 개입 실행 타이밍

### 현직 선생님의 실행 패턴

**선생님 텍스트 입력 기반 데이터:**
> "학원 수업 2시간 전 개념 요약, 30분 전 핵심 포인트 안내"
> "학원 수업 직후 30분 내 핵심 정리, 2시간 후 문제 풀이, 24시간 후 복습 확인"

### 1.1 학원 수업 전 개입 (Pre-Class Intervention)

#### 수업 2시간 전: 개념 요약 제공

**실행 조건:**
- 학원 수업 시간이 2시간 이내
- 학생이 수업 전 예습 가능 상태

**실행 방식:**
- 개념 요약 카드 제공
- 핵심 공식/정리 제공
- 수업 내용 미리보기

**목적:**
- 수업 이해도 향상
- 수업 집중도 향상

**개입 실행 룰:**
- `ACADEMY_01_pre_class_intervention` (priority: 91)

#### 수업 30분 전: 핵심 포인트 안내

**실행 조건:**
- 학원 수업 시간이 30분 이내
- 이전 개념 요약 제공 완료

**실행 방식:**
- 핵심 포인트 간단 요약
- 집중해야 할 부분 강조
- 질문 준비 안내

**목적:**
- 수업 직전 기억 상기
- 집중도 최대화

**개입 실행 룰:**
- `ACADEMY_02_pre_class_keypoints` (priority: 90)

### 1.2 학원 수업 후 개입 (Post-Class Intervention)

#### 수업 직후 30분 내: 핵심 정리

**실행 조건:**
- 학원 수업 종료 직후
- 30분 이내 실행

**실행 방식:**
- 수업 핵심 내용 정리
- 중요 공식/정리 요약
- 다음 수업 준비 안내

**목적:**
- 즉시 기억 강화 (망각 곡선 방지)
- 수업 내용 정리

**개입 실행 룰:**
- `ACADEMY_03_post_class_immediate` (priority: 89)

#### 수업 2시간 후: 문제 풀이 안내

**실행 조건:**
- 학원 수업 종료 후 2시간 경과
- 핵심 정리 완료

**실행 방식:**
- 수업 내용 관련 문제 풀이 안내
- 학원 교재 문제 추천
- 단계별 풀이 가이드

**목적:**
- 이해도 강화
- 적용 연습

**개입 실행 룰:**
- `ACADEMY_04_post_class_practice` (priority: 88)

#### 수업 24시간 후: 복습 확인

**실행 조건:**
- 학원 수업 종료 후 24시간 경과
- 복습 여부 확인

**실행 방식:**
- 복습 여부 확인 질문
- 복습 안 한 경우 복습 권장
- 복습 완료 시 다음 단계 안내

**목적:**
- 장기 기억 강화
- 복습 습관 형성

**개입 실행 룰:**
- `ACADEMY_05_post_class_review` (priority: 87)

### 1.3 학원 과제 수행 시 개입

**선생님 텍스트 입력 기반 데이터:**
> "과제 시작 전 예습 상기, 과제 중 막힘 시 학원 교재 참고, 과제 완료 후 정리"

#### 과제 시작 전: 예습 내용 상기

**실행 조건:**
- 학원 과제 시작 시점
- 관련 수업 내용 존재

**실행 방식:**
- 관련 수업 내용 요약 제공
- 핵심 개념 상기
- 과제 목표 안내

**목적:**
- 과제 이해도 향상
- 효율적 과제 수행

#### 과제 중 막힘: 학원 교재 참고 안내

**실행 조건:**
- 과제 수행 중 막힘 발생
- 관련 학원 교재 존재

**실행 방식:**
- 학원 교재 해당 페이지 안내
- 교재 예제 참고 권장
- 단계별 해결 방법 제시

**목적:**
- 독립적 문제 해결 능력 향상
- 교재 활용 능력 향상

#### 과제 완료 후: 정리

**실행 조건:**
- 과제 완료 시점

**실행 방식:**
- 완료한 과제 내용 정리
- 어려웠던 부분 분석
- 다음 과제 준비 안내

**목적:**
- 학습 내용 정리
- 오류 분석 및 개선

**개입 실행 룰:**
- `ACADEMY_06_homework_intervention` (priority: 86)

## 2. 실시간 문제 풀이 개입 실행 전략

### 현직 선생님의 실행 패턴

**선생님 체크리스트 기반 데이터:**
- "1분 막힘: 간단한 힌트 제공"
- "3분 막힘: 단계별 접근법 제시"
- "5분 막힘: 즉시 힌트 제공 + 쉬운 유사 문제"
- "10분 막힘: 문제 변경 권장"

### 2.1 막힘 시간별 개입 전략

#### 1분 막힘: 간단한 힌트 제공

**실행 조건:**
- 문제 풀이 시작 후 1분 경과
- 풀이 진행 없음

**실행 방식:**
- 매우 간단한 힌트 (예: "첫 번째 단계를 생각해보세요")
- 풀이 방향만 제시
- 직접적인 답은 제공하지 않음

**목적:**
- 초기 막힘 해소
- 풀이 방향 제시

**개입 실행 룰:**
- `REALTIME_01_stuck_1min_hint` (priority: 85)

#### 3분 막힘: 단계별 접근법 제시

**실행 조건:**
- 문제 풀이 시작 후 3분 경과
- 풀이 진행 없음 또는 매우 느림

**실행 방식:**
- 문제를 작은 단계로 나누어 제시
- 각 단계별 해결 방법 안내
- 풀이 과정 구조화

**목적:**
- 복잡한 문제 단순화
- 체계적 접근법 습득

**개입 실행 룰:**
- `REALTIME_02_stuck_3min_stepwise` (priority: 84)

#### 5분 막힘: 즉시 힌트 제공 + 쉬운 유사 문제

**실행 조건:**
- 문제 풀이 시작 후 5분 경과
- 풀이 진행 없음

**실행 방식:**
- 구체적인 힌트 제공
- 비슷하지만 쉬운 유사 문제 제공
- 유사 문제 풀이 후 원 문제 재시도 권장

**목적:**
- 즉시 막힘 해소
- 유사 문제를 통한 이해도 향상

**개입 실행 룰:**
- `REALTIME_03_stuck_5min_immediate_hint` (priority: 83)

#### 10분 막힘: 문제 변경 권장

**실행 조건:**
- 문제 풀이 시작 후 10분 경과
- 풀이 진행 없음

**실행 방식:**
- 현재 문제는 일시 보류
- 더 쉬운 난이도의 문제로 변경 권장
- 기초 개념 복습 권장

**목적:**
- 시간 낭비 방지
- 적절한 난이도 문제로 전환

**개입 실행 룰:**
- `REALTIME_04_stuck_10min_problem_change` (priority: 82)

### 2.2 오류 유형별 즉시 대응 전략

**선생님 텍스트 입력 기반 데이터:**
> "계산 실수 감지 시 즉시 확인 요청, 개념 오류 시 개념 설명, 풀이 과정 오류 시 단계별 안내"

#### 계산 실수: 즉시 확인 요청

**실행 조건:**
- 계산 실수 감지 (30초 내)
- 문제 풀이 중

**실행 방식:**
- "계산을 다시 확인해보세요" 같은 확인 요청
- 계산 과정 점검 안내
- 계산 실수 방지 팁 제공

**목적:**
- 즉시 오류 인식
- 계산 정확도 향상

**개입 실행 룰:**
- `REALTIME_05_error_type_calculation_mistake` (priority: 81)

#### 개념 오류: 개념 설명 제공

**실행 조건:**
- 개념 오류 감지 (1분 내)
- 잘못된 개념 적용

**실행 방식:**
- 해당 개념 재설명
- 개념 적용 예시 제공
- 개념 관련 컨텐츠 링크 제공

**목적:**
- 개념 이해도 향상
- 오개념 수정

**개입 실행 룰:**
- `REALTIME_06_error_type_concept_error` (priority: 80)

#### 풀이 과정 오류: 단계별 안내

**실행 조건:**
- 풀이 과정 오류 감지 (1.5분 내)
- 잘못된 풀이 단계

**실행 방식:**
- 올바른 풀이 단계 제시
- 단계별 풀이 과정 시각화
- 단계별 체크리스트 제공

**목적:**
- 풀이 과정 이해도 향상
- 체계적 풀이 습관 형성

**개입 실행 룰:**
- `REALTIME_07_error_type_process_error` (priority: 79)

### 2.3 풀이 중 정서 관리 전략

**선생님 텍스트 입력 기반 데이터:**
> "좌절 감지 → 즉시 격려, 집중도 하락 → 휴식 권장, 자신감 하락 → 성공 상기"

#### 좌절 감지: 즉시 격려

**실행 조건:**
- 좌절 감정 감지
- 문제 풀이 중

**실행 방식:**
- "조금만 더 해보면 풀 수 있어요" 같은 격려
- 작은 성공 경험 상기
- 긍정적 피드백 제공

**목적:**
- 동기 유지
- 지속성 강화

**개입 실행 룰:**
- `REALTIME_08_emotion_frustration` (priority: 78)

#### 집중도 하락: 휴식 권장

**실행 조건:**
- 집중도 0.5 이하
- 문제 풀이 중

**실행 방식:**
- "잠시 휴식하는 게 어떨까요?" 같은 휴식 권장
- 5-10분 휴식 제안
- 휴식 후 재시도 안내

**목적:**
- 효율적 학습 유지
- 피로도 관리

**개입 실행 룰:**
- `REALTIME_09_concentration_drop` (priority: 77)

#### 자신감 하락: 성공 경험 상기

**실행 조건:**
- 자신감 0.5 이하
- 문제 풀이 중

**실행 방식:**
- 과거 성공 경험 상기
- 비슷한 문제 풀이 성공 사례 제시
- "할 수 있어요" 같은 긍정적 메시지

**목적:**
- 자신감 회복
- 동기 부여

**개입 실행 룰:**
- `REALTIME_10_confidence_drop` (priority: 76)

## 3. 단원별 취약점 기반 개입 효과성 분석

### 현직 선생님의 실행 패턴

**선생님 텍스트 입력 기반 데이터:**
> "함수 단원 개입 효과성 85%, 도형 단원 60%"
> "취약 단원 개입 시 즉시 실행, 비취약 단원은 후순위"
> "함수 단원: 그래프 시각화 중심, 도형 단원: 도형 그리기 중심"

### 3.1 취약 단원 우선순위 조정

**실행 조건:**
- 취약 단원 개입 요청
- 취약 단원 식별 완료

**실행 방식:**
- 우선순위 1.3배 상향
- 즉시 실행
- 효과성 추적 활성화

**목적:**
- 취약 단원 집중 개입
- 효과성 향상

**개입 실행 룰:**
- `UNIT_01_vulnerable_unit_priority` (priority: 75)

### 3.2 단원별 효과성 추적

**실행 조건:**
- 개입 완료
- 대상 단원 식별 완료

**실행 방식:**
- 단원별 효과성 데이터 수집
- 응답률, 목표 달성도, 행동 변화도 추적
- 단원별 효과성 DB 업데이트

**목적:**
- 단원별 개입 전략 최적화
- 효과성 향상

**개입 실행 룰:**
- `UNIT_02_unit_effectiveness_tracking` (priority: 74)

### 3.3 단원별 개입 전략 차별화

#### 함수 단원: 그래프 시각화 중심

**실행 조건:**
- 함수 단원 개입
- 그래프 관련 문제

**실행 방식:**
- 그래프 시각화 컨텐츠 제공
- 그래프 이해도 강조
- 그래프 그리기 연습 안내

**목적:**
- 함수 이해도 향상
- 시각적 학습 강화

**개입 실행 룰:**
- `UNIT_03_unit_strategy_differentiation` (priority: 73)

#### 도형 단원: 도형 그리기 중심

**실행 조건:**
- 도형 단원 개입
- 도형 관련 문제

**실행 방식:**
- 도형 그리기 안내
- 도형 작도 연습 권장
- 도형 시각화 강조

**목적:**
- 도형 이해도 향상
- 작도 능력 향상

**개입 실행 룰:**
- `UNIT_04_geometry_strategy` (priority: 72)

## 데이터 수집 방법 (가정)

### 선생님 설문 항목
- "학원 수업 전후 언제 개입이 가장 효과적이었나요?"
- "학원 수업 전 예습 개입의 최적 타이밍은 언제인가요?"
- "학원 수업 후 복습 개입의 최적 타이밍은 언제인가요?"
- "문제 풀이 중 막힘 시 몇 분 이상이면 개입하시나요?"

### 선생님 체크리스트
- [ ] 1분 막힘: 간단한 힌트 제공
- [ ] 3분 막힘: 단계별 접근법 제시
- [ ] 5분 막힘: 즉시 힌트 제공 + 쉬운 유사 문제
- [ ] 10분 막힘: 문제 변경 권장

- "계산 실수 감지 시 즉시 확인 요청, 개념 오류 시 개념 설명, 풀이 과정 오류 시 단계별 안내"
- "함수 단원 개입 효과성 85%, 도형 단원 60%"
- "취약 단원 개입 시 즉시 실행, 비취약 단원은 후순위"

## 효과성 평가 기준

### 학원 수업 전후 개입 효과성

1. **수업 전 개입**
   - 수업 이해도 향상률
   - 수업 집중도 향상률
   - 수업 참여도 향상률

2. **수업 후 개입**
   - 기억 유지율
   - 문제 풀이 성공률
   - 복습 완료율

### 실시간 문제 풀이 개입 효과성

1. **막힘 시간별 개입**
   - 막힘 해소 시간 단축률
   - 힌트 제공 후 재시도 성공률
   - 문제 변경 후 성공률

2. **오류 유형별 개입**
   - 계산 실수 감소율
   - 개념 오류 감소율
   - 풀이 과정 오류 감소율

3. **정서 관리 개입**
   - 좌절 해소율
   - 집중도 회복율
   - 자신감 회복율

### 단원별 효과성

1. **취약 단원 개입**
   - 취약 단원 성적 향상률
   - 취약 단원 이해도 향상률

2. **단원별 전략 차별화**
   - 함수 단원 그래프 이해도 향상률
   - 도형 단원 작도 능력 향상률

   - 시간 가중치 적용

   - 막힘 시간이 길수록 높은 우선순위
   - 즉시 실행 필요

   - 취약 단원 1.3배 가중치

- 학원 수업 전후 개입 여러 개 동시 발생 시: 시간 가까운 것 우선
