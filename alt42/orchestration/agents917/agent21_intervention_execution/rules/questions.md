💬 포괄형 질문 1

“지금 이 학생에게 개입을 실행한다면, 어떤 방식·타이밍·강도로 해야 가장 효과적일까?”
→ 최근 활동, 감정상태(Agent 05), 집중시간대(Agent 01), 개입계획(Agent 20)을 종합해
실행 우선순위, 최적 실행 시점, 개입 강도(즉시/지연/완화)와 전달 방식(메시지·채팅·알림)을 결정하고
예상 반응 패턴을 예측해줘.

💬 포괄형 질문 2

“현재 대기 중인 개입들을 어떻게 조합·조정하면 과부하 없이 자연스럽게 실행할 수 있을까?”
→ 하루 메시지 제한, 연쇄 트리거 여부, 선행·후속 개입 관계를 고려해
병합·연기·우선 실행해야 할 개입을 구분하고
학생의 피로도·집중도 변화를 최소화하는 실행 시퀀스를 설계해줘.

💬 포괄형 질문 3

“최근 실행된 개입들의 반응과 성과를 바탕으로, 다음 개입 실행 전략을 어떻게 조정해야 할까?”
→ 실행 기록(읽음률, 응답시간, 효과성 점수), 행동 변화, 감정 회복 속도 등을 분석해
효과성이 높았던 패턴과 낮았던 패턴을 구분하고
다음 개입의 전달 톤, 시간대, 개입 주기를 최적화하는 개선 루프를 제안해줘.

## 🧠 2️⃣ 각 상황별 Agent 21에 던질 맞춤형 복합질문 예시

### ✅ ① 데이터 트리거 발생

> “침착도 하락 트리거가 발생했어.
> 현재 학생의 감정 상태와 집중 시간대를 기반으로 즉시 실행해야 할 개입을 추천하고 실행 시점을 결정해줘.”

**핵심 포함 요소**

* 트리거 유형
* 감정 상태(Agent 05)
* 집중 시간대(Agent 01)
* 실행 타이밍 결정

---

### ✅ ② 개입 계획 실행

> “Agent 20에서 보낸 ‘19시 10분 개념 요약 카드 전송’ 개입을 받았어.
> 현재 개인 개입 목록에 추가하고, 우선순위를 재조정해서 최적 실행 시점과 실행 방식을 결정해줘.”

**핵심 포함 요소**

* 개입 계획 데이터
* 기존 대기 목록
* 우선순위 재조정 로직
* 타이밍 결정

---

### ✅ ③ 메시지 과다 또는 충돌

> “오늘 이미 8개의 개입 메시지가 발송됐어.
> 대기 중인 3개의 개입(침착도 하락, 필기 지연, 휴식 없음)을 검토해서
> 하루 제한 내에서 우선순위를 다시 정하고, 선생님 승인 필요 여부를 판단해줘.”

**핵심 포함 요소**

* 메시지 한도 상태
* 대기 개입 목록
* 선생님 승인 트리거

---

### ✅ ④ 집중 시간대 또는 활동 전환 직전

> “학생이 30분 후 집중 시간대에 들어가.
> 그 시간대에 가장 효과적인 개입 2가지를 선택하고,
> 실행 순서와 시간 오프셋을 계산해줘.”

**핵심 포함 요소**

* 집중 시간대 데이터
* 효과성 예측 모델
* 타이밍 오프셋 계산

---

### ✅ ⑤ 선생님 개입 필요

> “학생의 감정 상태가 불안정해서 자동 메시지보단 직접 피드백이 필요해.
> 현재 대기 중인 개입 중 선생님 직접 전달이 더 효과적인 항목을 추천하고,
> 자동 개입은 보류 표시로 전환해줘.”

**핵심 포함 요소**

* 감정 상태 분석
* 개입 효과성 기록
* 자동 → 수동 전환 로직

---

### ✅ ⑥ 효과성 검증 및 리포트

> “어제 19시 개입(침착도 하락 대응)의 실행 결과를 요약해줘.
> 학생의 읽음·응답 패턴, 목표 달성도, 행동 변화도를 기반으로 효과성 점수를 계산하고,
> 다음 개입 개선점을 제안해줘.”

**핵심 포함 요소**

* 실행 히스토리
* 반응 기록
* 효과성 점수 계산식
* 개선 피드백
