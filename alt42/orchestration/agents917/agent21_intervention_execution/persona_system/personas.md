## 페르소나 구조 설명

각 페르소나는 다음 요소로 구성됩니다:

| 요소 | 설명 |
|------|------|
| **ID** | 반응코드_유형번호 (예: A_P1, R_P2) |
| **별칭** | 직관적으로 이해할 수 있는 유형명 |
| **심리적 특성** | 내면의 감정, 생각, 동기 |
| **행동적 특성** | 외부로 드러나는 행동 패턴 |
| **전형적 발화** | 해당 유형이 자주 하는 말 |
| **대응 전략** | 시스템/교사의 최적 대응 방법 |
| **전환 가능성** | 다른 반응 유형으로 전환될 가능성 |

### 반응 유형 코드 체계

```
A (Acceptance)   : 개입 수용 반응
R (Resistance)   : 개입 저항 반응
N (No Response)  : 무응답/무반응
D (Delayed)      : 지연 반응
```

### 반응 하위 유형

| 주 유형 | 하위 유형 | 설명 |
|---------|-----------|------|
| A | reinforcement | 강화 필요한 수용 |
| A | consolidation | 안정화 단계 수용 |
| A | enthusiastic | 적극적 수용 |
| R | explicit | 명시적 저항 |
| R | passive | 소극적 저항 |
| R | negotiation | 협상형 저항 |
| N | silent | 완전 무응답 |
| N | minimal | 최소 반응 |
| D | processing | 처리 중 지연 |
| D | avoidance | 회피성 지연 |

## A: 수용형 (Acceptance) 페르소나

### A_P1: 즉각적 순응자 (Immediate Complier)

| 항목 | 내용 |
|------|------|
| **하위 유형** | enthusiastic |
| **심리적 특성** | 권위에 대한 신뢰가 높음. 지시를 따르는 것이 편함. 갈등 회피 성향 |
| **행동적 특성** | 개입 제안 즉시 수용. 질문 없이 동의. "네, 알겠습니다" 즉답 |
| **전형적 발화** | "네, 그렇게 할게요!", "알겠습니다, 선생님", "바로 시작해도 돼요?" |
| **대응 전략** | 이해도 확인 필수. 단순 순응이 아닌 내면화 유도. 자기 결정권 강화 |
| **전환 가능성** | 과부하 시 N 또는 R로 전환 가능 |
| **성공 지표** | 행동 변화 지속 여부, 자발적 적용 증거 |

### A_P2: 신중한 수용자 (Thoughtful Accepter)

| 항목 | 내용 |
|------|------|
| **하위 유형** | consolidation |
| **심리적 특성** | 논리적 이해 후 수용. 납득해야 행동하는 성향. 합리적 사고 |
| **행동적 특성** | 질문 후 수용. 이유 확인. 계획 세우기를 원함 |
| **전형적 발화** | "왜 그렇게 하면 좋은 건가요?", "이해했어요, 해볼게요", "구체적으로 어떻게요?" |
| **대응 전략** | 충분한 설명 제공. 논리적 근거 제시. 질문 환영. 계획 수립 지원 |
| **전환 가능성** | 설명 부족 시 R(negotiation)으로 전환 가능 |
| **성공 지표** | 질문의 질적 향상, 응용 능력 |

### A_P3: 기대에 찬 도전자 (Eager Challenger)

| 항목 | 내용 |
|------|------|
| **하위 유형** | enthusiastic |
| **심리적 특성** | 변화에 대한 기대. 성장 욕구가 강함. 긍정적 마인드셋 |
| **행동적 특성** | 적극적 참여. 추가 활동 요청. 결과에 대한 관심 |
| **전형적 발화** | "더 해도 돼요?", "이거 하면 정말 좋아져요?", "빨리 해보고 싶어요!" |
| **대응 전략** | 열정 유지 지원. 과부하 방지. 단계적 목표 설정. 성공 경험 축적 |
| **전환 가능성** | 기대 미충족 시 D 또는 R로 전환 가능 |
| **성공 지표** | 지속적 참여, 자기주도 확장 |

### A_P4: 관계 중심 수용자 (Relationship-Driven Accepter)

| 항목 | 내용 |
|------|------|
| **하위 유형** | reinforcement |
| **심리적 특성** | 교사/시스템과의 관계 때문에 수용. 신뢰 기반. 인정받고 싶음 |
| **행동적 특성** | 교사 반응에 민감. 칭찬에 동기화. 관계 유지를 위한 노력 |
| **전형적 발화** | "선생님이 시키시니까요", "실망시키고 싶지 않아요", "칭찬해주실 거죠?" |
| **대응 전략** | 관계 강화와 내재적 동기 연결. 자기 결정감 부여. 진정성 있는 피드백 |
| **전환 가능성** | 관계 손상 시 R 또는 N으로 즉시 전환 |
| **성공 지표** | 내재적 동기 발현, 관계 없이도 행동 유지 |

### A_P5: 조건부 수용자 (Conditional Accepter)

| 항목 | 내용 |
|------|------|
| **하위 유형** | consolidation |
| **심리적 특성** | 특정 조건 하에서만 수용. 교환적 사고. 이익 계산 |
| **행동적 특성** | "~하면 ~할게요" 패턴. 보상/조건 확인. 협상 시도 |
| **전형적 발화** | "이거 하면 뭐가 좋아요?", "그럼 게임 시간은요?", "대신 이건 안 해도 되죠?" |
| **대응 전략** | 명확한 기대치 설정. 내재적 가치 전달. 단계적 조건 해제 |
| **전환 가능성** | 조건 미충족 시 R(negotiation)으로 전환 |
| **성공 지표** | 조건 없이도 수용하는 빈도 증가 |

### A_P6: 회복 중인 수용자 (Recovering Accepter)

| 항목 | 내용 |
|------|------|
| **하위 유형** | reinforcement |
| **심리적 특성** | 이전 저항/무응답에서 전환됨. 조심스러운 신뢰 회복. 재도전 |
| **행동적 특성** | 조심스러운 참여. 확인 질문 많음. 작은 성공에 안도 |
| **전형적 발화** | "이번에는 해볼게요...", "이게 맞아요?", "잘하고 있죠?" |
| **대응 전략** | 작은 성공 적극 인정. 안전한 환경 유지. 실패 허용 분위기. 점진적 확장 |
| **전환 가능성** | 실패 경험 시 원래 상태(R/N)로 회귀 가능 |
| **성공 지표** | 회귀 없이 안정적 수용 유지 기간 |

## R: 저항형 (Resistance) 페르소나

### R_P1: 명시적 거부자 (Explicit Rejector)

| 항목 | 내용 |
|------|------|
| **하위 유형** | explicit |
| **심리적 특성** | 자율성 침해에 대한 반발. 통제에 대한 저항. 자기 결정권 주장 |
| **행동적 특성** | 명확한 거부 표현. "싫어요", "안 해요" 직접 발화. 대안 미제시 |
| **전형적 발화** | "싫어요", "안 할래요", "왜 해야 해요?", "하기 싫어요" |
| **대응 전략** | 강요 금지. 선택권 부여. 저항의 이유 탐색. 시간 제공. 대안 협의 |
| **전환 가능성** | 자율성 존중 시 A 또는 R(negotiation)으로 전환 |
| **경고 신호** | 저항 강도 증가, 감정적 격앙 |

### R_P2: 논리적 반박자 (Logical Challenger)

| 항목 | 내용 |
|------|------|
| **하위 유형** | negotiation |
| **심리적 특성** | 개입의 타당성에 의문. 비판적 사고. 납득되어야 행동 |
| **행동적 특성** | 이유/근거 질문. 반례 제시. 대안 주장 |
| **전형적 발화** | "그게 왜 필요한데요?", "다른 방법은 없어요?", "이건 맞지 않는 것 같아요" |
| **대응 전략** | 논리적 설명 제공. 토론 환영. 학생 의견 존중. 합의점 도출 |
| **전환 가능성** | 논리적 설득 시 A(thoughtful)로 전환 높음 |
| **성공 지표** | 질문의 건설적 방향 전환 |

### R_P3: 소극적 저항자 (Passive Resistor)

| 항목 | 내용 |
|------|------|
| **하위 유형** | passive |
| **심리적 특성** | 직접 대립 회피. 내면적 불만. 표면적 순응, 실질적 저항 |
| **행동적 특성** | 말로는 동의하나 행동 미이행. 잊어버림. 미루기. 핑계 |
| **전형적 발화** | "네... (행동 안 함)", "깜빡했어요", "나중에 할게요", "할 거예요..." |
| **대응 전략** | 행동 모니터링 강화. 작은 단위 과제. 즉각적 실행 유도. 저항 원인 탐색 |
| **전환 가능성** | 원인 해결 시 A로 전환, 방치 시 N으로 전환 |
| **경고 신호** | 반복적 미이행, 핑계의 다양화 |

### R_P4: 감정적 반발자 (Emotional Reactor)

| 항목 | 내용 |
|------|------|
| **하위 유형** | explicit |
| **심리적 특성** | 개입이 정서적 상처를 건드림. 과거 트라우마. 감정적 반응 우선 |
| **행동적 특성** | 화남, 울음, 격한 반응. 감정적 발화. 비합리적 거부 |
| **전형적 발화** | "왜 저한테만 그래요!", "너무해요!", "맨날 저만 시켜요!" |
| **대응 전략** | 감정 먼저 수용. 진정 시간 제공. 원인 파악. 정서적 지지 우선 |
| **전환 가능성** | 감정 안정 후 A 또는 D로 전환 가능 |
| **위험 신호** | 자해 언급, 극단적 회피, 관계 단절 시도 |

### R_P5: 협상형 저항자 (Negotiating Resistor)

| 항목 | 내용 |
|------|------|
| **하위 유형** | negotiation |
| **심리적 특성** | 완전 거부가 아닌 조건 협상. 타협점 모색. 통제감 유지 욕구 |
| **행동적 특성** | 대안 제시. "이렇게 하면요?" 제안. 분량/방법 조정 요청 |
| **전형적 발화** | "절반만 하면 안 돼요?", "이것 대신 저거 하면요?", "내일 하면 안 돼요?" |
| **대응 전략** | 협상 가능 영역 제시. 학생 의견 일부 수용. 윈-윈 솔루션 도출 |
| **전환 가능성** | 협상 성공 시 A(conditional)로 전환 높음 |
| **성공 지표** | 협상 빈도 감소, 초기 수용 증가 |

### R_P6: 무력감 기반 저항자 (Helplessness-Based Resistor)

| 항목 | 내용 |
|------|------|
| **하위 유형** | passive |
| **심리적 특성** | "해도 안 된다"는 믿음. 학습된 무력감. 자기효능감 결여 |
| **행동적 특성** | 시작 전 포기. "못해요" 반복. 시도 회피 |
| **전형적 발화** | "어차피 못해요", "해봤자 안 돼요", "저 원래 못해요" |
| **대응 전략** | 극히 작은 성공 경험. 자기효능감 재건. 과거 성공 상기. 장기적 접근 |
| **전환 가능성** | 성공 경험 축적 시 A(recovering)로 점진적 전환 |
| **위험 신호** | 완전 포기 선언, 무기력 심화 |

## N: 무응답형 (No Response) 페르소나

### N_P1: 완전 침묵자 (Complete Silent)

| 항목 | 내용 |
|------|------|
| **하위 유형** | silent |
| **심리적 특성** | 의도적 무시 또는 관심 부재. 소통 자체에 대한 거부. 단절 |
| **행동적 특성** | 메시지 미확인/무시. 응답 제로. 읽씹. 로그아웃 상태 유지 |
| **전형적 발화** | (응답 없음) |
| **대응 전략** | 압박 금지. 다양한 채널 시도. 간접 접촉. 부모/다른 에이전트 협조 |
| **전환 가능성** | 외부 개입 시 D 또는 R로 전환 가능 |
| **경고 신호** | 장기 침묵, 다른 활동에서도 비활성 |

### N_P2: 최소 반응자 (Minimal Responder)

| 항목 | 내용 |
|------|------|
| **하위 유형** | minimal |
| **심리적 특성** | 의무적 최소 반응. 관심 없음. 에너지 절약 모드 |
| **행동적 특성** | 단답형 응답. "네", "아니요", "몰라요". 추가 대화 회피 |
| **전형적 발화** | "네", "아니요", "몰라요", "그래요" |
| **대응 전략** | 열린 질문보다 구체적 선택지. 흥미 유발 시도. 반응 패턴 분석 |
| **전환 가능성** | 흥미 영역 발견 시 A 또는 D로 전환 |
| **성공 지표** | 응답 길이/깊이 증가 |

### N_P3: 바쁨 핑계자 (Busy Excuse Maker)

| 항목 | 내용 |
|------|------|
| **하위 유형** | minimal |
| **심리적 특성** | 진짜 바쁘거나 바쁨을 핑계로 회피. 우선순위에서 낮음. 시간 관리 문제 |
| **행동적 특성** | "시간 없어요" 반복. 일정 이유 제시. 나중에 하겠다는 약속 |
| **전형적 발화** | "지금 바빠요", "학원 가야 해요", "내일 할게요", "숙제 먼저 해야 해요" |
| **대응 전략** | 짧은 개입 제안. 시간 협의. 우선순위 재조정 지원. 작은 단위 과제 |
| **전환 가능성** | 시간 확보 시 A 또는 D로 전환 |
| **확인 필요** | 진짜 바쁜지 vs 회피성 바쁨인지 구분 |

### N_P4: 기술적 무응답자 (Technical Non-Responder)

| 항목 | 내용 |
|------|------|
| **하위 유형** | silent |
| **심리적 특성** | 기술적 문제로 인한 무응답. 앱/시스템 문제. 알림 미확인 |
| **행동적 특성** | 다른 채널에서는 활동. 간헐적 접속. 기술 문제 언급 |
| **전형적 발화** | "알림이 안 왔어요", "앱이 안 돼요", "와이파이가 없었어요" |
| **대응 전략** | 대체 연락 채널 확보. 기술 지원. 정기적 알림 재설정. 다중 채널 활용 |
| **전환 가능성** | 기술 문제 해결 시 즉시 A 또는 D로 전환 |
| **확인 필요** | 기술 핑계 vs 실제 문제 구분 |

### N_P5: 선택적 무응답자 (Selective Non-Responder)

| 항목 | 내용 |
|------|------|
| **하위 유형** | minimal |
| **심리적 특성** | 특정 주제/개입에만 무응답. 선별적 회피. 불편한 영역 존재 |
| **행동적 특성** | 특정 질문 스킵. 다른 주제에는 반응. 패턴적 회피 |
| **전형적 발화** | (특정 주제에서만 응답 없음) |
| **대응 전략** | 회피 영역 파악. 우회적 접근. 안전한 환경 조성. 점진적 접근 |
| **전환 가능성** | 신뢰 구축 시 회피 영역에서도 D → A로 전환 |
| **분석 필요** | 회피 패턴과 원인 파악 |

### N_P6: 관망형 무응답자 (Wait-and-See Non-Responder)

| 항목 | 내용 |
|------|------|
| **하위 유형** | silent |
| **심리적 특성** | 지켜보는 중. 신뢰/검증 단계. 섣불리 반응하지 않음 |
| **행동적 특성** | 메시지 확인은 하지만 응답 안 함. 읽음 표시. 관찰 중 |
| **전형적 발화** | (확인하되 응답 없음) |
| **대응 전략** | 일관된 메시지 지속. 압박 없이 기다림. 신뢰 구축 행동. 첫 반응 시 긍정 피드백 |
| **전환 가능성** | 신뢰 형성 시 D → A로 자연스러운 전환 |
| **성공 지표** | 첫 자발적 응답 발생 |

## D: 지연반응형 (Delayed) 페르소나

### D_P1: 숙고형 지연자 (Deliberate Delayer)

| 항목 | 내용 |
|------|------|
| **하위 유형** | processing |
| **심리적 특성** | 신중한 성격. 생각할 시간 필요. 충동적 결정 회피 |
| **행동적 특성** | 질문/제안 후 시간을 두고 응답. 생각 중 표현. 나중에 답변 |
| **전형적 발화** | "생각해볼게요", "좀 이따 말씀드릴게요", "아직 결정 못 했어요" |
| **대응 전략** | 충분한 시간 제공. 마감 설정 시 사전 안내. 중간 체크인. 생각 과정 존중 |
| **전환 가능성** | 숙고 후 A로 전환 높음, 과한 압박 시 R로 전환 |
| **성공 지표** | 숙고 후 긍정적 결정 비율 |

### D_P2: 회피형 지연자 (Avoidant Delayer)

| 항목 | 내용 |
|------|------|
| **하위 유형** | avoidance |
| **심리적 특성** | 불편한 상황 회피. 결정/행동 미루기. 문제 직면 어려움 |
| **행동적 특성** | 계속 미루기. "나중에요" 반복. 마감 임박까지 미행동 |
| **전형적 발화** | "내일 할게요", "주말에요", "시험 끝나고요", "나중에..." |
| **대응 전략** | 작은 즉시 행동 유도. 시작의 문턱 낮추기. 동행 제안. 마감 관리 |
| **전환 가능성** | 첫 행동 성공 시 A로 전환, 방치 시 N으로 전환 |
| **경고 신호** | 무기한 연기, 마감 무시 반복 |

### D_P3: 우선순위 조정 지연자 (Priority-Based Delayer)

| 항목 | 내용 |
|------|------|
| **하위 유형** | processing |
| **심리적 특성** | 다른 것이 더 급함. 합리적 우선순위. 시간 분배 고려 |
| **행동적 특성** | "~먼저 하고요" 패턴. 순서 조정 요청. 현실적 일정 제시 |
| **전형적 발화** | "시험 끝나고 바로 할게요", "이번 주는 힘들고 다음 주요", "이것부터 끝내고요" |
| **대응 전략** | 우선순위 존중. 예약/스케줄링 지원. 유연한 마감. 리마인더 설정 |
| **전환 가능성** | 계획된 시점에 A로 전환 높음 |
| **성공 지표** | 약속된 시점에 이행 여부 |

### D_P4: 확인 필요 지연자 (Clarification-Seeking Delayer)

| 항목 | 내용 |
|------|------|
| **하위 유형** | processing |
| **심리적 특성** | 불확실성에 대한 불편함. 더 알고 싶음. 완벽한 이해 후 행동 |
| **행동적 특성** | 추가 질문. 확인 요청. 정보 수집 후 행동 |
| **전형적 발화** | "그게 정확히 뭔데요?", "좀 더 알아보고요", "예시 보여주세요" |
| **대응 전략** | 상세한 정보 제공. 예시 활용. 질문 환영. 단계별 안내 |
| **전환 가능성** | 정보 충족 시 A로 전환 높음 |
| **성공 지표** | 질문 → 행동 전환 속도 |

### D_P5: 기분/컨디션 기반 지연자 (Mood-Based Delayer)

| 항목 | 내용 |
|------|------|
| **하위 유형** | avoidance |
| **심리적 특성** | 기분/컨디션에 따라 반응. 오늘은 안 됨. 정서적 준비 필요 |
| **행동적 특성** | "오늘은 힘들어요" 패턴. 컨디션 언급. 다른 날 제안 |
| **전형적 발화** | "오늘 기분이 안 좋아요", "피곤해서요", "컨디션이 안 좋아서요" |
| **대응 전략** | 정서 상태 존중. 가벼운 활동 제안. 컨디션 좋을 때 리마인더. 정서 지원 |
| **전환 가능성** | 정서 회복 시 A로 전환, 지속적 저조 시 N으로 전환 |
| **확인 필요** | 일시적 vs 지속적 정서 문제 구분 |

### D_P6: 타인 의존 지연자 (Other-Dependent Delayer)

| 항목 | 내용 |
|------|------|
| **하위 유형** | processing |
| **심리적 특성** | 부모/친구 의견 필요. 혼자 결정 어려움. 의존적 성향 |
| **행동적 특성** | "엄마한테 물어보고요" 패턴. 다른 사람 언급. 허락 필요 |
| **전형적 발화** | "부모님한테 여쭤볼게요", "친구랑 상의해보고요", "선생님 허락받고요" |
| **대응 전략** | 관련인 개입 지원. 독립적 결정 기회 제공. 자기결정력 강화 |
| **전환 가능성** | 타인 동의 시 A로 전환, 반대 시 R로 전환 |
| **성공 지표** | 독립적 결정 빈도 증가 |

## 전환 패턴 가이드

### 긍정적 전환 (개입 성공)

```
R → A : 저항 극복, 가장 의미있는 성공
N → A : 무응답 돌파, 관계 형성 성공
N → D : 반응 유도 성공, 첫 단계
D → A : 지연 해소, 행동 시작
R → D : 저항 완화, 대화 시작
```

### 부정적 전환 (관계 악화)

```
A → R : 과부하, 신뢰 손상, 압박
A → N : 관심 상실, 번아웃
D → N : 지속적 미이행, 포기
R → N : 완전 단절, 최악의 시나리오
```

### 전환 촉진 전략

| 목표 전환 | 핵심 전략 |
|-----------|-----------|
| R → A | 자율성 존중, 선택권 부여, 논리적 설득 |
| N → D | 부담 없는 첫 접촉, 흥미 유발, 다양한 채널 |
| D → A | 시간 존중, 적절한 리마인더, 시작 지원 |
| N → A | 관계 구축, 성공 경험, 일관된 지지 |

### 에스컬레이션 기준

| 패턴 | 에스컬레이션 |
|------|-------------|
| R 3회 연속 | 부모 알림, 다른 에이전트 협조 |
| N 7일 이상 | 부모 개입, 직접 연락 시도 |
| R → N 전환 | 위기 대응, 전문가 연계 검토 |
| 자해/극단적 발언 | 즉시 에스컬레이션 |

## 복합 유형 처리

### 복합 유형 예시

| 복합 유형 | 특성 | 대응 |
|-----------|------|------|
| R+D | 저항하면서 미룸 | 저항 완화 우선, 이후 시간 제공 |
| N+D | 무응답이다가 늦게 반응 | 반응 시 즉시 지지, 압박 금지 |
| A+R | 수용했다가 저항 | 과부하 점검, 페이스 조절 |
| D+N | 지연하다가 무응답 | 리마인더 강화, 채널 다양화 |

### 상황별 유연한 적용

1. **첫 개입**: 모든 학생을 D(processing)로 가정하고 시작
2. **3회 상호작용 후**: 주요 반응 유형 식별
3. **지속적 모니터링**: 전환 신호 감지
4. **유형 재평가**: 주 1회 이상 페르소나 재검토
