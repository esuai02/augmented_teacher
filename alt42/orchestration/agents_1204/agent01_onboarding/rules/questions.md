💬 포괄형 질문 1

“이 학생의 현재 수학 학습 맥락을 종합해서, 첫 수업에서 무엇을 어떻게 시작해야 할지 알려줘.”
→ 온보딩 정보, 개념/심화 진도, 학습 스타일, 시험 대비 성향, 자신감 수준 등을 반영해서 수업 도입 루틴, 설명 전략, 자료 유형 등을 포함한 총체적 개입 전략 도출

💬 포괄형 질문 2

“이 학생의 성향과 목표를 기반으로, 커리큘럼과 루틴을 어떤 방향으로 최적화해야 할까?”
→ 목표(단기/중기/장기), 학습 성향, 스트레스/자신감, 부모 개입도 등을 고려하여 진도별 우선순위, 학습 흐름, 문제 유형 비중 조절 등 커스터마이징 설계 도출

💬 포괄형 질문 3

“이 학생이 중장기적으로 성장하기 위해 지금부터 어떤 부분을 특히 신경 써야 할까?”
→ 경시 준비, 진학 목표, 수학 자존감 성장, 피로 누적 방지, 루틴 유지 여부 등을 포괄적으로 분석하여 조기 리스크 예측 및 트래킹 우선요소 추천

## 🎯 2. 상황별 맞춤형 복합질문 예시

### 🟦 S0. 수학 특화 정보 수집 단계 (신규 추가)

* "이 학생의 수학 학습 스타일을 파악해주세요. (계산형/개념형/응용형)"
* "다니고 있는 수학학원 정보를 수집해주세요. (학원명, 등급, 수업 일정)"
* "최근 수학 시험 성적과 취약한 단원을 알려주세요."
* "사용 중인 수학 교재를 알려주세요. (학교 교과서, 학원 교재: 쎈/개념원리 등)"
* "지금까지 배운 수학 단원 중 완료한 단원, 진행 중인 단원을 알려주세요."

### 🟦 S1. 신규 학생 등록 직후

* "이 학생은 어떤 수학 진도에 있고 어떤 학습 스타일을 가지고 있는지 한 번에 요약해줄 수 있나요? (수학 학습 스타일 포함)"
* "초기 수업을 시작할 때 어떤 내용을 중심으로 준비하면 좋을까요? (학원 진도와 학교 진도 연계 고려)"

### 🟦 S2. 수업 전 학습 설계 필요

* "시험 스타일, 수학 자신감, 부모 개입 정도를 종합해서 어떤 루틴이 적절할까요? (학원 수업 일정 반영)"
* "이 학생에게는 개념 강의 위주로 갈지, 문제풀이를 먼저 해야 할지 판단해주세요. (수학 학습 스타일 반영)"

### 🟦 S3. 개념/심화 진도 판단

* "개념 진도와 심화 진도를 비교해서 현재 난이도 밸런스는 적절한가요? (단원 선후관계 고려)"
* "지금 진도에서 자주 혼란을 겪는 개념은 무엇일 가능성이 높나요? (단원 난이도 DB 활용)"

### 🟦 S4. 학부모 상담 정보 추출

* "이 학생의 학습 성향과 부모님의 스타일을 바탕으로 상담 전략을 알려주세요. (학원 정보 포함)"
* "학부모님과 주기적인 피드백 루틴이 필요한 학생인가요? (학원 수업 이해도 고려)"

### 🟦 S5. 장기 목표 기반 설계

* "이 학생의 장기 목표와 현재 성향을 연결해서 신경 써야 할 요소가 있다면 알려주세요. (학원 진도와 목표 연계)"
* "경시대회 준비가 목표라고 했는데, 지금 상태로 보면 어떤 접근이 현실적일까요? (학원 수업과 병행 전략)"
