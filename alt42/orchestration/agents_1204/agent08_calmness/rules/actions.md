# Agent 08 - Calmness Actions

## 수행 가능한 액션 목록

### 침착도 관리
- `calculate_calm_score`: 침착도 점수 계산
- `detect_calm_decline`: 침착도 저하 감지
- `recommend_break`: 휴식 권장

### 학습 조정
- `adjust_difficulty_by_calm`: 침착도 기반 난이도 조정
- `suggest_deep_content`: 심화 콘텐츠 제안
- `recommend_review`: 복습 권장

### 복구 전략
- `suggest_recovery_activities`: 복구 활동 제안
- `create_break_routine`: 휴식 루틴 생성
