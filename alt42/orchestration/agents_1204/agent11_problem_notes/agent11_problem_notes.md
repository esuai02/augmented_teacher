# Agent 11 Problem Notes 지식파일

## 노트 구조
- 풀이노트 : 시험 중 작성하는 실제 풀이 노트
- 준비노트 : 오답이 발생한 이후 해설지와 함께 공부하는 노트로 풀이노트가 전환된 형태. 오답원인을 수기로 기록함
- 서술평가 : 준비노트 완료되고 10분 후 생성. 해설지 없이 해당 문제를 다시 한번 풀게 함.

## 목적
- 오답 패턴을 분석하여 취약 영역을 규명하고 복습 전략을 설계
- 오답노트를 얼마나 의미있게 작성하는지 분석하고 습관, 행동 변화를 위한 학생만의 시그니처 루틴 찾아주기

## 핵심 지표 해석
- 총 필기량(nstroke): 활동 강도/몰입도 지표
- 마지막 필기시점(tlaststroke): 세션 종료 근접도/최근성
- 생성시각(timecreated): 기록 시작 타임라인
- 소요시간(usedtime) : 노트에 사용된 총 시간
- 개념 제목(contentstitle): 단원/주제 맥락

## 휴리스틱
- 오답원인 작성 잘하면 성찰 깊이 생김
- 실용적 풀이가 속도 빠르고 속도가 빠르면 내용에 대한 연결성이 향상됨
- 풀이습관은 고정적인 패턴을 가질 가능성이 크며 그것을 파악하는 것은 행동관찰, 필기관찰임. 이런 관찰을 기록하고 추적하고 피드백하면 행동변화가 가능함.
