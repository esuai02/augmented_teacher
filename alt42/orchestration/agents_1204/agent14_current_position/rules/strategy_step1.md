# Agent 14 - Current Position Strategy Step 1
# 수학 교과 특화 분석 전략

## 목적
현직 수학선생님이 학생의 현재 위치를 평가할 때 고려하는 수학 교과 특화 요소를 반영한 분석 전략

---

## 1. 단원별 진행 분석 전략

### 1.1 단원명 추출 방법
- **plan1~plan16 텍스트 분석**: "함수", "도형", "통계", "확률", "방정식", "부등식", "집합", "명제" 등 키워드 추출
- **학생 설문 수집**: "현재 학습 중인 수학 단원은 무엇인가요?"
- **선생님 체크리스트**: "단원별 진행 패턴: [단원명] → [진행 상태]"

### 1.2 단원별 분석 지표
- **단원별 완료율**: (단원별 완료 항목 수 / 단원별 전체 항목 수) × 100
- **단원별 평균 소요 시간**: 단원별 모든 항목의 평균 소요 시간
- **단원별 지연 빈도**: 단원별 지연 항목 수 / 단원별 전체 항목 수
- **단원별 만족도 분포**: 단원별 매우만족/만족/불만족 비율

### 1.3 단원별 인사이트 생성
- **지연 빈번 단원 식별**: 지연 빈도 > 50%인 단원
- **원활 진행 단원 식별**: 지연 빈도 < 20%인 단원
- **감정 하락 단원 식별**: 불만족 비율 > 50%인 단원

### 1.4 단원별 대응 전략
- **지연 단원**: 병목 해소용 마이크로 목표 추가, 쉬운 문제부터 해결
- **원활 단원**: 심화 학습, 추가 도전 과제 제시
- **감정 하락 단원**: 학습 방식 변경, 개념 재학습

## 2. 난이도별 진행 분석 전략

### 2.1 난이도 분류 방법
- **plan1~plan16 텍스트 분석**: "기본", "유형", "심화", "기출" 키워드 추출
- **학생 설문 수집**: "현재 풀고 있는 문제 난이도는? (A) 기본형 (B) 유형 (C) 심화"
- **선생님 체크리스트**: "난이도별 진행 패턴: [난이도] → [평균 소요 시간]"

### 2.2 난이도별 분석 지표
- **난이도별 평균 소요 시간**: 기본형/유형/심화별 평균 소요 시간
- **난이도별 지연 빈도**: 난이도별 지연 항목 수 / 난이도별 전체 항목 수
- **난이도별 완료율**: 난이도별 완료 항목 수 / 난이도별 전체 항목 수

### 2.3 난이도별 인사이트 생성
- **기본형 지연**: 학습 기초 부족 가능성 → 기초 개념 재학습
- **유형 지연**: 문제 유형 미숙 → 유형별 집중 연습
- **심화 지연**: 난이도 적합성 문제 → 난이도 조정 또는 쉬운 문제부터

### 2.4 난이도별 대응 전략
- **기본형 지연**: 기초 개념 재학습, 쉬운 문제부터 해결
- **유형 지연**: 유형별 집중 연습, 유형별 정리 강화
- **심화 지연**: 난이도 조정, 쉬운 문제부터 자신감 회복

## 3. 학습 단계별 분석 전략

### 3.1 학습 단계 분류 방법
- **plan1~plan16 텍스트 분석**: "개념", "유형", "심화", "기출" 키워드 추출
- **학생 설문 수집**: "현재 어떤 학습 단계인가요? (A) 개념 학습 (B) 유형 연습 (C) 심화 문제 (D) 기출 문제"

### 3.2 학습 단계별 분석 지표
- **단계별 평균 소요 시간**: 개념/유형/심화/기출별 평균 소요 시간
- **단계별 지연 빈도**: 단계별 지연 항목 수 / 단계별 전체 항목 수
- **단계별 완료율**: 단계별 완료 항목 수 / 단계별 전체 항목 수

### 3.3 학습 단계별 인사이트 생성
- **개념 단계 지연**: 개념 이해 부족 → 개념 재학습 필요
- **유형 단계 지연**: 유형 연습 부족 → 유형별 집중 연습
- **심화 단계 지연**: 난이도 적합성 문제 → 난이도 조정
- **기출 단계 지연**: 실전 대비 부족 → 기출 문제 집중 연습

### 3.4 학습 단계별 대응 전략
- **개념 단계 지연**: 개념 재학습, 개념 정리 강화
- **유형 단계 지연**: 유형별 집중 연습, 유형별 정리
- **심화 단계 지연**: 난이도 조정, 쉬운 문제부터
- **기출 단계 지연**: 기출 문제 집중 연습, 실전 대비 강화

## 4. 문제 유형별 분석 전략

### 4.1 문제 유형 분류 방법
- **선생님 텍스트 입력**: "함수 단원 유형: 일차함수 그래프, 함수식 구하기, 함수의 활용"
- **단원별 유형 DB 구축**: 각 단원별 표준 문제 유형 목록

### 4.2 문제 유형별 분석 지표
- **유형별 완료율**: 유형별 완료 항목 수 / 유형별 전체 항목 수
- **유형별 지연 빈도**: 유형별 지연 항목 수 / 유형별 전체 항목 수
- **유형별 평균 소요 시간**: 유형별 평균 소요 시간

### 4.3 문제 유형별 인사이트 생성
- **지연 빈번 유형**: 해당 유형 집중 연습 필요
- **원활 진행 유형**: 해당 유형 심화 학습 가능

### 4.4 문제 유형별 대응 전략
- **지연 유형**: 해당 유형 집중 연습, 유형별 정리 강화
- **원활 유형**: 해당 유형 심화 학습, 추가 도전 과제 제시

## 5. 계산 실수 vs 개념 오류 구분 전략

### 5.1 오류 원인 분류 방법
- **학생 설문 수집**: "문제를 틀렸을 때 주된 원인은 무엇인가요? (A) 계산 실수 (B) 개념을 잘못 이해함 (C) 문제를 잘못 읽음 (D) 시간 부족"

### 5.2 오류 유형별 분석 지표
- **계산 실수 비율**: 계산 실수로 인한 지연 비율
- **개념 오류 비율**: 개념 오류로 인한 지연 비율
- **문제 오독 비율**: 문제 오독으로 인한 지연 비율
- **시간 부족 비율**: 시간 부족으로 인한 지연 비율

### 5.3 오류 유형별 인사이트 생성
- **계산 실수 다수**: 반복 연습 필요, 계산 정확도 향상
- **개념 오류 다수**: 개념 재학습 필요, 개념 정리 강화
- **문제 오독 다수**: 문제 읽기 훈련 필요, 문제 이해도 향상
- **시간 부족 다수**: 시간 관리 훈련 필요, 문제 풀이 속도 향상

### 5.4 오류 유형별 대응 전략
- **계산 실수**: 반복 연습 강화, 계산 정확도 향상 훈련
- **개념 오류**: 개념 재학습, 개념 정리 강화
- **문제 오독**: 문제 읽기 훈련, 문제 이해도 향상
- **시간 부족**: 시간 관리 훈련, 문제 풀이 속도 향상
