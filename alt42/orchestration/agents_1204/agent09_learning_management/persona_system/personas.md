## 페르소나 구조 설명

각 페르소나는 다음 요소로 구성됩니다:

| 요소 | 설명 |
|------|------|
| **ID** | 상황코드_유형번호 (예: P_P1, A_P1) |
| **별칭** | 직관적으로 이해할 수 있는 유형명 |
| **심리적 특성** | 내면의 감정, 생각, 동기 |
| **행동적 특성** | 외부로 드러나는 행동 패턴 |
| **학습 데이터 특징** | 5대 지표(출결/목표/포모도로/오답노트/시험) 패턴 |
| **전형적 발화** | 해당 유형이 자주 하는 말 |
| **개입 전략** | 시스템/교사의 최적 대응 방법 |

## P-Series: 패턴 유형 기반 페르소나

> Agent09의 6가지 핵심 패턴 유형에 기반한 페르소나 분류

### P_P1: 데이터 희박형 (data_sparse) 🌫️
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 학습 기록에 무관심. 시스템 사용에 익숙하지 않거나 거부감. 자기 관리의 필요성을 못 느낌 |
| **행동적 특성** | 로그인 빈도 낮음. 데이터 입력 최소화. 포모도로/목표 기능 미사용 |
| **학습 데이터 특징** | 5대 지표 중 3개 이상 데이터 없음. 출결 외 기록 부재. 데이터 밀도 < 30% |
| **전형적 발화** | "그냥 공부하면 되지 뭘 기록해요", "귀찮아요", "이거 꼭 해야 해요?" |
| **개입 전략** | 데이터 기록의 이점을 구체적으로 설명. 가장 쉬운 기능(출결)부터 습관화. 자동 기록 기능 안내. 작은 인센티브 제공 |

### P_P2: 불균형형 (data_imbalanced) ⚖️
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 특정 기능에만 익숙함. 편향된 학습 관리. 전체적 시야 부족 |
| **행동적 특성** | 포모도로만 열심히 하거나, 출결만 기록. 오답노트는 안 하고 문제만 풀기 |
| **학습 데이터 특징** | 일부 지표만 풍부 (예: 포모도로만 100%, 오답노트 0%). 편차 3배 이상 |
| **전형적 발화** | "전 포모도로만 하면 되는 거 아니에요?", "오답노트는 귀찮아요" |
| **개입 전략** | 균형 잡힌 학습의 효과 설명. 부족한 영역 연결 제안 (예: 포모도로 중 오답 정리). 통합 대시보드 활용 |

### P_P3: 패턴 불안정형 (pattern_unstable) 🎢
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 의욕은 있으나 지속력 부족. 기분/컨디션에 따라 학습량 변동. 루틴 형성 어려움 |
| **행동적 특성** | 어떤 날은 10세션, 어떤 날은 0세션. 들쭉날쭉한 출석. 계획 미준수 |
| **학습 데이터 특징** | 일관성 점수(consistency_score) < 0.5. 일별 편차 큼. 연속 기록 짧음 |
| **전형적 발화** | "오늘은 의욕이 안 나요", "어제 너무 많이 해서 오늘은 쉴래요", "그때그때 달라요" |
| **개입 전략** | 현실적으로 낮춘 목표 설정. 최소 기준선(minimum viable) 제시. 일관성 시각화로 동기부여. 습관 형성 기법 적용 |

### P_P4: 자동화 저항형 (automation_resistant) 🛡️
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 시스템 추천에 대한 불신. 자기 방식 고수. 통제감 상실 우려 |
| **행동적 특성** | 추천 콘텐츠 무시. 설정된 루틴 변경. 시스템 알림 끄기 |
| **학습 데이터 특징** | 추천 수용률 < 30%. 사용자 지정 항목 다수. 알림 무시율 높음 |
| **전형적 발화** | "제가 알아서 할게요", "이건 제 방식이 있어요", "자동 추천 안 믿어요" |
| **개입 전략** | 선택권 제공 (추천 vs 자율). 추천 근거 투명하게 설명. 점진적 신뢰 구축. 자율성 존중하며 가이드 |

### P_P5: 이탈 위험형 (dropout_risk) ⚠️
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 학습 포기 고려 중. 효능감 저하. 무기력 또는 외부 요인 스트레스 |
| **행동적 특성** | 활동량 급감. 연속 결석. 목표 미달성 누적. 반응 지연 |
| **학습 데이터 특징** | dropout_risk_score ≥ 60. 최근 7일 결석 ≥ 3회. 포모도로 완료율 < 30% |
| **전형적 발화** | "그만둘까 생각 중이에요", "의미가 없는 것 같아요", "힘들어요" |
| **개입 전략** | 즉각적 정서 지원 우선. 부담 경감 (목표/분량 축소). 작은 성공 경험 제공. 담당 교사 직접 연락. 원인 파악 대화 |

### P_P6: 고성취 지속형 (high_achiever) ⭐
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 높은 자기 효능감. 성취 동기 강함. 더 높은 목표 추구. 번아웃 위험 내재 |
| **행동적 특성** | 목표 초과 달성. 추가 학습 요청. 꾸준한 기록. 자기 분석 활발 |
| **학습 데이터 특징** | 목표 달성률 > 80%. 시험 평균 > 80점. 포모도로 일관성 > 0.8 |
| **전형적 발화** | "더 어려운 거 없어요?", "다음 목표는 뭐로 할까요?", "시간을 더 효율적으로 쓰고 싶어요" |
| **개입 전략** | 도전적 목표 제시. 심화 학습 콘텐츠. 번아웃 예방 안내. 멘토 역할 기회 제공. 성과 인정 |

## A-Series: 출결 관리 상황 페르소나

### A_P1: 꾸준한 출석왕 📅
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 규칙적 생활 선호. 책임감 강함. 출석 자체에 의미 부여 |
| **행동적 특성** | 매일 정해진 시간 등원. 결석 시 미리 알림. 출석률 95% 이상 |
| **전형적 발화** | "매일 오는 게 습관이에요", "결석하면 찝찝해요" |
| **개입 전략** | 꾸준함 인정 및 격려. 출석 연속 기록 시각화. 다음 단계(학습 질) 도전 제안 |

### A_P2: 불규칙 출석형 📊
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 일정 조절 어려움. 우선순위에서 학습이 밀림. 상황 의존적 참여 |
| **행동적 특성** | 주 2-3회 불규칙 출석. 사전 공지 없는 결석. 패턴 예측 어려움 |
| **전형적 발화** | "오늘은 일이 있어서요", "내일은 꼭 갈게요", "바빠서요" |
| **개입 전략** | 출석 가능 요일/시간 협의. 유연한 일정 제안. 최소 출석 기준 설정. 결석 시 자가 학습 가이드 |

### A_P3: 지각 상습형 ⏰
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 시간 관리 어려움. 급하지 않으면 미루는 경향. 늦어도 괜찮다는 인식 |
| **행동적 특성** | 자주 10-30분 지각. 시작 시간 인식 부족. 사과 없이 입장 |
| **전형적 발화** | "조금 늦었네요", "시간 가는 줄 몰랐어요" |
| **개입 전략** | 지각의 학습 손실 수치화. 알림 시스템 활용. 도착 시간 목표 설정. 지각 줄이기 챌린지 |

### A_P4: 장기 결석형 🏥
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 건강/가정 문제 또는 학습 포기 신호. 복귀에 대한 부담감. 뒤처짐 걱정 |
| **행동적 특성** | 연속 7일 이상 결석. 연락 두절 또는 짧은 응답. 시스템 미접속 |
| **전형적 발화** | "요즘 못 갔어요", "다시 시작하기 겁나요", "많이 밀렸죠?" |
| **개입 전략** | 부담 없는 복귀 지원. 밀린 내용 요약 제공. 개인 상담 연결. 단계적 복귀 계획 |

## G-Series: 목표 설정/관리 상황 페르소나

### G_P1: 야심찬 목표 설정자 🎯
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 높은 성취욕. 도전을 즐김. 때로 비현실적 기대 |
| **행동적 특성** | 높은 목표 설정. 달성 실패 시 좌절. 수정보다 새 목표 선호 |
| **전형적 발화** | "이번 주에 50문제 풀 거예요", "100점 맞을 거예요" |
| **개입 전략** | 현실적 중간 목표 제안. 실패를 학습 기회로 리프레이밍. 달성 가능한 마일스톤 설정 |

### G_P2: 소극적 목표 회피자 😶
| 항목 | 내용 |
|------|------|

| **행동적 특성** | 목표 미설정. "선생님이 정해주세요" 요청. 모호한 목표 |
| **전형적 발화** | "잘 모르겠어요", "그냥 열심히 할게요", "목표 없어도 되나요?" |
| **개입 전략** | 작고 구체적인 목표부터 제안. 선택형 목표 제시. 성공 경험 축적 후 점진적 확장 |

### G_P3: 목표 달성 마스터 🏆
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 계획적 성향. 달성 쾌감 추구. 자기 효능감 높음 |
| **행동적 특성** | 목표 대부분 달성. 체계적 실행. 진행 상황 자주 확인 |
| **전형적 발화** | "목표 달성했어요!", "다음 목표 세울게요", "진행률 확인해주세요" |
| **개입 전략** | 달성 축하 및 인정. 더 도전적 목표 제안. 장기 목표 연계. 노하우 공유 기회 |

### G_P4: D-Day 압박형 ⏳

| **행동적 특성** | D-Day 3일 전부터 급증하는 활동. 평소 목표 무시. 막판 집중 |

## F-Series: 포모도로 학습 패턴 페르소나

### F_P1: 포모도로 마스터 🍅
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 시간 관리 능숙. 집중-휴식 리듬 체화. 효율성 추구 |
| **행동적 특성** | 일일 목표 세션 달성. 휴식 시간 준수. 방해 요소 관리 |
| **전형적 발화** | "오늘 6세션 완료했어요", "쉬는 시간에 진짜 쉬어요" |
| **개입 전략** | 성과 인정. 세션 수 점진적 증가 제안. 집중력 향상 팁 공유. 다른 학생 멘토 역할 |

### F_P2: 집중력 부족형 💨
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 집중 지속 어려움. 쉽게 산만해짐. 외부 자극에 민감 |
| **행동적 특성** | 세션 중단 빈번. 25분 완수 어려움. 방해 기록 다수 |
| **전형적 발화** | "집중이 안 돼요", "25분이 너무 길어요", "자꾸 딴 생각이..." |
| **개입 전략** | 짧은 세션(15분)부터 시작. 집중 환경 조성 가이드. 방해 요소 분석 및 제거. 점진적 시간 증가 |

### F_P3: 과도 집중형 🔥
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 쉬지 않고 하고 싶음. 휴식에 죄책감. 번아웃 위험 |

## 페르소나 활용 가이드

### 페르소나 식별 방법

1. **데이터 패턴 분석**: 5대 지표(출결/목표/포모도로/오답노트/시험) 데이터 기반 1차 분류
2. **행동 관찰**: 시스템 사용 패턴 및 메시지 분석으로 2차 확인
3. **복합 유형 고려**: 한 학생이 여러 페르소나 특성을 가질 수 있음
4. **시계열 변화**: 이전 상태와 비교하여 변화 추이 파악

### 데이터 기반 자동 분류

```php
// Agent09PersonaEngine의 자동 분류 로직
$patternType = $this->classifyPatternType($context);
$dropoutRisk = $this->calculateDropoutRisk($context);

// 패턴 유형 → 페르소나 매핑
$personaMapping = [
    'data_sparse' => 'P_P1',
    'data_imbalanced' => 'P_P2',
    'pattern_unstable' => 'P_P3',
    'automation_resistant' => 'P_P4',
    'dropout_risk' => 'P_P5',
    'high_achiever' => 'P_P6'
];
```

### 개입 우선순위

| 우선순위 | 페르소나 유형 | 개입 시점 |
|----------|--------------|----------|
| 🔴 긴급 | D_P3 (이탈 긴급) | 즉시 |
| 🟠 높음 | D_P2 (이탈 경고), P_P5 (이탈 위험) | 24시간 내 |
| 🟡 중간 | D_P1 (이탈 주의), E_P1 (스트레스), E_P2 (자신감 저하) | 3일 내 |
| 🟢 일반 | 기타 모든 페르소나 | 정기 점검 |

## 온톨로지 연계

각 페르소나는 다음 온톨로지 클래스와 연계될 수 있습니다:

```yaml
mk:LearningManagementPersona:
  - mk:hasPatternType        # 패턴 유형 (data_sparse, high_achiever 등)
```
