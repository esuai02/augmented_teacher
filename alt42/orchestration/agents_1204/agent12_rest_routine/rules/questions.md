💬 포괄형 질문 1

“이 학생의 휴식 패턴과 회복 리듬을 종합해서, 현재 학습 지속력에 어떤 영향을 주고 있는지 분석해줘.”
→ 휴식 버튼 사용 주기, 휴식 전후 집중도 변화, 피로 누적 지수, 감정 안정도 데이터를 종합해
회복 루틴의 질적 수준(즉흥적·계획적·보상형)을 진단하고,
수업 중 휴식과 학습 전환이 자연스럽게 이어지도록 리듬형 개입 전략을 도출.

💬 포괄형 질문 2

“이 학생의 휴식 습관과 정서 반응을 기반으로, 피로 누적을 최소화하고 몰입을 회복시키는 루틴은 어떻게 최적화해야 할까?”
→ 주간 피로 상승 패턴, 감정 회복 속도, 외부 일정 영향, 휴식 중 감정 톤 변화를 반영해
개인별 회복 루틴(짧은 휴식/감각 전환/정서 안정 중심)을 맞춤 설계하고,
학습-휴식 전환을 원활하게 만드는 리듬 재조정 알고리즘을 제안.

💬 포괄형 질문 3

“이 학생이 장기적으로 안정적인 학습 리듬을 유지하기 위해, 지금부터 어떤 휴식 관리가 가장 중요할까?”
→ 장기 데이터에서 피로 누적 지수, 회복탄력성 추세, 감정루틴 일관성, 슬럼프 예측 신호를 분석해
지속적 성장을 위한 정서 회복력 강화 루틴과
‘보편적 피로 구간(예: 시험기간)’에 대비한 예방형 휴식전략을 추천.

## 🧠 1. 이 에이전트를 필요로 하는 상황

| 상황                                      | 설명                             | 분석 목적                     |
| --------------------------------------- | ------------------------------ | ------------------------- |
| **① 학습 피로 누적 의심**                       | 학생이 최근 집중 시간이 줄거나 학습효율이 하락했을 때 | 누적 피로인지 일시적 컨디션 저하인지 구분   |
| **② 휴식 버튼 사용 패턴 변화**                    | 평소보다 자주 또는 거의 사용하지 않는 경우       | 스트레스 반응 패턴 또는 과몰입 경향 탐지   |
| **③ 시험 대비기간 진입**                        | 몰입학습이 이어지며 휴식이 불균형해질 때         | 회복 탄력성 유지 및 리듬 조정 필요      |
| **④ 감정기복 감지 후 안정화 필요**                  | 감정 로그(짜증, 포기 등) 빈도가 높아진 경우     | 정서 기반 휴식 루틴 개선 및 재설계      |
| **⑤ 외부 일정(학교행사·시험 등)으로 인한 피로 상승 예상 시점** | 수면부족, 일정 과부하 등으로 인한 변동 감지      | 예방적 조정 및 루틴 보정            |
| **⑥ 귀가검사 에이전트 연동 시점**                   | 하루 종료 시 피로-집중 데이터 종합           | 다음날 루틴 피드백 생성 및 부모 보고용 요약 |
| **⑦ 장기 학습 프로젝트 진행 중**                   | 3주 이상 지속 학습에서 슬럼프 진입 가능성       | 슬럼프 예측 및 정서 회복 전략 발동      |

## 💬 2. 각 상황별 맞춤형 복합질문 예시

> (교사가 에이전트에게 던질 질문의 형태)

| 상황                     | 맞춤형 복합질문 예시                                                                              |
| ---------------------- | ---------------------------------------------------------------------------------------- |
| **① 학습 피로 누적 의심**      | “최근 3일간 학생의 휴식 전후 집중 흐름 데이터를 비교해봐. 휴식이 회복 효과를 주지 못하고 있다면 원인을 감정 로그와 시선 분포 데이터를 함께 분석해줘.” |
| **② 휴식 버튼 사용 패턴 변화**   | “이번 주 학생이 휴식 버튼을 누른 시점의 학습 밀도 변화를 분석해줘. 불안정한 패턴인지, 의식적 리듬 조절인지 구분해.”                     |
| **③ 시험 대비기간 진입**       | “시험 대비 2주차에 학생의 피로 누적 신호가 상승하는지 확인해줘. 집중도와 감정기복이 동시에 증가하면 조정 루틴을 추천해.”                   |
| **④ 감정기복 감지 후 안정화 필요** | “휴식 직전의 감정 상태와 휴식 후 감정 안정도 변화를 비교해줘. 감정 회복이 충분하지 않다면, 개인화된 회복 루틴(산책, 음악, 대화 등)을 제안해.”    |
| **⑤ 외부 일정으로 인한 피로 상승** | “학교 일정과 학원 일정이 겹친 날의 피로 데이터를 모니터링해줘. 이게 일시적 과부하인지 누적된 패턴인지 분류해.”                         |
| **⑥ 귀가검사 에이전트 연동**     | “오늘 하루의 휴식루틴 효과를 요약해서 귀가검사 에이전트로 전달해줘. 피로 누적지수와 회복점수, 감정 안정도를 포함해.”                      |
| **⑦ 장기 프로젝트 중 슬럼프 예측** | “최근 2주간 휴식의 회복효과가 점차 약화되는지 분석해줘. 슬럼프 위험 단계라면 원인군(정서/인지/환경)을 분류하고 조정안을 제안해.”              |
