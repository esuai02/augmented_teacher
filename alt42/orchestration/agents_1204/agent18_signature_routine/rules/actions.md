# Agent 18 - Signature Routine Actions

## 수행 가능한 액션 목록

### 발견
- `discover_signature_routine`: 시그너처 루틴 발견
- `analyze_time_performance`: 시간대별 성과 분석
- `find_optimal_session_length`: 최적 세션 길이 찾기

### 정교화
- `refine_routine`: 루틴 정교화
- `match_subject_to_routine`: 과목-루틴 매칭
- `fix_signature_routine`: 시그너처 루틴 고정
