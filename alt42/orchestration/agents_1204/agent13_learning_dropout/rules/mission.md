# Agent 13 - Learning Dropout Mission

## 목적/스코프
- 최근 24시간(롤링 윈도우) 동안의 학습 이탈 조짐을 조기 탐지하고 예방 개입을 설계한다
- **수학 학습 특화**: 단원별/난이도별/학원 맥락 통합 분석으로 수학 학습 이탈 원인을 정확히 파악한다
- 이탈이 빈번한 학생을 식별하고 원인을 추정하여 맞춤형 피드백을 제공한다
- 쉬운 승리 구간을 제안하여 필요한 활동으로 빠르게 재진입하도록 가이드한다

## 핵심 지표 정의(24h 롤링)

### 기본 지표
- 이탈 이벤트 수(ninactive): 5분 쿨다운 정책으로 증가하는 경고 카운터
- 지연 시청(👀): timespent ≥ 5분인 이벤트 수
- 무입력 지속시간(tlaststroke)
- 루틴 지연(nlazy): 20분당 1 블록으로 환산
- 포모도로 상태: npomodoro, kpomodoro, pmresult

### 수학 특화 지표 (추가)
- **단원별 이탈 빈도**: 현재 학습 단원에서의 이탈 발생 횟수
- **난이도별 이탈 빈도**: 기본형/유형/심화 문제별 이탈 발생 횟수
- **학습 단계별 이탈 빈도**: 개념/유형/심화/기출 단계별 이탈 발생 횟수
- **학원 맥락 지표**: 학원 수업 이해도, 학원 과제 부담
- **수학 성적 수준**: 하위권/중위권/상위권별 이탈 원인 차별화

## 위험 등급(24h)
- **Low**: ninactive ≤ 1 그리고 npomodoro ≥ 5
- **Medium**: ninactive ∈ [2,3] 또는 npomodoro ∈ [2,4] 또는 👀 ≥ 2회
- **High**: ninactive ≥ 4 또는 npomodoro < 2 또는 tlaststroke ≥ 30분 1회 이상

## 수학 특화 위험 등급 (추가)
- **단원별 위험도**: 어려운 단원(함수, 도형, 이차)에서 이탈 증가 시 Medium → High 상향
- **난이도별 위험도**: 심화 문제에서 이탈 증가 시 Medium → High 상향
- **학원 맥락 위험도**: 학원 수업 이해 못함 + 이탈 증가 시 Medium → High 상향

## 휴리스틱

### 기본 휴리스틱
- 반복적 필기 지연 → 동기/행동활성 저하 가능성 높음
- npomodoro < 5 반복 → 단기목표 루틴 미정착
- 로그인 후 목표 입력 지연/누락 → 시작 장벽 높음

### 수학 특화 휴리스틱 (추가)
- **단원별 이탈 패턴**: 함수/도형/이차 단원에서 이탈 빈번 → 난이도 조정 필요
- **난이도별 이탈 패턴**: 심화 문제에서 이탈 증가 → 기본형 문제로 전환 필요
- **학습 단계별 이탈 패턴**: 
  - 개념 학습 중 이탈 → 설명 보강 필요 (시각 자료/예시 활용)
  - 문제 풀이 중 이탈 → 쉬운 문제로 전환 필요
- **학원 맥락 이탈 패턴**: 
  - 학원 수업 이해 못함 → 자습 시 이탈 증가 → 예습/복습 강화 필요
  - 학원 과제 과다 → 집에서 이탈 증가 → 과제 우선순위 조정 필요
- **수준별 이탈 원인**:
  - 하위권: 너무 어려워서 포기 → 쉬운 문제부터
  - 중위권: 집중력 부족 → 루틴 강화
  - 상위권: 권태로움 → 도전 과제 제시

## 개입 액션 플레이북

### 기본 개입
- 즉시 개입: 짧은 리포커스 메시지 전송, 쉬운 승리 태스크 제시
- 루틴 보정: 세션 길이 15→10분 시범, 휴식 알람 50→40분
- 에스컬레이션: High 등급 2일 연속 시 보호자/담임 알림

### 수학 특화 개입 (추가)
- **단원별 개입**: 어려운 단원에서 이탈 시 기본 개념 재학습 제안
- **난이도별 개입**: 심화 문제에서 이탈 시 기본형 문제로 전환 제안
- **학습 단계별 개입**: 
  - 개념 학습 이탈 → 시각 자료/예시 활용 설명 보강
  - 문제 풀이 이탈 → 한 단계 쉬운 문제로 전환
- **학원 맥락 개입**: 
  - 학원 수업 이해 못함 → 학원 교재 개념 부분 재학습
  - 학원 과제 과다 → 과제 우선순위 조정, 쉬운 과제부터 완료
- **수준별 개입**:
  - 하위권: 가장 쉬운 기본형 문제 1개 완료 → 성취감 → 다음 문제
  - 중위권: 10분 세션 완료 → 1분 휴식 → 다음 세션 (루틴 강화)
  - 상위권: 창의 문제/변형 문제 도전 → 흥미 유발 → 재진입

## 데이터 수집 계획 (현재 데이터 없는 경우)

### 학생 설문 항목
1. "현재 학습 중인 수학 단원은 무엇인가요?"
2. "현재 풀고 있는 문제 난이도는? (A) 기본형 (B) 유형 (C) 심화"
3. "현재 어떤 학습 단계인가요? (A) 개념 학습 (B) 유형 연습 (C) 심화 문제 (D) 기출 문제"
4. "오늘 학원 수업을 이해하셨나요? (A) 완전히 이해 (B) 대부분 이해 (C) 반만 이해 (D) 거의 이해 못함"
5. "오늘 학원 과제 양이 적절하신가요? (A) 적절함 (B) 조금 많음 (C) 너무 많음"
6. "현재 수학 성적 수준은? (A) 하위권 (B) 중위권 (C) 상위권"

### 선생님 체크리스트 항목
1. "단원별 이탈 패턴: [단원명] 단원에서 이탈 빈번 → [대응 전략]"
2. "난이도별 이탈 패턴: [난이도] 문제에서 이탈 증가 → [대응 전략]"
3. "학원 수업 이해도: 이해 못함 → 이탈 패턴 분석 → 예습/복습 강화"
4. "학원 과제 부담: 과제 과다 → 이탈 증가 → 과제 우선순위 조정"

### 선생님 텍스트 입력 예시
- "함수 단원에서 이탈 빈번 → 난이도 조정 필요"
