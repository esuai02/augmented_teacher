## 🆕 추가된 룰 상세

### S8: 수학 학습 특화 감정 룰 (R44 ~ R52)

#### R44: 수학 문제 유형별 감정 패턴 감지
- **목적**: 계산/개념/응용/증명 문제별 감정 패턴 분류
- **설문 항목**: "어떤 유형의 문제를 풀 때 가장 자주 막히나요?"
- **데이터 수집 방법**: 학생 설문
- **기대 효과**: 문제 유형별 맞춤형 피드백 제공

#### R45: 계산 실수 vs 개념 오류 감정 차이 매핑
- **목적**: 오류 유형별 감정 반응 패턴 분류
- **설문 항목**: "문제를 틀렸을 때, 계산 실수였을 때와 개념을 몰랐을 때 감정이 다르나요?"
- **데이터 수집 방법**: 학생 설문 + 오답노트 데이터 분석
- **기대 효과**: 오류 유형별 맞춤형 학습 전략 제안

#### R46: 수학 단원별 감정 패턴 분석
- **목적**: 단원별 어려움/흥미도/성취감 감정 패턴 분류
- **설문 항목**: "어떤 단원에서 가장 어려움을 느끼나요?"
- **데이터 수집 방법**: 학생 설문 + 학습 로그 분석
- **기대 효과**: 단원별 맞춤형 학습 전략 제안

#### R47: 난이도별 감정 반응 패턴 분석
- **목적**: 기본/유형/심화 문제별 감정 반응 분류
- **설문 항목**: "어려운 문제를 마주했을 때 어떤 감정이 드나요?"
- **데이터 수집 방법**: 학생 설문
- **기대 효과**: 난이도 조정 전략 제안

#### R48: 학원 맥락 감정 요인 분석
- **목적**: 학원 수업 이해도, 학원 친구 비교, 학원 과제 부담감 감정 패턴
- **설문 항목**: 
  - "학원 수업을 이해하지 못했을 때 어떤 감정이 드나요?"
  - "학원에서 다른 친구들보다 문제를 늦게 풀 때 어떤 기분이 드나요?"
  - "학원 과제가 많을 때 집에서 공부할 때 어떤 감정이 드나요?"
- **데이터 수집 방법**: 학생 설문 + 학원 정보 연동
- **기대 효과**: 학원-집 학습 연계 최적화

#### R49: 풀이 과정별 감정 변화 추적
- **목적**: 문제 읽기 → 풀이 시작 → 중간 계산 → 답 확인 단계별 감정 추적
- **설문 항목**: "문제를 풀 때, 어느 단계에서 가장 스트레스를 받나요?"
- **데이터 수집 방법**: 학생 설문 + 실시간 학습 로그 분석
- **기대 효과**: 취약 단계 파악 및 맞춤형 지원

#### R50: 시간 압박 감정 패턴 분석
- **목적**: 시험 시간 부족 시 감정 패턴 분류
- **설문 항목**: "시간이 부족할 때 어떤 감정이 드나요?"
- **데이터 수집 방법**: 학생 설문 + 시험 로그 분석
- **기대 효과**: 시험 대비 전략 및 침착도 관리

#### R51: 단원 간 감정 전이 패턴 분석
- **목적**: 이전 단원 실패 → 다음 단원 두려움 패턴 분석
- **설문 항목**: "이전 단원을 못하면 다음 단원에 대한 감정이 달라지나요?"
- **데이터 수집 방법**: 학생 설문 + 단원 성취도 데이터 분석
- **기대 효과**: 선행 단원 보완 전략 제안

#### R52: 복합 페르소나 프로필 구축
- **목적**: 활동별 페르소나를 통합한 학생별 통합 페르소나 프로필
- **데이터 수집 방법**: 활동별 페르소나 자동 수집 (최소 3개 활동)
- **기대 효과**: 개인화된 학습 경험 설계

## 📋 학생 설문 데이터 가정

### 설문 1: 수학 문제 유형별 감정
**질문**: "어떤 유형의 문제를 풀 때 가장 자주 막히나요?"

**가정 응답 분포** (100명 설문 기준):
- 계산 문제: 25명 (25%)
  - 주요 감정: 계산 실수 두려움, 집중력 필요
- 개념 이해 문제: 35명 (35%)
  - 주요 감정: 이해도 부족 좌절, 개념 재학습 필요
- 응용 문제: 30명 (30%)
  - 주요 감정: 도전 의욕 또는 두려움
- 증명 문제: 10명 (10%)
  - 주요 감정: 논리적 사고 즐거움 또는 좌절

**분석 결과**:
- 개념 이해 문제에서 막히는 비율이 가장 높음 (35%)
- 응용 문제에서의 감정이 이분화됨 (도전 의욕 vs 두려움)
- 증명 문제는 소수만 경험하지만 경험자들의 감정 강도가 높음

### 설문 2: 오류 유형별 감정 차이
**질문**: "문제를 틀렸을 때, 계산 실수였을 때와 개념을 몰랐을 때 감정이 다르나요?"

**가정 응답 분포**:
- 계산 실수: 40명 (40%)
  - 주요 감정: 아쉬움 (60%), 다음에 조심해야겠다는 생각 (30%), 집중력 상승 (10%)
- 개념 오류: 35명 (35%)
  - 주요 감정: 좌절감 (50%), 개념 재학습 필요성 인식 (40%), 포기하고 싶음 (10%)
- 문제 오독: 15명 (15%)
  - 주요 감정: 짜증 (50%), 다음에 꼼꼼히 읽어야겠다는 생각 (40%), 자책 (10%)
- 시간 부족: 10명 (10%)
  - 주요 감정: 긴장감 (60%), 시간 관리 필요성 인식 (30%), 체념 (10%)

**분석 결과**:
- 계산 실수는 비교적 경미한 감정 반응 (아쉬움 수준)
- 개념 오류는 강한 부정적 감정 반응 (좌절감)
- 시간 부족은 긴장감 증가로 이어짐

- 포기하고 싶어요: 15명 (15%)

- 나만 못하는 것 같아요: 20명 (20%)
- 괜찮아요, 나만의 속도가 있어요: 15명 (15%)
