## 페르소나 구조 설명

각 페르소나는 다음 요소로 구성됩니다:

| 요소 | 설명 |
|------|------|
| **ID** | 진정도레벨_유형번호 (예: C95_P1, C_crisis_P3) |
| **별칭** | 직관적으로 이해할 수 있는 유형명 |
| **진정도 범위** | 해당 레벨의 점수 범위 |
| **심리적 특성** | 내면의 감정, 생각, 불안 요인 |
| **행동적 특성** | 외부로 드러나는 행동 패턴 |
| **신체적 특성** | 관찰 가능한 신체 반응 |
| **전형적 발화** | 해당 유형이 자주 하는 말 |
| **권장 개입** | 최적의 개입 전략 및 호흡/그라운딩 기법 |

### 진정도(Calmness) 레벨 체계

| 레벨 | 점수 범위 | 상태 설명 | 개입 우선순위 |
|------|----------|----------|--------------|
| **C95** | 95-100 | 매우 안정적, 최적 학습 상태 | ProgressRecognition |
| **C90** | 90-94 | 안정적, 양호한 학습 상태 | SkillBuilding |
| **C85** | 85-89 | 적당히 안정, 가벼운 긴장 | MindfulnessSupport |
| **C80** | 80-84 | 약간 불안, 주의 필요 | FocusGuidance |
| **C75** | 75-79 | 불안, 적극적 지원 필요 | EmotionalSupport |
| **C_crisis** | <75 | 위기, 즉각적 개입 필요 | CrisisIntervention |

## C95: 매우 안정적 상태 페르소나

> **점수 범위**: 95-100점
> **기본 개입**: ProgressRecognition, InformationProvision

### C95_P1: 마음챙김 마스터 🧘

| 항목 | 내용 |
|------|------|
| **심리적 특성** | 자기 감정을 잘 인식하고 조절함. 스트레스 상황에서도 평정심 유지. 내면의 평화를 중시함 |
| **행동적 특성** | 차분하고 일관된 학습 패턴. 어려운 문제에도 침착하게 접근. 실패에도 감정 흔들림 없음 |
| **신체적 특성** | 이완된 자세, 규칙적 호흡, 안정된 시선. 표정이 부드럽고 긴장 없음 |
| **전형적 발화** | "괜찮아요, 천천히 해볼게요", "어려워도 할 수 있어요", "모르면 다시 배우면 되죠" |
| **권장 개입** | 현재 상태 인정 및 강화. 고급 집중력 기법 소개. 다른 학생 멘토링 기회 제공 |

### C95_P2: 자연스러운 낙관주의자 ☀️

| 항목 | 내용 |
|------|------|
| **심리적 특성** | 긍정적 관점을 자연스럽게 유지. 실패를 성장 기회로 봄. 미래에 대한 희망적 태도 |
| **행동적 특성** | 밝은 표정으로 학습 참여. 어려움을 가볍게 받아들임. 주변에 긍정적 에너지 전파 |
| **신체적 특성** | 열린 자세, 자연스러운 미소, 활기찬 에너지. 신체 긴장 거의 없음 |
| **전형적 발화** | "재미있네요!", "이것도 배울 수 있어서 좋아요", "실수해도 다음에 더 잘하면 되죠" |
| **권장 개입** | 긍정성 인정 및 강화. 도전적 과제 제공. 리더십 역할 부여 |

### C95_P3: 차분한 분석가 📊

| 항목 | 내용 |
|------|------|
| **심리적 특성** | 감정과 사고를 분리해서 처리. 객관적 자기 관찰 능력 보유. 논리적 접근 선호 |
| **행동적 특성** | 문제 상황에서 감정보다 해결책에 집중. 자신의 상태를 명확하게 표현함 |
| **신체적 특성** | 안정적 호흡, 균형 잡힌 자세, 집중된 시선. 과도한 움직임 없음 |
| **전형적 발화** | "지금 제 상태는 괜찮아요", "이 부분이 어려운 건 이해가 안 돼서예요", "차근차근 해볼게요" |
| **권장 개입** | 자기 조절 능력 인정. 메타인지 전략 심화. 문제 해결 중심 접근 유지 |

## C90: 안정적 상태 페르소나

> **점수 범위**: 90-94점
> **기본 개입**: SkillBuilding, InformationProvision

### C90_P1: 균형 잡힌 학습자 ⚖️

| 항목 | 내용 |
|------|------|
| **심리적 특성** | 학습과 휴식의 균형을 잘 맞춤. 적당한 긴장감을 긍정적 동기로 활용. 자기 관리 능력 양호 |
| **행동적 특성** | 규칙적인 학습 습관. 어려움이 있어도 꾸준히 시도. 필요시 도움 요청 |
| **신체적 특성** | 편안한 자세, 정상적 호흡. 가끔 가벼운 긴장 표현이 있으나 금방 해소 |
| **전형적 발화** | "좀 어려운데 해볼게요", "잠깐 쉬고 다시 할게요", "도움이 필요할 것 같아요" |
| **권장 개입** | 자기 조절 전략 강화. 어려운 상황 대처 기술 교육. 현재 상태 유지 격려 |

### C90_P2: 성실한 노력가 📚

| 항목 | 내용 |
|------|------|
| **심리적 특성** | 꾸준한 노력을 중시. 결과보다 과정에 집중. 작은 성취에도 만족감 느낌 |
| **행동적 특성** | 계획대로 학습 진행. 실패해도 재시도. 점진적 향상에 동기부여 |
| **신체적 특성** | 안정적 자세 유지. 집중 시 살짝 긴장되나 정상 범위. 규칙적 호흡 |
| **전형적 발화** | "조금씩 나아지고 있어요", "어제보다 나은 것 같아요", "계속 하다 보면 될 거예요" |
| **권장 개입** | 성장 과정 인정 및 기록. 효과적 학습 전략 공유. 지속적 격려 |

### C90_P3: 사교적 협력자 🤝

| 항목 | 내용 |
|------|------|
| **심리적 특성** | 관계를 통해 안정감을 느낌. 함께 공부할 때 더 편안함. 소속감 중시 |
| **행동적 특성** | 또래와 협력 학습 선호. 어려울 때 대화로 해소. 교사와 긍정적 관계 형성 |
| **신체적 특성** | 열린 자세, 적절한 눈 맞춤, 자연스러운 표정. 상호작용 시 편안함 |
| **전형적 발화** | "같이 하면 더 재미있어요", "선생님 도와주세요", "친구랑 얘기하면 기분이 나아져요" |
| **권장 개입** | 협력 학습 기회 제공. 사회적 지지 네트워크 강화. 관계 기반 격려 |

## C85: 적당히 안정 상태 페르소나

> **점수 범위**: 85-89점
> **기본 개입**: MindfulnessSupport, FocusGuidance

### C85_P1: 가벼운 긴장형 😊

| 항목 | 내용 |
|------|------|
| **심리적 특성** | 약간의 긴장감을 느끼지만 통제 가능. 어려운 과제에 살짝 불안. 도움으로 빠르게 안정됨 |
| **행동적 특성** | 시작 전 잠깐 망설임. 진행하면서 점차 안정됨. 확인 질문을 가끔 함 |
| **신체적 특성** | 가벼운 어깨 긴장, 살짝 빠른 호흡. 집중하면 정상화됨 |
| **전형적 발화** | "좀 긴장되네요", "맞게 하고 있는 거죠?", "이거 어렵지 않아요?" |
| **권장 개입** | 심호흡 한 번 권유. 긍정적 피드백으로 안심. 4-7-8 호흡법 가볍게 소개 |

### C85_P2: 완벽주의적 신중함 🎯

| 항목 | 내용 |
|------|------|
| **심리적 특성** | 실수하고 싶지 않은 마음. 정확성 추구로 인한 가벼운 압박. 자기 검열 경향 |
| **행동적 특성** | 답하기 전 오래 생각함. 여러 번 확인 후 제출. 실수에 민감하게 반응 |
| **신체적 특성** | 집중 시 이마 찡그림. 손에 살짝 힘이 들어감. 체크 행동 반복 |
| **전형적 발화** | "다시 확인해볼게요", "이게 맞아요?", "틀리면 어떡해요..." |
| **권장 개입** | 완벽하지 않아도 된다는 메시지. 과정 칭찬. 박스 호흡법으로 긴장 완화 |

### C85_P3: 에너지 관리 필요형 🔋

| 항목 | 내용 |
|------|------|
| **심리적 특성** | 컨디션에 따라 안정도가 달라짐. 피로 시 감정 조절이 약해짐. 자원 관리 필요 |
| **행동적 특성** | 시간대별 집중력 차이. 피곤하면 짜증 증가. 휴식 필요성 인식 |
| **신체적 특성** | 가끔 하품, 자세 흐트러짐, 눈 피로감 호소 |
| **전형적 발화** | "오늘 좀 피곤해요", "잠깐 쉬어도 될까요?", "아침에는 더 잘 되는데..." |
| **권장 개입** | 에너지 관리 전략 교육. 적절한 휴식 허용. 깊은 복식 호흡으로 활력 회복 |

## C80: 약간 불안 상태 페르소나

> **점수 범위**: 80-84점
> **기본 개입**: FocusGuidance, EmotionalSupport

### C80_P1: 불안 인식형 😰

| 항목 | 내용 |
|------|------|
| **심리적 특성** | 자신의 불안을 인식하고 있음. 걱정이 많지만 표현할 줄 앎. 도움받고 싶은 욕구 |
| **행동적 특성** | 불안을 언어로 표현. "걱정돼요" 발언 빈번. 안심시켜주면 나아짐 |
| **신체적 특성** | 자주 한숨, 어깨 긴장, 손 만지작거림. 불안한 표정 |
| **전형적 발화** | "걱정돼요", "잘 할 수 있을까요?", "불안해요", "어떻게 해야 해요?" |
| **권장 개입** | 감정 수용 및 공감. 4-7-8 호흡법 실습. 구체적 안심 메시지 제공 |

### C80_P2: 산만한 회피형 🦋

| 항목 | 내용 |
|------|------|
| **심리적 특성** | 불안을 회피하려고 다른 곳에 집중. 어려운 것 직면을 피함. 분산으로 불안 관리 시도 |
| **행동적 특성** | 주제 이탈, 잡담 시도, 휴대폰 확인. 과제 미루기. 집중 시간 짧음 |
| **신체적 특성** | 안절부절못함, 자세 자주 바꿈, 시선 분산 |
| **전형적 발화** | "잠깐만요", "그건 나중에 해도 되죠?", "다른 거 먼저 할게요" |
| **권장 개입** | 집중 훈련 (물체 집중 그라운딩). 작은 단위로 과제 분할. 타이머 활용 제안 |

### C80_P3: 비교 불안형 📉

| 항목 | 내용 |
|------|------|
| **심리적 특성** | 다른 사람과 비교하며 불안해짐. 뒤처진다는 두려움. 자존감 흔들림 |
| **행동적 특성** | 다른 학생 진도 확인. "남들은 어때요?" 질문. 결과에 민감 |
| **신체적 특성** | 긴장된 표정, 주변 살피기, 초조한 손동작 |
| **전형적 발화** | "다른 애들은 잘해요?", "저만 못하는 거 아니에요?", "평균이 어때요?" |
| **권장 개입** | 개인 성장 초점 유도. 비교가 아닌 자기 발전 강조. 안전한 장소 시각화로 자존감 회복 |

### C80_P4: 숨기는 불안형 🎭

| 항목 | 내용 |
|------|------|
| **심리적 특성** | 불안을 드러내고 싶지 않음. 괜찮은 척함. 내면의 긴장을 숨김 |
| **행동적 특성** | "괜찮아요" 반복. 도움 거절. 혼자 해결하려 함. 감정 표현 회피 |
| **신체적 특성** | 표정과 신체 긴장의 불일치. 주먹 꽉 쥠. 억지 미소 |
| **전형적 발화** | "괜찮아요, 진짜로", "혼자 할 수 있어요", "별거 아니에요" |
| **권장 개입** | 안전한 환경 조성으로 표현 유도. 감정 표현의 가치 전달. 신체 스캔으로 자기 인식 촉진 |

## C75: 불안 상태 페르소나

> **점수 범위**: 75-79점
> **기본 개입**: EmotionalSupport, CalmnessCoaching

### C75_P1: 떨리는 긴장형 😨

| 항목 | 내용 |
|------|------|
| **심리적 특성** | 강한 긴장감으로 정상 기능 어려움. 실패에 대한 공포. 회피하고 싶은 충동 |
| **행동적 특성** | 말이 빨라지거나 더듬음. 집중력 현저히 저하. 실수 증가 |
| **신체적 특성** | 손 떨림, 땀, 빠른 심장박동, 호흡 얕아짐. 안색 변화 |
| **전형적 발화** | "못하겠어요", "너무 떨려요", "머리가 하얘졌어요", "숨이 안 쉬어져요" |

## 페르소나 활용 가이드

### 진정도 수준 식별 방법

1. **초기 관찰**: 신체적 신호 (호흡, 자세, 표정) 확인
2. **언어적 단서**: 발화 내용과 말하는 방식 분석
3. **행동 패턴**: 회피, 참여도, 집중력 수준 평가
4. **맥락 고려**: 상황(시험, 발표 등)과 개인 이력 참고

### 개입 원칙

| 원칙 | 설명 |
|------|------|
| **안전 우선** | 위기 상황에서는 학습보다 안전이 최우선 |
| **공감 기반** | 모든 감정을 수용하고 비판하지 않음 |
| **점진적 접근** | 작은 성공에서 시작하여 자신감 구축 |
| **개별화** | 같은 점수라도 유형에 따라 다른 개입 |
| **지속성** | 일회성이 아닌 꾸준한 지원과 모니터링 |

### 호흡/그라운딩 기법 선택 가이드

| 진정도 수준 | 권장 기법 |
|------------|----------|
| C95-C90 | 마음챙김 명상, 고급 호흡법 |
| C85 | 4-7-8 호흡법, 간단한 그라운딩 |
| C80 | 박스 호흡법, 물체 집중 |
| C75 | 깊은 복식 호흡, 5-4-3-2-1 감각 그라운딩 |
| C_crisis | **즉시 안정화** - 함께 호흡, 신체 스캔, 안전한 장소 시각화 |

## 온톨로지 연계

각 페르소나는 다음 온톨로지 클래스와 연계됩니다:

```yaml
mk:CalmnessPersona:
  - mk:hasCalmnessLevel      # 진정도 레벨 (C95, C90, C85, C80, C75, C_crisis)
  - mk:hasEmotionalState     # 감정 상태
  - mk:hasAnxietyTrigger     # 불안 유발 요인
  - mk:hasRecoveryPattern    # 회복 패턴
  - mk:hasBehaviorPattern    # 행동 패턴
  - mk:hasPhysicalSymptoms   # 신체 증상
  - mk:requiresIntervention  # 필요 개입 유형
  - mk:prefersBreathingExercise  # 선호 호흡 기법
  - mk:prefersGroundingExercise  # 선호 그라운딩 기법
```
