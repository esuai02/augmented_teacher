# Strategy Step 2: 복습 전략 설계

## 목적
오답 패턴 분석 결과를 바탕으로 학생에게 최적화된 복습 전략을 설계하고, 복습 효과를 검증하여 지속적으로 개선한다.

---

## 1. 복습 우선순위 산정

### 1.1 단원별 복습 우선순위
```
우선순위 점수 = (오답률 * 0.4) + (오답 수 * 0.3) + (최근성 * 0.2) + (시험 중요도 * 0.1)

- 오답률: 해당 단원의 오답 비율
- 오답 수: 해당 단원의 총 오답 수
- 최근성: 최근 오답 발생 시각 (최근일수록 높은 점수)
- 시험 중요도: 시험 범위에 포함되는지 여부 (agent02 연계)
```

### 1.2 오류 유형별 복습 우선순위
```
우선순위 점수 = (오류 빈도 * 0.5) + (오류 심각도 * 0.3) + (개선 가능성 * 0.2)

- 오류 빈도: 해당 오류 유형 발생 횟수
- 오류 심각도: 개념 오해 > 공식 오용 > 계산 실수 > 부주의한 실수
- 개선 가능성: 노트 품질이 높으면 개선 가능성 높음
```

### 1.3 복합 우선순위 결정
```
최종 우선순위 = 단원별 우선순위 * 오류 유형별 우선순위 가중치

예시:
- 함수 단원 + 개념 오해 → 최우선 복습
- 통계 단원 + 계산 실수 → 중간 우선순위
```

## 2. 오류 유형별 복습 전략

### 2.1 계산 실수 (calculation_error)
```
복습 전략: 반복 연습 (repetitive_practice)
복습 주기: 3일, 7일, 14일

1단계 (3일 후):
- 동일 유형 문제 5-10문제 재풀이
- 계산 과정을 천천히 다시 확인
- 부호, 단위 변환 등 실수하기 쉬운 부분 강조

2단계 (7일 후):
- 유사 문제 10-15문제 풀이
- 시간 제한 두고 풀이 (속도 향상)
- 오답 노트에 계산 실수 원인 기록

3단계 (14일 후):
- 통합 문제 풀이
- 실제 시험과 유사한 환경에서 풀이
- 계산 실수 없이 풀 수 있는지 검증
```

### 2.2 개념 오해 (concept_misunderstanding)
```
복습 전략: 개념 재학습 (concept_relearning)
복습 주기: 7일, 14일, 30일

1단계 (7일 후):
- 개념 설명 다시 읽기
- 개념의 핵심을 한 문장으로 정리
- 개념 적용 예시 문제 풀이

2단계 (14일 후):
- 개념을 다르게 설명하는 자료 찾아보기
- 개념과 관련된 실생활 예시 찾기
- 개념 이해도 확인 문제 풀이

3단계 (30일 후):
- 개념 연결 문제 풀이 (다른 단원과 연계)
- 개념을 활용한 응용 문제 풀이
- 개념 이해도 종합 평가
```

### 2.3 문제 이해 오류 (problem_understanding_error)
```
복습 전략: 유사 문제 연습 (similar_problem_practice)
복습 주기: 3일, 10일

1단계 (3일 후):
- 동일 문제 조건 해석 연습
- 핵심 키워드 찾기 연습
- 문제 조건을 단계별로 정리하는 연습

2단계 (10일 후):
- 유사한 문제 유형 10-15문제 풀이
- 문제 조건 해석 연습
- 문제 이해도 확인 문제 풀이
```

### 2.4 공식 오용 (formula_misapplication)
```
복습 전략: 공식 재검토 및 연습 (formula_review_and_practice)
복습 주기: 5일, 12일, 21일

1단계 (5일 후):
- 공식 다시 외우기
- 공식 사용 조건 정리
- 공식 적용 예시 문제 풀이

2단계 (12일 후):
- 공식 변형 문제 풀이
- 공식 적용 조건 확인 문제 풀이
- 공식 사용 연습 문제 풀이

3단계 (21일 후):
- 공식 종합 문제 풀이
- 다른 공식과의 연계 문제 풀이
- 공식 활용도 종합 평가
```

### 2.5 적용 실수 (application_error)
```
복습 전략: 단계별 재검토 (step_by_step_review)
복습 주기: 4일, 10일, 20일

1단계 (4일 후):
- 풀이 단계를 하나씩 다시 확인
- 단계별 검증 연습
- 단계 생략하지 않고 풀이하는 연습

2단계 (10일 후):
- 유사 문제 단계별 풀이 연습
- 단계별 오류 확인 연습
- 단계별 풀이 습관 형성

3단계 (20일 후):
- 통합 문제 단계별 풀이
- 단계별 풀이 습관 유지 확인
- 단계별 풀이 종합 평가
```

## 3. 복습 주기 설계 (스페이싱)

### 3.1 에빙하우스 망각 곡선 기반
```
기본 복습 주기:
- 1일 후: 즉시 복습 (서술평가)
- 3일 후: 1차 복습
- 7일 후: 2차 복습
- 14일 후: 3차 복습
- 30일 후: 4차 복습
```

### 3.2 오류 유형별 조정
```
- 계산 실수: 3일, 7일, 14일 (빠른 주기)
- 개념 오해: 7일, 14일, 30일 (느린 주기)
- 문제 이해 오류: 3일, 10일 (중간 주기)
```

### 3.3 학생 수준별 조정
```
- 하위권: 기본 주기보다 빠르게 (2일, 5일, 12일, 25일)
- 중위권: 기본 주기 유지
- 상위권: 기본 주기보다 느리게 (5일, 12일, 21일, 40일)
```

## 4. 복습 방법 설계

### 4.1 단원별 복습 방법
```
함수 단원:
- 개념 재학습 → 기본 문제 → 유형 문제 → 심화 문제
- 그래프와 함께 학습
- 실생활 예시 활용

통계 단원:
- 개념 간단히 확인 → 문제 풀이 중심
- 계산 실수 주의
- 데이터 해석 연습

도형 단원:
- 증명 과정 다시 확인
- 도형 성질 정리
- 증명 문제 연습
```

### 4.2 학생 수준별 복습 방법
```
하위권:
- 개념 중심 (60%) + 유형 연습 (30%) + 기출 (10%)
- 반복 연습 강조
- 단계별 설명

중위권:
- 개념 확인 (30%) + 유형 연습 (50%) + 심화 (10%) + 기출 (10%)
- 유형별 연습 강조
- 오답 정리

상위권:
- 개념 확인 (10%) + 유형 연습 (30%) + 심화 (40%) + 기출 (20%)
- 심화 문제 강조
- 변형 문제 연습
```

## 5. 복습 효과 검증

### 5.1 정답 전환율 계산
```
정답 전환율 = (복습 후 정답 수 / 복습 전 오답 수) * 100

- 80% 이상: 매우 효과적
- 60-80%: 효과적
- 40-60%: 보통
- 40% 미만: 비효과적 (전략 수정 필요)
```

### 5.2 노트 품질 개선 여부
```
복습 전후 노트 품질 점수 비교:
- reflection_score 변화
- completeness_score 변화
- overall_score 변화

개선 여부:
- 점수 증가: 복습 효과 있음
- 점수 유지: 복습 효과 보통
- 점수 감소: 복습 효과 없음 (전략 수정 필요)
```

### 5.3 이해도 검증 (서술평가)
```
서술평가 결과 분석:
- 해설지 없이 동일 문제 재풀이
- 정답 여부 확인
- 풀이 과정 정확성 확인

이해도 점수:
- 정답 + 풀이 과정 정확: 100점
- 정답 + 풀이 과정 부분 정확: 80점
- 오답 + 풀이 과정 부분 정확: 50점
- 오답 + 풀이 과정 부정확: 0점
```

출력:

최적화 목표:

- 동기 부여 메시지

형식: 리포트 형태 (요약 카드)

## 8. 복습 전략 조정

### 8.1 효과 없는 전략 수정
```
조건:
- 정답 전환율 < 40%
- 노트 품질 개선 없음
- 이해도 점수 개선 없음

조정 방법:
- 복습 주기 변경
- 복습 방법 변경
- 복습 강도 조정
```

### 8.2 효과 있는 전략 강화
```
조건:
- 정답 전환율 > 80%
- 노트 품질 지속적 개선
- 이해도 점수 지속적 개선

강화 방법:
- 복습 주기 확대 (더 긴 주기로)
- 복습 강도 증가 (더 많은 문제)
- 다음 단계로 진행
```

## 10. 복습 전략 모니터링

### 10.1 복습 이행률 추적
```
복습 이행률 = (실제 복습 수 / 계획된 복습 수) * 100

- 90% 이상: 우수
- 70-90%: 양호
- 50-70%: 보통
- 50% 미만: 개선 필요
```

### 10.2 복습 효과 지속성 추적
```
장기 복습 효과 확인:
- 복습 후 1개월 이상 기억 유지 여부
- 다른 단원 문제 해결 능력 향상 여부
- 전반적 성적 향상 여부
```
