# 상황별 상호작용 타겟팅 전략 상세

> **목적**: 9가지 상황별 구체적인 타겟팅 전략 및 사례  
> **활용**: Strategy Step 2의 세부 전략 보완

---

## 1. 실시간 고민 상황

### 1.1 상황 정의
학생이 즉시 질문하거나 도움을 요청하는 상황

### 1.2 타겟팅 전략

#### 즉시 개입 원칙
- **타이밍**: 즉시 (최대 30초 이내)
- **우선순위**: 최우선
- **응답 속도**: 실시간

#### 상황별 모드 선택

**질문 유형: 개념 이해**
```yaml
IF (질문_유형 == "개념_이해") THEN
    모드 = "도제학습"
    전략 = {
        접근: "단계별 설명",
        내용: "개념 설명 → 예제 → 학생 이해 확인",
        톤: "친절하고 명확한 설명"
    }
END IF
```

**질문 유형: 문제 풀이 막힘**
```yaml
IF (질문_유형 == "문제_풀이_막힘") THEN
    모드 = "도제학습" 또는 "단기미션"
    
    IF (학생_수준 == "하위권") THEN
        모드 = "도제학습"  # 단계별 가이드
        전략 = "풀이 과정 단계별 안내"
    ELSE
        모드 = "단기미션"  # 힌트 제공
        전략 = "힌트 제공 → 학생 스스로 풀이 유도"
    END IF
END IF
```

**질문 유형: 학습 방법 고민**
```yaml
IF (질문_유형 == "학습_방법_고민") THEN
    모드 = "맞춤학습" 또는 "자기성찰"
    
    전략 = {
        접근: "학생 상황 분석 → 맞춤 방법 제시",
        내용: "현재 수준 파악 → 개선 방안 제시",
        톤: "공감하고 격려하는 톤"
    }
END IF
```

**질문 유형: 감정/동기 문제**
```yaml
IF (질문_유형 == "감정_동기_문제") THEN
    모드 = "자기성찰" 또는 "맞춤학습"
    
    전략 = {
        접근: "감정 공감 → 원인 파악 → 해결 방안",
        내용: "학생 감정 인정 → 동기 회복 방법 제시",
        톤: "따뜻하고 공감하는 톤"
    }
END IF
```

### 1.3 수학 특화 고려사항

**단원별 응답 전략**
- 함수 단원: 추상적 개념 → 시각적 예시 활용
- 통계 단원: 실용적 예시 → 일상 생활 연결
- 도형 단원: 시각적 설명 → 그림/도형 활용

**오류 유형별 응답**
- 계산 실수: 즉시 지적 → 정확도 강조
- 개념 오류: 개념 재설명 → 이해 확인
- 풀이 전략 오류: 올바른 전략 제시 → 비교 설명

## 2. 포모도르 상황 중 선택

### 2.1 상황 정의
포모도르 타이머가 실행 중인 학습 세션 내 상호작용

### 2.2 세션 단계별 타겟팅

#### 시작 단계 (0-5분)
```yaml
타겟팅_전략 = {
    목적: "학습 진입 촉진",
    모드: "단기미션",
    내용: "오늘 목표 확인, 첫 작은 목표 설정",
    톤: "격려하고 동기 부여"
}
```

#### 집중 단계 (5-20분)
```yaml
타겟팅_전략 = {
    목적: "집중 유지",
    모드: "자기주도",
    내용: "최소 개입, 집중 방해 안 함",
    조건: "집중도 유지 시 개입 안 함"
}
```

#### 집중도 하락 단계 (20분 이후)
```yaml
IF (포모도르_경과 >= 20분 AND 집중도 < 5) THEN
    타겟팅_전략 = {
        목적: "집중도 회복 또는 휴식 유도",
        모드: "시간성찰",
        내용: "휴식 권장 또는 짧은 동기 부여",
        톤: "부드럽고 이해하는 톤"
    }
END IF
```

#### 성취감 저하 단계
```yaml
IF (포모도르_경과 >= 15분 AND 성취감 < 3) THEN
    타겟팅_전략 = {
        목적: "동기 부여",
        모드: "단기미션",
        내용: "작은 성취 강조, 다음 목표 제시",
        톤: "격려하고 성취 인정"
    }
END IF
```

### 2.3 수학 학습 단계별 포모도르 타겟팅

**개념 학습 중**
- 모드: 도제학습
- 개입 시점: 개념 설명 후 이해 확인
- 내용: 개념 정리, 예제 풀이 가이드

**유형 연습 중**
- 모드: 단기미션
- 개입 시점: 유형별 목표 달성 시
- 내용: 성취 인정, 다음 유형 제시

**심화 문제 중**
- 모드: 자기성찰
- 개입 시점: 막힘 시 (5분 이상)
- 내용: 힌트 제공, 풀이 전략 안내

## 3. 수업준비 상황

### 3.1 학원 수업 준비

#### 수업 전 2시간 이내
```yaml
타겟팅_전략 = {
    모드: ["맞춤학습", "도제학습", "단기미션"],
    목적: "예습 가이드",
    내용: {
        개념_미리보기: "오늘 배울 단원 개념 요약",
        기본_문제: "기본 문제 3-5개 풀어보기",
        질문_준비: "궁금한 점 정리하기"
    },
    타이밍: "수업 2시간 전까지"
}
```

#### 수업 직전 (30분 이내)
```yaml
타겟팅_전략 = {
    모드: "시간성찰",
    목적: "수업 준비 완료 확인",
    내용: {
        준비물_확인: "교재, 필기구 준비",
        마음_준비: "집중 준비, 긍정적 마인드셋",
        목표_설정: "오늘 수업에서 배울 내용 목표 설정"
    }
}
```

### 3.2 학교 수업 준비

#### 수업 전 1시간 이내
```yaml
타겟팅_전략 = {
    모드: ["맞춤학습", "단기미션"],
    목적: "수업 준비 및 복습",
    내용: {
        전날_복습: "전날 배운 내용 빠르게 복습",
        예습: "오늘 배울 내용 미리보기 (선택)",
        준비물: "준비물 확인"
    }
}
```

### 3.3 수학 단원별 수업 준비

**함수 단원 (추상적)**
- 모드: 도제학습
- 내용: 함수 개념 기본 이해, 그래프 시각화 예습

**통계 단원 (실용적)**
- 모드: 맞춤학습
- 내용: 일상 생활 예시 준비, 데이터 해석 연습

**도형 단원 (시각적)**
- 모드: 탐구학습
- 내용: 도형 그리기 연습, 공간 지각 연습

## 4. 오늘목표 설정 상황

### 4.1 하루 시작 시 (오전)

```yaml
타겟팅_전략 = {
    모드: "단기미션",
    목적: "오늘 목표 설정 및 동기 부여",
    내용: {
        목표_설정: "오늘 달성할 학습 목표 설정",
        우선순위: "중요한 목표부터 순서 정하기",
        시간_배분: "목표별 시간 배분 계획"
    },
    톤: "긍정적이고 동기 부여하는 톤"
}
```

### 4.2 목표 변경 시

```yaml
IF (목표_변경_이유 == "목표_달성_완료") THEN
    타겟팅_전략 = {
        모드: "자기성찰",
        내용: "성취 인정 → 다음 목표 제시",
        톤: "축하하고 격려"
    }
ELSE IF (목표_변경_이유 == "목표_어려움") THEN
    타겟팅_전략 = {
        모드: "맞춤학습",
        내용: "목표 조정 → 현실적 목표 재설정",
        톤: "이해하고 지원"
    }
END IF
```

### 4.3 수학 학습 목표 설정

**단원별 목표**
- 개념 학습: "오늘 함수 개념 이해하기"
- 유형 연습: "함수 유형 문제 10개 풀기"
- 심화 문제: "함수 심화 문제 3개 풀기"

**수준별 목표**
- 하위권: 작은 목표, 성취감 강조
- 중위권: 중간 목표, 단계별 달성
- 상위권: 도전 목표, 성장 강조

## 9. 장기목표(커리큘럼) 설계 상황

### 9.1 커리큘럼 조정

```yaml
타겟팅_전략 = {
    모드: "맞춤학습",
    목적: "개인 맞춤 커리큘럼 설계",
    내용: {
        현재_수준_분석: "학생 현재 수준 종합 분석",
        목표_수준: "목표 수준 설정",
        학습_경로: "목표 달성을 위한 학습 경로 설계",
        난이도_조정: "학생 수준에 맞는 난이도 조정"
    }
}
```

### 9.2 수학 커리큘럼 설계

**학년별 커리큘럼**
- 중1: 정수와 유리수 → 일차방정식 → 함수
- 중2: 일차함수 → 연립방정식 → 도형
- 중3: 이차함수 → 이차방정식 → 통계

**단원 간 연계성 고려**
- 선후관계 단원 순서
- 연계 단원 동시 학습
- 기초 단원 우선 학습

**학생 수준별 커리큘럼**
- 하위권: 기초 개념 중심, 반복 학습
- 중위권: 개념 + 유형 병행
- 상위권: 심화 문제 중심, 창의 문제
