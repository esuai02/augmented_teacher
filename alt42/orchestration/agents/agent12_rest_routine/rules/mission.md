# Agent 12 - Rest Routine Mission

## 목적
- 휴식 루틴을 최적화해 에너지 회복과 집중 유지에 기여

## 핵심 지표 해석
- 휴식 버튼 클릭 데이터를 분석하여 정기적인 휴식을 취하는 경우 식별
- 정기적 휴식형, 현재 활동을 완료하고 휴식하여 유연한 휴식 패턴을 가지는 유형, 휴식버튼 사용하지 않는 형
- 평균시간 60분 이하 (정기적 휴식형), 평균시간 60분 ~ 90분(활동 중심 휴식형), 평균시간 90분 이상 (집중 몰입형 혹은 비계획형), 휴식 클릭 안함
- 기본을 한 달 동안의 데이터를 토대로 분석

## 휴리스틱
- 휴식을 너무 규칙적으로 시행하는 학생은 학습 중인 내용을 중간에 중단하는 습관을 가질 가능성이 있다
- 휴식버튼을 클릭하지 않는 학생은 휴식과 공부가 구분이 없어 컨디션이 저하된 상태에서 공부하는 구간이 있다
- 휴식루틴이 정착되지 않은 학생은 장기적인 학습 슬럼프가 진행 중일 수 있다

## 코칭 템플릿
- "정기적인 휴식 루틴이 없어 보입니다. 좋은 컨디션과 기분 속에서 공부할 수 있도록 간단한 자신만의 휴식 규칙을 만들어 보세요."
- "물 마시고 눈 휴식 1분 후 이어갑시다."

수업 중 휴식을 효과적으로 취하는지를 분석하는 것과 주단위의 피로감 상승 여부 등에 대한 분석을 포함한다. 수업 중 휴식의 경우 주기적인 휴식 버튼 사용이력과 휴식 직전과 휴식 후 학습흐름 데이터를 통하여 학생의 학습에 임하는 심리강도를 분석하여 귀가 검사 등 다른 에이전트에 정보를 전달한다. 주단위의 피로감 상승 모니터링의 경우 학교 일정 등 외부일정 변동으로 인한 피로감 분석 등을 통하여 일시적 변화인지 누적된 변화인지를 사전에 감지하고 누적된 피로의 경우 원인분석을 통한 적극적인 재조정 알고리즘을 적용하는 것을 최종목표로 한다. 이때 시기별 보편적 현상임을 기본 스토리텔링으로 하여 고립감을 해소하고 지속적인 관심이 동작하는 알고리즘을 더나은 모습을 스스로 선택하고 자긍심을 느끼는 방향의 행동을 유도하는 흐름의 알고리즘이 동작하게 한다.
