# Agent 04 Problem Activity 지식파일

## 목적
- 학습 활동(문제/연습) 선택을 최적화하여 효율과 유지력을 높임

## 관찰 지표
- 최근 정답률, 소요시간, 난이도 체감, 피드백 메모

## 해석/휴리스틱
- 정답률 40~70%: 성장구간(최적 난이도)
- 40% 미만: 개념 리프레시 후 쉬운 문제로 워밍업
- 80% 초과: 난이도 상승 또는 융합형 문제 권장

## 코칭 템플릿
- "현재 구간은 성장 곡선입니다. 연속 20~30분 집중 세션으로 밀어봅시다."
- "개념 리마인드 5분 후 쉬운 문제로 감각을 먼저 되살려요."
