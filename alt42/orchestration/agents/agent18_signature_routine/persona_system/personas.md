# Agent18 Signature Routine Personas

## Overview
Agent18의 시그너처 루틴 발견 미션을 위한 페르소나 정의.
학습자의 최적 학습 리듬과 패턴을 발견하기 위한 심리적/행동적 특성 분류.

---

## Persona Categories

### R (Rhythm) - 리듬 기반 페르소나
학습 시간대, 세션 길이, 휴식 패턴에 따른 분류

### A (Achievement) - 성취 기반 페르소나
목표 달성 방식과 동기부여 패턴에 따른 분류

### E (Explorer) - 탐험 기반 페르소나
새로운 학습 방식 시도와 호기심 패턴에 따른 분류

### C (Challenge) - 도전 기반 페르소나
난이도 선호와 도전 성향에 따른 분류

### M (Meaning) - 의미 기반 페르소나
학습의 목적과 가치 인식 패턴에 따른 분류

## R (Rhythm) Personas

### R1: 아침형 집중러 (Morning Focuser)
- **ID**: R1_MORNING_FOCUSER
- **별칭**: 새벽 독수리, 아침 전사
- **심리적 특성**:
  - 일찍 시작하면 하루가 뿌듯함
  - 오전에 에너지가 최고조
  - 규칙적인 생활 패턴 선호
- **행동적 특성**:
  - 오전 6-10시 학습 성과 최고
  - 오후로 갈수록 집중력 저하
  - 일찍 자고 일찍 일어남
- **전형적 발화**:
  - "아침에 공부하면 머리가 맑아요"
  - "저녁에는 집중이 안 돼요"
  - "일찍 일어나서 공부하는 게 좋아요"
- **응답 전략**:
  - 오전 학습 스케줄 강화 제안
  - 저녁 학습은 복습/가벼운 활동으로
  - 수면 리듬 유지 중요성 강조

### R2: 저녁형 야행러 (Night Owl)
- **ID**: R2_NIGHT_OWL
- **별칭**: 올빼미, 야간 전사
- **심리적 특성**:
  - 밤에 더 창의적이고 집중됨
  - 조용한 밤 시간을 선호
  - 아침에 에너지가 낮음
- **행동적 특성**:
  - 저녁 8시-새벽 2시 최고 성과
  - 오전에는 멍한 상태
  - 늦게 자고 늦게 일어남
- **전형적 발화**:
  - "밤에 공부가 더 잘 돼요"
  - "아침에는 머리가 안 돌아가요"
  - "조용할 때 집중이 잘 돼요"
- **응답 전략**:
  - 저녁 집중 학습 시간 확보 제안
  - 오전은 가벼운 복습으로
  - 수면 부족 경고 및 밸런스 조언

### R3: 점심 후 집중러 (Afternoon Peak)
- **ID**: R3_AFTERNOON_PEAK
- **별칭**: 오후 전사, 점심 후 파이터
- **심리적 특성**:
  - 점심 후 활력이 상승
  - 워밍업 시간이 필요함
  - 안정적인 리듬 선호
- **행동적 특성**:
  - 오후 2-6시 최고 성과
  - 오전에는 천천히 준비
  - 저녁에는 서서히 하락
- **전형적 발화**:
  - "점심 먹고 나면 힘이 나요"
  - "오전에는 좀 느려요"
  - "오후에 집중이 잘 돼요"
- **응답 전략**:
  - 오후 핵심 학습 배치
  - 오전은 예습/계획 시간으로
  - 저녁은 정리 시간으로 활용

### R4: 분산형 학습러 (Distributed Learner)
- **ID**: R4_DISTRIBUTED
- **별칭**: 쪼개기 마스터, 짧은 집중러
- **심리적 특성**:
  - 긴 시간 집중이 어려움
  - 자주 쉬어야 효율적
  - 다양한 활동을 번갈아 선호
- **행동적 특성**:
  - 25-30분 집중 + 5-10분 휴식
  - 하루에 여러 번 짧게 학습
  - 한 과목 오래 안 함
- **전형적 발화**:
  - "오래 하면 머리가 안 돌아가요"
  - "쉬면서 해야 효과가 있어요"
  - "조금씩 자주 하는 게 좋아요"
- **응답 전략**:
  - 포모도로 기법 추천
  - 짧은 세션 여러 개로 분산
  - 과목 전환 전략 제안

### R5: 몰입형 마라토너 (Deep Focus Marathon)
- **ID**: R5_MARATHON
- **별칭**: 마라톤 러너, 장시간 집중러
- **심리적 특성**:
  - 한번 집중하면 오래 지속
  - 흐름이 끊기면 다시 시작 어려움
  - 깊은 몰입을 좋아함
- **행동적 특성**:
  - 2-3시간 연속 학습 가능
  - 방해 없는 환경 선호
  - 한 과목 완료 후 다음 이동
- **전형적 발화**:
  - "한번 시작하면 끝까지 해야 해요"
  - "중간에 끊기면 다시 집중하기 힘들어요"
  - "깊이 파고들 때 재미있어요"
- **응답 전략**:
  - 긴 학습 블록 확보 지원
  - 방해 요소 제거 조언
  - 적절한 휴식 필요성 안내

## A (Achievement) Personas

### A1: 목표 달성형 (Goal Achiever)
- **ID**: A1_GOAL_ACHIEVER
- **별칭**: 목표 사냥꾼, 체크리스트 마스터
- **심리적 특성**:
  - 명확한 목표가 있어야 동기부여
  - 완료 표시에서 성취감
  - 계획적이고 체계적
- **행동적 특성**:
  - 목표 설정 후 학습 시작
  - 진도 추적을 좋아함
  - 일정 준수 노력
- **전형적 발화**:
  - "오늘 목표는 여기까지예요"
  - "체크할 때 기분이 좋아요"
  - "계획대로 되면 뿌듯해요"
- **응답 전략**:
  - 명확한 단계별 목표 제시
  - 진도 시각화 제공
  - 달성 시 축하/인정

### A2: 작은 성취 누적형 (Small Wins Accumulator)
- **ID**: A2_SMALL_WINS
- **별칭**: 한 걸음씩, 작은 승리자
- **심리적 특성**:
  - 작은 성공이 큰 동기부여
  - 큰 목표는 부담스러움
  - 점진적 발전 선호
- **행동적 특성**:
  - 작은 단위로 쪼개서 학습
  - 자주 성취감을 느끼려 함
  - 실패 회피 성향
- **전형적 발화**:
  - "조금씩 해나가는 게 좋아요"
  - "너무 큰 목표는 부담돼요"
  - "하나씩 하면 할 수 있어요"
- **응답 전략**:
  - 작은 단위 과제 제공
  - 자주 성취 피드백
  - 누적 진도 시각화

### A3: 경쟁형 성취러 (Competitive Achiever)
- **ID**: A3_COMPETITIVE
- **별칭**: 승부사, 1등 추구러
- **심리적 특성**:
  - 다른 사람과 비교로 동기부여
  - 순위/점수에 민감
  - 지는 것을 싫어함
- **행동적 특성**:
  - 랭킹 시스템에 반응
  - 다른 학생 성과에 관심
  - 도전적 태도
- **전형적 발화**:
  - "다른 애들은 어디까지 했어요?"
  - "내가 몇 등이에요?"
  - "이기고 싶어요"
- **응답 전략**:
  - 순위/비교 정보 제공
  - 경쟁 요소 활용
  - 건강한 경쟁심 유도

### A4: 자기기록 갱신형 (Personal Best Seeker)
- **ID**: A4_PERSONAL_BEST
- **별칭**: 자기 경쟁러, 기록 갱신러
- **심리적 특성**:
  - 과거의 나와 비교
  - 자기 성장에 집중
  - 타인 비교는 불편함
- **행동적 특성**:
  - 이전 기록 확인
  - 개인 성장 추적
  - 꾸준한 개선 추구
- **전형적 발화**:
  - "저번보다 더 잘하고 싶어요"
  - "내 기록을 깨고 싶어요"
  - "성장하는 게 보이면 좋아요"
- **응답 전략**:
  - 개인 성장 그래프 제공
  - 이전 대비 개선점 강조
  - 자기 비교 중심 피드백

## E (Explorer) Personas

### E1: 호기심 탐험가 (Curious Explorer)
- **ID**: E1_CURIOUS
- **별칭**: 왜왜왜, 궁금이
- **심리적 특성**:
  - 새로운 것에 대한 호기심
  - 깊이 파고드는 성향
  - 질문이 많음
- **행동적 특성**:
  - 관련 주제로 확장
  - 추가 자료 요청
  - 원리/이유 탐구
- **전형적 발화**:
  - "왜 그런 거예요?"
  - "더 알고 싶어요"
  - "이것도 연결되나요?"
- **응답 전략**:
  - 추가 학습 자료 제공
  - 연결 개념 안내
  - 탐구심 격려

### E2: 다양성 추구형 (Variety Seeker)
- **ID**: E2_VARIETY
- **별칭**: 변화 추구러, 다재다능
- **심리적 특성**:
  - 반복은 지루함
  - 다양한 경험 선호
  - 새로운 방식 환영
- **행동적 특성**:
  - 학습 방식 자주 변경
  - 여러 과목 번갈아
  - 새 도구/방법 시도
- **전형적 발화**:
  - "다른 방법으로 해볼래요"
  - "같은 거 계속하면 지루해요"
  - "새로운 거 없어요?"
- **응답 전략**:
  - 다양한 학습 방법 제안
  - 주기적 변화 제공
  - 새로운 도전 과제

### E3: 실험형 학습러 (Experimenter)
- **ID**: E3_EXPERIMENTER
- **별칭**: 시험삼아, 실험가
- **심리적 특성**:
  - 직접 해봐야 이해
  - 시행착오 학습
  - 실패도 학습으로 인식
- **행동적 특성**:
  - 먼저 시도하고 배움
  - 다양한 접근 시도
  - 결과 관찰 및 조정
- **전형적 발화**:
  - "일단 해볼게요"
  - "이렇게 하면 어떻게 되나요?"
  - "실패해도 괜찮아요"
- **응답 전략**:
  - 안전한 실험 환경 제공
  - 시도 후 피드백
  - 실패에서 배움 추출

## C (Challenge) Personas

### C1: 난이도 도전형 (Difficulty Challenger)
- **ID**: C1_DIFFICULTY
- **별칭**: 어려운 거 좋아요, 도전자
- **심리적 특성**:
  - 어려운 문제에 끌림
  - 쉬운 건 재미없음
  - 성장 마인드셋
- **행동적 특성**:
  - 상위 난이도 선택
  - 실패해도 재도전
  - 한계 테스트
- **전형적 발화**:
  - "더 어려운 거 주세요"
  - "쉬운 건 재미없어요"
  - "도전해볼래요"
- **응답 전략**:
  - 점진적 난이도 상승
  - 도전 과제 제공
  - 성장 과정 인정

### C2: 안정권 유지형 (Comfort Zone Keeper)
- **ID**: C2_COMFORT
- **별칭**: 안전 제일, 확실한 거
- **심리적 특성**:
  - 실패 두려움
  - 확실한 것 선호
  - 점진적 확장 원함
- **행동적 특성**:
  - 아는 것 반복 연습
  - 새 영역 조심스럽게
  - 충분히 익숙해지면 진행
- **전형적 발화**:
  - "좀 더 연습하고 싶어요"
  - "아직 자신 없어요"
  - "확실히 알고 넘어가고 싶어요"
- **응답 전략**:
  - 충분한 연습 기회
  - 점진적 난이도 증가
  - 자신감 형성 지원

### C3: 단계적 성장형 (Gradual Climber)
- **ID**: C3_GRADUAL
- **별칭**: 한 계단씩, 꾸준히
- **심리적 특성**:
  - 체계적 성장 선호
  - 기초부터 탄탄히
  - 건너뛰기 싫어함
- **행동적 특성**:
  - 순서대로 학습
  - 이전 단계 확인
  - 빈틈없이 진행
- **전형적 발화**:
  - "순서대로 하고 싶어요"
  - "기초가 중요해요"
  - "차근차근 할게요"
- **응답 전략**:
  - 체계적 커리큘럼 제공
  - 단계별 완료 확인
  - 연결성 설명

## M (Meaning) Personas

### M1: 목적 지향형 (Purpose Driven)
- **ID**: M1_PURPOSE
- **별칭**: 왜 배워야 해요?, 의미 추구러
- **심리적 특성**:
  - 배움의 이유가 중요
  - 실생활 연결 원함
  - 무의미함 싫어함
- **행동적 특성**:
  - 활용처 질문
  - 실제 사례 관심
  - 관련성 파악
- **전형적 발화**:
  - "이거 왜 배워야 해요?"
  - "어디에 쓰이는 거예요?"
  - "실제로 필요한가요?"
- **응답 전략**:
  - 실생활 활용 예시
  - 배움의 목적 설명
  - 연결성 강조

### M2: 미래 연결형 (Future Connector)
- **ID**: M2_FUTURE
- **별칭**: 미래를 위해, 꿈 추구러
- **심리적 특성**:
  - 미래 목표와 연결
  - 장기적 관점
  - 꿈/직업과 연결
- **행동적 특성**:
  - 진로 관련 질문
  - 미래 활용 고려
  - 관련 분야 탐색
- **전형적 발화**:
  - "이게 나중에 도움이 될까요?"
  - "내 꿈이랑 관련 있어요?"
  - "미래에 필요한 건가요?"
- **응답 전략**:
  - 진로 연결 설명
  - 미래 비전 제시
  - 장기 목표 설정 지원

### M3: 현재 집중형 (Present Focused)
- **ID**: M3_PRESENT
- **별칭**: 지금 여기, 현재 집중러
- **심리적 특성**:
  - 지금 이 순간에 집중
  - 미래는 너무 멀게 느낌
  - 즉각적 보상 선호
- **행동적 특성**:
  - 당장의 결과 중시
  - 현재 상황에 집중
  - 즉각적 피드백 선호
- **전형적 발화**:
  - "지금 중요한 게 뭐예요?"
  - "바로 확인하고 싶어요"
  - "오늘 할 것에 집중할래요"
- **응답 전략**:
  - 즉각적 피드백 제공
  - 현재 단계 명확화
  - 단기 목표 설정
