## 페르소나 구조 설명

각 페르소나는 다음 요소로 구성됩니다:

| 요소 | 설명 |
|------|------|
| **ID** | 상황코드_유형번호 (예: S0_P1) |
| **별칭** | 직관적으로 이해할 수 있는 유형명 |
| **심리적 특성** | 내면의 감정, 생각, 동기 |
| **행동적 특성** | 외부로 드러나는 행동 패턴 |
| **전형적 발화** | 해당 유형이 자주 하는 말 |
| **대응 전략** | 시스템/교사의 최적 대응 방법 |

## S0: 수학 특화 정보 수집 단계 페르소나

### S0_P1: 솔직한 자기 분석가 🔍
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 자신의 학습 상태를 객관적으로 인식하고 있음. 도움받기 위해 정확한 정보를 제공하려는 의지가 있음 |
| **행동적 특성** | 질문에 상세하고 정확하게 답변함. 추가 정보를 자발적으로 제공함. 모르는 것도 솔직히 인정함 |
| **전형적 발화** | "솔직히 함수 부분이 제일 약해요", "제가 문제를 풀 때 실수가 많아서요" |
| **대응 전략** | 솔직함을 인정하고 격려. 제공된 정보를 바탕으로 정밀한 맞춤 계획 수립. 정확한 피드백 제공 |

### S0_P2: 방어적 최소 응답자 🛡️
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 자신의 약점을 드러내기 싫어함. 평가받는 느낌에 불안함. 정보 공개에 대한 거부감 |
| **행동적 특성** | 질문에 짧고 모호하게 답변함. "몰라요", "그냥요" 등 회피성 답변. 필수 정보만 최소한으로 입력 |
| **전형적 발화** | "그냥 평범해요", "잘 모르겠어요", "딱히..." |
| **대응 전략** | 압박하지 않고 천천히 진행. 쉬운 질문부터 시작. 정보 제공의 이점을 부드럽게 설명. 선택형 질문 활용 |

### S0_P3: 과대 포장형 자신감 ✨
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 자신의 실력을 과대평가하는 경향. 약점을 인정하기 어려워함. 좋은 인상을 주고 싶은 욕구 |
| **행동적 특성** | 성적/실력을 부풀려서 말함. 취약 단원을 축소하거나 숨김. 자신감 점수를 높게 입력 |
| **전형적 발화** | "수학 원래 잘해요", "지난 시험은 운이 안 좋았을 뿐이에요", "다 알긴 아는데..." |
| **대응 전략** | 직접적 반박 피하기. 진단 테스트를 통한 객관적 데이터 수집. 강점을 인정하면서 보완점 발견 유도 |

### S0_P4: 불안한 완벽주의자 😰
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 정확하게 답변해야 한다는 압박감. 틀린 정보를 제공할까 봐 걱정. 모든 선택에 대한 고민이 많음 |
| **행동적 특성** | 질문마다 오래 고민함. 답변 후 수정을 여러 번 함. "이게 맞나요?"라고 확인함 |
| **전형적 발화** | "정확히 뭐라고 해야 하지...", "이렇게 답하는 게 맞나요?", "다시 고쳐도 되나요?" |
| **대응 전략** | 정답이 없음을 안내. 언제든 수정 가능함을 강조. 진행 중 피드백으로 안심감 제공 |

### S0_P5: 무관심한 수동적 참여자 😐
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 학습에 대한 동기가 낮음. 온보딩 과정에 의미를 느끼지 못함. "시키는 대로 하자"는 태도 |
| **행동적 특성** | 대충 빠르게 넘기려 함. 선택지 중 첫 번째만 선택하는 경향. 질문을 제대로 읽지 않음 |
| **전형적 발화** | "아무거나요", "빨리 끝내고 싶어요", "그냥 넘어가도 되나요?" |
| **대응 전략** | 온보딩의 개인적 이점 강조. 간소화된 핵심 질문만 진행. 재미 요소 추가. 이후 단계적 정보 수집 |

### S0_P6: 호기심 많은 탐색자 🌟
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 새로운 시스템에 대한 호기심. 자신에 대해 알아가는 것을 즐김. 적극적으로 참여하고 싶은 의지 |
| **행동적 특성** | 질문의 의도를 묻기도 함. 추가 설명을 요청함. 자발적으로 상세 정보 입력 |
| **전형적 발화** | "이 질문은 왜 하는 건가요?", "더 알려드릴 게 있어요!", "재미있네요" |
| **대응 전략** | 호기심을 만족시키는 상세 설명 제공. 적극적 참여에 대한 긍정적 피드백. 심층 프로필 작성 유도 |

## S1: 신규 학생 등록 직후 페르소나

| **행동적 특성** | 구체적인 목표와 기간을 제시함. 성과 측정 방법에 관심. 불필요한 과정 생략 요구 |

| **대응 전략** | 구체적인 로드맵 제시. 측정 가능한 목표 설정. 효율적 학습 방법 강조. 진도 추적 시스템 활용 |

## S2: 수업 전 학습 설계 단계 페르소나

### S2_P1: 계획 수용형 따르는 학습자 📋
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 전문가의 계획을 신뢰함. 안내 받는 것을 편안하게 여김. 책임을 덜고 싶은 마음 |
| **행동적 특성** | 제시된 루틴을 그대로 수용. 질문 없이 동의. 계획대로 실행하려 노력 |
| **전형적 발화** | "네, 그렇게 할게요", "선생님이 정해주세요", "알려주신 대로 하면 되죠?" |
| **대응 전략** | 명확하고 구체적인 지시 제공. 단계별 체크리스트 활용. 자율성을 점진적으로 부여 |

### S2_P2: 자기주도형 협상가 🤝
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 자신의 학습에 대한 통제권을 원함. 계획 수립에 참여하고 싶음. 자율성 중시 |
| **행동적 특성** | 루틴에 대해 의견 제시. 수정 요청. 대안 제안. 협의 과정 선호 |
| **전형적 발화** | "이 부분은 제가 조절해도 될까요?", "이렇게 하면 어떨까요?", "왜 이 순서인가요?" |
| **대응 전략** | 계획 수립에 참여 기회 제공. 선택지 부여. 의견 반영 후 피드백. 자율성과 가이드의 균형 |

### S2_P3: 과부하 회피형 최소주의자 ⚡
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 너무 많은 계획에 부담감. 실행 가능성에 대한 의심. 최소한만 하고 싶음 |
| **행동적 특성** | "이거 다 해야 해요?" 질문. 분량 줄여달라 요청. 쉬운 것만 선택하려 함 |
| **전형적 발화** | "너무 많아요", "이거 줄일 수 없나요?", "꼭 다 해야 돼요?" |
| **대응 전략** | 핵심만 추린 미니멀 버전 제공. 우선순위 명확화. 단계적 확장. 성취감 먼저 경험시키기 |

### S2_P4: 완벽주의 과다 계획형 📚
| 항목 | 내용 |
|------|------|
| **심리적 특성** | 더 많이, 더 완벽하게 하고 싶음. 부족하다는 불안감. 과도한 목표 설정 |
| **행동적 특성** | 추가 학습 요청. 더 어려운 문제 요구. 스스로를 과도하게 채찍질 |
| **전형적 발화** | "이것만 하면 되나요? 더 해야 할 것 같은데", "심화 문제도 추가해주세요", "하루에 3시간은 해야죠" |
| **대응 전략** | 현실적 계획의 중요성 설명. 번아웃 위험 안내. 지속 가능한 루틴 강조. 휴식의 가치 전달 |

### S2_P5: 시험 중심 전략가 📝
| 항목 | 내용 |
|------|------|

## 페르소나 활용 가이드

### 페르소나 식별 방법

1. **초기 반응 관찰**: 첫 질문에 대한 반응으로 1차 분류
2. **패턴 확인**: 2-3개 상호작용 후 일관된 패턴 확인
3. **복합 유형 고려**: 한 학생이 여러 페르소나 특성을 가질 수 있음
4. **상황별 변화**: 같은 학생도 상황에 따라 다른 페르소나를 보일 수 있음

### 대응 원칙

| 원칙 | 설명 |
|------|------|
| **개별화** | 동일 상황이라도 페르소나에 따라 다른 접근 |
| **유연성** | 고정관념 피하고 변화 가능성 열어둠 |
| **존중** | 모든 유형을 존중하고 판단하지 않음 |
| **적응** | 학생의 반응에 따라 전략 조정 |
| **일관성** | 신뢰 구축을 위한 일관된 지지 |

### 페르소나 전환 신호

학생의 페르소나가 변화하는 신호:
- 평소와 다른 언어 사용
- 참여도의 급격한 변화
- 정서적 표현의 변화
- 새로운 상황에 대한 반응

mk:StudentPersona:

  - mk:requiresInterventionType  # 필요 개입 유형
