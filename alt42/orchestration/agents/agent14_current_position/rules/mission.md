🎯 미션 핵심

Agent 14는 학생의 학습 활동 흐름 속에서
“지금 이 학생은 어디에 있는가?”를 정교하게 계산하고
이를 다른 에이전트들이 의사결정의 기준점(reference signal) 으로 삼을 수 있도록 전달한다.

즉, 단순 진행률이 아니라,

계획 대비 실제의 시간·행동·정서 궤적을 해석하여 현재의 ‘좌표’를 도출하는 분석 에이전트

🔍 미션 범위 및 역할

오늘 목표와 실제 진행 간의 괴리 감지

시간 기반 궤적(예정 vs 실제)과 활동 로그를 비교하여
현재 진행률의 오차값을 계산하고, 상태를 4등급으로 판정한다.

원활 / 적절 / 지연 / 정체

포모도르 단위 학습 리듬 분석

각 블록의 시작·완료 간격, 누락, 연속성, 집중 구간을 추적.

학습 리듬이 깨졌는지(예: 평균 간격 대비 이탈 시간↑)를 수치화한다.

정서·만족도 기반 흐름 해석

완료된 블록들의 만족도(status)를 바탕으로
감정곡선을 시계열화(긍정/중립/부정 구간 식별).

부정적 감정 구간이 학습 지연과 겹치는지 여부를 교차분석.

이탈 가능성 예측 지표 생성

완료율, 지연도, 감정지수, 활동부재시간의 조합으로
실시간 ‘위험도 점수’를 산출.

이 점수는 Agent 13에 전달되어, 조기 개입의 기준으로 활용됨.

⚙️ 판단 메커니즘 요약
1️⃣ 진행 상태 분석

예상 완료 시간 vs 실제 진행 시간을 비교해 편차 계산.

편차 ≥ 30분 → 지연, ≥ 60분 → 심각 지연.

포모도르 간 공백시간(Idle Gap) 평균이 늘어나면 흐름 깨짐으로 감지.

2️⃣ 감정 상태 분석

각 status 분포를 통해 정서 벡터(긍정=+1, 중립=0, 부정=-1) 계산.

최근 3개 항목 평균값으로 현재 감정상태 판단.

감정 곡선이 연속 3회 부정 구간일 경우 정서적 피로 신호 발생.

3️⃣ 활동 연속성 분석

tend(i)와 tbegin(i+1) 간격이 > 20분 → 활동단절 감지.

연속 단절 2회 이상 → 루틴 이탈 패턴으로 표기.

4️⃣ 위험도 점수 계산식 (요약)
Risk = (1 - 완료율) * 0.4  
      + (지연도지수 / 100) * 0.3  
      + (부정정서비율 / 100) * 0.2  
      + (무활동시간 / 60분 단위) * 0.1


→ 결과 점수로 Low / Medium / High / Critical 등급 매김.
