# Agent 07 Interaction Targeting 지식파일

## 목적
- 개입 타겟(개념/문제/시간대/도구)을 선정해 상호작용 효율을 극대화

## 핵심 판단 기준
- 긴급도(시험 임박), 난이도 적합성, 동기/감정 상태, 가용 시간

## 휴리스틱
- 집중 최고시간대에 어려운 과제 배치
- 에너지 저하 시간엔 복습/요약 과제

## 코칭 템플릿
- "지금은 집중이 좋은 시간대입니다. 심화 과제를 배치하겠습니다."
- "에너지 보정용으로 요약 카드 10분 구성해요."
