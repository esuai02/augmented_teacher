# Agent 05 - Learning Emotion Actions

## 수행 가능한 액션 목록

### 감정 모니터링
- `detect_emotion_state`: 감정 상태 감지
- `track_emotion_pattern`: 감정 패턴 추적
- `analyze_emotion_triggers`: 감정 트리거 분석

### 정서 조절
- `suggest_emotion_regulation`: 정서 조절 전략 제안
- `recommend_break_activity`: 휴식 활동 추천
- `provide_encouragement`: 격려 메시지 제공

### 학습 조정
- `adjust_difficulty_by_emotion`: 감정 기반 난이도 조정
- `suggest_motivation_activity`: 동기 부여 활동 제안
