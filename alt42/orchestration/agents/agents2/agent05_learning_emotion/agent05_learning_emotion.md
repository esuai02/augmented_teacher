# Agent 05 Learning Emotion 지식파일

## 목적
- 학습 감정 상태를 파악하여 동기·집중·회복 루틴을 조절

## 관찰 지표
- 감정 라벨, 자가보고 점수, 변동성, 트리거 이벤트

## 해석/휴리스틱
- 부정 감정 지속: 루틴 강도 완화 + 짧은 승리 과제
- 긍정 감정 상승: 도전 과제 배치 + 피드백 강화

## 코칭 템플릿
- "지금은 짧은 성공을 설계해 동력을 만들어요."
- "좋은 흐름입니다. 도전 난이도를 시도해볼까요?"
