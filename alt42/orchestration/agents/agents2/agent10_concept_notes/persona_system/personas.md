## 페르소나 구조 설명

각 페르소나는 다음 요소로 구성됩니다:

| 요소 | 설명 |
|------|------|
| **ID** | 상황코드_유형번호 (예: N1_P1) |
| **별칭** | 직관적으로 이해할 수 있는 유형명 |
| **필기 특성** | 노트 작성 방식과 패턴 |
| **행동적 특성** | 학습 중 보이는 행동 패턴 |
| **데이터 지표** | 노트 데이터에서 관찰되는 특징 |
| **대응 전략** | 시스템/교사의 최적 대응 방법 |

### 핵심 지표 정의

| 지표 | 설명 | 활용 |
|------|------|------|
| nstroke | 총 필기 획 수 | 활동 강도/몰입도 측정 |
| tlaststroke | 마지막 필기 시점 | 최근성/세션 종료 근접도 |
| timecreated | 노트 생성 시각 | 학습 시작 타임라인 |
| usedtime | 노트 사용 총 시간 | 학습 지속성 측정 |
| contentstitle | 개념 제목 | 단원/주제 맥락 파악 |

## N1: 노트 탐색 시작 단계 페르소나

### N1_P1: 체계적 탐색자 (Systematic Explorer)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 노트를 시작하기 전 전체 구조를 파악하려 함. 목차나 개요를 먼저 확인 |
| **행동적 특성** | 여러 노트를 훑어보며 전체 맥락 파악. 체계적으로 접근 |
| **데이터 지표** | 첫 노트 생성 전 탐색 시간이 김. 여러 개념 페이지 순차 방문 |
| **전형적 발화** | "전체적으로 어떤 개념들이 있는지 먼저 볼게요" |
| **대응 전략** | 전체 학습 맵 제공. 개념 간 연결 관계 시각화. 체계적 학습 경로 제안 |

### N1_P2: 빠른 스캔러 (Quick Scanner)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 빠르게 훑어보고 바로 필기 시작. 세부 내용보다 핵심 키워드에 집중 |
| **행동적 특성** | 탐색 시간 최소화. 즉시 행동 선호. 직관적 접근 |
| **데이터 지표** | 탐색-필기 간격이 짧음. 초기 필기량 급증 패턴 |
| **전형적 발화** | "일단 시작하고 보자", "하면서 알게 되겠지" |
| **대응 전략** | 즉시 시작 가능한 간단한 과제 제공. 핵심 개념 요약 먼저 표시. 점진적 구조화 유도 |

### N1_P3: 신중한 관찰자 (Cautious Observer)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 노트를 시작하기 주저함. 어디서부터 시작해야 할지 고민 |
| **행동적 특성** | 많은 시간을 관찰에 사용. 첫 필기까지 시간이 오래 걸림 |
| **데이터 지표** | 첫 획까지 시간이 김. 초기 필기량 매우 적음 |
| **전형적 발화** | "뭐부터 적어야 하지?", "이렇게 하는 게 맞나?" |
| **대응 전략** | 시작 가이드 제공. 첫 필기 템플릿 제안. 정답이 없음을 안내하여 부담 감소 |

### N1_P4: 호기심 주도형 (Curiosity-Driven)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 흥미로운 개념부터 자유롭게 탐색. 순서에 구애받지 않음 |
| **행동적 특성** | 여러 주제를 넘나들며 탐색. 관심 끄는 것에 집중 |
| **데이터 지표** | 비순차적 개념 방문. 특정 주제에 필기 집중 |
| **전형적 발화** | "이거 재미있어 보인다", "이건 뭐지?" |
| **대응 전략** | 관심 기반 학습 경로 허용. 연결된 흥미로운 개념 추천. 탐색의 가치 인정 |

## N2: 개념 이해도 분석 페르소나

### N2_P1: 개념 완벽주의자 (Concept Perfectionist)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 하나의 개념을 완전히 이해할 때까지 반복 필기. 상세한 노트 작성 |
| **행동적 특성** | 같은 노트를 여러 번 수정. 추가 설명 요구. 확인 질문 많음 |
| **데이터 지표** | 단일 노트 수정 횟수 높음. usedtime 높음. nstroke 매우 높음 |
| **전형적 발화** | "완전히 이해해야 넘어갈 수 있어요", "이 부분 더 설명해주세요" |
| **대응 전략** | 단계별 이해도 체크 제공. 완벽하지 않아도 진행 가능함을 안내. 80% 이해 후 진행 권장 |

### N2_P2: 핵심 파악형 (Core Grasper)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 핵심 개념만 간결하게 정리. 불필요한 세부사항 생략 |
| **행동적 특성** | 효율적 학습 선호. 빠른 개념 습득. 응용으로 빠르게 이동 |
| **데이터 지표** | nstroke 중간. usedtime 대비 커버리지 높음 |
| **전형적 발화** | "핵심만 알면 되죠", "이게 제일 중요한 거죠?" |
| **대응 전략** | 핵심 요약 제공. 심화 내용은 선택적으로 안내. 빠른 피드백 루프 |

### N2_P3: 연결 학습자 (Connection Learner)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 개념 간 관계를 찾아 연결 짓는 필기. 마인드맵 형태 선호 |
| **행동적 특성** | "이거랑 저거랑 관련 있지 않아요?" 질문. 통합적 사고 |
| **데이터 지표** | 여러 노트 간 교차 참조 패턴. 시각적 연결 요소 많음 |
| **전형적 발화** | "이 개념이랑 저번에 배운 거랑 연결되네요" |
| **대응 전략** | 개념 연결 다이어그램 제공. 관련 개념 자동 추천. 통합 복습 기회 |

### N2_P4: 암기 의존형 (Memorization-Dependent)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 그대로 베껴 적는 방식. 정의나 공식 그대로 복사 |
| **행동적 특성** | 이해보다 암기 우선. 변형 문제에 약함 |
| **데이터 지표** | 원본 텍스트와 유사도 높음. nstroke 높지만 독창적 내용 적음 |
| **전형적 발화** | "이거 외워야 해요?", "공식이 뭐였지?" |
| **대응 전략** | 이해 기반 설명 추가. 암기 내용의 의미 설명. 변형 문제로 이해도 확인 |

## N3: 학습 흐름 해석 페르소나

### N3_P1: 순차 학습자 (Sequential Learner)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 정해진 순서대로 차근차근 진행. 단계별 노트 작성 |
| **행동적 특성** | 커리큘럼 순서 준수. 건너뛰기 싫어함. 안정적 진행 |
| **데이터 지표** | timecreated 순서와 개념 순서 일치. 균일한 진행 속도 |
| **전형적 발화** | "앞에서부터 순서대로 할게요", "건너뛰면 안 될 것 같아요" |
| **대응 전략** | 순차적 학습 경로 유지. 각 단계 완료 확인. 선행 개념 연결 강화 |

### N3_P2: 점프 학습자 (Jump Learner)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 관심 있거나 필요한 개념으로 바로 이동. 비순차적 학습 |
| **행동적 특성** | 목표 지향적 학습. 필요할 때 선행 개념으로 돌아감 |
| **데이터 지표** | 개념 방문 순서 비순차적. 특정 개념 집중 패턴 |
| **전형적 발화** | "이거 먼저 하고 싶어요", "필요하면 나중에 볼게요" |
| **대응 전략** | 선수 개념 알림 제공. 필요시 기초 복습 추천. 유연한 학습 경로 허용 |

### N3_P3: 집중-분산 교대형 (Focus-Scatter Alternator)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 집중적 필기 후 휴식, 다시 집중의 패턴 반복 |
| **행동적 특성** | 에너지 관리를 위한 학습 리듬 보유. 번아웃 방지 |
| **데이터 지표** | 필기 간격에 주기적 패턴. 집중-휴식 사이클 |
| **전형적 발화** | "잠깐 쉬고 다시 할게요", "한 번에 다 못해요" |
| **대응 전략** | 휴식 타이밍 존중. 적정 학습량 제안. 포모도로 기법 안내 |

### N3_P4: 몰입형 (Deep Diver)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 한번 시작하면 오랜 시간 집중. 깊이 있는 노트 작성 |
| **행동적 특성** | 방해받기 싫어함. 플로우 상태 추구. 시간 감각 상실 |
| **데이터 지표** | usedtime 매우 높음. 연속 필기 시간 김. nstroke 높음 |
| **전형적 발화** | "벌써 이렇게 됐어?", "더 하고 싶어요" |
| **대응 전략** | 몰입 환경 지원. 적절한 휴식 알림. 성취 인정 및 격려 |

## N4: 복습 권장 판단 페르소나

### N4_P1: 정기 복습형 (Regular Reviewer)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 일정한 주기로 노트 재방문. 복습 노트 별도 작성 |
| **행동적 특성** | 스케줄에 따른 복습 습관. 장기 기억 형성에 효과적 |
| **데이터 지표** | 동일 노트 재방문 주기적 패턴. 복습 표시 있음 |
| **전형적 발화** | "이번 주 복습할 차례예요", "주기적으로 봐야 기억나요" |
| **대응 전략** | 복습 일정 자동 제안. 에빙하우스 망각 곡선 기반 알림. 복습 효과 피드백 |

### N4_P2: 직전 복습형 (Last-Minute Reviewer)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 시험이나 평가 직전에만 집중 복습. 벼락치기 스타일 |
| **행동적 특성** | 평소에는 노트 미방문. 필요할 때만 급하게 복습 |
| **데이터 지표** | 장기간 미방문 후 갑자기 집중 방문. usedtime 급증 |
| **전형적 발화** | "시험 전에 한 번에 볼게요", "지금은 괜찮아요" |
| **대응 전략** | 분산 복습의 효과 안내. 작은 복습 습관 형성 유도. 시험 전 요약 제공 |

### N4_P3: 선택적 복습형 (Selective Reviewer)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 어려웠던 개념만 선별적으로 복습. 효율 추구 |
| **행동적 특성** | 자기 약점 인식. 필요한 부분만 집중 |
| **데이터 지표** | 특정 노트만 반복 방문. 다른 노트는 미방문 |
| **전형적 발화** | "이 부분만 다시 볼게요", "이건 알아서 안 봐도 돼요" |
| **대응 전략** | 약점 노트 자동 식별. 간과하기 쉬운 개념 알림. 전체 복습 필요성 안내 |

### N4_P4: 복습 회피형 (Review Avoider)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 새로운 내용만 학습. 이전 노트 재방문 꺼림 |
| **행동적 특성** | "이미 한 거"에 대한 흥미 상실. 새것 선호 |
| **데이터 지표** | 노트 재방문율 매우 낮음. tlaststroke가 timecreated와 유사 |
| **전형적 발화** | "그건 이미 했잖아요", "새로운 거 하고 싶어요" |
| **대응 전략** | 복습의 중요성 재미있게 안내. 게임화된 복습 도입. 새로운 관점으로 복습 유도 |

## N5: 노트 활용 전략 페르소나

### N5_P1: 노트 정리왕 (Note Organizer)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 체계적이고 깔끔한 노트 정리. 분류와 태그 활용 |
| **행동적 특성** | 노트 자체를 학습 도구로 활용. 정리된 노트에서 성취감 |
| **데이터 지표** | 노트 간 연결 체계적. 일관된 형식. 메타데이터 풍부 |
| **전형적 발화** | "노트 정리하면 공부가 돼요", "깔끔하게 정리해야 해요" |
| **대응 전략** | 고급 정리 도구 제공. 노트 템플릿 추천. 정리 팁 공유 |

### N5_P2: 실전 활용형 (Practical Applier)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 문제 풀이와 연결된 노트 작성. 적용 중심 |
| **행동적 특성** | 이론보다 실전. 노트를 문제 풀이에 직접 활용 |
| **데이터 지표** | 문제 풀이 영역과 노트 연결. 예제 중심 필기 |
| **전형적 발화** | "이거 어디에 쓰이는 건데요?", "문제에 어떻게 적용해요?" |
| **대응 전략** | 개념별 예제 연결. 실전 적용 가이드. 문제-개념 매핑 |

### N5_P3: 창의적 재구성형 (Creative Reconstructor)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 자신만의 방식으로 개념 재해석. 창의적 노트 작성 |
| **행동적 특성** | 그림, 비유, 스토리 활용. 개인화된 학습 |
| **데이터 지표** | 다양한 형태의 필기 요소. 원본과 다른 구조화 |
| **전형적 발화** | "이렇게 이해하면 될까요?", "제 방식대로 정리할게요" |
| **대응 전략** | 창의적 접근 격려. 다양한 표현 도구 제공. 재해석의 정확성 확인 |

### N5_P4: 최소 노트형 (Minimal Noter)
| 항목 | 내용 |
|------|------|
| **필기 특성** | 필요 최소한만 기록. 머릿속 정리 선호 |
| **행동적 특성** | 노트 의존도 낮음. 기억력에 자신감 |
| **데이터 지표** | nstroke 낮음. usedtime 대비 필기량 적음 |
| **전형적 발화** | "다 적을 필요 없어요", "기억하면 되죠" |
| **대응 전략** | 핵심 요약만 유도. 기억 보조 도구 제안. 복습 시 노트 필요성 안내 |

## 페르소나 간 관계

### 상황 전환 패턴

```
N1 (탐색) → N2 (분석) → N3 (학습) → N4 (복습) → N5 (활용)
     ↓           ↓           ↓           ↓
   P1~P4      P1~P4       P1~P4       P1~P4       P1~P4
```

### 대응 전략 우선순위

1. **동기 부여**: N1_P3(신중한 관찰자), N4_P4(복습 회피형)
2. **구조화 지원**: N1_P2(빠른 스캔러), N2_P4(암기 의존형)
3. **유연성 허용**: N1_P4(호기심 주도형), N3_P2(점프 학습자)
4. **깊이 강화**: N2_P1(개념 완벽주의자), N3_P4(몰입형)
