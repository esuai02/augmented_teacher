💬 포괄형 질문 1

“이 학생의 개념 공부 패턴을 종합해서, 첫 개념 수업에서는 무엇을 어떻게 다뤄야 할까?”
→ 필기량, 체류시간, TTS 활용, 재방문 패턴, 단계별 이해도 등을 종합하여 학생의 개념 이해 방식(시각형/청각형/요약형 등) 을 판별하고, 첫 개념 수업에서의 설명 방식, 교재 구성, 필기 유도 전략을 포함한 맞춤형 도입 루틴을 제안하도록 함.

💬 포괄형 질문 2

“이 학생의 개념 학습 데이터를 기반으로, 어떤 루틴이 가장 효율적일까?”
→ 개념요약–이해–체크–예제–대표유형–테스트 간의 전이 효율, TTS·필기·체류 로그 간 상관관계, 오답 발생 구간 등을 분석하여 학생 개인에게 가장 적합한 개념학습 루틴 시퀀스(순서·시간·형태) 와 집중 블록 구조(포모도르/세션단위) 를 설계하도록 요청.

💬 포괄형 질문 3

“이 학생이 중장기적으로 개념 학습 효율을 높이기 위해 지금부터 어떤 부분을 보완해야 할까?”
→ 오답 원인(이해 부족/주의력 저하/필기누락), 루틴 유지율, 단계별 이해도 편차, 재방문 이유 분석을 통합하여 개념이해 안정성 지표, 청각/시각 입력 의존도, 루틴 지속 리스크 요인을 추출하고, 향후 개념학습 지속성 향상 전략(습관·도구·피드백 루프) 을 제안하도록 함.

### 🧩 1. 에이전트를 필요로 하는 주요 상황

| 구분      | 상황 설명                                          | 주요 목표                   |
| ------- | ---------------------------------------------- | ----------------------- |
| **S1**  | 학생이 개념공부 중 **오답**을 냈을 때                        | 오답 원인 진단 및 피드백 루프 전달    |
| **S2**  | 학생의 개념노트 필기량이 **비정상적으로 적거나 많을 때**              | 학습 몰입도·필기 효율성 점검        |
| **S3**  | 특정 단원에서 **체류 시간이 과도하게 짧거나 긴 경우**               | 개념 이해도 대비 집중시간 효율 분석    |
| **S4**  | 학생이 **TTS를 과도하게 반복 재생하거나 전혀 사용하지 않을 때**        | 학습 스타일 기반 루틴 조정         |
| **S5**  | 이전 학습단계(개념요약, 이해, 체크 등) 간의 **연결성이 약할 때**       | 단계 간 전이(transfer) 품질 분석 |
| **S6**  | 학생이 **재방문 패턴을 보이거나 동일 페이지를 반복 탐색할 때**          | 불안감, 확신 부족, 이해도 불균형 감지  |
| **S7**  | 선생님이 학생의 개념이해가 약하다고 판단할 때                      | 개념노트 로그 기반 근거 리포트 생성    |
| **S8**  | 침착도, 감정, 문제노트 에이전트 등 다른 에이전트가 **원인 데이터 요청할 때** | 복합 분석을 위한 근거 데이터 제공     |
| **S9**  | 주기적 학습분석 리포트(예: 주간/단원 피드백) 생성 시                | 루틴 최적화 제안용 데이터 추출       |
| **S10** | 신규 단원 진입 전, 이전 단원의 **개념노트 완성도 진단 필요 시**        | 누적학습 기반 루틴 재설계 입력 제공    |

### 🎯 2. 각 상황별 맞춤형 복합질문 (에이전트 호출용 프롬프트 예시)

| 상황코드    | 에이전트에게 던질 복합질문                                                                       |
| ------- | ------------------------------------------------------------------------------------ |
| **S1**  | “학생이 이번 개념테스트에서 틀린 문제의 원인을 이전 단계 필기데이터와 비교해서 찾아줘. 필기 누락, 오개념, 시간부족 중 어떤 유형인지도 분류해줘.” |
| **S2**  | “이번 단원에서 학생의 필기량이 평균보다 적은데, 체류시간과 이해도 로그를 함께 분석해서 몰입 저하인지, 개념이 단순했는지 구분해줘.”          |
| **S3**  | “학생의 페이지별 체류시간이 불균형한데, 어느 구간에서 사고가 멈췄는지 탐색해줘. 해당 구간의 오답·재방문 로그도 함께 확인해.”             |
| **S4**  | “TTS 사용 패턴을 분석해서 학생이 청각적 입력에 의존하는지 아니면 시각적 학습을 선호하는지 분류하고, 그에 맞는 루틴 조정안을 제안해줘.”      |
| **S5**  | “개념요약 → 개념이해 → 개념체크 단계별 로그를 비교해서 연결이 약한 단계가 어디인지, 원인이 무엇인지 진단해줘.”                    |
| **S6**  | “학생이 동일 페이지를 여러 번 방문했어. 불확실감 때문인지, 복습 루틴 때문인지 판단해줘. 감정 데이터와 비교 분석해.”                 |
| **S7**  | “선생님이 학생의 개념이해 부족을 느꼈어. 최근 개념노트 데이터를 기반으로 근거 리포트를 만들어줘. 필기 완성도와 재방문 로그 중심으로.”        |
| **S8**  | “침착도 에이전트가 원인을 찾고 있어. 오답 전후 개념노트 필기패턴과 체류시간 데이터를 제공해줘.”                              |
| **S9**  | “이번 주 학생의 개념공부 루틴을 요약해줘. 개념요약→체크→테스트로 이어지는 패턴의 효율성과 변화 추세를 포함해.”                     |
| **S10** | “새 단원 진입 전에 이전 단원 개념노트의 완성도를 평가하고 루틴 개선점을 제시해줘. 필기 충실도와 오답률 중심으로.”                   |
