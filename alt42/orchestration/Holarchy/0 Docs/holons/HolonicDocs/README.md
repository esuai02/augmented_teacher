# HolonicDocs - 계층적 문서 코드 체계

> **버전**: 1.0
> **생성일**: 2025-12-07

## 네이밍 컨벤션

### 코드 구조

```
{사업부}_{미션에이전트}_{프로젝트에이전트}_{태스크에이전트A}_{태스크에이전트B}...
```

### 계층 구조

| 레벨 | 접두사 | 의미 | 예시 |
|------|--------|------|------|
| 0 | HTE | HighTouchEdu 사업부 | HTE |
| 1 | M## | 미션 에이전트 | M01, M02, M03... |
| 2 | P## | 프로젝트 에이전트 | P01, P02, P03... |
| 3 | TA## | A타입 태스크 에이전트 | TA01, TA02... |
| 3 | TB## | B타입 태스크 에이전트 | TB01, TB02... |
| 3 | TC## | C타입 태스크 에이전트 | TC01, TC02... |
| 3 | TD## | D타입 태스크 에이전트 | TD01, TD02... |

### 예시

```
HTE_M01_P01_TA01_TB01_TC01_TD01
│   │   │   │    │    │    └── D타입 태스크 #01
│   │   │   │    │    └────── C타입 태스크 #01
│   │   │   │    └─────────── B타입 태스크 #01
│   │   │   └──────────────── A타입 태스크 #01
│   │   └──────────────────── 프로젝트 #01
│   └──────────────────────── 미션 #01
└──────────────────────────── HighTouchEdu 사업부
```

## 협업 구조

### 수직 관계 (Hierarchy)

```
       ┌─────────┐
       │  M01    │  ← 미션 에이전트 (전략/방향)
       └────┬────┘
            │
       ┌────▼────┐
       │  P01    │  ← 프로젝트 에이전트 (실행 계획)
       └────┬────┘
            │
    ┌───────┼───────┐
    ▼       ▼       ▼
┌──────┐┌──────┐┌──────┐
│ TA01 ││ TB01 ││ TC01 │ ← 태스크 에이전트 (실행)
└──────┘└──────┘└──────┘
```

- **상향**: 진행 상황 리포트, 결과 보고
- **하향**: 지시사항 전달, 업데이트 배포

### 수평 관계 (Collaboration)

```
┌──────┐     ┌──────┐     ┌──────┐
│ TA01 │◄───►│ TB01 │◄───►│ TC01 │
└──────┘     └──────┘     └──────┘
    협업 매핑 / 정보 공유 / 동기화
```

- **매핑**: 상호 참조 및 의존성 정의
- **협업**: 공유 리소스, 공통 인터페이스
- **동기화**: 상태 업데이트, 버전 관리

## 문서 타입별 코드

| 타입 | 접미사 | 설명 |
|------|--------|------|
| 회의록 | _MTG## | Meeting |
| 결정사항 | _DEC## | Decision |
| 태스크 | _TSK## | Task |
| 리포트 | _RPT## | Report |
| 기능명세 | _FTR## | Feature |
| 전략문서 | _STR## | Strategy |

### 회의록 예시

```
HTE_M01_P01_MTG001  # 미션01-프로젝트01 회의록 #001
HTE_M02_MTG001      # 미션02 레벨 회의록 #001
```

