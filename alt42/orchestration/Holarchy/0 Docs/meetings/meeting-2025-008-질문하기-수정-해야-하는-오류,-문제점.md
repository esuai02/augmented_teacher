# 📋 질문하기 수정 해야 하는 오류, 문제점

## 원본 회의록

질문하기 수정 해야 하는 오류, 문제점
1. 오류 발생 및 새로고침 시 문제가 사라지고 학생에게 빈칸이 전달되며 다시 생성할 수 없는 문제
 -> 학생이 재 풀이 요청을 할 수 있는 버튼이 필요
2. 학생 화면에서 풀이를 보기 눌렀을 때, 음성 TTS가 자동으로 재상될 때 2~3개가 겹쳐서 재생되는 현상이 있습니다. 자동재생이 아니라 처음에 재생버튼을 눌렀을 때 재생이 되도록 하면 여러개의 음성이 동시에 나오는 오류를 없앨 수 있어 보입니다.
3. 질문하기를 올렸지만 직접 해결 해 줬을 때 선생님 화면에서 그 질문을 삭제할 수 있는 버튼 필요
4. 한명의 질문을 해결 해 주면, 새로고침을 해야 다음 사람의 질문을 생성해서 제작할 수 있고, 새로고침하지 않고 생성을 누르면 1번과 같은 오류가 생김
-> 한번에 여러 사람의 질문을 진행 할 수 있도록 제작하거나, 한 사람의 질문 제작이 완성이 되면 즉시 새로고침 등이 되어서 오류가 없도록 해야 할듯 합니다.
5. 핸드폰 사진의 경우 화이트보드에 잘려서 크게 나오는데 일정 픽셀 이내로 축소시켜서 등장하게 해야 합니다.
6. 우리반의 질의응답 현황판이 연동되지 않습니다. (지금까지 질문 온 것과 해결한 것을 볼 수 없음)
7. 질문하기 요청을 해도 수학일기에 질문 순번이 나타나지 않습니다.
8. 카피 페이스트로 사진을 집어넣을 수 있도록 만들면 더욱 유용해보입니다(파일을 집어넣는 방법과 ctrl+V를 눌러 사진을 넣는 방법 둘 다 사용 가능)
9. 학생이 질문요청을 올리면 자동으로 처리하도록 하는 모드와 수동으로 처리하도록 하는 모드를 온/오프로 스위칭 할 수 있도록 하면 좋아보입니다.(4번의 오류와 여러명의 동시에 질문에 많이 쌓일 경우 똑같이 학생이 대기시간이 길게 생겨 몰입이 깨지는 일이 있습니다.)
