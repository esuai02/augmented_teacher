# Questions
# 상황별 포괄질문 + 세부질문 목록

---

## _global (모든 상황 공통)
```yaml
포괄질문:
  W_ALIGN: 이 작업이 상위 의지(W_ROOT)와 정렬되는가?
  RISK: 진행 시 위험 요소는 무엇인가?
  ENTROPY: 복잡도가 증가하고 있는가?

세부질문:
  - 상위 홀론의 의지가 보존되고 있는가?
  - 의지 약화 경로로 진입하지 않았는가?
  - 비협상 항목(non_negotiables)을 위반하지 않는가?
```

---

## init (초기화 상황)
```yaml
포괄질문:
  GOAL: 목표가 명확히 정의되었는가?
  RESOURCE: 필요한 리소스가 준비되었는가?
  STRUCTURE: 홀론 구조가 올바르게 설계되었는가?

세부질문:
  - W(세계관/의지)가 정의되었는가?
  - S(구조/리소스)가 식별되었는가?
  - P(절차)의 첫 단계가 명확한가?
  - 첨부할 지식파일이 있는가?
```

---

## develop (개발 진행 상황)
```yaml
포괄질문:
  PROGRESS: 현재 진행률은 얼마인가?
  NEXT: 다음 단계는 무엇인가?
  BOTTLENECK: 병목 지점이 있는가?

세부질문:
  - E(실행) 단계가 정상 진행 중인가?
  - 예상치 못한 예외가 발생했는가?
  - 다른 홀론과 충돌이 있는가?
  - 진단 파라미터(SEI, EC, ES, BV, GR)가 정상 범위인가?
```

---

## review (검토 상황)
```yaml
포괄질문:
  REVIEW_ITEM: 검토 항목은 무엇인가?
  HUMAN_NEED: 사람의 판단이 필요한 부분은?
  APPROVAL: 승인 조건은 무엇인가?

세부질문:
  - 위험이 제거되었음을 확인할 수 있는가?
  - 사람에게 제공할 정보가 충분히 가독성 있는가?
  - R(성찰) 결과가 기록되었는가?
  - 새 홀론 생성이 필요한가? (사람 검증 필수)
```

---

## blocked (차단 상황)
```yaml
포괄질문:
  CAUSE: 차단 원인은 무엇인가?
  OPTIONS: 해결 옵션들은 무엇인가?
  ESCALATION: 상위 홀론에 에스컬레이션이 필요한가?

세부질문:
  - 위험 수준은? (caution/danger/critical)
  - 자동 해결 가능한가, 사람 개입이 필요한가?
  - 다른 홀론에 영향을 주는가?
  - 롤백이 필요한가?
```

---

## complete (완료 상황)
```yaml
포괄질문:
  RESULT: 결과물은 무엇인가?
  LEARNING: 배운 것은 무엇인가?
  NEXT_HOLON: 다음 홀론 확장이 필요한가?

세부질문:
  - A(고도화) 단계로 추상화할 것이 있는가?
  - T(전파) - 다른 홀론에 전달할 것이 있는가?
  - 지식파일로 저장할 것이 있는가?
  - 진단 파라미터가 개선되었는가?
```
