# 플러그인 파일 팝업 테스트 가이드

## 구현 내용

플러그인 클릭 시 지정된 파일을 팝업으로 열 수 있는 기능이 구현되었습니다.

### 주요 변경사항

1. **executePluginAction 함수 개선**
   - 모든 플러그인 타입에서 파일 경로 지원
   - 팝업 창 크기 커스터마이징 지원
   - 팝업 차단 감지 및 대체 방안 제공

2. **플러그인 설정 UI 개선**
   - 파일 경로 입력 필드 추가
   - 팝업 창 크기 설정 옵션 추가
   - 더 명확한 입력 가이드 제공

3. **파일 경로 처리**
   - 상대 경로 지원 (./file.html, ../docs/guide.pdf)
   - 절대 URL 지원 (https://example.com)
   - 자동 경로 변환 기능

## 테스트 방법

### 1. 기본 카드(default_card) 플러그인으로 테스트

1. 플러그인 추가 버튼(+) 클릭
2. "기본 카드" 선택
3. 다음 정보 입력:
   - **카드 제목**: 테스트 파일 팝업
   - **카드 설명**: 팝업 테스트용 파일
   - **링크 URL 또는 파일 경로**: ./test_popup_file.html
   - **팝업 창 크기**: 너비 1000, 높이 700 (선택사항)

4. 저장 후 생성된 카드 클릭

### 2. 내부 링크(internal_link) 플러그인으로 테스트

1. 플러그인 추가 버튼(+) 클릭
2. "내부링크" 선택
3. 다음 정보 입력:
   - **카드 제목**: 주간 리포트
   - **내부 파일 경로**: ./weekly/test_report.html
   - **팝업 창 크기**: 너비 1200, 높이 900

4. 저장 후 생성된 카드 클릭

### 3. 테스트 파일 목록

생성된 테스트 파일:
- `/test_popup_file.html` - 기본 팝업 테스트 파일
- `/weekly/test_report.html` - 주간 리포트 예시 파일

### 4. 다양한 경로 테스트

다음과 같은 경로들을 테스트해볼 수 있습니다:

```
./test_popup_file.html          (현재 디렉토리)
./weekly/test_report.html       (하위 디렉토리)
../studenthome/index.php        (상위 디렉토리)
/absolute/path/to/file.html     (절대 경로)
https://example.com/file.pdf    (외부 URL)
```

## 기능 특징

1. **팝업 차단 대응**
   - 팝업이 차단되면 사용자에게 알림
   - 새 탭으로 열기 대체 옵션 제공

2. **유연한 경로 처리**
   - 상대 경로 자동 변환
   - HTTP/HTTPS URL 직접 지원
   - 파일 확장자 제한 없음

3. **커스터마이징 옵션**
   - 팝업 창 크기 조절 가능
   - 기본값: 1200x800 픽셀

## 주의사항

1. 파일 경로는 웹 서버에서 접근 가능해야 함
2. 상대 경로는 현재 페이지 기준으로 계산됨
3. 팝업 차단기가 활성화된 경우 수동으로 허용 필요

## 문제 해결

- **팝업이 열리지 않는 경우**: 브라우저 팝업 차단 설정 확인
- **파일을 찾을 수 없는 경우**: 파일 경로 확인 및 웹 서버 접근 권한 확인
- **창 크기가 적용되지 않는 경우**: 일부 브라우저는 최소/최대 크기 제한이 있음
