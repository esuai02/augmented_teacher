## 📌 1. 핵심 철학: 삼각 협력 구조

### 1.1 왜 선생님 참여가 중요한가

AI 튜터는 실시간 데이터 분석과 패턴 인식에 강점이 있지만, **인간적 통찰**과 **맥락적 판단**은 선생님만이 제공할 수 있습니다.

```
    ┌─────────────────────────────────────────────────────┐
    │                   학생 성장                         │
    │         (인지·정서·메타인지의 균형 발달)             │
    └──────────────────────┬──────────────────────────────┘
                           │
          ┌────────────────┼────────────────┐
          │                │                │
    ┌─────▼─────┐    ┌─────▼─────┐    ┌─────▼─────┐
    │  AI 튜터   │◄──►│   학생    │◄──►│  선생님   │
    │           │    │           │    │           │
    │ ▪ 실시간   │    │ ▪ 학습 주체│    │ ▪ 맥락 이해│
    │   패턴 감지│    │ ▪ 감정 표현│    │ ▪ 관계 형성│
    │ ▪ 즉시 개입│    │ ▪ 메타인지 │    │ ▪ 장기 관찰│

```

### 1.2 역할 분담 원칙

| 영역 | AI 튜터 담당 | 선생님 담당 |
|------|-------------|------------|
| **실시간 반응** | 즉시 피드백, 감정 감지 | 심층 대화, 동기 탐색 |

| **데이터 분석** | 학습 이력 통계, 추이 분석 | 인사이트 도출, 교육적 의미 부여 |
| **관계 형성** | 일관된 튜터링 제공 | 신뢰 관계, 정서적 지지 |

## 🎯 2. 선생님 참여 시점 (When)

### 2.1 AI 튜터가 선생님을 부르는 순간

AI 튜터는 다음 상황에서 **선생님 개입 요청 신호**를 보냅니다:

#### 🔴 즉시 개입 필요 (High Priority)

| 상황 | 학생 신호 | 요청 이유 |
|------|----------|----------|
| **정서적 위기** | 좌절감 3회 연속, 포기 발언 | AI의 공감에 한계, 인간적 위로 필요 |
| **고착 상태** | 동일 단계 10분 이상 정체 | 근본적 이해 결핍 의심 |
| **페르소나 급변** | 30분 내 3회 이상 스위칭 | 불안정 상태, 진단 재검토 필요 |
| **반복 실수** | 동일 유형 오류 5회 이상 | 개념적 오해 가능성 |

#### 🟡 참고 알림 (Medium Priority)

| 상황 | 의미 | 권장 행동 |
|------|------|----------|
| **새 인지관성 발견** | 미식별 패턴 감지 | 다음 수업에서 관찰 강화 |
| **마스터리 달성** | 단원/개념 정복 완료 | 칭찬 및 다음 목표 안내 |
| **학습 속도 변화** | 평소 대비 ±30% 이상 | 컨디션/외부 요인 체크 |

#### 🟢 정기 리뷰 (Low Priority)

| 주기 | 내용 |
|------|------|
| **일간** | 오늘의 감정 변화, 풀이 패턴 요약 |
| **주간** | 인지관성 정복 현황, 성장 곡선 |
| **월간** | 페르소나 변화 추이, 장기 목표 점검 |

### 3.2 상황별 피드백 스크립트

#### 막힘 상황에서

**AI 튜터가 시도한 것**:
> "어느 부분이 막히나요? 이 단계만 먼저 해볼까요?"

**선생님이 추가할 수 있는 것**:

| 학생 유형 | 선생님 멘트 예시 |
|----------|-----------------|
| 막힘-회피형 | "선생님도 이거 처음 봤을 때 어려웠어. 같이 천천히 해보자." |
| 감정출렁형 | "지금 답답한 거 알아. 잠깐 숨 고르고 다시 보자." |
| 무기력형 | "이 한 문제만. 딱 이것만 해보고 어떤 느낌인지 말해줘." |

#### 성공 순간에

**AI 튜터의 기본 반응**:
> "정답이에요! 🎉"

**선생님이 추가할 수 있는 것**:

```
┌────────────────────────────────────────────────────────────────┐
│  💬 선생님 칭찬 코멘트 입력                                     │
│                                                                │
│  [빠른 선택]                                                   │
│  ├─ "이 부분 정말 잘 이해했네!"                                │
│  ├─ "저번에 막혔던 건데, 이제 완전 정복이다!"                  │
│  ├─ "네가 쓴 이 풀이 방식, 교과서보다 멋지다"                  │
│  └─ [직접 작성...]                                             │
│                                                                │
│  🎵 음성 녹음: [🔴 REC] (10초 이내)                            │
└────────────────────────────────────────────────────────────────┘
```

#### 오개념 발견 시

**AI 튜터가 감지한 것**:
> "등식과 항등식을 혼동하고 있어요"

**선생님이 할 수 있는 것**:

1. **즉시 개입**: "잠깐, 선생님이 설명할게"
2. **추가 자료 연결**: 개념 노트 링크 첨부
3. **다음 수업 예약**: "이거 내일 수업에서 집중적으로 다루자"
4. **비유 제공**: 학생이 좋아하는 것과 연결한 설명
