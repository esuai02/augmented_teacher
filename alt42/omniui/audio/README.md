## 파일 정보

### 파일명
- `last_chunking_intro.m4a` (M4A 형식 - 필수)
- `last_chunking_intro.mp3` (MP3 형식 - 선택사항, 브라우저 호환성을 위해)

### 업로드 위치
```

```

### 권장 사양
- 형식: M4A (필수), MP3 (선택사항)
- 비트레이트: 128kbps 이상
- 샘플레이트: 44.1kHz
- 채널: 모노 또는 스테레오

### 업로드 방법
1. 이 audio 폴더에 음성 파일을 복사합니다
2. 파일명을 정확히 `last_chunking_intro.m4a`로 지정합니다
3. 브라우저에서 exam_preparation_system.php 페이지를 새로고침하면 자동으로 로드됩니다

### 참고사항
- M4A 형식은 대부분의 최신 브라우저에서 지원됩니다
- MP3 형식은 구형 브라우저 호환성을 위한 대체 형식입니다
- 파일 크기는 웹 성능을 위해 10MB 이하로 권장합니다
- M4A는 AAC 코덱을 사용하여 더 나은 음질과 압축률을 제공합니다
